## Чому

Для багатьох програмістів, робота з командними аргументами може бути новою та незвичною. Але розуміння того, як працювати з командними аргументами може значно полегшити створення ефективних інструментів та програм.

## Як

Щоб зрозуміти як читати командні аргументи в Haskell, перш за все потрібно перевірити що командної стрічки (command line) аргументи є частинами вебі-застосунку. Наприклад, якщо ми хочемо передати ім'я користувача програмі, ми можемо використовувати наступну команду:

```Haskell
username :: IO ()
username = do
  name <- getArgs
  putStrLn ("Hello " ++ head name ++ "!")
```

Ця функція приймає команду із інтерфейсом командного рядка та виводить привітання з іменем користувача.

Вивід:
```
Hello Haskell!
```

## Глибока погляд

Крім переліченої вище функції `getArgs`, є ще багато інших корисних функцій, які дозволяють читати та обробляти командні аргументи в Haskell. Наприклад, є функція `getProgName`, яка повертає ім'я запущеної програми, а функція `getEnv` дозволяє читати змінні середовища.

Також важливо розуміти як обробляти помилки, якщо командні аргументи не введені правильно або відсутні. Для цього можна використовувати функцію `catch` з модуля `Control.Exception`.
