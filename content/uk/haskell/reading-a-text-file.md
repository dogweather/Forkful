Завантаження та обробка файлів тексту у Haskell

## Чому

Читання файлів тексту є важливою частиною процесу розробки програм на мові програмування Haskell. Велика кількість даних зазвичай зберігається у текстовому форматі, тому вміння читати та обробляти ці файли є важливою навичкою для будь-якого програміста.

## Як

Щоб прочитати текстовий файл у Haskell, спочатку потрібно відкрити його за допомогою функції `openFile` з модуля `System.IO`. Потім можна використовувати функцію `hGetContents` для отримання вмісту файлу у вигляді рядка. Наприклад:

```Haskell
import System.IO

main = do
    -- Відкриття файлу у режимі читання
    handle <- openFile "text_file.txt" ReadMode
    -- Отримання вмісту файлу
    contents <- hGetContents handle
    -- Виведення вмісту на екран
    putStrLn contents
    -- Закриття файлу
    hClose handle
```

Вищенаведений код відкриває файл з назвою "text_file.txt" у режимі читання та виводить його зміст на екран. Після закінчення роботи з файлом, його необхідно закрити за допомогою функції `hClose`.

## Глибоке занурення

Розуміння того, як саме працює читання файлів у Haskell, може створити покращений процес розробки. У цьому прикладі ми використали функцію `hGetContents`, яка повертає цілий файл у вигляді одного рядка. Однак, для більшої ефективності можна використовувати функцію `hGetLine`, яка повертає по одному рядку кожного разу.

Наприклад, якщо у файлі "text_file.txt" є наступний вміст:

```
Hello,
World!
```

Тоді застосування функції `hGetLine` виглядатиме так:

```Haskell
main = do
    -- Відкриття файлу у режимі читання
    handle <- openFile "text_file.txt" ReadMode
    -- Отримання першого рядка
    line1 <- hGetLine handle
    -- Отримання другого рядка
    line2 <- hGetLine handle
    -- Виведення рядків на екран
    putStrLn line1
    putStrLn line2
    -- Закриття файлу
    hClose handle
```

Виведення цього коду буде видавати:

```
Hello,
World!
```

Використання функції `hGetLine` дозволяє працювати з файлом більш ефективно за рахунок читання по одному рядку, а не всього файлу цілком.
