## Чому

Створення випадкових чисел є важливою складовою будь-якої програми, що працює з критичними для безпеки даними або в експериментальних вимірах. У Haskell є потужні засоби для генерування випадкових чисел, що дозволяє забезпечити безпеку та відтворюваність цих чисел.

## Як це зробити

Використовуємо функцію `randomR` для генерування випадкового числа у заданому інтервалі. Наприклад, щоб отримати випадкове число від 1 до 10, введіть наступний код:

```Haskell
import System.Random

main = do
    gen <- getStdGen
    let (rand, newGen) = randomR (1, 10) gen
    print rand
```

Виходом буде число у вигляді `Int`.

Також можна скористатися функцією `randomRIO`, яка автоматично створює генератор випадкових чисел і повертає результат у зазначеному інтервалі. Наприклад:

```Haskell
import System.Random

main = do
    rand <- randomRIO (1, 10)
    print rand
```

В цьому випадку виходом також буде число у вигляді `Int`.

## Розглиблене дослідження

Haskell має багато вбудованих функцій для роботи з випадковими числами, таких як `random`, `randomR`, `randomRs` та багато інших. Вони дозволяють налаштувати генератор випадкових чисел для досягнення більшої волатильності або забезпечення безпеки.

Наприклад, функція `random` генерує псевдовипадкове число, яке змінюється з кожним викликом. Щоб забезпечити стабільність цих чисел у кожному виклику, можна використовувати функцію `newStdGen`, яка повертає новий генератор за кожним викликом.

Також важливо зазначити, що вбудований генератор випадкових чисел у Haskell запам'ятовує останній використаний генератор, тому його необхідно часто оновлювати, викликаючи функцію `newStdGen`.
