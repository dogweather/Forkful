## Чому

Довгі і складні рядки є звичайними в програмуванні, тож здійснення різних операцій з ними є важливим завданням для програміста. Знаходження довжини рядка є однією з таких необхідних операцій, яку ми розглянемо у цій статті.

## Як

Для знаходження довжини рядка використовується функція `length` у Haskell. Ця функція приймає рядок як аргумент і повертає його довжину у вигляді цілого числа. Давайте розглянемо приклад реалізації цієї функції:

```Haskell
length :: [a] -> Int
length [] = 0
length (_:xs) = 1 + length xs
```

Дана реалізація базується на рекурсії – функція `length` зменшує рядок на один елемент на кожному кроці, доки він не стане пустим. Через це, ми можемо обчислити довжину рядка, додаючи по одиниці для кожного елементу, що був видалений.

Давайте спробуємо цей код у дії:

```Haskell
length "Привіт, світ!" -- поверне 13
length "Це український текст" -- поверне 23
```

Як бачимо, функція дійсно повертає довжину рядка.

## Глибокий занурення

Існує ще один спосіб реалізувати функцію `length` у Haskell, використовуючи вбудовану функцію `foldl`. Ця функція приймає два аргументи – функцію і початкове значення. В нашому випадку, функція буде додавати одиницю до початкового значення для кожного елементу у списку.

Лямбда-функція, яку ми будемо використовувати, виглядає наступним чином:

```Haskell
\x y -> x + 1
```

Це просто функція з двома аргументами – `x` і `y`, яка повертає суму `x` і 1. Тож, застосовуючи цю функцію разом з `foldl`, ми отримаємо нашу функцію `length`:

```Haskell
length :: [a] -> Int
length xs = foldl (\x y -> x + 1) 0 xs
```

Ця реалізація також діє правильно і повинна повертати довжину рядка. Цей підхід може бути корисним, коли ви хочете використовувати функцію `length` у комбінації з іншими функціями для обробки списків.
