## Чому

Питання незовнішній, але все ж таки важливий: чому програмісти взагалі хвилюються про капіталізацію рядка? Чи вона є дійсно такою важливою? В цьому блозі ми розглянемо різні причини, чому капіталізація рядка може бути корисною та як саме її реалізувати.

## Як

Капіталізація рядка є процесом перетворення всіх букв у рядку на заголовні. Це може бути корисно у випадках, коли нам потрібно вирізняти окремі слова у рядку або використовувати пошук за ключовими слівами. Давайте розглянемо приклади, як це можна зробити в Haskell:

```Haskell
-- Завантажуємо необхідні бібліотеки
import Data.Char (toUpper)

-- Функція для капіталізації рядка
capitalize :: String -> String
capitalize str = [toUpper c | c <- str]

-- Приклади використання
main = do
    let str1 = "привіт, світ!"
    putStrLn (capitalize str1)
    -- Вивід: "ПРИВІТ, СВІТ!"

    let str2 = "Haskell це найкраща мова програмування"
    putStrLn (capitalize str2)
    -- Вивід: "HASKELL ЦЕ НАЙКРАЩА МОВА ПРОГРАМУВАННЯ"
```

Як бачимо, функція `capitalize` приймає рядок та повертає той же рядок, але з усіма буквами у верхньому регістрі. Це можна зробити за допомогою циклу `for` та функції `toUpper` з бібліотеки `Data.Char`.

### Операційна капіталізація

Іноді нам потрібно капіталізувати тільки першу літеру в кожному слові рядка. Наприклад, у заголовках новин, де зазвичай кожне слово починається із великої літери. Це можна зробити за допомогою функцій `words` та `unwords`:

```Haskell
-- Функція для капіталізації першої літери в кожному слові рядка
capitalizeFirst :: String -> String
capitalizeFirst str = unwords [capitalize word | word <- words str]

-- Приклад використання
main = do
    let str = "ukrainian bloggers are amazing"
    putStrLn (capitalizeFirst str)
    -- Вивід: "Ukrainian Bloggers Are Amazing"
```

Цей підхід дозволяє зробити оперативну капіталізацію, не змінюючи загальної структури рядка.
