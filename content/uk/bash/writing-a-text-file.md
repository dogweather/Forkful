## What & Why? (Що і чому?)
Запис текстового файлу - це процес збереження данних у текстовому форматі. Програмісти роблять це для зберігання конфігурацій, звітів, логів чи автоматизації задач.

## How to (Як це зробити):
```Bash
# Створення текстового файлу з написом "Привіт, Україно!"
echo "Привіт, Україно!" > hello.txt

# Додавання тексту до існуючого файлу
echo "Додатковий рядок" >> hello.txt

# Перегляд вмісту файлу
cat hello.txt
```
Sample output (Приклад виводу):
```
Привіт, Україно!
Додатковий рядок
```

## Deep Dive (Поглиблено):
Запис файлів у UNIX-подібних системах має довгу історію, що йде з часів створення системи файлів. Є команди як `echo`, `printf`, за допомогою яких можна записувати дані у файли. Символи `>` та `>>` використовуються для перенаправлення виводу у файл, відповідно замінюючи та добавляючи вміст. Варто знати, що також є команди `touch` для створення порожнього файлу і `tee` для запису у файл та виводу на екран одночасно.
