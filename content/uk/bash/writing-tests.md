## Чому
Написання тестів є важливою частиною розробки програмного забезпечення, оскільки допомагає виявляти помилки та забезпечує стабільність коду.

## Як
Написання тестів за допомогою Bash є простим та ефективним способом перевіряти роботу вашого коду. Починаємо зі створення нового файлу з розширенням `.sh` та вкладення в нього наступного коду:
 
```Bash
#!/bin/bash

# Перевірка, чи виконується програма нормально
if [ $? -eq 0 ]; then 
  echo "Тест пройшов успішно!" 
else 
  echo "Тест не пройшов, щось пішло не так." 
fi
```

Цей код перевіряє статус виконання програми та повідомляє результат. Ви можете також додати більше умов для перевірки конкретної функціональності вашого коду.

## Deep Dive
Тестування є необхідним кроком у процесі розробки програмного забезпечення, оскільки без нього важко забезпечити надійність та стабільність програми. Якщо ви плануєте розробку великого проекту, рекомендується створити окремий каталог для збереження тестів та використовувати спеціальні фреймворки для тестування, які допоможуть зробити цей процес більш організованим.
