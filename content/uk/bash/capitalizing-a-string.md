# Чому

Можливо, ви бачили попередні пости про Баш програмування та хочете дізнатися більше про цю потужну мову. А можливо, ви просто шукаєте спосіб, як виконати певне завдання або задачу. Незважаючи на причину, з якої ви читаєте цю статтю, я впевнений, що ви будете задоволені тим, як легко можна використовувати Баш для різноманітних завдань. Одна з таких задач - це капіталізація рядка.

# Як

Кодування у Баш стає дуже простим, коли маєте правильні знання та інструменти. Для капіталізації рядка нам знадобиться використовувати команду "tr", яка може виконувати різні маніпуляції з текстом. Нижче ви побачите приклад коду та його вихід, який зробить першу літеру рядка великою.

```Bash
str="приклад тексту"
capitalized_str=$(echo $str | tr '[:lower:]' '[:upper:]')
echo $capitalized_str
```

Вихід: "Приклад тексту"

# Deep Dive

Коли ми використовуємо команду "tr" для капіталізації рядка, ми вказуємо перетворення одного набору символів у інший. У нашому випадку, ми вказуємо перетворення з набору малих літер у набір великих літер. Ця команда може бути корисною для різних завдань, не тільки для капіталізації рядка. Наприклад, ви можете використовувати її для заміни символів у рядку або для зміни регістру всіх літер.
