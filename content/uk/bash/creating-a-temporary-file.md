## Чому

Створення тимчасових файлів є важливою частиною програмування в Bash. Це може бути корисно для передачі даних, збереження проміжних результатів або для інших потреб. 

## Як

Для створення тимчасового файлу використовується спеціальна команда `mktemp`. Приклад використання:
```Bash
tmp_file=$(mktemp) 
```
Це створить тимчасовий файл з унікальним іменем та поверне шлях до цього файлу у змінну `tmp_file`. Можна також вказати бажане префіксне ім'я для тимчасового файлу. Наприклад:
```Bash
tmp_file=$(mktemp myprefixXXXXX)
```
де `X` буде замінено на унікальне число. Після завершення роботи з тимчасовим файлом, його можна видалити за допомогою команди `rm`:
```Bash
rm "$tmp_file"
```

## Глибше

Створення тимчасових файлів також може бути корисним для застосування в шаблонах файлів. Наприклад, ми можемо створити тимчасовий файл з текстом і використовувати його як шаблон для кількох файлів замість повторної передачі тексту у кожному файлі. Приклад використання:
```Bash
template=$(mktemp) # створюємо шаблонний файл
echo "Це шаблонний текст" > "$template" # додаємо текст до шаблону
# створюємо дві нові копії змінного шаблону
new_file1=$(mktemp) 
new_file2=$(mktemp) 
cat "$template" > "$new_file1" # заповнюємо першу копію текстом з використанням шаблону
echo "Додатковий текст" >> "$new_file1" # додаємо нові рядки до першої копії
cat "$template" > "$new_file2" # заповнюємо другу копію текстом з використанням шаблону
# видаляємо шаблонний файл
rm "$template" 
```
