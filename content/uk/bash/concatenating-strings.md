## Зачем

Уведомите читачів про те, що показуєць цей блог пост. Вкажіть, що контатенување рядав дуже важлива техніка у програмуњнні, яка дозволяє об'єднати кілька рядків в один і дає більший контроль над виводом.

## Як

```bash
# Приклад 1
word1="Привіт,"
word2="світ!"
echo "$word1 $word2"
```

Результат:
```bash
Привіт, світ!
```

```bash
# Приклад 2
phrase1="Я хочу"
phrase2="вчитися"
phrase3="програмувати"
echo "$phrase1 $phrase2 $phrase3"
```

Результат:
```bash
Я хочу вчитися програмувати
```

У цьому прикладі ми використовуємо змінні та команду `echo` для виведення з'єднаних рядків.

## Вдосконалена техніка контатенування рядків

Крім простого з'єднання двох рядків, існує багато способів вдосконалити техніку контатенування рядків.

- Додавання пробілів між рядками для структурованішого виводу.
- Використання циклів для з'єднання більшої кількості рядків.
- Видалення або заміна певних символів зі з'єднаних рядків.

```bash
# Приклад 3
greeting="Привіт"
name="Олег"
echo "$greeting, $name!"
```

Результат:
```bash
Привіт, Олег!
```

```bash
# Приклад 4
# З'єднання кількох рядків з використанням циклу
colors=("червоний" "жовтий" "синій")
result=""
for color in "${colors[@]}"
do
  result="$result$color "
done
echo "Це масив кольорів: $result"
```

Результат:
```bash
Це масив кольорів: червоний жовтий синій
```

## Детальний огляд

Контатенување рядків є важливою технікою у програмуванні та може бути використана для різних цілей. Для поглибленого розуміння, вивчайте додаткові функції та можливості, які пропонує Bash.
