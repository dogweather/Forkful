## Чому

Із появою сучасних інструментів та технологій програмування, може здатися, що розробку програм можна проводити безпосередньо на графічному інтерфейсі користувача. Але командний рядок залишається основним інструментом для багатьох програмістів. Іноді важливо мати можливість передати програмі параметри при її запуску. У цьому пості я покажу вам, як це зробити з використанням командного рядка (command line arguments).

## Як це зробити

Спочатку, давайте розглянемо синтаксис для читання командних аргументів за допомогою Bash. Це простий приклад, але дуже корисний при роботі з командним рядком.

```Bash
#!/bin/bash

# Отримуємо аргументи, передані програмі
args=$@

# Перебираємо всі аргументи і виводимо їх на екран
for arg in $args
do
  echo $arg
done
```

Збережіть цей скрипт в файлі з розширенням .sh, наприклад, "read_args.sh". Тепер ви можете запустити його з командному рядку з будь-якими аргументами, наприклад:

```Bash
./read_args.sh hello world
```

Ви побачите, що на екрані буде виведено:

```
hello
world
```

Також, ви можете передавати аргументи в якості змінних, наприклад:

```Bash
#!/bin/bash

# Передаємо 1-й параметр з командному рядку в змінну "name"
name=$1

echo "Привіт, $name!"
```

Запустіть цей скрипт з командному рядку і передайте як аргумент своє ім'я, наприклад:

```Bash
./greet.sh John
```

На екрані ви побачите:

```
Привіт, John!
```

## Глибше занурення

Іноді, ви можете отримати багато аргументів з командного рядку. У такому випадку, їх може бути зручно зчитувати в масив. Для цього використовуйте форму запису `${array[@]}`:

```Bash
#!/bin/bash

# Збираємо аргументи в масив "arguments"
arguments=("${@}")

# Виводимо на екран кожен аргумент, як елемент масиву
for arg in "${arguments[@]}"
do
  echo $arg
done
```

Запустіть цей скрипт з кількома аргументами і ви побачите, що вони виводяться окремими елементами масиву.
