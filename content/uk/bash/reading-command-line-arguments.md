## Що і чому?
Введення аргументів командного рядка – це механізм, який дозволяє передавати додаткову інформацію вашій програмі під час її запуску. Це корисно для забезпечення гнучкості та многоразового використання коду.
  
## Як це зробити:
Для прикладу, давайте створимо скрипт Bash з іменем test.sh, який прийматиме аргументи командного рядка.
```Bash
#!/bin/bash
echo "Перший аргумент: $1"
echo "Другий аргумент: $2"
```
Запустіть цей скрипт, вказавши *arg1* і *arg2* як аргументи командного рядка.
```Bash
bash test.sh arg1 arg2
```
Виправлення буде виглядати так:
```Bash
Перший аргумент: arg1
Другий аргумент: arg2
```
## Занурення в деталі:
Введення аргументів командного рядка було використано в Unix з самого початку його існування, згодом передбачено в Bash. Альтернатива цьому - використання вводу та виводу файлів, але це менш зручно для користувачів. 
В Bash, аргументи командного рядка зберігаються в особливих змінних, які називаються позиційними параметрами, з $0, що містить ім'я скрипту, та $1, $2 та ін., що містять позиційні аргументи.
