## Чому

Генерація випадкових чисел є важливою функцією в багатьох програмах та скриптах. Вона дозволяє створювати випадковий результат, що може бути корисним для широкого діапазону завдань, включаючи тестування програм та генерацію випадкового контенту.

## Як це зробити

Для генерування випадкових чисел використовуйте вбудовану команду ```Bash $RANDOM```. Наприклад, цей код створить випадкове число від 0 до 999:

```Bash
echo $(( RANDOM % 1000 ))
```

Відсоток визначає максимальне значення, яке може бути згенероване, тому замінюючи його на більше число, ви можете отримати більші випадкові числа.

## Глибоке погруження

Хоча команда ```$RANDOM``` зручна та проста для використання, вона не забезпечує повної випадковості. Наприклад, при кожному запуску вашої програми буде згенерований один і той самий послідовний набір чисел. Також можуть бути певні кореляції та шаблони між згенерованими числами. Якщо вам потрібен більш високий рівень випадковості, варто розглянути використання спеціальних програм для генерації випадкових чисел.
