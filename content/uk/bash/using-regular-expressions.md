## Що і чому?
Регулярні вирази — це шаблони для пошуку тексту, що дозволяють нам з легкістю знаходити, замінювати чи аналізувати дані. Програмісти використовують їх для ефективної автоматизації обробки текстів та складних задач фільтрування.

## How to:
### Знайти слово "кіт" у файлі:
```Bash
echo "Кіт сидів на підвіконні" | grep "кіт"
```
Вивід: 
```
Кіт сидів на підвіконні
```

### Замінити "кіт" на "пес" у текстовій стрічці:
```Bash
echo "Кіт сидів на підвіконні" | sed 's/кіт/пес/'
```
Вивід:
```
пес сидів на підвіконні
```

### Вибрати всі слова на "к" у текстовому файлі:
```Bash
grep "\bк\w*" текстовий_файл.txt
```

## Deep Dive:
Регулярні вирази (RegExp/Regex) з'явились у 1950-х і постійно еволюціонували. Існують інші інструменти, як awk чи Perl, що також мають сильні можливості текстової обробки. В Bash, grep, sed, і awk — найбільш популярні утиліти, які використовують регулярні вирази.
