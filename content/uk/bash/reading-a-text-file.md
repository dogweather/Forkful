## Чому

Текстові файли є неодмінною частиною програмування в будь-якій мові, включаючи Bash. Їх часто використовують для зберігання та обробки даних, що робить їх важливим інструментом для програмістів. Читати текстовий файл - це навичка, яка буде корисна у будь-якій області програмування і дозволить зрозуміти, як зчитати та обробити дані.

## Як

```Bash
# Створення текстового файлу з даними
echo "Привіт світ!" > file.txt

# Читання та виведення вмісту файлу
cat file.txt
```

В цьому прикладі ми використали команду `echo` для створення нового текстового файлу з одним рядком даних - "Привіт світ!". Далі ми використовуємо команду `cat` для зчитування та виведення вмісту файлу. В результаті ми повинні побачити наш рядок на екрані.

Для зчитування текстових файлів також можна використовувати інші команди, такі як `head` (для виведення перших декількох рядків файлу) і `tail` (для виведення останніх декількох рядків файлу).

```Bash
# Виведення перших 3 рядків файлу
head -n 3 file.txt

# Виведення останніх 3 рядків файлу
tail -n 3 file.txt
```

Більш детальні команди та опції для зчитування текстових файлів можна знайти в документації Bash.

## Глибші води

Під час читання текстового файлу також можна використовувати пайпи для обробки даних і фільтрації результатів. Наприклад, ми можемо використати команди `grep` і `wc` для пошуку певного слова у файлі і підрахунку кількості його входжень.

```Bash
# Знаходження кількості входжень слова "Привіт" у файлі
grep -o "Привіт" file.txt | wc -l
```

Також варто зазначити, що зчитування текстових файлів можна робити не тільки з командного рядка, але й у скриптах Bash для автоматизації деяких процесів.
