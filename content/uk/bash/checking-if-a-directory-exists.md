## Що і чому?
Перевірка наявності директорії - це процес, за допомогою якого ми визначаємо, чи існує певна папка в системі. Програмісти роблять це, щоб уникнути помилок при спробі відкрити або працювати з неіснуючим каталогом.

## Як це робиться:
Перевірка наявності директорії в Bash виглядає дуже просто. Використовуючи ```if``` з командою ```[-d DIR]```, ви можете легко розібратися в ситуації.  

```Bash
if [ -d "/tmp/test" ]; then
    echo "Каталог існує."
else
    echo "Каталог не існує."
fi
```
Якщо буде існувати каталог /tmp/test, Bash виведе "Каталог існує.", інакше "Каталог не існує."

## Поглиблено:
Історично в Bash, для перевірки наявності директорії, було багато варіантів, але ```[-d DIR]``` виявився найбільш простим і надійним. Альтернативи включають використання ```ls```, але це може призвести до помилок, якщо назва вашої директорії містить спеціальні символи.

Ця команда використовує POSIX-шелову команду ```test``` з опцією ```-d``` для перевірки директорії, це своєрідний "історичний" аспект цього рішення.
