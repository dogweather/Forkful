## Що і Навіщо?
Перевірка існування директорії у Bash - це спосіб уникнути помилок, коли ваш скрипт залежить від певних папок. Робимо це для того, щоб забезпечити правильну роботу скрипта і для захисту від можливих помилок при невідомих структурах файлової системи.

## Як зробити:
```Bash
# Перевірка існування директорії
if [ -d "/path/to/your/directory" ]; then
  echo "Директорія існує."
else
  echo "Директорії не існує."
fi
```
Вивід залежить від результату:
```
Директорія існує.
```
або
```
Директорії не існує.
```

## Поглиблений Розгляд
Перевірка існування директорії в Bash - це стара практика, існує з часів, коли Bash був створений у 1989 році. `-d` опція в `test` команді ([) — це те, що перевіряє директорію. Є альтернативні рішення, як `[[ ]]` для вбудованих умовних виразів у нових версіях Bash, або команди типу `find` та `stat` для детальнішої інформації про файли та директорії. Перевага `-d` полягає в тому, що вона є простою і портативною між Unix-подібними системами.
