## Чому

Перевірка наявності каталогу є важливою частиною написання програм на мові Bash. Цей процес допомагає уникнути помилок та забезпечити коректну роботу програми. 

## Як

Існує декілька способів перевірити наявність каталогу в Bash. Один з найпростіших і найчастіше використовуваних методів - це використання умовного оператора `if` та команди `test`. Нижче наведено приклад коду та його вихідний результат:

```Bash
#!/bin/bash

# Перевірка наявності каталогу за допомогою умовного оператора
if [ -d "Назва каталогу" ]; then
  echo "Каталог існує!"
else
  echo "Каталог не знайдено!"
fi
```

**Вихід:**
```
Каталог існує!
```

Іншим способом є використання команди `ls` та оператора `grep` для пошуку потрібного каталогу. Нижче наведено приклад, як це можна зробити:

```Bash
#!/bin/bash

# Перевірка наявності каталогу за допомогою команди ls та оператора grep
if ls | grep -q "Назва каталогу"; then
  echo "Каталог існує!"
else
  echo "Каталог не знайдено!"
fi
```

**Вихід:**
```
Каталог існує!
```

Ще одним варіантом є використання функції `test` та оператора `-d`, який перевіряє, чи є переданий аргумент каталогом. Перевіримо наявність каталогу "Назва каталогу":

```Bash
#!/bin/bash

# Перевірка наявності каталогу за допомогою функції test та оператора -d
if test -d "Назва каталогу"; then
  echo "Каталог існує!"
else
  echo "Каталог не знайдено!"
fi
```

**Вихід:**
```
Каталог існує!
```

## Deep Dive

Більш глибоко розберемося з командою `test` і її різними параметрами. Існує ряд операторів, які можна використати для перевірки наявності каталогу, серед яких:

- `-d` - перевіряє, чи є переданий аргумент каталогом
- `-e` - перевіряє, чи існує заданий файл або каталог
- `-f` - перевіряє, чи є переданий аргумент файлом
- `-s` - перевіряє, чи має переданий аргумент ненульовий розмір
- `-r` - перевіряє, чи є у користувача право читати заданий файл або каталог
