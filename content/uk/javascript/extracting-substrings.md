# Чому

Виділення підрядків є важливим процесом у програмуванні, оскільки дозволяє вам дістати певну частину тексту або рядка для подальшої обробки. Це може бути корисно при роботі з великими наборами даних або при створенні функцій для роботи зі строками.

# Як

Для того, щоб виконати екстракцію підрядків, ми скористаємося методом `substring()` у JavaScript. Даний метод дозволяє нам використовувати два параметри - початковий індекс та кінцевий індекс - для вибору потрібної частини рядка. Давайте розглянемо приклад коду:

```Javascript
const str = "Привіт, світ!";
const newStr = str.substring(7, 12); // поверне "світ"
console.log(newStr);
```

У цьому прикладі ми створили змінну `str`, що містить рядок "Привіт, світ!" та використали метод `substring()` для виділення підрядка з індексами 7 та 12. Результатом виконання цього коду буде рядок "світ", який потім виводиться у консоль.

Важливо зауважити, що індекси для методу `substring()` починаються з нуля, а кінцевий індекс не включає у себе останній символ. Також, можна використовувати лише один параметр - початковий індекс - для виділення всієї частини рядка після цього індексу.

# Глибоке погруження

У методу `substring()` є багато корисних використань і додаткових можливостей. Наприклад, ви можете використовувати від’ємні індекси для вибору підрядка з кінця рядка. Також, даний метод може бути поєднаним з іншими методами для більш складних операцій, наприклад, для заміни певної частини рядка на інший текст.

При використанні `substring()` варто також звертати увагу на можливі проблеми з українською кодуванням символів. Якщо у вас виникнуть труднощі з коректним відображенням тексту, ви можете спробувати використати методи `slice()` або `substr()`, які мають подібні можливості та мають більш широку підтримку символів.
