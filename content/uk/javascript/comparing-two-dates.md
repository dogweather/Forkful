"## Чому"

Написання комп'ютерних програм - це завжди цікаво і зворушливо. Крім того, іноді нам потрібно порівняти дві дати, щоб визначити, яка з них є більш пізньою або ранішою. Це може бути корисно при створенні планувальників або організації важливих подій.

"## Як"

Для порівняння двох дат у Javascript, ми можемо використовувати вбудовану функцію `Date()`. Вона приймає три аргументи: рік, місяць і день. Наприклад, якщо нам потрібно порівняти дату 12 грудня 2021 року з поточною датою, код буде виглядати наступним чином:

```
```Javascript
let date1 = new Date(2021, 11, 12); // Місяці починаються з 0, тому грудень - 11
let date2 = new Date(); // Поточна дата
```

Ми створили два об'єкти `date1` та `date2` з типом даних `Date` і передали потрібні нам дати. Далі, ми можемо використовувати умови `if` і `else` для порівняння цих дат:

```
```Javascript
if (date1 > date2) {
  console.log("Дата 12 грудня 2021 року є більш пізньою.");
} else {
  console.log("Поточна дата є більш пізньою.");
}
```

Наш код повинен висвітлити повідомлення "Дата 12 грудня 2021 року є більш пізньою." Але що, якщо нам потрібно порівняти дати з різницею всього у кілька годин або хвилин? Тоді ми можемо використовувати метод `getTime()` для отримання часу в мілісекундах і порівняти ці значення.

"## Глибокий занурення"

У глибинних зануреннях нам потрібно дізнатися більше про методи `Date()` і як вони працюють. Ці методи дозволяють нам отримати і встановити різні значення дати, такі як рік, місяць, день, година, хвилина, секунда і навіть мілісекунда. Також, є методи для відображення дати у різних форматах, наприклад, `toDateString()` і `toTimeString()`. Більше про ці методи можна прочитати у [офіційній документації](https://developer.mozilla.org/uk/docs/Web/JavaScript/Reference/Global_Objects/Date) Javascript.
