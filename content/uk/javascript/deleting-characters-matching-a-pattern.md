## Причина

Коли ми програмуємо, ми часто змушені видаляти частини тексту, які відповідають певному шаблону, наприклад, певні символи або слова, щоб виконати необхідні операції над текстом. Це може бути запит до користувача про введення даних, перетворення тексту у вигляді таблиці або робота зі списком елементів. У цілому, видалення маленьких частин тексту є важливою частиною програмування і варто навчитися цього.

## Як це зробити

Існує багато способів видалення символів або слов, які відповідають певному шаблону, у Javascript. Один з найпростіших способів - використання методу `replace()` з регулярним виразом. Наприклад, якщо ми хочемо видалити всі пропуски у рядку, можемо використати такий код:

```Javascript
let string = "Це рядок із пробілами.";
let newString = string.replace(/\s/g, "");
console.log(newString);
```

Результат виконання цього коду буде наступним:

`"Церядокізпробілами."`

У цьому прикладі, ми використали регулярний вираз `/s/g`, який вибирає всі пропуски у рядку і замінює їх на порожній рядок. Подібним чином можна також видаляти певні символи або слова, застосовуючи потрібний регулярний вираз.

Існує також більш розширений спосіб робити це, використовуючи функцію `filter()` у поєднанні зі стрілковими функціями (arrow function). Наприклад, якщо ми хочемо видалити всі числа зі списку, можемо написати наступний код:

```Javascript
let numbers = [1, 2, 3, 4, 5];
let newNumbers = numbers.filter((number) => typeof number !== "number");
console.log(newNumbers);
```

Результатом буде масив, у якому не буде чисел:

`[1, 2, 3, 4, 5]`

Існує також багато інших способів видалення символів або слів у Javascript, і краще вибрати той, який найбільше підходить для конкретної ситуації.
