## Чому

Причиною для читання аргументів командного рядка може бути потреба передати вхідні дані до програми ззовні або зробити її більш гнучкою для користувачів. Це може бути корисно при створенні скриптів або програм, які повинні працювати з різними наборами даних.

## Як це зробити

Мова програмування Javascript має вбудований об'єкт process, який зберігає всю інформацію про поточний процес виконання. Цей об'єкт містить властивість argv, яка представляє масив з усіма переданими аргументами командного рядка. Для того, щоб отримати доступ до цих аргументів, достатньо просто звернутися до певного індексу в цьому масиві. Наприклад:

```Javascript
// Код для читання аргументів командного рядка
var num1 = process.argv[2];
var num2 = process.argv[3];

// Приклад виклику скрипта з аргументами командного рядка
// node script.js 5 10

// Результат виконання:
console.log(num1); // виведе 5
console.log(num2); // виведе 10
```

## Глибокий занурення

Зверніть увагу, що властивість argv містить не тільки передані аргументи командного рядка, але і назву виконуваного файлу. Наприклад, якщо виконуваний файл називається "script.js", то перший елемент масиву argv буде рівний "node", а наступними будуть передані аргументи. Це можна корисно використати для перевірки наявності певних аргументів та зменшення ймовірності помилок.
