## Чому

Отримання поточної дати є важливою частиною багатьох веб-додатків, таких як онлайн-календарі, завдання та бронювання подій. Тому знання, як отримати поточну дату, є необхідним для будь-якого програміста, який працює з Javascript.

## Як

Для отримання поточної дати в Javascript існує кілька методів. Розглянемо кожен з них за допомогою кодових прикладів і відображення результату в консолі.

### Метод #1: ```new Date()```

Цей метод створює новий екземпляр об'єкта ```Date``` і повертає поточну дату та час. Наприклад:

```
let currentDate = new Date();
console.log(currentDate);
```

Вихідний результат буде виглядати приблизно так:

```
Thu Jul 15 2021 15:50:22 GMT+0300 (Eastern European Summer Time)
```

### Метод #2: ```Date.now()```

Цей метод повертає кількість мілісекунд, що пройшли з моменту 01 січня 1970 року до поточного часу. Це дуже корисно для обчислення різниці часу між двома подіями. Приклад використання:

```
let currentTime = Date.now();
console.log(currentTime);
```

Результат буде виглядати приблизно так:

```
1626354828675
```

### Метод #3: ```Date.parse()```

Цей метод приймає рядок у форматі дати та часу і повертає кількість мілісекунд, що пройшли з моменту 01 січня 1970 року до вказаного часу. Наприклад:

```
let birthday = Date.parse('Aug 30, 1995 08:45:00');
console.log(birthday);
```

Результат буде виглядати приблизно так:

```
810232500000
```

## Глибоке погруження

Тепер, коли ми розібралися з різними методами отримання поточної дати в Javascript, давайте поглибимось трохи. За замовчуванням Javascript використовує часовий пояс, який встановлений в операційній системі комп'ютера. Але, якщо потрібно, ми можемо змінити цей часовий пояс за допомогою методу ```setUTC()```.

Наприклад, для встановлення UTC-часу:

```
let currentDate = new Date();
currentDate.setUTCHours(10);
```

Або для встановлення залишку часу:

```
currentDate.setMinutes(30);
```
