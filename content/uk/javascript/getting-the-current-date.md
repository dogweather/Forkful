# Контрольний час: Захоплення поточної дати в JavaScript
## Що і чому?
Отже, що таке отримання поточної дати? Це процес отримання актуального контрольного часу з нашого системного годинника. А чому програмісти це роблять? Залежно від вимоги, це може бути необхідним для створення часових відміток, відслідковування часу або для других цілей, пов'язаних з часом.

## Як це зробити:
За допомогою об'єкта `Date` в JavaScript, ми можемо отримати поточну дату й час. Давайте пройдемося швидко:

```Javascript
let currentDate = new Date();
console.log(currentDate);
```
Коли ви виконуєте цей код, він виведе поточну дату і час, подібне до наступного:

```Javascript
2022-08-20T19:20:30.45Z
```
Це основний спосіб отримати поточну дату в JavaScript!

## Поглиблена інформація
Історично, використання `Date()` в JavaScript з'явилось з самого початку мови та вважається одним з фундаментальних об'єктів. 

Щодо альтернатив, ми можемо вжити більш високорівневі бібліотеки, такі як moment.js або date-fns для більш складних операцій дати.

Що стосується виконання, `Date()` в JavaScript повертає дату як рядок в форматі ISO, виходячи з часової зони UTC. Це важливо мати на увазі, коли ви працюєте з часовими зонами.
