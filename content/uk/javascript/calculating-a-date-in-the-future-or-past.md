## Чому

Якщо ви розробник програмного забезпечення, або тільки починаєте вивчати Javascript, то навряд чи зможете уникнути розрахунків дат у вашому коді. Незалежно від проекту, часто потрібно обробляти дату і час, виконувати аналіз, змінювати формат або просто маніпулювати ними. Давайте розглянемо, як це зробити в Javascript!

## Як це зробити

Для початку, необхідно зрозуміти, як в Javascript працює об'єкт `Date`. Цей об'єкт містить всю необхідну інформацію про час та дату, включаючи рік, місяць, день тижня, день місяця, години, хвилини та секунди. Щоб створити новий об'єкт `Date`, можна використовувати конструктор `new Date()`, який поверне поточний час і дату.

За замовчуванням, наш об'єкт `Date` містить поточний час і дату. Але що робити, якщо нам потрібно розрахувати дату в майбутньому або минулому? Для цього існує декілька корисних методів, які дозволяють змінити та отримати значення дати.

Давайте розглянемо детальніше кожен із цих методів за допомогою кодових прикладів.

### Зміна дати

Щоб змінити дату в об'єкті `Date`, потрібно використовувати методи `setFullYear()`, `setMonth()`, `setDate()`, `setHours()`, `setMinutes()` та `setSeconds()`. Наприклад:

```Javascript
let currentDate = new Date(); // створюємо об'єкт Date з поточною датою
currentDate.setFullYear(2022); // змінюємо рік на 2022
currentDate.setMonth(11); // змінюємо місяць на грудень (індекс місяця починається з 0)
currentDate.setDate(25); // змінюємо день на 25-е
currentDate.setHours(0); // змінюємо години на 0
currentDate.setMinutes(0); // змінюємо хвилини на 0
currentDate.setSeconds(0); // змінюємо секунди на 0

console.log(currentDate); // 2022-12-25T00:00:00.000Z
```
