## Чому
Написання тестів є важливою складовою роботи програміста. Це допомагає перевірити правильність роботи програми та попередити появу помилок у майбутньому.

## Як це зробити
Для написання тестів використовуються спеціальні фреймворки, такі як Jest, Chai або Mocha. Нижче наведено приклад використання Jest та виведення результатів тестів.

```Javascript
test('2 + 2 дорівнює 4', () => {
    expect(2 + 2).toBe(4);
});
```

Результат виконання цього тесту буде таким:

```
PASS  ./test.js
✔ 2 + 2 дорівнює 4 (5ms)
```

Таким чином, за допомогою фреймворків можна перевірити роботу навіть найскладніших функцій та впевнитися в їхній правильній роботі.

## Глибоке дослідження
Написання тестів дозволяє проаналізувати код програми та з'ясувати, які саме частини можуть призвести до помилок. Також це допомагає знайти ефективніші шляхи написання коду та зробити його більш структурованим.
