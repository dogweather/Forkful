##Чому

Перетворення рядка в нижній регістр є однією з основних операцій у роботі з рядками в програмуванні. Це дозволяє зробити рядок легше для обробки та порівняння з іншими рядками.

## Як це зробити

```C#
string str = "Привіт світ!";
Console.WriteLine(str.ToLower());
// виведе: привіт світ!
```

Ви також можете використовувати метод `ToLowerInvariant()`, який виконує перетворення в нижній регістр з незмінною локалізацією.

```C#
string str = "Hello World!";
Console.WriteLine(str.ToLowerInvariant());
// виведе: hello world!
```

Якщо ви хочете перетворити рядок з певною мовною локалізацією, ви можете передати `CultureInfo` об'єкт як параметр методу `ToLower()`.

```C#
string str = "Привіт світ!";
Console.WriteLine(str.ToLower(new CultureInfo("uk-UA")));
// виведе: привіт світ!
```

Ви також можете використовувати оператор присвоєння з явним перетворенням типу для змінення регістру рядка.

```C#
string str = "HELLO WORLD!";
Console.WriteLine((string)str.ToLower());
// виведе: hello world!
```

##Глибоке занурення

Значення рядків в C# є незмінними, тобто не можна змінити вже існуючий рядок, але можна створити новий на його основі. При перетворенні рядка в нижній регістр, створюється новий рядок з новими значеннями символів.

Це важливо пам'ятати, оскільки якщо програма використовує багато рядків та виконує перетворення в нижній регістр багато разів, це може спричинити зайве використання пам'яті.

Крім того, необхідно враховувати особливу локалізацію при використанні методів `ToLower()` та `ToLowerInvariant()`. Це може призвести до непередбачуваних результатів, якщо програма виконується на системі з іншою мовною інсталяцією.
