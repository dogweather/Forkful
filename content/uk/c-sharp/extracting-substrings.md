## Чому

Ви, безсумнівно, часто зустрічалися з ситуацією, коли вам потрібно було виділити певну частину рядка зі строки. Це може бути необхідно для багатьох різних завдань, наприклад, для пошуку конкретних слів або для заміни певних символів. У цьому пості ми поглиблено розглянемо процес вилучення підрядків (substrings) в C# і покажемо вам, яким чином це можна зробити зі зручністю та ефективністю.

## Як виконати

Кодування для вилучення підрядків у C# досить простий і ми покажемо вам це на прикладах. У першому прикладі ми використовуємо метод `Substring` на рядку "Hello World!" для вилучення підрядка "Hello". Код виглядає наступним чином:

```C#
string str = "Hello World!";
string subStr = str.Substring(0, 5);
Console.WriteLine(subStr); // Output: Hello
```

В цьому прикладі ми визначаємо початковий і кінцевий індекси для підрядка, які обидва включаються у вилучений підрядок. Якщо нам потрібно вилучити підрядок з індексу 5 до кінця рядка, ми можемо просто використовувати один параметр `str.Substring(5)`, що поверне підрядок "World!".

В другому прикладі ми використовуємо метод `Split` для розбиття рядка за роздільником. Роздільником може бути будь-який символ, наприклад, кома, пробіл або навіть певне слово. У нашому прикладі ми розбиваємо рядок "Hello, World!" за комою та отримуємо масив стрінгів з двома елементами: "Hello" та "World!". Код буде виглядати так:

```C#
string str = "Hello, World!";
string[] subStrs = str.Split(',');
Console.WriteLine(subStrs[0]); // Output: Hello
Console.WriteLine(subStrs[1]); // Output: World!
```

Є безліч інших методів та способів вилучення підрядків в C#, але ми навели лише два найпростіших приклади.

## Поглиблене дослідження

Під час виконання вилучення підрядків в C#, важливо пам'ятати, що будь-які зміни, зроблені з підрядком, не впливають на вихідний рядок. Також варто знати, що методи `Substring` та `Split` повертають нові об'єкти рядків, тому вони не будуть впливати на оригінальні дані.
