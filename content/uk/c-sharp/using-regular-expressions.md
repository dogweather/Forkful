## Чому

У програмуванні часто зустрічаються випадки, коли потрібно шукати та витягувати певну інформацію з тексту. Використання регулярних виразів є одним з найкращих способів для цього. Вони дозволяють створити шаблон, за допомогою якого можна знайти і витягнути потрібні дані з будь-якого тексту, будь-то логи, веб-сторінки або документи.

## Як

Для початку, необхідно підключити вбудований в C# клас `Regex`, який містить всі необхідні методи та властивості для роботи з регулярними виразами.

```C#
using System.Text.RegularExpressions;

// Створення регулярного виразу для пошуку email адреси
Regex emailRegex = new Regex(@"\b[\w\.-]+@[\w\.-]+\.\w{2,4}\b");

// Перевірка наявності email адреси в тексті
string text = "Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed quis elit@domain.com tristique.";

bool hasEmail = emailRegex.IsMatch(text);

// Отримання email адреси з тексту
Match emailMatch = emailRegex.Match(text);
string emailAddress = emailMatch.Value;

// Виведення результату
if(hasEmail)
{
    Console.WriteLine($"Знайдено email адресу: {emailAddress}");
}
else
{
    Console.WriteLine("Email адреса відсутня.");
}
```

При запуску програми, в консолі буде виведено: `Знайдено email адресу: elit@domain.com`.

Також, регулярні вирази можна використовувати для заміни частин тексту на інші дані. Наприклад, якщо потрібно змінити формат дати з `dd/mm/yyyy` на `yyyy/mm/dd`, можна використати наступний код:

```C#
// Створення регулярного виразу для знаходження дати з форматом dd/mm/yyyy
Regex dateRegex = new Regex(@"(\d{2})/(\d{2})/(\d{4})");

// Заміна формату з dd/mm/yyyy на yyyy/mm/dd
string newFormattedDate = dateRegex.Replace("18/05/2021", "$3/$2/$1");

// Виведення результату
Console.WriteLine(newFormattedDate);
```
Результатом виконання цього коду буде: `2021/05/18`.
