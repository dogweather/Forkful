## What & Why?

Конвертація дати в рядок - це процес перетворення об'єкта дати в текстовий формат. Програмісти роблять це для легшого відображення та обробки інформації.

## How to:

В Lua для конвертації дати в рядок використовується built-in функція `os.date()`.

```Lua
dateString = os.date("%d.%m.%Y")
print(dateString)
```

Цей код повертає дату в форматі "день.місяць.рік". Ви отримаєте виведення схоже на:

```Lua
"15.10.2022"
```

## Deep Dive:

1. Історичний контекст: Lua була створена у 1993 році з метою надати зручні вбудовані функції для роботи з датами та часом.

2. Альтернативи: Використання вбудованої функції `os.date()` - найпростіше рішення, але також можливо створення власних функцій для специфічних форматів дати. 

3. Деталі реалізації: Функція `os.date()` може приймати рядок формату як аргумент, що дає можливість перетворювати дату в будь-який потрібний формат.
