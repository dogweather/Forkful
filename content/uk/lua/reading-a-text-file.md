## Що і чому?

Читання текстового файлу - це процес, під час якого программа отримує дані з текстового файлу і використовує їх для реалізації своїх функцій. Програмісти роблять це для ефективного обміну даними між різними застосунками та маніпулювання даними.

## Як це зробити:

Вам потрібно відкрити файл, прочитати його вміст, а потім закрити файл. Ось простий приклад в Lua:

```Lua
-- Відкриваємо текстовий файл
local file = io.open("textfile.txt", "r")

-- Читаємо увесь файл і записуємо в змінну
local content = file:read("*all")

-- Закриваємо файл
file:close()

print(content)
```

Коли ви запустите цей скрипт, ви побачите на виводі вміст вашого файлу.

## Поглиблений занурення:

1. Контекст: У минулому програмісти приймали потоки даних безпосередньо з пристроїв вводу/виводу. З появою файлових систем та текстових файлів, програмісти змогли зберігати та передавати великі об'єми даних дуже легко.

2. Альтернативи: Lua надає можливість для асинхронного читання файлу через Luvit, який працює на основі libuv. Lua також має бібліотеку Penlight, яка надає високорівневі функції для роботи з файлами.

3. Особливості реалізації: Функція `io.open` насправді повертає два значення - власне файл і помилку. Ви повинні перевіряти наявність помилки перед тим, як працювати з файлом.
