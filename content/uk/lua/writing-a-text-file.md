## What & Why? / Що та чому?
Запис текстових файлів — це процес збереження даних у файлі. Програмісти це роблять для зберігання налаштувань, зберігання даних користувача, або логування інформації.

## How to / Як робити:
Запис файликів у Lua дуже простий. Ось приклад:

```Lua
local file, err = io.open("hello.txt", "w")  -- Відкриття файлу для запису
if file then
    file:write("Привіт, світ!")             -- Запис тексту у файл
    file:close()                             -- Закриття файлу
else
    print("Помилка при відкритті файлу:", err)
end
```

Після запуску коду файл `hello.txt` міститиме текст "Привіт, світ!".

## Deep Dive / Поглиблено:
З записом файлів у Lua свої нюанси. Початково, Lua призначалася для вбудованих систем і мала обмеження за обсягом, тому її вбудована бібліотека вводу/виводу є дуже ефективною але й простою. Існують альтернативи, як-от LFS (LuaFileSystem), для більш гнучкого управління файлами. При запису файла, важливо використовувати правильний режим доступу до файлу (наприклад, "w" для запису), та не забувати закривати файл аби звільнити ресурси.
