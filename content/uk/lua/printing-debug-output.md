## Що це таке й чому це потрібно?

Вивід відлагоджувальної інформації - це процес, при якому програма показує деталі про її виконання. Програмісти використовують це для виявлення та усунення помилок у коді.

## Як це зробити:

Ось приклад, який показує, як у Lua можна вивести відлагоджувальну інформацію:

```Lua
print("Hello, Debug!")
```

Якщо ви застосуєте описаний вище код, отримаєте такий результат:

```
Hello, Debug!
```

## Занурення в деталі

1. Історичний контекст: Lua з самого початку була призначена для вбудованого використання, включаючи відлагоджувальний вивід.
2. Альтернативи: Вбудована функція `print` є найпростішим способом вивести відлагоджувальну інформацію, але ви також можете використовувати бібліотеку `io` для більш складних задач.
3. Деталі реалізації: Вивід відлагоджування у Lua можна реалізувати через перенаправлення виводу функції `print` в файл, замість консолі.
