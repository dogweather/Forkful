## Що і чому?

Починання нового проєкту — це створення первинного коду для ваших ідей. Програмісти роблять це, аби втілити софтверні рішення, що вирішують конкретні завдання або задовольняють потреби.

## Як це зробити:

Створімо простий проєкт. Перш за все, запишемо "Hello, world!" скрипт.

```Lua
print("Привіт, світ!")
```

Коли запустите, отримаєте наступне:

```
Привіт, світ!
```

Тепер додамо трохи логіки, щоб програма могла відповідати на просте запитання.

```Lua
local name = "Іванко"
print("Як тебе звати?")
if name == "Іванко" then
    print("Привіт, " .. name .. "!")
else
    print("Ти не Іванко!")
end
```

Результат:

```
Як тебе звати?
Привіт, Іванко!
```

## Поглиблене занурення:

Lua — це легка і гнучка мова програмування. Розроблена у Бразилії, у 1993 році, як інструмент для конфігурації програмного забезпечення. Через свою простоту та інтеграційні можливості, Lua швидко стала популярною для вбудованого скриптування в ігри та програми.

Як альтернативу, можна використовувати Python або JavaScript для старту нових проєктів, але Lua вирізняється завдяки своїй швидкості і мінімалізму.

З погляду реалізації, Lua використовує віртуальну машину для виконання байткоду. Це означає, що Lua скрипти можна використовувати на багатьох платформах без змін у коді.
