## Чому
У програмуванні існує багато різних завдань, і часом важливо змінити формат тексту для зручності або виконання певних дій. Капіталізація рядка - це один з таких завдань, яке допомагає перетворити рядок таким чином, щоб всі перші літери слів були з великої літери.

## Як це зробити
Існує декілька способів капіталізувати рядок в PHP. Один з найпростіших - використання функції ```ucwords()```, яка дозволяє змінити тільки першу літеру кожного слова в рядку. Наприклад:
```
<?php
$name = "максим";
echo ucwords($name); // Виведе "Максим"
?>
```
Існує також можливість використовувати цей метод у разі, коли рядок містить слова з великої літери. Також можна змінити весь рядок з використанням функції ```strtoupper()```. Наприклад:
```
<?php
echo strtoupper("привіт"); // Виведе "ПРИВІТ"
?>
```
Інша цікава функція - ```mb_convert_case()```. Вона дозволяє не тільки капіталізувати рядок, але і змінити його на будь-яку іншу кастомну форму, наприклад, на всі великі літери кирилиці. Наприклад:
```
<?php
echo mb_convert_case("привіт", MB_CASE_UPPER, "UTF-8"); // Виведе "ПРИВІТ"
?>
```

## Глибока капсулація
Функції для капіталізації рядка можуть бути корисними, коли ви хочете створити більш зручний та красивий вивід для користувача, або коли необхідно виконати певний тип операцій з текстом.
