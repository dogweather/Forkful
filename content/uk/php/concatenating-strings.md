## Чому

Конкатенація стрічок - це корисна техніка у PHP, яка дозволяє об'єднувати різні значення в одну стрічку. Це дозволяє робити наш код більш гнучким і ефективним.

## Як

Один з найпростіших способів конкатенації стрічок - використання оператора `.`. Приклад коду:

```PHP
$name = "Марія";
$age = 25;

echo "Привіт! Я " . $name . " і мені " . $age . " років.";
```

Вивід:

```
Привіт! Я Марія і мені 25 років.
```
Інший спосіб - використання функції `sprintf`. Приклад коду:

```PHP
$name = "Петро";
$age = 32;

echo sprintf("Привіт! Я %s і мені %d років.", $name, $age);
```

Вивід:

```
Привіт! Я Петро і мені 32 років.
```

## ProfDeep Dive

Конкатенація стрічок у PHP також може бути корисною для форматування даних. Наприклад, використовуючи функцію `sprintf`, ми можемо вказати, скільки знаків повинно бути виведено після коми. Також можна використовувати спеціальні символи, наприклад, `%s` для стрічок і `%d` для чисел.
