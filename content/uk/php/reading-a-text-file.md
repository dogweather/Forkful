## Чому

Програмування на PHP є надзвичайно корисним навичкою для будь-якого, хто зацікавлений у розробці веб-сайтів та додатків. Один з базових елементів програмування на PHP - це робота з текстовими файлами. Це важливий навичок, який дозволяє PHP розпізнати та обробляти інформацію з текстових файлів, таких як CSV або TXT.

## Як

Для початку, нам потрібно відкрити наш текстовий файл з допомогою функції `fopen()`. Наприклад, якщо наш файл називається "data.csv", ми можемо використати такий код:

```PHP
$file = fopen("data.csv", "r");
```

Ми вказуємо, що файл відкривається для читання (`"r"`). Далі, можна використовувати цей об'єкт для отримання даних з файлу. Наприклад, ми можемо використати функцію `fgetcsv()`, щоб зчитати дані з файла у вигляді масиву.

```PHP
while (($data = fgetcsv($file, 1000, ",")) !== FALSE) {
  print_r($data);
}
```

Цей приклад виведе на екран усі рядки з нашого файла у вигляді масивів, де значення розділяються комами. Зверніть увагу, що ми використовуємо `while` для того, щоб отримати усі рядки файлу доти, доки не дойдемо до його кінця.

## Deep Dive

Якщо ви хочете докладніше ознайомитися з роботою з текстовими файлами на PHP, ви можете скористатися іншими функціями, такими як `fgets()` для читання файлу по рядку чи `fseek()` для навігації по файлу.

Також, можливо, ви захочете виконати певні дії з отриманими даними, наприклад, додати їх до бази даних. Для цього ви можете використати функцію `explode()`, щоб розділити значення між комами та зберегти їх у змінні. Потім ви можете використати запит до бази даних, щоб додати ці дані.
