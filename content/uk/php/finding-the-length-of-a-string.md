## Чому

Знаходження довжини рядка - це важлива задача при програмуванні, оскільки це дозволяє отримати інформацію про кількість символів в рядку. Це може бути корисно для різних операцій з рядками, таких як обрізання, зміна регістру і багато іншого.

## Як

```PHP
$my_string = "Привіт світе!";
echo strlen($my_string); // Виведе 13
```

У цьому прикладі ми використовуємо функцію `strlen()` для знаходження довжини рядка `Привіт світе!`. Результатом буде число 13, оскільки в рядку є 13 символів.

### Ще один приклад:

```PHP
$language = "PHP";
echo strlen($language); // Виведе 3
```

Цей приклад показує, що функція `strlen()` також може використовуватись для вимірювання довжини змінної.

## Deep Dive

Звертаємо увагу, що функція `strlen()` повертає кількість байтів, а не символів. Якщо текст містить кириличні символи або інші не-ASCII символи, то кількість байтів не буде відповідати кількості символів. У такому випадку слід використовувати функцію `mb_strlen()`.

Крім того, можуть виникати проблеми з підрахунком кількості символів в рядках, які містять HTML теги, оскільки `<` і `>` вважаються окремими символами. Тому, якщо ми хочемо отримати правильну кількість символів, слід використовувати функцію `strip_tags()` для видалення HTML тегів перед використанням `strlen()`.
