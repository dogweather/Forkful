## Що і чому?

Перевірка наявності директорії є фундаментальним завданням у програмуванні на PHP, оскільки вона дозволяє верифікувати наявність директорії перед виконанням операцій, таких як читання з файлів або запис у них, що знаходяться у ній. Ця операція допомагає уникнути помилок, які могли б виникнути при спробі доступу до неіснуючих директорій, і є невід'ємною для динамічного управління файлами у вашому застосунку.

## Як це зробити:

Рідним способом перевірки наявності директорії у PHP є використання функції `is_dir()`. Ця функція приймає як аргумент шлях до файлу та повертає `true`, якщо директорія існує і є директорією, або `false` у протилежному випадку.

```php
$directoryPath = "/path/to/your/directory";

if(is_dir($directoryPath)) {
    echo "Директорія існує.";
} else {
    echo "Директорія не існує.";
}
```

Приклад виводу:
```
Директорія існує.
```
Або, якщо директорія не існує:
```
Директорія не існує.
```

Хоча стандартна бібліотека PHP достатньо надійна для більшості завдань по роботі з директоріями та файлами, іноді вам може знадобитися більш комплексне рішення. Для таких випадків популярною сторонньою бібліотекою є компонент Symfony Filesystem. Він пропонує широкий спектр утиліт для роботи з файловою системою, включаючи простий спосіб перевірки наявності директорії.

Спершу вам потрібно встановити компонент Symfony Filesystem. Якщо ви користуєтесь Composer (менеджером залежностей для PHP), ви можете запустити наступну команду у вашій проєктній директорії:

```
composer require symfony/filesystem
```

Після встановлення компонента Symfony Filesystem ви можете використовувати його, щоб перевірити наявність директорії так:

```php
use Symfony\Component\Filesystem\Filesystem;

$filesystem = new Filesystem();
$directoryPath = '/path/to/your/directory';

if($filesystem->exists($directoryPath)) {
    echo "Директорія існує.";
} else {
    echo "Директорія не існує.";
}
```

Приклад виводу:
```
Директорія існує.
```
Або, якщо директорія не існує:
```
Директорія не існує.
```

Обидва методи надають надійні способи для перевірки наявності директорії у PHP. Вибір між використанням вбудованих функцій PHP або сторонньою бібліотекою, як-от компонент Filesystem від Symfony, залежить від конкретних потреб вашого проєкту і того, чи вам потрібні додаткові маніпуляції з файловою системою, які можуть бути більш ефективно оброблені за допомогою бібліотеки.
