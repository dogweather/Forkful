## Чому 

Перевірка наявності директорії є важливою частиною програмування в PHP. Це дозволяє перевіряти, чи існує певна директорія на сервері та виконувати відповідні дії з нею. Наприклад, створювати нову директорію або копіювати файли у вже існуючу директорію. 

## Як це зробити 

Для перевірки наявності директорії використовується функція `is_dir()`. Вона приймає аргументом шлях до директорії та повертає `true`, якщо директорія існує, або `false`, якщо директорія не існує. Це виглядає наступним чином: 

```PHP 
if(is_dir("шлях_до_директорії")) {
    echo "Директорія існує";
} else {
    echo "Директорія не існує";
}
```

Приклад виводу: 

Директорія існує

## Глибокий аналіз 

Щоб краще зрозуміти, як працює перевірка наявності директорії, потрібно звернутися до внутрішнього механізму процесу. Спочатку, функція `is_dir()` викликає системний виклик `opendir()`, який відкриває файл, який містить директорію, та повертає вказівник на цей файл. Якщо ця операція вдається, відбувається перевірка вказівника за допомогою функції `readdir()`, яка перевіряє, чи є це директорія, а потім повертає вказівник на неї. Якщо цей вказівник не пустий, то функція `is_dir()` повертає `true`, інакше - `false`.
