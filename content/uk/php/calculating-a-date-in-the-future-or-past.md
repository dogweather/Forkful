## Чому

Розрахунок дати в майбутньому або в минулому може бути корисним для багатьох задач програмування, наприклад, запланування подій або взаємодії з даними, які змінюються з плином часу.

## Як це зробити

Для розрахунку дати в майбутньому або в минулому використовуються функції `strtotime` та `date` у мові програмування PHP. Наприклад, щоб отримати дату, яка буде через 10 днів, можна використовувати наступний код:

```PHP
$date = strtotime("+10 days");
echo date("Y-m-d", $date); // виведе: 2021-05-29
```

Або, якщо потрібно отримати дату, яка була 20 днів тому, можна використовувати наступний код:

```PHP
$date = strtotime("-20 days");
echo date("Y-m-d", $date); // виведе: 2021-05-09
```

## Глибоке дослідження

Функція `strtotime` використовує ряд ключових слів для позначення різних періодів часу, наприклад "days" - для днів, "months" - для місяців, "years" - для років. Крім того, у функції можна використовувати і цифрові значення для позначення періоду часу.

Функція `date` використовує формати для виведення дати у зручному для нас вигляді. Наприклад, `Y-m-d` виводить дату у вигляді року-місяця-дня.
