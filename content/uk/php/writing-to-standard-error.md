## Чому

Запис до стандартного помилкового потоку є важливою частиною програмування в PHP. Це дозволяє вам отримувати інформацію про помилки та відлагоджувати код з більшою ефективністю.

## Як

Для запису до стандартного помилкового потоку використовуйте функцію `fwrite()` разом з файловим дескриптором `STDERR`. Наприклад:

```PHP
<?php
// Відкриття файлу дескриптору
$fh = fopen('php://stderr', 'w');
// Запис до стандартного помилкового потоку
fwrite($fh, 'Це текст помилки!\n');
// Зачинення файлу дескриптору
fclose($fh);
```

Ви отримаєте наступний вихід:

```
Це текст помилки!
```

## Глибше

При використанні функції `fwrite()` важливо враховувати, що ця функція повертає кількість байтів, які були успішно записані до файлу. Якщо ця кількість не дорівнює довжині рядка, то це означає, що виникла помилка при записі.

Також, ви можете записувати до стандартного помилкового потоку за допомогою функції `error_log()`. Ця функція приймає три аргументи: повідомлення помилки, тип помилки (який відображатиметься у логах), і файл, куди буде записана помилка. Наприклад:

```PHP
<?php
// Запис до стандартного помилкового потоку за допомогою функції error_log()
error_log('Це інше повідомлення про помилку!', 3, 'errors.log');
```
