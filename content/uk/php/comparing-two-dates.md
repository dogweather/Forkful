##Чому

У програмуванні нерідко виникає необхідність порівняти дві дати для виконання певних дій або встановлення послідовності подій. Це може бути корисно, наприклад, при оформленні замовлення або визначенні давності події.

##Як

У PHP існує кілька способів порівняти дві дати, і ми розглянемо два найпоширеніших з них.

```PHP
$date1 = '2020-10-15';
$date2 = '2020-10-20';

// Перший спосіб: використання функції strtotime
if (strtotime($date1) > strtotime($date2)) {
  echo 'Дата 1 пізніше Дати 2';
} elseif (strtotime($date1) < strtotime($date2)) {
  echo 'Дата 1 раніше Дати 2';
} else {
  echo 'Обидві дати однакові';
}

// Другий спосіб: використання класу DateTime
$d1 = new DateTime($date1);
$d2 = new DateTime($date2);
if ($d1 > $d2) {
  echo 'Дата 1 пізніше Дати 2';
} elseif ($d1 < $d2) {
  echo 'Дата 1 раніше Дати 2';
} else {
  echo 'Обидві дати однакові';
}
```

Результат виконання коду буде таким:

```
Дата 1 раніше Дати 2
Дата 1 раніше Дати 2
```

Як бачимо, результати обох способів порівняння дат однакові, тому ви можете вибрати той, який більш зручний для вас.

##Глибше

Для порівняння дат у PHP можна використовувати такі функції, як `strtotime`, `DateTime`, `date_create` та інші. Ці функції дозволяють не тільки порівняти дати, але і виконувати різноманітні операції з ними, такі як додавання чи віднімання певної кількості часу.

Наприклад, для визначення різниці між двома датами можна використовувати клас `DateInterval`:

```PHP
$date1 = new DateTime('2020-10-10');
$date2 = new DateTime('2020-10-15');
$diff = $date1->diff($date2);

echo $diff->days;
// Виведе 5 – кількість днів між датами
```

Також варто знати, що в PHP встановлений стандартний формат дати `YYYY-MM-DD`, тому перед порівнянням дати треба перетворювати до цього формату, щоб уникнути помилок.
