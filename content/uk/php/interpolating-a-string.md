## Що це та навіщо?

Інтерполяція рядків - це процес вставки значень змінних безпосередньо в рядок. Програмісти використовують це для лаконічності та кращої читабельності коду.

## Як це зробити: 

PHP підтримує два види рядків, одинарні та подвійні лапки. Інтерполяція рядків працює тільки в подвійних лапках. 

Наприклад:

```PHP
$name = "Василь";
echo "Привіт, $name!";
```
Виконання цього коду дасте вам результат: 

```
Привіт, Василь!
```
## Занурення вглиб:

Історично PHP був створений для роботи з рядками і веб-сторінками, тому інтерполяція рядків була однією з його ключових особливостей.

Замість інтерполяції рядків, ви також можете використовувати функцию `sprintf()`, або конкатенацію з допомогою `.`. 

Використання:

```PHP
$name = "Василь";
echo sprintf("Привіт, %s!", $name);
```
або

```PHP
$name = "Василь";
echo 'Привіт, ' . $name . '!';
```
Інтерполяція рядків в PHP використовується в runtime, тож виконання цієї операції може мати вплив на продуктивність, особливо при великій кількості великих змінних.
