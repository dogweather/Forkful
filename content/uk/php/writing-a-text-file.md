## Чому

Над цінною інформацією ви працюєте довго і наполегливо. Напевно, ви не хочете втрачати її через випадкові помилки чи втрату даних. Тому важливо мати можливість зберігати свою інформацію в надійному та структурованому форматі - такому як текстовий файл. Це дозволяє не тільки зберігати інформацію, але і зберігати її в порядку, щоб в майбутньому легше було отримати доступ до неї.

## Як

Написання текстових файлів у PHP досить просте та зручне завдання. Для цього можна використовувати функції `fopen()`, `fwrite()`, `fclose()` та інші. Нижче наведено приклад коду, який демонструє, як створити та записати інформацію у текстовий файл:

```PHP
$file = fopen("mydata.txt", "w"); // відкрити файл для запису

$data = "Це приклад інформації, яка буде записана в файл."; // дані, які будуть записані

fwrite($file, $data); // записати дані в файл
fclose($file); // закрити файл
```

Якщо треба додати інформацію до вже існуючого файлу, можна використати функцію `a+` у відкритті файлу, яка автоматично переносить курсор до кінця файлу перед додаванням нової інформації. Наприклад:

```PHP
$file = fopen("mydata.txt", "a+"); // відкрити файл для додавання інформації

$data = "Це нова інформація, яка буде додана до файлу."; // дані, які будуть додані

fwrite($file, $data); // додати дані до файлу
fclose($file); // закрити файл
```

## Deep Dive

Написання текстових файлів у PHP також надає багато додаткових можливостей, наприклад, можна використовувати цикли для записування більшої кількості інформації або використовувати функцію `file_put_contents()` для зручного запису даних у файл. Також варто пам'ятати про те, що робота з текстовими файлами може бути пов'язана з певними ризиками, тому важливо завжди перевіряти дані перед записом та робити резервну копію файлів для запобігання втрати даних.
