## Що і чому?

Створення тимчасового файлу - це процес створення файлу, який використовується для зберігання інформації тимчасово. Програмісти роблять це, щоб обробляти великі дані, уникнути зайвої витрати пам'яті і зберегти дані під час виконання складних задач.

## Як це робиться:

```PHP
<?php
$tempFile = tmpfile();

fwrite($tempFile, 'Тимчасовий текст для прикладу');
rewind($tempFile);

echo fgets($tempFile);

fclose($tempFile);
?>
```
Запускаючи цей код, ви побачите 'Тимчасовий текст для прикладу' виведений у консолі.

## Занурення в глибину:

Історично PHP завжди надавала можливість створювати тимчасові файли для обробки значних об'ємів даних. Є альтернативи, такі як використання пам’яті (наприклад, пам'ять RAM), але це може бути небезпечно для критично важливих програм, які не можуть дозволити собі заповнити всю доступну пам’ять. Під час створення тимчасового файлу, PHP автоматично видаляє файл, коли закривається дескриптор файлу або закінчується скрипт PHP.
