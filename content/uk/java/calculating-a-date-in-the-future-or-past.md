# Чому

Ви, напевно, питаєтесь, чому хтось міг би зацікавитися обчисленням дати у майбутньому або минулому. Ну, це досить корисний навик для розрахунку термінів та виконання інших завдань, пов'язаних з часом. Наприклад, може бути потрібно порахувати, скільки днів залишилося до важливої події або отримати дату із зазначеним розривом в часі.

# Як

Для розрахунку дати у майбутньому або минулому нам знадобиться використати клас ```LocalDate``` з пакету ```java.time```. Ось приклад коду для отримання дати на 10 днів у майбутньому:

```Java
LocalDate today = LocalDate.now();
LocalDate futureDate = today.plusDays(10);
System.out.println(futureDate);
```

В даному прикладі ми спочатку отримуємо поточну дату за допомогою методу ```now()```, а потім застосовуємо метод ```plusDays()```, який дозволяє додати необхідну кількість днів до поточної дати. Ми використовуємо метод ```println``` для виводу результату у консоль.

Також можна використати інші методи, такі як ```minusDays()```, ```plusMonths()```, ```minusYears()``` тощо, щоб отримати дати у минулому чи з іншими змінами.

# Глибокий погляд

У Java є багато різноманітних класів та методів для роботи з датами. Для більш складних операцій, наприклад, порівняння дат, можна використати клас ```ZonedDateTime```. Також варто зазначити, що в піклувальництві у Java 9 було введено новий клас ```YearMonth```, який дозволяє працювати зі змінними, що містять лише рік та місяць.

Загалом, розуміння роботи із датами та використання відповідних класів та методів може зайняти деякий час і вимагати певної практики, але у кінці кінців, це надзвичайно корисний навик для майбутніх Java-розробників.
