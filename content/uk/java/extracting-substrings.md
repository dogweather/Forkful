# Чому
Витягування підрядків є важливою частиною програмування на Java і може бути корисним для різних завдань, таких як обробка рядків або пошук певної інформації у великих текстових документах.

## Як
Витягування підрядків в Java можна зробити шляхом використання методу `substring()` для рядків. Для цього необхідно вказати початковий та кінцевий індекси в дужках, наприклад `myString.substring(2,5)`, щоб витягнути підрядок з 2-го по 4-й символи (індекси рахуються з нуля). Для зручності також можна використовувати метод `indexOf()`, який повертає індекс першого входження певної підрядка в рядок.

```Java
String str = "Привіт, це приклад рядку.";
System.out.println(str.substring(2,5)); // витягне "иві"
System.out.println(str.indexOf("приклад")); // поверне 9
```

## Вдивитися
Витягування підрядків може бути більш складним застосуванням, якщо потрібно врахувати різні випадки, такі як відсутність підрядка або використання змінної кількості символів. Також можна використовувати метод `substring()` для модифікації рядків, що може бути корисним для конвертації форматів дат або видалення певних символів. При використанні в більш складних програмах, важливо врахувати ефективність та продуманість підходу до витягування підрядків.
