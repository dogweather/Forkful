## Чому 

В цьому пості ми поговоримо про запис до стандартного виводу помилок у програмуванні на Java та чому це може бути корисно.

## Як це зробити

В Java, є функція ```System.err.println ()```, яка дозволяє нам записувати повідомлення до стандартного виводу помилок. Наприклад:

```
Java
System.err.println("Цей рядок буде виведено до стандартного виводу помилок.");
```

В результаті ви побачите наступну виведену інформацію:

```
Цей рядок буде виведено до стандартного виводу помилок.
```

## Глибші відомості

Записування до стандартного виводу помилок дозволяє нам вивести інформацію про помилки в нашій програмі, що може стати цілком корисним при відлагодженні. Крім того, це також може бути корисним при роботі з більш складними системами, де необхідно моніторити роботу додатків.
