## Чому

Генерація випадкових чисел є важливою задачею в багатьох програмах, таких як ігри, криптографія та статистичні дослідження. Випадкові числа дозволяють створити різноманіття в роботі програм та забезпечують більш безпечну обробку конфіденційної інформації.

## Як

Існує кілька способів генерування випадкових чисел в Java. Один з них - використання класу `Random` з пакету `java.util`. Для цього необхідно створити об'єкт `Random` та використовувати його методи для отримання випадкових значень у потрібному діапазоні.

```Java
import java.util.Random;

public class RandomNumberGenerator {
    public static void main(String[] args) {
        // Створення об'єкту Random
        Random rand = new Random();

        // Отримання випадкового цілого числа
        int randomInt = rand.nextInt();

        // Отримання випадкового числа в діапазоні від 0 до 10
        int randomInRange = rand.nextInt(11);

        // Отримання випадкового числа типу double
        double randomDouble = rand.nextDouble();

        // Отримання випадкового числа в діапазоні від 0 до 1
        double randomDoubleInRange = rand.nextDouble() * 10;

        // Виведення результату
        System.out.println("Випадкове ціле число: " + randomInt);
        System.out.println("Випадкове число в діапазоні від 0 до 10: " + randomInRange);
        System.out.println("Випадкове число типу double: " + randomDouble);
        System.out.println("Випадкове число в діапазоні від 0 до 1: " + randomDoubleInRange);

    }
}
```

Вихідні дані можуть бути, наприклад:

```
Випадкове ціле число: -1522065779
Випадкове число в діапазоні від 0 до 10: 7
Випадкове число типу double: 0.42560781723578953
Випадкове число в діапазоні від 0 до 1: 6.024018264636473
```

## Глибоке дослідження

У Java також є можливість використовувати клас `Math` для генерування випадкових чисел. Цей клас містить багато методів, які дозволяють отримувати випадкові значення у потрібному діапазоні або типі даних.

```Java
// Отримання випадкового цілого числа в діапазоні від 0 до 100
int randomIntInRange = (int) (Math.random() * 100);

// Отримання випадкового числа типу double в діапазоні від 0 до 5
double randomDoubleInRange = Math.random() * 5;
```
