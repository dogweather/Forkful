## Чому

Однією з основних задач програмування є пошук і обробка даних. Іноді нам потрібно знайти та видалити певні символи з тексту згідно заданого шаблону. Це може виявитися корисним для роботи з даними, які містять непотрібну інформацію або серед підтвердження введеного користувачем даних.

## Як

Існує кілька способів видалення символів відповідно до шаблону в Java. Ми розглянемо три популярні шляхи для досягнення цієї мети.

Припустимо, що у нас є текстовий рядок "Привіт, це прикладний текст для видалення символів." Ми хочемо видалити всі знаки пунктуації. Нижче наведені приклади коду, які показують різні способи видалення символів.

```
// З використанням методу replaceAll ()
String str = "Привіт, це прикладний текст для видалення символів.";
String output1 = str.replaceAll("\\p{P}", "");
System.out.println(output1); // Виведе: Привіт це прикладний текст для видалення символів

// З використанням методу replaceAll () та регулярних виразів
String str = "Привіт, це прикладний текст для видалення символів.";
String output2 = str.replaceAll("[^a-zA-Z0-9]", "");
System.out.println(output2); // Виведе: Привітцеприкладнийтекстдлявидаленнясимволів

// З використанням циклу for
String str = "Привіт, це прикладний текст для видалення символів.";
StringBuilder output3 = new StringBuilder();
for(int i = 0; i<str.length(); i++){
    char c = str.charAt(i);
    if(Character.isLetterOrDigit(c)){
        output3.append(c);
    }
}
System.out.println(output3.toString()); // Виведе: Привітцеприкладнийтекстдлявидаленнясимволів
```

Незалежно від способу, результат буде такий самий - всі знаки пунктуації будуть видалені з тексту.

## Глибокий занурення

У Java є кілька вбудованих методів для роботи з рядками, таких як `replaceAll ()`, який замінює всі співпадіння заданого регулярного виразу, та `charAt ()`, який повертає символ за вказаним індексом у рядку. Ми також можемо використовувати регулярні вирази для знаходження та заміни певних символів у рядку.
