### Чому

Маніпулювання текстовими рядками є необхідною частиною програмування, і одна з таких операцій - це капіталізація рядка. Капіталізація дозволяє перетворити першу букву кожного слова у рядку на велику, що може бути корисно при форматуванні даних або приватній інформації. Давайте подивимося, як це зробити за допомогою мови програмування Java.

### Як це зробити

Для початку нам потрібно створити метод, який приймає вхідний рядок та повертає капіталізований результат. Ми можемо перебрати кожен символ у рядку, перетворити перший символ кожного слова на велику літеру, а решту залишити без змін. Тут приведений приклад коду, який виконує цю операцію:

```java
public static String capitalizeString(String input){
    String[] words = input.split(" ");
    StringBuilder capitalized = new StringBuilder();

    for(String word: words){
        char firstLetter = Character.toUpperCase(word.charAt(0));
        String restOfWord = word.substring(1);
        String capitalizedWord = firstLetter + restOfWord;
        capitalized.append(capitalizedWord + " ");
    }

    return capitalized.toString().trim();
}
```

Якщо ми виконаємо цей метод на рядку "привіт світ!", ми отримаємо результат "Привіт Світ!". Іншим методом, який також можна використовувати для капіталізації рядка, є використання класса `StringTokenizer` або методу `String.toUpperCase()`. Обидва ці методи також дають змогу легко капіталізувати рядок.

### Глибоке занурення

Основний підхід до капіталізації рядків полягає в перетворенні першої букви кожного слова на велику. Однак існують випадки, коли нам необхідно врахувати виключення, такі як мали підслова або абревіатури в середині рядка. Також важливо використовувати правильну локалізацію при капіталізації. Наприклад, українська мова має використовувати правило капіталізації, де початкова ініціали написані великими буквами, якщо вони складаються лише з кількох символів.
