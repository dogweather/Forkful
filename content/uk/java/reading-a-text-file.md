## Чому
Робота з текстовими файлами є важливою частиною програмування на Java. В цьому пості ми поговоримо про те, чому варто вчитися читати текстові файли, а також як це робити за допомогою коду Java.

## Як це зробити
Із цими декількома кроками ви будете в стані легко читати текстові файли на Java.

```Java
// Відкриваємо потік для читання текстового файлу
FileReader fileReader = new FileReader("файл.txt");

// Створюємо об'єкт, який буде зчитувати дані з файлу
BufferedReader bufferedReader = new BufferedReader(fileReader);

// Створюємо змінну для зберігання прочитаного рядка
String line;

// Читаємо рядки з файлу, поки не дійдемо до кінця
while ((line = bufferedReader.readLine()) != null) {
    // Виконуємо дії з отриманим рядком
    System.out.println(line);
}

// Закриваємо потоки
bufferedReader.close();
fileReader.close();
```

Припустимо, що наш файл має наступний вміст:

```
Привіт!
Якщо ви читаєте цей файл, то ви, безсумнівно, дуже розумні люди.
Дякуємо за ваш інтерес до програмування на Java!
```

В результаті виконання коду ми отримаємо наступний вивід:

```
Привіт!
Якщо ви читаєте цей файл, то ви, безсумнівно, дуже розумні люди.
Дякуємо за ваш інтерес до програмування на Java!
```

## Глибоке погруження
Тепер давайте трохи розберемося в деталях, щоб краще зрозуміти, як саме відбувається читання текстового файлу на Java.

У першому рядку ми створили об'єкт `FileReader`, який відповідає за відкриття потоку для читання файлу з заданою назвою. Потім ми створили об'єкт `BufferedReader`, який буде використовуватися для зчитування даних з відкритого файлу.

У циклі `while` ми використовуємо метод `readLine()` об'єкту `BufferedReader`, який читає рядок з файлу та повертає його як рядок типу `String`. Коли доходимо до кінця файлу, метод повертає `null`, що дає нам змогу завершити цикл.

Останні два рядки закривають відповідні потоки для очищення ресурсів.
