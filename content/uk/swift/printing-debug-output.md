## Чому

Іноді, під час програмування на Swift, може знадобитися виводити різноманітну інформацію для допомоги в розумінні процесу виконання програми. В найпростішому варіанті, це може бути просто текст, який допоможе вам зрозуміти, що відбувається в певному місці вашого коду. Тому виведення відладкових повідомлень (debug output) - це необхідний інструмент у вашому програмістському арсеналі.

## Як

У Swift існує кілька способів виводу відладкових повідомлень. Один із них - використання функції `print()`, яка дозволяє виводити будь-яку інформацію, включаючи рядки, числа, змінні, та навіть результати виразів. Спробуйте наступний приклад коду:

```Swift
let name = "Іван"
let age = 30
print("Мене звати \(name) і мені \(age) років")
```

В результаті на консолі ви побачите `Мене звати Іван і мені 30 років`.

Інший спосіб - використання функцій `dump()` або `debugPrint()`, які дозволяють виводити структури даних та об'єкти з більш детальною інформацією. Наприклад:

```Swift
struct Point {
    var x: Int
    var y: Int
}

let point = Point(x: 4, y: 7)
dump(point)
debugPrint(point)
```

Результатом буде виводу на консоль такого:

```
▿ Point
  - x: 4
  - y: 7
struct Point #0 ({
  // Схоже на вивід за допомогою print(), але тут також вказується назва типу даних та кожне поле на окремому рядку.
  x: 4,
  y: 7,
}) 
```

Як бачите, виведення відладкових повідомлень може бути дуже корисним для розуміння виконання вашої програми.

## Детальна інформація

У деяких ситуаціях, виведення простого тексту або значень змінних недостатнє для вирішення проблем з вашим кодом. Тоді необхідно дослідити більш детальну інформацію про стан вашої програми. Наприклад, ви можете вивести значення окремих властивостей об'єкта, вивести наступні рядки коду та виведення впереміш:

```Swift
class Person {
    var name: String
    var age: Int
    
    init(name: String, age: Int) {
        self.name = name
        self.age = age
    }
}

let person = Person(name: "Олексій", age: 25)
print("Ім'я: \(person.name)")
print("Вік: \(person.age)")
```
