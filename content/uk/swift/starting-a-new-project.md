## Що & Навіщо?
Старт нового проєкту — це наче відкриття чистої сторінки для кодування. Програмісти розпочинають нові проєкти, щоб втілити ідеї в життя, вирішити проблеми або просто для самовдосконалення.

## Як це зробити:
```Swift
// Встановлення Xcode з App Store.
// Створення нового проєкту в Xcode:
// 1. Відкрити Xcode
// 2. Файл -> Створити новий проєкт (File -> New -> Project)
// 3. Вибрати шаблон для вашого додатку
// 4. Надати назву проекту та іншу інформацію

import SwiftUI

@main
struct MyApp: App {
    var body: some Scene {
        WindowGroup {
            ContentView()
        }
    }
}

struct ContentView: View {
    var body: some View {
        Text("Ласкаво просимо до мого нового проєкту!")
            .padding()
    }
}

// При запуску, ви побачите наступний текст:
// "Ласкаво просимо до мого нового проєкту!"
```

## Поглиблений Розбір:
Створення нового проєкту в Swift веде своє походження від традиційних методів програмування, де кожен файл і папка мали бути організовані вручну. З появою Xcode, Apple спростила процес до декількох кліків, автоматизувавши багато нудної роботи. Хоча Swift і є вибором за замовчуванням для розробки на Apple платформах, існують альтернативи, як-от Objective-C, але Swift є більш сучасним та безпечним. Важливо розуміти, що коли ви створюєте новий проєкт в Xcode, інтегроване середовище розробки (IDE) налаштовує все необхідне, щоби ви могли зосередитися на написанні коду.
