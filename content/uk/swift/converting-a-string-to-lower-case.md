## Чому

Конвертація рядка до нижнього регістру є важливою операцією в розробці програмного забезпечення, оскільки багато функцій і алгоритмів працюють тільки з нижнім регістром. Використання цього методу допоможе забезпечити правильну інтеракцію з іншими функціями та збільшить точність обробки даних.

## Як

Для конвертації рядка до нижнього регістру використовується метод `lowercased()`. Наприклад, якщо ми маємо рядок "Hello World", застосування методу виглядатиме наступним чином:

```Swift
let string = "Hello World"
let lowercasedString = string.lowercased()
print(lowercasedString) // результат: "hello world"
```

Як бачимо, метод повертає новий рядок зі зміненим регістром. Важливо також зазначити, що цей метод не змінює оригінальний рядок, а повертає новий результат.

## Глибокий розгляд

При конвертації рядка до нижнього регістру використовується стандарт Unicode для визначення регістру літер. Це означає, що для різних мов та символів, можуть бути використані різні правила конвертації. Наприклад, для мови з кириличним алфавітом будуть використовуватися правила для цієї мови, тоді як для латинських символів будуть використовуватися правила для латинської мови.

Додатково, існує також метод `localizedLowercase`, який використовує правила, специфічні для мови та регіону налаштувань пристрою.
