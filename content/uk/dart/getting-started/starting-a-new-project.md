## Як зробити:
1. **Встановіть Dart**:
   Переконайтеся, що Dart встановлено на вашій системі. Якщо це не так, ви можете завантажити його з [https://dart.dev/get-dart](https://dart.dev/get-dart). Перевірте інсталяцію за допомогою:

   ```shell
   dart --version
   ```

2. **Створіть новий проект Dart**:
   Використовуйте Dart CLI, щоб згенерувати новий проект:

   ```shell
   dart create hello_dart
   ```

   Ця команда створює нову директорію `hello_dart` з простим прикладом веб- або консольного додатку, залежно від вашого вибору.

3. **Огляньте структуру проекту**:
   
   Перейдіть до директорії вашого проекту:

   ```shell
   cd hello_dart
   ```

   Типовий проект Dart містить наступні ключові файли та директорії:

   - `pubspec.yaml`: Файл конфігурації, який включає залежності вашого проекту та обмеження SDK.
   - `lib/`: Директорія, де розміщена більшість коду Dart.
   - `test/`: Директорія для тестів проекту.

4. **Додайте залежності**:
   Відредагуйте `pubspec.yaml`, щоб додати залежності. Для веб-проектів розгляньте можливість додавання `http`, популярного пакета для виконання HTTP-запитів:

   ```yaml
   dependencies:
     flutter:
       sdk: flutter
     http: ^0.13.3
   ```

   Після редагування отримайте залежності:

   ```shell
   dart pub get
   ```

5. **Напишіть свій перший код на Dart**:
   
   У директорії `lib/` створіть новий файл Dart, `main.dart`, і додайте простий код Dart:

   ```dart
   // Імпорт основної бібліотеки Dart
   import 'dart:core';

   void main() {
     print('Hello, Dart!');
   }
   ```

6. **Запустіть свій додаток Dart**:

   Виконайте вашу програму на Dart за допомогою:

   ```shell
   dart run
   ```

   Вивід має бути:

   ```
   Hello, Dart!
   ```

Слідуючи цим крокам, ви успішно розпочали новий проект на Dart, починаючи від інсталяції до запуску вашого першого шматка коду на Dart. Ці базові знання створюють фундамент для глибшого занурення в багату екосистему Dart та її можливості для створення масштабованих додатків.
