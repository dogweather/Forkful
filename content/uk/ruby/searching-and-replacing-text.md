## Що та навіщо?
Пошук та заміна тексту - це процес, коли ви вказуєте деякий текст для пошуку й інший для його заміни в прогресії або рядку. Програмісти роблять це, щоб керувати та маніпулювати даними безпосередньо в коді, зекономивши потрібний час та зусилля.

## Як це робити:
В Ruby, ви можете використовувати метод `gsub` для пошуку та заміни тексту. Давайте зробимо приклад:

```Ruby
text = "Я люблю кодувати на Ruby"
new_text = text.gsub("кодувати", "програмувати")
puts new_text
```
Вивід:
```Ruby
"Я люблю програмувати на Ruby"
```

Ви можете також використовувати регулярні вирази для комплексного пошуку тексту. Ось приклад:

```Ruby
text = "Я люблю кодувати на Ruby"
new_text = text.gsub(/кодувати|Ruby/, "Python")
puts new_text
```
Вивід:
```Ruby
"Я люблю Python на Python"
```

## Поглиблений розділ
Пошук та заміна тексту були ключовими складовими комп'ютерного програмування ще з його ранніх днів. В різних мовах ця функція має різні реалізації, але суть залишається тією ж. 

Більшість мов, включаючи Ruby, Perl та Python, мають вбудовані функції для цього. У Ruby `gsub` - це поширений спосіб зробити це. 

У мовах таких як Java або C++, вам, можливо, доведеться писати більше коду для виконання такої операції.
