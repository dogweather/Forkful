## Чому

Програмування - це незрівнянно цікаве і потужне заняття, але іноді у ньому потрібно робити однотипні дії, такі як заміна тексту. Великі обсяги тексту можна замінити швидше і ефективніше за допомогою пошуку та заміни в Ruby. Дізнайтеся, як це зробити у нашому блозі.

## Як це зробити

Для пошуку та заміни тексту в Ruby використовується метод `gsub`. Приклад застосування методу виглядає так:

```Ruby
"Привіт, друзі!".gsub("друзі", "милі")
```

Результат виконання коду буде:

```Ruby
"Привіт, милі!"
```

Якщо потрібно замінити всі входження слова, можна додати флаг `g`, наприклад:

```Ruby
"Привіт, друзі! Друзі - це так гарно!".gsub("друзі", "милі") # замінить лише перше входження слова
"Привіт, друзі! Друзі - це так гарно!".gsub("друзі", "милі", g) # замінить всі входження слова
```

Також можна використовувати регулярні вирази для пошуку та заміни тексту. Наприклад, якщо потрібно замінити всі символи, окрім латинських літер, на пустий рядок, можна використати такий регулярний вираз:

```Ruby
"Привіт, це 2020 рік!".gsub(/[^a-z]/, "") # видалить всі символи, крім латинських літер
```

Результат буде:

```Ruby
"vtitr" # "vtitr" - це транслітерований текст речення "gthfvjcrfz 2020 d jcnfywbz"
```

## Глибоке занурення

Метод `gsub` використовується для заміни всіх входжень заданого тексту на новий. Його також можна використовувати для видалення певних вкладень, наприклад:

```Ruby
"Привіт, це [бот]!".gsub(/\[.*\]/, "") # видалить підрядок в квадратних дужках
```

Результат виконання коду буде:

```Ruby
"Привіт, це !" # усе, що було між квадратними дужками було видалено
```

Існують також інші схожі методи для роботи з текстом, такі як `sub`, який замінює лише перше входження, або `scan`, який шукає певні відповідності в рядку. Вивчення регулярних виразів допоможе вам більш ефективно працювати з текстом у Ruby.
