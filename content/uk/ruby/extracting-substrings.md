## Чому

Існує багато ситуацій, коли робота з рядками є необхідною частиною програмування на Рубі. Наприклад, вам може знадобитися витягти частину тексту зі строку для подальшої обробки або валідації. Використання функції витягування підрядків дозволяє легко здійснити це завдання.

## Як це зробити

Існує кілька способів витягування підрядків у Рубі. Один з них - це використання методу `[]`, який приймає аргументи для вказівки початкової та кінцевої позицій в підрядку. Дивіться приклад нижче:

```Ruby
str = "Привіт, світ!"
puts str[0, 6] # "Привіт"
```

У цьому прикладі, ми вказали початкову позицію `0` та кінцеву позицію `6`, тому отримали підрядок з 6 перших символів зі строки `str`.

Також існує можливість вказати індекс або діапазон символів, які потрібно виключити з підрядка. Наприклад:

```Ruby
str = "Привіт, світ!"
puts str[7..-1] # "світ!"
```

У цьому прикладі, ми виключили перші 7 символів із строки `str` та отримали лише підрядок з решти символів.

Є ще багато інших методів для витягування підрядків у Рубі, таких як `slice`, `scan`, `match` тощо. Варто вивчити їх усі, щоб мати можливість ефективно працювати з рядками.

## Вдосконалений погляд

Крім використання вбудованих методів для витягування підрядків, також можна створити свої власні функції для цього. Зверніть увагу наступний приклад:

```Ruby
str = "Рубі є найкрасивішою мовою програмування"
def extract_substring(str, start, length)
  substr = ""
  start.upto(start + length - 1) do |i|
    substr << str[i]
  end
  return substr
end

puts extract_substring(str, 7, 9) # "є найкра"
```

У цьому прикладі, ми створили функцію `extract_substring`, яка приймає строку, початкову позицію та довжину підрядка і повертає відповідний результат.
