## Що це і чому?

Видобуток підрядків (Extracting substrings) - це процес вилучення частини рядка з існуючого рядка. Програмісти роблять це для маніпуляций із специфічними частинами інформації.

## Як це робити:

Отже, ось як ви можете використовувати Ruby для видобутку підрядків. В Ruby це виконується за допомогою оператора індексації.

```Ruby 
str = "Привіт, світе!"
p str[7,5]     # "світе"
```

В цьому прикладі, ми вилучаємо підрядок починаючи з 7-го символу і довжиною 5 символів. Оператор індексації в Ruby виконує подвійну функцію - видобуток підрядків і доступ до окремих символів.

```Ruby
p str[7]      # "с"
```

В цьому випадку ми отримуємо 7-й символ.

## Поглиблено:

1. **Історичний контекст**: Видобуток підрядків був частиною Ruby з самого початку, що відображає його силу як мови для роботи з текстом.
2. **Альтернативи**: Метод `slice` в Ruby також можна використовувати для видобутку підрядків. 
```Ruby
p str.slice(7,5)     # "світе"
```
3. **Деталі реалізації**: Видобуток підрядків в Ruby реалізований через метод `_substring_`, як метод класу `String`. Цей метод приймає два аргументи: початковий індекс і довжину підрядка.
