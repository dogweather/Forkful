## Що та чому?

Читання текстового файлу - це процес, коли програма відкриває та читає інформацію з текстового файлу. Програмісти роблять це, щоб організувати та маніпулювати данними, що зберігаються в цих файлах.

## Как делать:

Ruby - мова з простим синтаксисом. Для читання текстового файлу ми можемо використати `File` клас та його методи. Ось приклад:

```Ruby
# Відкриваємо файл з ім'ям 'example.txt' в режимі 'r' (читання)
file = File.open('example.txt', 'r')

# Читаємо весь файл
puts file.read

# Не забуваємо закрити файл після читання
file.close
```
Результат може виглядати так:

```
Привіт, світ! Це приклад текстового файлу.
```

## Поглиблений матеріал:

Історично, читання файлів було одним з основних способів обміну даними між програмами. Особливо це було актуально в епоху командного рядка, коли графічних інтерфейсів ще не було.

Щодо альтернатив, у Ruby також існують різні бібліотеки та геми для роботи з файлами. Ви також можете використовувати `IO` для відкриття та читання файлів.

Механізм читання файлу в Ruby є деталлю реалізації виртуальної машини Ruby (MRI). Так, наприклад, метод `File.read` зчитує весь файл в одній строкі, а метод `File.readlines` зчитує його построково.
