## Чому

Хочете дізнатися, як читати текстовий файл за допомогою Ruby? Цей блог-пост для вас! Читати тестові файли може бути корисно при роботі з програмами, що працюють з текстовими даними, або при аналізуванні логів сервера.

## Як це зробити

Для читання текстового файлу в Ruby використовується метод `File.read()`. Давайте створимо простий текстовий файл з назвою `text_file.txt`. У ньому знаходиться наступний текст:

```Ruby
Hello world!
This is a test file.
```

Тепер створимо новий файл з назвою `read_text_file.rb` і додамо такий код:

```Ruby
text = File.read("text_file.txt")

puts text
```

Після запуску цього файлу за допомогою команди `ruby read_text_file.rb` у терміналі, ми отримаємо наступний вихід:

```Ruby
Hello world!
This is a test file.
```

Також, можна використовувати метод `File.readlines()`, який повертає масив рядків, у якому кожен елемент є окремим рядком з файлу. Наприклад:

```Ruby
lines = File.readlines("text_file.txt")

puts lines[0] # вивід першого рядка
puts lines[1] # вивід другого рядка
```

Вивід буде той самий, що і в попередньому прикладі.

## Глибока розробка

При читанні текстового файлу, можна задавати різні параметри, такі як енкодинг файлу або режим читання. Також, можна зберігати прочитаний текст у змінну для подальшої обробки даних. Документація до методу `File.read()` допоможе вам розібратися з іншими варіантами використання.
