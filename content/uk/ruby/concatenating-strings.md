## Чому

У програмуванні часто потрібно об’єднувати (конкатенувати) рядки, щоб створити новий рядок зі зчеплених частин. Це може бути корисно для виводу повідомлень, створення динамічних URL або будь-якого іншого завдання, яке вимагає об’єднання декількох стрічок. У цій статті я покажу, як об’єднувати стрічки у Ruby і дам корисні поради для впровадження цього в своїх проектах.

## Як це зробити

Об’єднання стрічок у Ruby - це досить проста операція. Існує декілька шляхів зробити це, але ми розглянемо два основних. Спочатку ми можемо використовувати оператор "+" для простого з’єднання двох стрічок:

```Ruby
puts "Hello" + "world"
```

Виходом цієї програми буде:

```
Helloworld
```

Ми також можемо використовувати метод `concat`, який дозволяє додавати необмежену кількість стрічок разом:

```Ruby
puts "Hello".concat(" ", "world", " ", "!")
```

Виходом буде:

```
Hello world!
```

Якщо потрібно об’єднати більшому кількість стрічок, то корисно використовувати метод `join`. Він приймає масив стрічок і з’єднує їх разом з використанням вказаного роздільника. Наприклад:

```Ruby
arr = ["It", "is", "a", "beautiful", "day"]
puts arr.join(" ")
```

Виходом буде:

```
It is a beautiful day
```

## Глибокий занурення

Під капотом всі зв’язування стрічок в Ruby використовують метод `+`. Тому, якщо ви використовуєте оператор `+` для об’єднання стрічок у вашому коді, це еквівалентно виклику методу `+`. Це може бути корисно для розуміння того, як робиться зв’язування стрічок.

Також варто зазначити, що оператор `+` і метод `concat` відрізняються у тому, що метод `concat` сприймає лише один аргумент, тоді як оператор `+` може приймати більше. Тому, якщо ви використовуєте оператор `+` для з’єднання більшу кількість стрічок, то, можливо, має сенс перейти на метод `concat` для покращення читабельності коду.
