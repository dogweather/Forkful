## Чому

Написання текстового файлу є важливою складовою частиною програмування, оскільки це дозволяє зберігати дані і корисну інформацію для подальшого використання. Текстові файли є універсальним форматом, який може бути прочитаний як людиною, так і комп'ютером.

## Як виконати

Нижче наведений приклад коду на мові Ruby, який демонструє створення і запис даних в текстовий файл.

```Ruby
# відкриття файлу для запису даних з можливістю використання кирилиці
out_file = File.open("привіт.txt", "w:UTF-8")

# запис тексту до файлу
out_file.puts "Привіт, світ!"

# закриття файлу
out_file.close

# читання з файлу
in_file = File.open("привіт.txt", "r:UTF-8")

# виведення тексту з файлу
puts in_file.read

# закриття файлу
in_file.close

# використання циклу для запису декількох рядків до файлу
out_file = File.open("привіт.txt", "w:UTF-8")
3.times do
  out_file.puts "Привіт, світ!"
end

# закриття файлу
out_file.close

# виведення тексту з файлу
puts File.read("привіт.txt")

# видалення файлу після виконання програми
File.delete("привіт.txt")
```

Результат:

```Привіт, світ!
Привіт, світ!
Привіт, світ!
```

## Глибше погляньмо

Текстові файли можуть бути використані для зберігання будь-яких типів даних, таких як числа, рядки, булеві значення та інші. Крім того, можна зчитувати і записувати дані з файлу, використовуючи цикли та умовні оператори. Наприклад, для розбирання файлу по рядках можна використати цикл `each_line`. Також існує багато додаткових методів для роботи з текстовими файлами, які можна дослідити для покращення навиків програмування.
