# Чому

Інколи при програмуванні нам потрібно видалити певні символи зі строки, які відповідають певному шаблону. Це може здатися складним, але насправді це досить просто зробити за допомогою деяких Ruby методів. Далі у статті ми розглянемо, чому це важливо та як це зробити.

# Як

Для видалення символів за певним шаблоном, нам потрібно скористатися методом `gsub`, який стоїть за "глобальну заміну". В якості аргументів він приймає шаблон та замінюючий символ. Давайте розглянемо це на прикладі:

```Ruby
string = "Привіт, світ!"
string.gsub(/[і]/, "и")
```
Цей код поверне нам строку "Привїт, свїт!" де і будуть замінені на и. Ви можете також використовувати регулярні вирази та глобальну заміну, щоб видалити більше, ніж один символ. Наприклад:

```Ruby
string = "Привіт, світ!"
string.gsub(/[іт]/, "")
```

Цей код видалить всі символи і, та т зі строки, тому результат буде "Прв, св!"

# Глибокий аналіз

У той же час, ви можете застосовувати глобальну заміну до всієї строки, але це не завжди корисно. Наприклад, якщо ви хочете видалити всі коми з числа у форматі `1,000`, ви можете використовувати `gsub` з шаблоном `[,]`, але це також зможе видалити кому з рядка "мільйони, нуль". В таких випадках краще використовувати метод `delete`, який буде видаляти лише вказані символи, не загальний шаблон. Наприклад:

```Ruby
string = "Число у форматі 1,000"
string.delete(",")
```

Цей код поверне нам строку "Число у форматі 1000", де кома буде видалена.
