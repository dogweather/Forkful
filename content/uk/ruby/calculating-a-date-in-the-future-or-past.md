## Чому
Розрахунок дати в майбутньому або минулому може бути корисним для різноманітних задач, наприклад, для створення звіту або активації події на конкретну дату. Також це може бути корисно для виконання різноманітних планувань та організації подій.

## Як це зробити
Для розрахунку дати в майбутньому або минулому використовуйте вбудовану бібліотеку Date в Ruby. Далі показано приклади коду та виводу для різних задач.

### Розрахунок дати в майбутньому
Примірник: поточна дата + 7 днів

```Ruby
require 'date'
current_date = Date.today
future_date = current_date + 7
puts "Теперішня дата: #{current_date}"
puts "Дата в майбутньому: #{future_date}"
```
Вивід:
```
Теперішня дата: 2021-10-12
Дата в майбутньому: 2021-10-19
```

### Розрахунок дати в минулому
Примірник: поточна дата - 30 днів

```Ruby
require 'date'
current_date = Date.today
past_date = current_date - 30
puts "Теперішня дата: #{current_date}"
puts "Дата в минулому: #{past_date}"
```
Вивід:
```
Теперішня дата: 2021-10-12
Дата в минулому: 2021-09-12
```

## Глибокий занурення
Клас Date має деякі корисні методи для роботи з датами, такі як знаходження різниці між двома датами, перевірка на належність до вихідного дня тижня та багато іншого. Також існує багато гемів для розширення можливостей роботи з датами в Ruby.
