## Тому чому

Регулярні вирази є потужним інструментом для обробки текстових даних у програмуванні. Вони дозволяють швидко та ефективно шукати та змінювати певні шаблони у тексті, що полегшує виконання складних завдань.

## Як користуватися регулярними виразами в Ruby

Для того, щоб почати використовувати регулярні вирази в своєму коді, потрібно використовувати об'єкт класу `Regexp`. Почнемо зі створення простого регулярного виразу для пошуку слова "привіт" у рядку:

```Ruby
hello_regex = /привіт/
```

Можна також використовувати додаткові опції, наприклад `i` для ігнорування регістру букв. Для пошуку у всьому тексті, можна скористатися методом `scan`:

```Ruby
txt = "Привіт, це цікава стаття про програмування!"
matches = txt.scan(hello_regex)
puts matches
# => привіт
```

Також можна змінювати текст за допомогою регулярних виразів, використовуючи метод `gsub`:

```Ruby
txt = "Привіт, це цікава стаття про програмування!"
new_txt = txt.gsub(hello_regex, "бувай")
puts new_txt
# => бувай, це цікава стаття про програмування!
```

## Глибше про регулярні вирази

Для створення складних регулярних виразів можна використовувати спеціальні метасимволи та конструкції. Наприклад, `.` використовується для позначення будь-якого символу, `+` для позначення одного або більше входжень попереднього символу, а `*` для будь-якої кількості входжень.

Крім того, можна використовувати конструкції `[]` для вказання діапазону символів, `()` для групування та `|` для вказівки альтернативних варіантів.

Для більш докладної інформації про регулярні вирази в Ruby, рекомендуємо ознайомитися з [документацією по регулярним виразам](https://ruby-doc.org/core-3.0.1/Regexp.html) та [онлайн-туторіалами](https://www.tutorialspoint.com/ruby/ruby_regular_expressions.htm).
