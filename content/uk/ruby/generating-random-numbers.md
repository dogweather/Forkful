## Чому

У програмуванні існує безліч ситуацій, коли потрібно згенерувати випадковий число. Наприклад, для створення гри чи для тестування коду, необхідні випадкові дані. Для цього існують спеціальні функції у більшості мов програмування, в тому числі і в Ruby.

## Як

Щоб згенерувати випадкове число в Ruby, використовуйте вбудовану функцію `rand`, яка приймає аргументом діапазон від 0 до 1 (для довгої десяткової форми) або від 0 до 99 (для короткої десяткової форми). Наприклад:

```Ruby
rand # випадкове число від 0 до 1
rand(10) # випадкове число від 0 до 9
rand(1..100) # випадкове число у заданому діапазоні
```

## Глибокий погляд

Можна також використовувати функцію `srand` для того, щоб встановити постійну випадковість, що дозволяє генерувати однакові випадкові числа. щоразу, як програма запускається. Крім того, у Ruby також є клас `Random`, який має більш широкі можливості для генерації випадкових чисел, таких як створення декількох генераторів та налаштування генерування випадкових чисел за певної логіки.
