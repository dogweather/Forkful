## Що це таке та навіщо?
Перевірка на існування директорії дає знати, чи є папка, яку ми збираємось використовувати, на диску. Програмісти це роблять, щоб уникнути помилок при роботі з файлами або папками, яких немає.

## Як це зробити:
```Ruby
require 'fileutils'

# Перевірка чи директорія існує:
if Dir.exist?('/шлях/до/вашої/директорії')
  puts 'Директорія існує!'
else
  puts 'Директорії не знайдено.'
end
```
Вивід буде або `Директорія існує!`, або `Директорії не знайдено.` залежно від результату.

## Поглиблений розгляд:
Назад у часи, `FileTest` модуль був тим, що ми використовували для перевірки файлів і директорій. Тепер ми маємо `Dir.exist?` і `File.exist?`, що зберігаються у стандартній бібліотеці. Крім `Dir.exist?`, можна використовувати `File.directory?` для тих самих цілей.

Реалізація в Ruby є крос-платформною, але є деталі на що варто звернути увагу. Наприклад, у Windows шляхи записуються інакше через зворотній слеш `\`. Також, Ruby має кілька методів для створення директорій (`Dir.mkdir`, `FileUtils.mkdir_p`), якщо перевірка показує їх відсутність.
