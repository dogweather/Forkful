# Почему

Перевірка наявності директорії є важливою частиною програмування на Ruby. Це дозволяє гарантувати, що програма буде працювати з правильними шляхами та уникне непотрібних помилок.

# Як

Для перевірки наявності директорії у Ruby існує метод `Dir.exist?`. Нижче наведений приклад використання цього методу:

```Ruby
if Dir.exist?("folder")
    puts "Директорія існує"
else
    puts "Директорія не існує"
end
```

В результаті виконання цього коду, буде виведено повідомлення про існування або неіснування директорії "folder".

# Глибоке дослідження

Метод `Dir.exist?` перевіряє наявність шляху до директорії і повертає значення `true` або `false`. Якщо перевіряється лінійний шлях, метод буде додавати символ `/` до кінця рядка, щоб переконатися, що шлях вказаний правильно. Також, цей метод не враховує права доступу до файлів або директорій, тому якщо у вас немає прав на доступ до певної директорії, метод все одно поверне значення `true` і ви отримаєте помилку з правами доступу при виконанні подальших дій.
