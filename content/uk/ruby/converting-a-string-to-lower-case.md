## Чому
Заміна рядка на нижній регістр - це корисний навик для програмування на Ruby. Це допомагає забезпечити правильне порівняння рядків та уникнути помилок через різницю у верхньому та нижньому регістрах.

## Як
```Ruby
my_string = "ЦЕ Рядок З Верхнім Регістром"
puts my_string.downcase
# виводить "це рядок з верхнім регістром"
```
Цей приклад показує, як використовувати метод `.downcase` для заміни всіх символів рядка на нижній регістр. Також можна використовувати цей метод на змінні для заміни її значення.

```Ruby
my_string = "Це рядок з верхнім регістром"
my_string.downcase!
puts my_string
# виводить "це рядок з верхнім регістром"
```
Цей приклад показує, як використовувати метод `.downcase!` для заміни всіх символів рядка на нижній регістр, змінюючи його значення без необхідності використовувати нову змінну. Цей метод змінює рядок із самого початку.

## Глибокий розбір
Заміна рядка на нижній регістр є достатньо простою операцією. При використанні методів `.downcase` та `.downcase!` враховується не тільки алфавіт, але й різні мови та символи. Це означає, що ви можете використовувати ці методи для будь-яких рядків незалежно від їхнього походження.

Приклади використання включають в себе порівняння рядків для пошуку повторних значень, зберігання даних для пошуку, а також забезпечення правильності та уніфікації даних.
