## Чому

Багато програмістів використовують аргументи командного рядка в своїх проектах. Це дозволяє їм змінювати поведінку програми, не змінюючи її коду. Читання аргументів командного рядка є важливою частиною програмування на Ruby, тому що воно дозволяє проектувати більш гнучкі та корисні додатки.

## Як це зробити

Для початку, необхідно відкрити свій улюблений текстовий редактор і створити новий файл. Далі, в цьому файлі необхідно створити змінну для збереження значення аргументів командного рядка, як наприклад ```args```. Потім, використовуючи метод ```ARGV``` ми можемо отримати список аргументів та зберегти їх у змінну ```args```. Наприклад:

```Ruby
args = ARGV
```

Тепер ми можемо вивести аргументи на екран, використовуючи цикл, такий як ```each```. Наприклад, програма, яка виводить на екран усі аргументи, може виглядати наступним чином:

```Ruby
args.each do |arg|
  puts arg
end
```

Програма буде виводити на екран усі аргументи, які були передані при запуску скрипта.

## Глибоке занурення

Щоб краще зрозуміти, як працюють аргументи командного рядка, важливо знати процес запуску Ruby програми. Коли ми запускаємо файл з розширенням ```.rb```, система перш за все запускає програму Ruby, а потім передає цей файл як аргумент командного рядка. Таким чином, всі аргументи, які були введені під час запуску програми, будуть знаходитись у змінній ```args```.

Для того щоб використати аргументи командного рядка в більш продуктивних програмах, можна використовувати спеціальну бібліотеку ```optparse```. Вона дозволяє створювати опції для запуску програми, які можуть приймати значення. Наприклад, з допомогою цієї бібліотеки можна додати опцію для задання імені файлу, який програма буде обробляти.
