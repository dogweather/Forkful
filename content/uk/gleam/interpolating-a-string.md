## Що і чому?

Інтерполяція рядків - це процес, який дозволяє вставляти змінні або вирази безпосередньо в текстовий рядок. Програмісти використовують це для створення більш чистого і зрозумілого коду.

## Як це зробити:

Gleam дозволяй використовувати інтерполяцію рядків з використанням вбудованої функції `string.concat`. Нижче ви можете побачити як це працює.

```Gleam
import gleam/string

fn main() {
    let имя = "Taras"
    let привіт = string.concat(["Привіт, ", имя, "!"])
    io.println(привіт) // Друкує: 'Привіт, Taras!'
}
```

## Занурення глибше

Історічно, інтерполяція рядків була частиною мов програмування від самого початку їх зародження. Мови такі як Perl, Ruby та мови сімейства C використовують її в основному по всьому коду. Gleam, на відміну має менш встроєний підхід, але все одно є ефективним. Щодо альтернатив, можна використовувати конкатенацію рядків, але це зазвичай займає більше місця.

На рівні реалізації, інтерполяція рядків в Gleam використовує функцію `string.concat`, яка приймає список рядків та об'єднує їх у один рядок.
