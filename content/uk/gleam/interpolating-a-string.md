## Що та Чому?
Інтерполяція рядків - це включення змінних чи виразів у середину рядкового літералу. Програмісти роблять це для зручного форматування тексту та динамічного створення рядків.

## Як це зробити:
Gleam має простий спосіб для інтерполяції - використовуйте функцію `String.append` або оператор `++`. Ось приклад:

```gleam
fn main() {
  let name = "Мішко"
  let greeting = "Привіт, " ++ name ++ "!"
  greeting
}
```

Вивід буде: `Привіт, Мішко!`

## Поглиблене вивчення:
Інтерполяція рядків існує майже в усіх сучасних мовах програмування. В інших мовах, таких як Python чи JavaScript, вона може виглядати як пряме включення змінних всередину рядка за допомогою спеціального синтаксису. В Gleam, на даний момент, явної підтримки для синтаксису інтерполяції рядків немає, тому використовують методи, що своєю функцією дуже нагадують інтерполяцію, але вимагають трохи більше написання коду.

Щодо історії, інтерполяція була частиною багатьох мов від їхніх ранніх версій, оскільки забезпечує чистий та ефективний спосіб об'єднання рядків з даними. У разі потреби форматування складніших речей у Gleam, можна використовувати функції, які надають більше контролю, такі як `String.concat` або `String.insert`.
