# Чому

У глобальному світі програмування, об'єднання різних рядків тексту є необхідним процесом. Воно дозволяє встановлювати логічні зв'язки та створювати зрозумілі команди для комп'ютерів. У Gleam програмуванні, оператор додавання строк є основним інструментом для об'єднання рядків.

# Як це зробити

У Gleam, об'єднання рядків є простою задачею, яка використовує оператор додавання "+" для об'єднання двох або більше рядків. Наприклад, якщо ми хочемо об'єднати рядки “Привіт” та “Україно”, то ми можемо використовувати код:

```Gleam
Привіт + Україно
```

Це простий спосіб об'єднати рядки та створити новий рядок “ПривітУкраїно”. Також, можна використовувати змінні в об'єднанні рядків, наприклад:

```Gleam
let country = "Україно"
let hello = "Привіт"
hello + country
```

Цей код також виведе рядок “ПривітУкраїно”. 

# Вдосконалене об'єднання рядків

У Gleam є також більш потужний спосіб об'єднання рядків, використовуючи форматування рядків. Це дозволяє зберегти рядок шаблону та додавати значення змінних в цей шаблон. Наприклад:

```Gleam
let num = 5
let str = "Значення x дорівнює {{ num }}"
str
```

Цей код виведе рядок “Значення x дорівнює 5”, де {{ num }} буде замінено змінною num. Це можливо використати для складніших рядків та збереження шаблонів для подальшого використання.
