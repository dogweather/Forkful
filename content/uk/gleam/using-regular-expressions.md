## Чому

Використання регулярних виразів є важливою складовою частиною програмування. Вони дозволяють швидко та ефективно шукати та замінювати текст у великих обсягах даних, що зробить ваш код більш компактним та ефективним.

## Як користуватися регулярними виразами

Для початку, необхідно імпортувати модуль для регулярних виразів у свій проект:

```Gleam
import gleam/regex
```

Зараз ми готові створити наш перший регулярний вираз. Давайте припустимо, що ми хочемо знайти всі слова у рядку тексту, які починаються злітери "G" та "S". Ми можемо це зробити наступним чином:

```Gleam
let regex = regex/compile("G\S+")
let text = "Hello, Gopher and Snake!"
let result = regex/find_all(text)
```

У цьому прикладі, ми створили регулярний вираз, який шукає слова, що починаються з букв "G" та "S". За допомогою функції `find_all`, ми отримуємо список всіх слова, що задовольняють цей шаблон у рядку тексту. У нашому випадку, результат буде наступним:

```Gleam
["Gopher", "Snake"]
```

Існує багато інших функцій і можливостей для роботи з регулярними виразами в Gleam, які ви можете вчитися та вдосконалювати.
