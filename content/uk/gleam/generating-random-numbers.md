## Чому

Створення випадкових чисел є важливим елементом багатьох програм і може бути корисним для вирішення різноманітних задач. Наприклад, випадкові числа можуть використовуватися для генерації унікальних ідентифікаторів, симуляції випадкових подій або тестування алгоритмів.

## Як

Для генерації випадкових чисел у Gleam можна використовувати модуль `random`. Спочатку потрібно підключити цей модуль за допомогою ключового слова `import`:

```
import random
```

Для генерації випадкового числа з певним діапазоном, використовуйте функцію `random.int(min, max)`, де `min` - це мінімальне значення, а `max` - максимальне. Наприклад, щоб отримати випадкове число від 1 до 10, потрібно написати такий код:

```
let random_number = random.int(1, 10)
```

Також можна отримати випадкове дійсне число за допомогою функції `random.float()`, що повертає число від 0 до 1:

```
let random_float = random.float()
```

Нижче знаходиться приклад коду з вебсайту для генерації випадкового паролю з рядка символів. Обережно, це не стиль безпеки:

```
let characters = "abcdefghijklmnoprstuwxyz"
let limit = string.length(characters)

let password = loop(i -> i > 0) |> map(_ -> random.int(0, limit)) |> map(string.from_char_code)

let 8 = password
    |> list.to_tuple()
    |> option.to_result("The password array was invalid, please submit a bug report.")
    ?=>
        tuple.to_list()
        |> string.join
```

## Глибоке заанлядування

Для генерації випадкових чисел у Gleam використовується алгоритм на основі мікросекунди системного часу. Це означає, що якщо програма запускається за допомогою однієї й тієї ж системи, випадкові числа будуть генеруватися в одній і тій же послідовності. Це може призвести до проблем з безпекою у деяких випадках. Якщо вам потрібна криптографічно безпечна генерація випадкових чисел, то краще використовувати вбудовану бібліотеку `crypto` для генерації випадкових ключів.
