## Почему

Конвертация даты в строку может быть полезна для отображения даты в удобном формате для пользователя или для обработки даты в коде программы. Например, можно использовать строку с датой в качестве названия файла или для построения таблицы с данными за определенный период времени.

## Как

Прежде чем начать, необходимо импортировать модуль `Date` в Gleam:

```Gleam
import Date 
```

Далее можно использовать функцию `to_string` из этого модуля, передав ей дату в качестве аргумента. Например, мы хотим преобразовать дату 25 декабря 2020 года в стандартный формат даты:

```Gleam
let date = Date.from_fields(~year=2020, ~month=12, ~day=25)
let date_string = Date.to_string(date)
```

В результате получим строку `"2020-12-25"`. Кроме того, можно указать необходимый формат, передав его как второй аргумент функции `to_string`. Например, чтобы получить строку в формате "25.12.20", мы можем использовать следующий код:

```Gleam
let date = Date.from_fields(~year=2020, ~month=12, ~day=25)
let date_string = Date.to_string(date, "DD.MM.YY")
```

В результате получим строку `"25.12.20"`.

## Подробно

В функции `to_string` для форматирования даты можно использовать различные специальные символы, такие как `YYYY` для года, `MM` для месяца и `DD` для дня. Кроме того, можно использовать специальные символы для отображения времени, например, `hh` для часа и `mm` для минут. Полный список символов и их значения можно найти в официальной документации по модулю `Date`.
