## Що і чому?

Знаходження довжини рядка - це процес визначення кількості символів у конкретному текстовому рядку. Програмісти роблять це, щоб сконструювати необхідну логіку алгоритму або обробити текстові дані.

## Як це зробити:

Використовуючи мову програмування Gleam, ми можемо використовувати вбудовану функцію `size` для знаходження довжини рядка. Ось як це виглядає:

```Gleam
import gleam/string 

fn main() {
  let my_string = "Привіт, Світ!"
  let length = string.size(my_string)
  io.println(to_string(length)) // Результат: 12
}
```

При виконанні цього коду ви отримаєте число 12, що відповідає кількості символів у рядку "Привіт, Світ!".

## Занурення глибше

Мова програмування Gleam розвивається з Erlang, яка виникла в 80-ті роки 20-го століття. Вона призначена для створення надійних, розподілених систем, тому Gleam прагне утримати ті ж ідеали, додаючи до них статичну типізацію. Протягом свого розвитку Gleam розширив можливості функції `size`, роблячи її зручною для роботи як з рядками, так і зі списками.

Як альтернативу, у деяких мовах програмування, таких як JavaScript або Python, довжина рядка зазвичай знаходиться за допомогою властивості або методу `length`.

Але не варто забувати, що в глибокому зануренні у роботу з рядками важливо розуміти, що деякі символи, такі як ті, що використовуються для представлення емодзі або особливих символів, можуть відрізнятися від звичайних символів за кількістю байт, які вони займають. Це може змінити отримувану довжину рядка.
