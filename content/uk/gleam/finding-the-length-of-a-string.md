##Чому

Стрічки є важливою частиною багатьох програм, і вміння знаходити їх довжину може бути корисним для багатьох завдань програмування. Прикладом може бути перевірка коректності введених користувачем даних або обробка текстових даних для подальшого аналізу.

##Як

Для того, щоб знайти довжину стрічки в Gleam, нам потрібно використати функцію `length` з модуля `String`. Давайте розглянемо кілька прикладів коду для кращого розуміння.

```Gleam
import String

my_string = "Привіт, світ!"
length = String.length(my_string)
```
У цьому прикладі ми імпортуємо модуль `String`, де міститься функція `length`. Потім ми створюємо змінну `my_string`, якій присвоюємо вміст нашої стрічки. На останок, застосовуємо функцію `length` до нашої стрічки і зберігаємо результат у змінну `length`. Виводом буде число, яке представляє довжину стрічки.

```Gleam
import String

my_other_string = "I'm learning Gleam!"
print(String.length(my_other_string))
```

У цьому прикладі ми використовуємо функцію `print`, щоб одразу вивести результат на екран, а не зберігати його у змінну. Результат буде такий самий - число, яке представляє довжину стрічки.

##Глибока підготовка

Якщо ви хочете краще зрозуміти, як саме виконується знаходження довжини стрічки у Gleam, ви можете прочитати документацію модуля `String` або розглянути вихідний код самої функції `length`. Також можна навчитися робити цю операцію з власною реалізацією за допомогою інших функцій та методів Gleam.
