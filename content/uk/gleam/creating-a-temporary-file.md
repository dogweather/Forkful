## What & Why?
Що таке створення тимчасового файлу та чому це роблять програмісти? Тимчасові файли – це тимчасове сховище для даних під час роботи вашої програми. Вони корисні для зберігання даних, які не потребують постійного збереження, або коли вам потрібно запобігти конфліктам доступу у багатокористувацьких додатках.

## How to:
На жаль, остання версія мови програмування Gleam (на момент досягнення знань) не має вбудованої підтримки для роботи з файловою системою, включаючи створення тимчасових файлів, безпосередньо в стандартній бібліотеці. Робота з файлами в Gleam, зазвичай, потребує інтеграції з кодом на Erlang через FFI (Foreign Function Interface).

Проте, ми можемо використовувати Erlang-функції через FFI для створення тимчасових файлів. Ось приклад такої інтеграції:

```gleam
external fn mktemp() -> Result(String, Nil) =
  "erlang" "mktemp"

pub fn create_temp_file() {
  let result = mktemp()
  case result {
    Ok(filename) -> 
      io.println("Created a temporary file: " ++ filename)
    Error(_) -> 
      io.println("Failed to create a temporary file.")
  }
}
```

Примітка: `"erlang"` `"mktemp"` повинно вказувати на реалізацію Erlang-функції, що створює тимчасовий файл і повертає шлях до нього як рядок.

## Deep Dive
Створення тимчасових файлів датується днями ранніх операційних систем, де управління місцем на диску було критично. Сьогодні, хоча диски стали більшими, проблема виникає під час мульти-таскінгу і мульти-юзерного доступу. Щоб забезпечити безпеку і уникнути колізій, тимчасові файли мають унікальні імена. Зазвичай, такі файли створюються в системній тимчасовій папці.

У Gleam, доступ до файлової системи, включаючи створення тимчасових файлів, опосередкований через Erlang. Це означає, що потрібно виконувати додаткові кроки для роботи з файлами, такі як використання FFI та знання відповідних Erlang-функцій.

Щодо альтернатив, можна використовувати сторонні бібліотеки, якщо такі існують, або звернутися до сторонніх мов, таких як Elixir, які можуть мати більш зручні інструменти для роботи з filesystem.
