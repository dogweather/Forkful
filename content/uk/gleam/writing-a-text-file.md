## Чому

Робота з текстовими файлами є важливим елементом програмування, оскільки вони дають змогу зберігати інформацію на довгий термін та обробляти її. Також, вони є необхідним для зберігання програмного коду та документації проекту. Навіть якщо ви не є професійним програмістом, вміння працювати з текстовими файлами неодмінно знадобиться вам під час навчання програмування.

## Як це зробити

Перш за все, вам потрібно створити новий файл з розширенням ".txt" за допомогою будь-якого текстового редактора. Наприклад, у Gleam ви можете створити файл, назвавши його "example.txt" і використовуючи команду ```Gleam new example.txt```.

Далі, ви можете відкрити цей файл для редагування та додавати потрібну інформацію. Наприклад, вставивши змінну, значення якої ви хочете зберігати, як показано у наступному прикладі:

```Gleam
name = "Василь"
age = 25
hobby = "програмування"
```

В результаті, ваш файл має буде виглядати приблизно так:

```Gleam
Name: Василь
Age: 25 years old
Hobby: програмування
```

## Глибше дослідження

Існує багато різноманітних способів роботи з текстовими файлами. Ви можете зчитувати та записувати інформацію з файлів, редагувати їх, а також виконувати різні дії з даними, що зберігаються. Рекомендуємо детальніше ознайомитися з цією темою для покращення своїх програмістських навичок.
