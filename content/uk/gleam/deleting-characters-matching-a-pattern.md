## Чому
Програмування часто включає в себе вирішення проблем та управління даними. Видалення символів, що відповідають певному шаблону, може бути потрібно у випадку, якщо потрібно очистити дані від зайвих символів або виконати операції над певною часткою рядка. Одне з рішень цієї проблеми може бути використання мови програмування Gleam.

## Як виконати
Для початку, нам потрібно створити функцію, яка буде видаляти символи, відповідні певному шаблону. Наприклад, ми хочемо видалити всі цифри з рядка, тому наш шаблон буде [0-9]. У функції ми будемо перебирати всі символи рядка та порівнювати їх з шаблоном. Якщо символ відповідає шаблону, то він буде видалений з рядка. Ось приклад коду на Gleam:

```Gleam
fn delete_matching_chars(message: String, pattern: String) {
	let chars = Seq String.from(message)
	let filtered = Seq.filter chars (fn(c) -> c != pattern)
	String.from_array filtered
}

let message = "Відьма 098 домовилась з --- домовить до дому"
let filtered_message = delete_matching_chars(message, [0-9])

IO.print(filtered_message)
```

Вивід:
```
Відьма  домовилась з --- домовить до дому
```

У цьому прикладі ми створили функцію ```delete_matching_chars```, яка приймає два параметри - рядок для обробки та шаблон для відповідних символів. Ми використовуємо функцію ```Seq.filter``` для фільтрації символів, що відповідають шаблону, та повертаємо результат у вигляді рядка.

## Заглиблення
Видалення символів за шаблоном може бути потрібно в багатьох випадках, наприклад, для очищення даних перед їх подальшою обробкою або для зміни формату рядка. У мові Gleam також є інші корисні функції для роботи з рядками, які можуть спростити цей процес. Наприклад, функція ```String.split``` дозволяє розділити рядок за певним роздільником та повернути масив розділених частин. Також є функції для пошуку та заміни певних підрядків у рядку.
