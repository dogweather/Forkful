# Для чого

Получення поточної дати є однією з найбільш поширених задач у програмуванні. Це дозволяє отримати поточну дату та час для подальшого використання у програмі.

# Як це зробити

Для отримання поточної дати використовується вбудована функція `Date.now()`, яка повертає кількість мілісекунд, що пройшли з 1 січня 1970 року. Щоб перетворити це значення на читабельну дату, можна скористатися методом `Date.from_millis()`, який приймає кількість мілісекунд та повертає об'єкт типу `Date`.

```Gleam
let current_date = Date.now()
// 1601439080557

let readable_date = Date.from_millis(current_date)
// <Date>
```

Наприклад, для виведення поточної дати у форматі `дд-мм-рррр` можна скористатися методом `format()` та вказати потрібний формат у вигляді рядка `"dd-MM-yyyy"`.

```Gleam
let formatted_date = readable_date.format("dd-MM-yyyy")
// "30-09-2020"
```

# Глибокий занурення

Крім отримання поточної дати, Gleam також має вбудовану можливість роботи з датами і часом, таку як знаходження різниці між датами, перетворення часових зон, робота зі змінними, що містять дату, та багато іншого. Для більш детального ознайомлення з можливостями Gleam у роботі з датами, радимо переглянути документацію.
