## Що і чому?
Видалення лапок з рядка означає зняття цих зайвих шарів – лапок – з ваших текстових даних. Програмісти роблять це, щоб очистити вхідні дані, підготувати рядки для обробки, або просто щоб утримувати речі охайними і послідовними у своїх застосунках. Усе зводиться до чистих, придатних для використання даних в кінці.

## Як:
Видалення лапок у Gleam є простим процесом. Ми можемо використовувати зіставлення взірців або вбудовані функції рядків. Ось швидкий приклад, щоб проілюструвати:

```gleam
pub fn remove_quotes(text: String) -> String {
  let without_quotes = string.trim(text, "\"")
  without_quotes
}

pub fn main() {
  let text_with_quotes = "\"Hello, World!\""
  let cleaned_text = remove_quotes(text_with_quotes)
  io.println(cleaned_text)
}
```

Приклад виводу:
```
Hello, World!
```

## Поглиблений Огляд
Історично, робота з лапками в рядках була загальним завданням у текстовій обробці та скриптових мовах. Оскільки рядки часто є вхідними даними користувача або читаються з файлів, вони можуть містити лапки, які потрібно видалити з різних причин, таких як вставка в базу даних або форматування.

У Gleam ми використовуємо функцію `string.trim`, щоб відстригти лапки. Є альтернативи! Ми могли б перебирати рядок або застосовувати регулярні вираження, але `string.trim` є вашим зручним інструментом для цієї роботи завдяки його лаконічності та продуктивності.

Якщо ми заглибимося в деталі реалізації, `string.trim` працює, видаляючи символи з початку та кінця рядка, які збігаються з наданим шаблоном. Тож, якщо у вас є лапки на обох кінцях вашого рядка, вони обрізаються одразу. Зауважте, що вона видаляє лапки тільки якщо вони знаходяться на краях; лапки, що міцно розміщені у середині вашого тексту, залишаться на місці.
