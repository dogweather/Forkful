## Чому

Ви можете запитатися, чому краще обрати Gleam для своєї нової проекту. Одна з головних причин - це потужний і масштабований мова, яка дозволяє вам швидко та ефективно розробляти ваші проекти. Крім того, Gleam використовує BEAM виртуальну машину, яка дозволяє вам отримати високу продуктивність та надійність вашої програми.

## Як

У цьому розділі ми покажемо вам, як почати проект в Gleam. Ми розглянемо основні функції мови та продемонструємо їх використання у прикладах коду. Нижче ви можете побачити приклад Hello World програми:

```Gleam
fn main() {
    IO.println("Привіт, світе!");
}
```

Ви можете запустити цей код, використовуючи gleam run команду і побачити наступний результат:

`Привіт, світе!`

За допомогою Gleam ви також можете створювати модулі та функції, щоб організувати свій код. Ось приклад функції, яка повертає суму двох чисел:

```Gleam
fn sum(x, y) {
    x + y
}
```

## Поглиблене дослідження

Якщо ви більш зацікавлені в детальнішому дослідженні Gleam та розпочинанні нових проектів, ви можете звернутися до офіційної документації, яка містить детальні пояснення та приклади для різних аспектів мови. Також, ви можете приєднатися до спільноти Gleam на форумі, де ви можете задавати питання та обговорювати свої ідеї з іншими розробниками.
