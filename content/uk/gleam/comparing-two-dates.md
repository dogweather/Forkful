## Що і Чому?

Порівняння двох дат - це процес, який визначає, яка дата є ранішою, пізнішою або чи обидві дати є однаковими. Програмісти роблять це для сортування подій у хронологічному порядку, відслідковування часових проміжків або для визначення дати через певний період часу.

## Як?

Ось приклад порівняння двох дат в Gleam:

```gleam
fn main() {
  let date_1 = Date(2021, 12, 11)
  let date_2 = Date(2021, 12, 12)

  case date_1.compare(date_2) {
    Order.Equal   -> io.println("Дати однакові")
    Order.Greater -> io.println("Дата 1 більша за Дату 2")
    Order.Less    -> io.println("Дата 1 менша за Дату 2")
  }
}
```

##  Занурення в тему

Порівняння дат має довгу історію у програмуванні, оскільки час і дата є важливими складовими багатьох додатків.

Як альтернатива, ви можете використовувати функцію comparing, яка вимагає від вас направлення порівняння. Вам потребується функція, яка отримує два елементи і повертає булеве значення, замість використання прямих порівнянь.

В глибині, функція порівняння використовує числа для порівняння, перевіряючи рік, місяць і день почергово.
