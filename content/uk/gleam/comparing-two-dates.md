## Чому

Компарація двох дат може бути корисною для багатьох проектів, де потрібно порівняти час проміжку між двома подіями або зробити певні обчислення з датами. Також, це може бути корисно для створення функцій планування та обчислення термінів кінцевих строків.

## Як

Для порівняння двох дат в Gleam, ми можемо використовувати модуль `gleam/time` та функцію `compare` для отримання результату компарації. Приклад коду нижче показує як порівняти дві дати та вивести результат у змінну `result`.

```Gleam
import gleam/time

let date1 = 2021-01-01
let date2 = 2021-02-01

let result = time.compare(date1, date2)
```

Очікуване значення у змінній `result` буде `-1`, `0` або `1`, відповідно до того, чи перша дата менша, рівна або більша за другу дату.

Для отримання більш детальної інформації про обидві дати, ми можемо використовувати функції `to_parts` та `from_parts` модуля `gleam/time`. Ці функції дозволяють розбити дату на його складові та створити нову дату з цими складовими.

```Gleam
import gleam/time

let date1 = 2021-01-01
let date2 = 2021-02-01

let parts1 = time.to_parts(date1)
let parts2 = time.to_parts(date2)

let new_date = time.from_parts(2021, 2, 1)
```

## Глибоке дослідження

Компарація дат є більш складним процесом, ніж здається на перший погляд. Є багато нюансів та підводних каменів, з якими потрібно бути обережним при порівнянні дат.

Наприклад, дати можуть мати різні часові зони або форматування, що може призводити до неточності у порівнянні. Також, не завжди варто порівнювати дати лише за значеннями дня, місяця чи року, оскільки це може не враховувати час, який пройшов між подіями.

Щоб уникнути цих проблем, рекомендується використовувати бібліотеки та функції, які дозволяють враховувати всі аспекти дати та часу при порівнянні.
