## З чого почати

Якщо ви цікавитеся програмуванням та вивчаєте мову програмування Go, ви, напевно, знаєте про важливість роботи з рядками. Часто виникає потреба з перетворення їх в нижній регістр. Це може бути обов'язковим для забезпечення правильної роботи програми або для порівняння текстів. У цій статті ми розглянемо як перетворити рядок в нижній регістр у мові програмування Go.

## Як це зробити

Процес конвертації рядка в нижній регістр у мові Go дуже простий і може бути здійснений у кілька рядків коду. Перш за все, нам потрібно імпортувати пакет strings, який містить функцію ToLower(). Потім ми можемо застосувати цю функцію до нашого рядка, який потрібно перетворити. Нижче наведений код для цього:

```Go
import (
	"fmt"
	"strings"
)

func main() {
	// змінна з початковим рядком
	str := "Привіт, Світ!"
	// використання функції ToLower() для перетворення рядка в нижній регістр
	lowerStr := strings.ToLower(str)
	// вивід результату
	fmt.Println(lowerStr)
}
```

Після виконання цього коду ви отримаєте наступний результат:

```
привіт, світ!
```

Ви можете спробувати змінити початковий рядок та виконати код знову, щоб переконатись в його роботі для різних рядків.

## Глибоке занурення

Функція ToLower() з пакету strings працює наступним чином: вона перетворює всі символи в рядку на їх нижній регістр. Це означає, що будь-який символ, який був введений у верхньому регістрі, буде перетворений на його еквівалент у нижньому регістрі. Значення, які не є буквами, залишаться без змін. Наприклад, у рядку "Привіт, Світ!" букви "П" і "С" будуть перетворені на "п" і "с" відповідно, а такі символи як коми та пробіли залишаться такими ж, які були.

Ця функція також підтримує мови з багатобайтовими символами, такими як китайська чи японська. Це означає, що вона може правильно перетворити символи цих мов у нижній регістр без зміни їх значення.
