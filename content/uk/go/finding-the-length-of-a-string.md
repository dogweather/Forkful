## Чому

В програмуванні немає нічого зайвого. Знаходження довжини рядка - важлива операція, яка може допомогти в різних завданнях. Наприклад, ви можете використовувати цю функцію для перевірки введеного користувачем пароля або для обробки даних зі зовнішнього джерела.

## Як

```Go
package main

import "fmt"

func main() {
  // Оголошуємо змінну з рядком
  str := "Привіт, світе!"

  // Використовуємо функцію len() для знаходження довжини рядка
  length := len(str)

  // Виводимо результат
  fmt.Println("Довжина рядка:", length)
}

// Вивід: Довжина рядка: 13
```

## Глибше

Функція len() визначає довжину рядка, рахуючи кожен символ як одну одиницю. У Go також є можливість отримати довжину байтів в рядку за допомогою функції utf8.RuneCountInString(). Крім того, ви можете використовувати ці функції для роботи з багатобайтовими символами, такими як кириличні чи китайські.
