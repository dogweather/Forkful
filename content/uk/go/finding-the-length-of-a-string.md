## Що це та навіщо це потрібно?
Знаходження довжини рядка - це процес отримання кількості символів у даному рядку. Програмісти роблять це для обробки, перевірки та маніпуляції даними.

## Як це робити:
Ось базовий приклад коду, який показує, як знайти довжину рядка в Go:

```Go
package main

import "fmt"

func main() {
    str := "Просто рядок"
    fmt.Println(len(str))
}
```

Під час виконання цей код поверне:

```Go
12
```
Це звичайний спосіб використання вбудованої функції `len` для отримання довжини рядка в Go.

## Докладніше:
- Історичний контекст: Go було створено з метою спрощення програмування та забезпечення високої продуктивності. Його вбудована функція `len` є простим і швидким способом отримання довжини рядка.
- Альтернатики: Хоча `len` є основним методом для отримання довжини рядка в Go, можна використовувати інші текніки, наприклад, ітерація по рядку за допомогою циклу `for`.
- Деталі реалізації: Функція `len` обчислює кількість байтів у рядку, а не кількість 'рун'. Це має значення, якщо ви працюєте з рядками у мультибайтових словах, наприклад, в UTF-8.
