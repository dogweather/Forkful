## Що і чому?

Отримання поточної дати - це процес, коли ваша програма визначає, який зараз день, місяць та рік. Це корисно для відслідковування подій, встановлення таймстампів та планування оновлень.

## Як це зробити:

Отримати поточну дату в Go можна за допомогою пакету ```time```. 
```Go
package main

import (
    "fmt"
    "time"
)

func main() {
    t := time.Now()
    fmt.Println(t)
}
```
Після запуску цього коду ви отримаєте поточну дату і час для вашої системи.

## Глибоке занурення

Отримання поточної дати - це загальноприйнята проблема програмування. В інших мовах програмування, таких як Java або Python, також є вбудованими бібліотеками для цієї задачі.

У Go ви можете використовувати різні форматувальники часу для отримання подробиць дати, такі як ```t.Year()```, ```t.Month()```, ```t.Day()```.

Можна також використовувати пакет ```time``` для обробки часових зон. Пам'ятайте, що отримання поточного часу завжди пов'язане з системною конфігурацією.
