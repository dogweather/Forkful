## Чому

Проста заміна тексту є важливою частиною розробки програм на будь-якій мові програмування, включаючи Go. Незалежно від того, чи ви шукайте і замінюєте струки в своєму коді, чи в рядку даних, ефективний і точний пошук і заміна може зекономити багато часу та запобігти помилкам. У даній статті ми дамо огляд того, як виконувати пошук і заміну тексту в мові програмування Go.

## Як

В Go існують кілька способів здійснення пошуку і заміни тексту, але ми розглянемо найбільш простий і універсальний спосіб, використовуючи методи пакету `strings`. Ви можете переглянути різні методи для пошуку і заміни тексту в [документації Go](https://golang.org/pkg/strings/).

Для початку, нам потрібно імпортувати пакет `strings` до нашого коду:

```Go
import "strings"
```

Потім, для заміни всіх входжень певного рядка в іншому рядку, ми можемо використовувати метод `ReplaceAll`:

```Go
str := "Привіт, світ!"
newStr := strings.ReplaceAll(str, "Привіт", "Hello")
fmt.Println(newStr)
// Output: Hello, світ!
```

Ми також можемо використовувати метод `Replace` для заміни першого входження певного рядка в іншому рядку:

```Go
str := "Привіт, світ!"
newStr := strings.Replace(str, "світ", "world", 1)
fmt.Println(newStr)
// Output: Привіт, world!
```

Іншим корисним методом є `Index`, який повертає позицію першого входження даного рядка:

```Go
str := "Привіт, світ!"
index := strings.Index(str, "світ")
fmt.Println(index)
// Output: 6
```


## Поглиблене дослідження

Якщо вам потрібна більш точна та ефективна заміна тексту, ви можете скористатися регулярними виразами. Для цього вам знадобиться імпортувати пакет `regexp`.

```Go
import "regexp"
```

Наприклад, якщо ви хочете замінити всі числа у рядку на слово "number", ви можете використовувати такий код:

```Go
str := "15+20=35"
var re = regexp.MustCompile("[0-9]+")
result := re.ReplaceAllString(str, "number")
fmt.Println(result)
// Output: number+number=number
```
