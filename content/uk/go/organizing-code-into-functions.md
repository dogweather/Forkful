## Що і Чому?
Організація коду в функції полягає в зламанні вашого коду на предмети з можливістю повторного використання. Це робить ваш код чистішим, легшим для читання та простішим для налагодження.

## Як це робити:
Ось уривок коду на Go, що показує блок коду, потім перероблену версію за допомогою функцій:

```go
package main

import "fmt"

func main() {
    // До: Вбудований код
    fmt.Println("Розрахунок суми...")
    total := 0
    for i := 1; i <= 10; i++ {
        total += i
    }
    fmt.Println("Загальна сума:", total)

    // Після: Використання функції
    fmt.Println("Розрахунок суми за допомогою функції...")
    sum := getSum(1, 10)
    fmt.Println("Загальна сума:", sum)
}

// Функція для розрахунку суми у діапазоні
func getSum(start, end int) int {
    total := 0
    for i := start; i <= end; i++ {
        total += i
    }
    return total
}
```

Приклад виведення для обидвох варіантів коду, вбудованого та з функціями, буде однаковим:

```
Розрахунок суми...
Загальна сума: 55
Розрахунок суми за допомогою функції...
Загальна сума: 55
```

## Поглиблюючи
Перед тим як з’явились функції, програмування було переважно процедурним, з кодом, що виконувався згори донизу. Як росли програми, такий підхід породжував неефективність та повторення коду.

Мови програмування ввели функції як механізм абстракції. У Go, функції інкапсулюють блоки коду з конкретним завданням, заохочуючи принцип DRY (Don’t Repeat Yourself - Не Повторюй Себе). Вони приймають параметри та можуть повертати результати.

Корисні поради:
- Іменуйте функції ясно; хороше ім’я пояснює, що робить функція.
- Тримайте їх короткими; якщо функція робить надто багато, розбивайте її на частини.
- Функції можуть повертати багато значень, використовуйте це для контролю помилок.
- Функції вищого порядку (функції, що приймають або повертають інші функції) є потужними інструментами у Go.

Альтернативи функціям включають вбудований код (заплутаний для складних завдань) та методи об’єктів (частина об’єктно-орієнтованого парадигму, доступного у Go через структури).
