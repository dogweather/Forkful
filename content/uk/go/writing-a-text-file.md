## Чому

Написання текстового файлу є важливим аспектом розробки програмного забезпечення у Go. Це дає можливість зберігати дані, які можна використовувати пізніше або надіслати іншим користувачам.

## Як це зробити

Для того, щоб створити текстовий файл у Go, нам потрібно використати функцію `WriteFile` з пакету `io/ioutil`. Ось як ви можете зробити це:

```Go
package main

import (
	"io/ioutil"
	"log"
)

func main() {
	// Створюємо змінну з нашим текстом
	text := "Привіт, світе! Це приклад текстового файлу."

	// Використовуємо функцію WriteFile для створення файлу з нашим текстом
	err := ioutil.WriteFile("example.txt", []byte(text), 0644)
	if err != nil {
		log.Fatal(err)
	}
}
```

Після запуску цього коду ви побачите новий файл з назвою `example.txt` у вашій папці проекту. Його можна відкрити і прочитати.

## Глибинний аналіз

У цьому прикладі ми використовували функцію `WriteFile`, яка отримує три аргументи: ім'я файлу, дані, які потрібно записати, і права доступу до файлу. Функція `WriteFile` створить новий файл з вказаним ім'ям і запише в нього дані, які ми передали.

У нашому прикладі ми використали значення `0644` для прав доступу. Це означає, що файл може бути прочитаним іншими користувачами, але тільки власник файлу може змінювати його.

Цей приклад є дуже простим, але ви можете вдосконалити його, додавши умови і обробку помилок.
