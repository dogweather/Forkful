## Чому

У програмуванні часто виникає потреба порівняти дві дати. Це корисно для сортування, пошуку, фільтрації та інших операцій з датами. В даній статті ми розглянемо як порівняти дві дати в мові програмування Go.

## Як це зробити

Ми використаємо пакет `time` для роботи з датами в Go. Цей пакет містить функцію `Before()` та `After()`, які дозволяють порівнювати дві дати.

```Go
package main
 
import (
	"fmt"
	"time"
)
 
func main() {
	date1 := time.Date(2021, time.January, 1, 0, 0, 0, 0, time.UTC)
	date2 := time.Date(2020, time.December, 31, 23, 59, 59, 0, time.UTC)
	
	if date1.Before(date2) {
		fmt.Println("Перша дата перед другою")
	}
	
	if date2.After(date1) {
		fmt.Println("Друга дата після першої")
	}
	
	if date1.Equal(date2) {
		fmt.Println("Дати рівні")
	}
}
```

Запустивши цей код, ми отримаємо наступний результат:

```
Перша дата перед другою
Друга дата після першої
```

## Детальний аналіз

У попередньому прикладі, ми використали функції `Before()` та `After()` для порівняння дат. Зверніть увагу, що ці функції приймають аргументом інший об'єкт дати та повертають булеве значення в залежності від того, чи перша дата передує другій чи навпаки.

Також важливо зазначити, що об'єкти дат повинні бути створені з однаковою зоною часу, щоб правильно порівнюватися. У нашому прикладі, ми використовували зону `time.UTC`.
