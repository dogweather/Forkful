## Чому

У кожної мови програмування є свої власні унікальні функції та можливості. Однією з них є можливість управління строковими значеннями, включаючи таку операцію, як капіталізація рядка. Це корисна функція, яка дозволяє легко змінити формат тексту для відображення його у більш зрозумілому або привабливішому вигляді. У цій статті ми розглянемо, як саме виконати капіталізацію у мові програмування Go.

## Як

Кодування в Go вимагає використання певних методів та складних алгоритмів, які допомагають в роботі зі строковими значеннями. Одним з найпростіших методів капіталізації рядків в Go є використання функції `strings.ToUpper()`. Нижче наведений приклад коду, який демонструє використання цієї функції:

```Go
package main

import (
    "fmt"
    "strings"
)

func main() {
    // Рядок, який буде капіталізовано
    str := "привіт друзі!"

    // Використання функції ToUpper() для капіталізації рядка
    capitalizedStr := strings.ToUpper(str)

    // Виведення капіталізованого рядка на екран
    fmt.Println(capitalizedStr)
}

// Виведення: ПРИВІТ ДРУЗІ!
```

Як бачите, за допомогою всього лише одного рядка коду ми змогли капіталізувати рядок у нашій програмі. В результаті функція `strings.ToUpper()` перетворила наш рядок з нижнього регістру в верхній.

## Глибоке занурення

Якщо ви хочете більш детально ознайомитися з процесом капіталізації строкових значень в Go, то вам може бути цікаво дізнатися, що функція `strings.ToUpper()` використовує в собі метод `unicode/simpleFold`, який дозволяє застосовувати правила капіталізації до різних мов.

Крім того, у мові Go є ще багато інших корисних методів та функцій, які допомагають працювати з строковими значеннями. Ви можете ознайомитися з ними у [документації Go](https://golang.org/pkg/strings/) та використовувати їх у своїх програмах.
