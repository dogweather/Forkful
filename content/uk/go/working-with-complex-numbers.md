## Що та Чому?
Комплексні числа, які складаються з дійсної та уявної частин (наприклад, 5 + 7i), є важливими у таких сферах як інженерія, фізика та обробка сигналів. Програмісти працюють з ними для вирішення проблем у цих доменах, що було б складно зробити лише з дійсними числами.

## Як це зробити:
Go має вбудовану підтримку комплексних чисел. Ось короткий огляд:

```go
package main

import (
	"fmt"
	"math/cmplx"
)

func main() {
	// Створення комплексних чисел
	a := complex(2, 3)
	b := 4 + 5i

	// Основні операції
	fmt.Println("Додавання:", a+b)
	fmt.Println("Віднімання:", a-b)
	fmt.Println("Множення:", a*b)
	fmt.Println("Ділення:", a/b)

	// Властивості комплексного числа
	fmt.Println("Дійсна частина:", real(b))
	fmt.Println("Уявна частина:", imag(b))
	fmt.Println("Спряжене:", cmplx.Conj(b))
	fmt.Println("Модуль:", cmplx.Abs(b))
	fmt.Println("Кут фази (радіани):", cmplx.Phase(b))
}

```

Приклад виводу:

```
Додавання: (6+8i)
Віднімання: (-2-2i)
Множення: (-7+22i)
Ділення: (0.5609756097560976+0.0487804878048781i)
Дійсна частина: 4
Уявна частина: 5
Спряжене: (4-5i)
Модуль: 6.4031242374328485
Кут фази (радіани): 0.8960553845713439
```

## Поглиблений Розбір
Давно комплексні числа розглядали з підозрою—деякі вважали їх безполезними! З часом їхня спроможність описувати фізичні явища стала очевидною. Вони є фундаментальними в квантовій фізиці, теорії управління та електротехніці, серед інших сфер. 

У Go комплексні числа представлені за допомогою типу даних `complex128` (64 біти для дійсної та уявної частини кожної) або `complex64` (32 біти кожної). Під капотом це насправді просто два `float64` або `float32` з'єднані разом. Стандартна бібліотека Go, `math/cmplx`, пропонує функції для математичних операцій з комплексними числами. Це заощаджує вам клопіт з математикою і дозволяє зосередитись на вирішенні проблем.

Альтернативи вбудованій підтримці Go включають використання зовнішніх бібліотек або самостійну реалізацію обробки комплексних чисел. Але це рідко потрібно, оскільки вбудована підтримка Go ефективна і добре інтегрована в мову.
