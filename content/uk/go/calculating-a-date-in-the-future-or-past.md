## Що і чому?
Обчислення дати в майбутньому або минулому - це процес визначення точного дня, місяця та року, відстоючи від сьогоднішнього дня на певну кількість днів. Програмісти часто використовують це для різних ситуацій - наприклад, для визначення терміну дії ліцензії або для розрахунку терміну доставки товару.

## Як це робиться:
```Go
package main

import (
	"fmt"
	"time"
)

func main() {
	now := time.Now()
	fmt.Println("Сьогодні:", now)

	future := now.AddDate(0, 0, 7)
	fmt.Println("Через тиждень:", future)

	past := now.AddDate(0, 0, -7)
	fmt.Println("Тиждень назад:", past)
}
```
Виконання цього коду дасте вам дату через тиждень від сьогодні та дату тижнем раніше.

## Поглиблений вивчення:
В історичному контексті, програми використовували декілька методів врахування часу для обчислення дати в минулому або майбутньому. Можливі альтернативи включають використання Unix timestamp і використання спеціальних часових бібліотек. Але Go забезпечує вбудовані функції, такі як AddDate(), для обчислення дат. AddDate() додає вказану кількість років, місяців та днів до дати.
