## Чому

У програмуванні, об'єднання рядків є важливою задачею. Це дозволяє нам створювати більш складні рядки шляхом комбінування існуючих. В Go мові, є кілька способів об'єднання рядків, і ми подивимося на них у цій статті.

## Як це зробити

Існують декілька шляхів об'єднання рядків в Go мові, але одним з найпростіших і популярних методів є використання оператора "+". Давайте подивимося на приклад коду нижче, щоб побачити, як це працює:

```Go
str1 := "Привіт"
str2 := "світ!"
result := str1 + " " + str2

fmt.Println(result)
// Вивід: Привіт світ!
```

Ми створили дві змінні з рядками і об'єднали їх за допомогою оператора "+". Також було додано додатковий пропуск між рядками для належної калькуляції. Можна об'єднати більше двох рядків шляхом додавання більшої кількості операторів "+". Наприклад:

```Go
str1 := "Hello"
str2 := "world"
str3 := "!"

result := str1 + " " + str2 + str3

fmt.Println(result)
// Вивід: Hello world!
```

Інший спосіб об'єднання рядків - використання функції `fmt.Sprintf()`. Вона працює подібно до функції `Printf`, за винятком того, що вона повертає рядок замість виводу на екран. Подивимося на наступний приклад:

```Go
str1 := "Це"
str2 := "тест"
str3 := "!"

result := fmt.Sprintf("%s %s%s", str1, str2, str3)

fmt.Println(result)
// Вивід: Це тест!
```

## Глибший аналіз

Ми бачимо, що об'єднання рядків в Go можливе за допомогою оператора "+" або функції `fmt.Sprintf()`. Однак, варто зазначити, що використання оператора "+" для об'єднання багатьох рядків може призводити до витрати багато пам'яті. Це пов'язано з тим, що при кожному використанні оператора "+" створюється новий рядок у пам'яті. Тому, якщо ви працюєте з великою кількістю рядків, рекомендується використовувати функцію `fmt.Sprintf()`.
