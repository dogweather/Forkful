## Що й Навіщо?
Друкування відлагоджувального виводу - це процес відображення внутрішніх значень або стану програми для зручності відстеження. Програмісти роблять це для пошуку та усунення помилок у коді шляхом стеження за виконанням програми.

## Як це зробити:
Нижче представлений код у Go:

```Go 
package main
import "log"

func main() {
    name := "World"
    log.Printf("Hello, %s!", name)
}
```
У вищезазначеному випадку виводиться наступне:

```
2022/02/20 12:08:03 Hello, World!
```
Тут `Printf` використовується для форматування виводу і використовує значення змінних.

## Поглиблений Розділ:
1. **Історичний контекст**: Go був створений у 2007 році і сама можливість відлагоджувального виводу була інтегрована в нього з самого початку. Вивчити зміни до цієї функції можна через [архів Go](https://golang.org/doc/go1compat).

2. **Альтернативи**: Як альтернатива, ви можете використовувати пакет `fmt` для виводу інформації про відлагодження, але `log` має перевагу в тому, що автоматично додає час і дату до кожного повідомлення.
```Go
package main
import "fmt"

func main() {
    name := "World"
    fmt.Printf("Hello, %s!", name)
}
```
3. **Деталі реалізації**: Функція `Printf` з пакету `log` працює за принципом форматування рядка, а потім викликає `Print` з сформатованим рядком як аргументом.
