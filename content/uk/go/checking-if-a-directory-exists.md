## Що й Навіщо?

Перевірка чи існує директорія - це процес, коли ваша програма "питає" операційну систему, чи існує певна директорія. Програмісти роблять це, щоб уникнути помилок вводу/виводу при спробі відкрити місце, якого не існує.

## Як це робити:

Простий і зручний спосіб перевірити, чи існує директорія в Go, полягає у використанні стандартної бібліотеки `os`.

```Go
package main

import (
	"fmt"
	"os"
)

func main() {
	if _, err := os.Stat("/path/to/directory"); !os.IsNotExist(err) {
		fmt.Println("Директорія існує.")
	} else {
		fmt.Println("Директорія не існує.")
	}
}
```
Якщо директорія існує, програма поверне "`Директорія існує.`". В інакшому випадку, ви побачите "`Директорія не існує.`".

## Поглиблений матеріал

Дана функція з'явилася с самого початку мови Go і використовується для превентивних маніпуляцій з файлами або директоріями. Й інші мови програмування мають подібні механізми. 

Альтернативно, можна використати бібліотеку `ioutil`, але станом на Go 1.16, `ioutil` позначено як застаріле.

Щодо деталей реалізації, `os.Stat` використовує системний виклик `stat` або `lstat` (залежно від ОС) для отримання інформації про файл або директорію.
