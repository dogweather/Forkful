## Чому

Перевірка наявності директорії є важливою частиною програмування на Go. Це допомагає забезпечити, що ваш код буде працювати з потрібними файлами та директоріями з правильними правами доступу.

## Як це зробити

Для перевірки наявності директорії у Go, ми можемо використовувати функцію `os.Stat()`, яка повертає інформацію про файл або директорію за заданим шляхом. Потім, зібравши цю інформацію, ми можемо використовувати умовні оператори, щоб перевірити, чи існує директорія і вивести відповідне повідомлення.

```Go
package main

import (
    "fmt"
    "os"
)

func main() {
    dirPath := "/Users/username/Documents"
    if dirInfo, err := os.Stat(dirPath); os.IsNotExist(err) {
        fmt.Println("Директорія не існує")
    } else if dirInfo.IsDir() {
        fmt.Println("Директорія існує")
    } else {
        fmt.Println("Таке ім'я вже використовується для файлу")
    }
}
```

Вивід з цього прикладу буде `Директорія існує`, якщо шлях до директорії `dirPath` існує та відповідає директорії.

## Глибоке погруження

Крім функції `os.Stat()`, існує також вбудована функція `os.IsExist()`, яка перевіряє, чи існує файл або директорія за заданим шляхом. Ми також можемо використовувати `os.IsNotExist()` для перевірки, чи не існує директорії.

Є кілька варіантів реалізації перевірки наявності директорії в Go, наприклад використання `os.Stat()` у поєднанні з умовними операторами або повернення помилки `err` при використанні `os.IsExist()` або `os.IsNotExist()`. Ви можете вибрати той варіант, який найбільш підходить для вашого проекту.
