## What & Why? (Що і Чому?)
Програми часто потребують даних для роботи. Аргументи командного рядка – це спосіб передати ці дані програмі при її запуску. Програмісти використовують їх для забезпечення гнучкості та налаштування поведінки програм без зміни коду.

## How to: (Як це зробити:)
```Go
package main

import (
	"fmt"
	"os"
)

func main() {
	args := os.Args[1:] // Відкидаємо назву програми з списку аргументів

	for i, arg := range args {
		fmt.Printf("Аргумент %d: %s\n", i+1, arg)
	}
}
```

Якщо зберегти код в файл `args.go` і запустити `go run args.go аргумент1 аргумент2`, отримаємо:

```
Аргумент 1: аргумент1
Аргумент 2: аргумент2
```

## Deep Dive (Занурення у глибину):
Аргументи командного рядка – концепція стара як самі операційні системи. Go, як сучасна мова, використовує пакет `os` для роботи з ними. Альтернативою є бібліотеки, такі як `flag` для більш складного парсингу аргументів. В Go, `os.Args` надає прямий доступ до аргументів: `os.Args[0]` це шлях до виконуваного файлу, а `os.Args[1:]` – власне аргументи, подані користувачем.
