## Чому

Прочитання аргументів командного рядка є важливим елементом у програмуванні на мові Go. Воно дозволяє взаємодіяти з користувачем та передавати вхідні дані для обробки програмою. Це корисна навичка, яка допоможе вам створювати більш гнучкі та ефективні програми.

## Як

Для читання аргументів командного рядка у Go існує спеціальний пакет `flag`, який має вбудовані функції для отримання та обробки вхідних аргументів. Давайте подивимося на приклад коду:

```Go
package main

import (
	"flag"
	"fmt"
)

func main() {
	// Використовуємо функцію String для визначення нової флагової змінної "name"
	// Тут "name" - це ім'я аргументу командного рядка, "-n" - це коротка назва флагу,
	// а "default" - це значення за замовчуванням для цієї змінної
	// Параметр "Usage" дозволяє вказати, як користувач повинен використовувати це поле
	name := flag.String("name", "World", "Enter your name")
	// Після визначення флагів, ми повинні викликати функцію "Parse",
	// яка буде зчитувати та обробляти вхідні аргументи командного рядка
	// Також ми використовуємо функцію "flag.PrintDefaults" для виведення
	// інформації про флаги у вихідному потоці в разі, якщо користувач вказав неіснуючий флаг
	flag.Parse()
	// Виводимо привітання за допомогою вказаного імені
	fmt.Printf("Hello, %s!", *name)
}
```

Тепер збережіть цей код у файлі з розширенням `.go` (наприклад, `hello.go`) та скомпілюйте його за допомогою команди `go build hello.go`. Після цього ви зможете виконати програму, вказавши значення для флага `name`, наприклад: `./hello -n Joe`, та отримаєте відповідне привітання.

Існує також можливість використовувати інші типи флагових змінних, наприклад числа або булеві значення. Дослідіть документацію для пакету `flag`, щоб дізнатися більше про всі можливості.
