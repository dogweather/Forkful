## Чому

Програмування в Go є потужним і корисним навичком в сучасному світі програмування. І час від часу ми можемо потрібувати читати текстовий файл для отримання інформації або для обробки даних. У цій статті ми розглянемо, як це зробити в Go.

## Як це зробити

Для початку, потрібно відкрити файл, за допомогою функції `Open` з пакету `os`, який надає нам можливість зчитувати дані з файлу. Приклад коду нижче демонструє цей процес:

```Go
file, err := os.Open("filename.txt")
if err != nil {
    log.Fatal(err)
}
defer file.Close()
```

Ми використали функцію `Open` із пакету `os`, щоб відкрити файл з назвою "filename.txt". Зверніть увагу на перевірку помилки за допомогою умовного оператора `if`. Якщо у процесі відкриття файлу сталася помилка, то ми виведемо її за допомогою функції `log.Fatal` і припинимо виконання програми.

Після відкриття файлу, ми можемо використовувати функцію `Read` з пакету `bufio`, щоб зчитувати дані з файлу пошарово. Наприклад, таким чином ми можемо зчитати кожен рядок з файлу та вивести його у консоль:

```Go
scanner := bufio.NewScanner(file)
for scanner.Scan() {
    fmt.Println(scanner.Text())
}
```

Функція `Scan` зчитує наступний рядок з файлу та переходить до наступного позиціонування. Ми використовуємо цикл `for` для того, щоб продовжувати зчитувати дані до тих пір, поки не досягнемо кінця файла.

Наприклад, якщо наш файл містить такий текст:

```
Hello,
World!
```

то в результаті в консолі ми побачимо:

```
Hello,
World!
```

## Глибокий занурення

При роботі з файлами, можуть виникнути різні ситуації, які потребують додаткової обробки. Наприклад, ми можемо хотіти зчитати лише певну частину файлу, а не весь вміст. Для цього можна використовувати функцію `ReadSlice` з пакету `bufio`, яка дозволяє зчитати певну кількість байтів з файлу.

Також, може бути потрібно перетворити текст з файлу у числа або інші типи даних для подальшої обробки. Для цього можна використовувати функції `strconv.Atoi` та `strconv.ParseFloat` з пакету `strconv`.
