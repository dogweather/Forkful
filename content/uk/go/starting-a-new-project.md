## Що і чому?

Створення нового проекту - це створення структури програми з нуля. Ми це робимо, бо так набагато зручніше стежити за кодом і колаборувати з командами.
 
## Як робити:
```Go 
package main

import (
	"fmt"
)

func main() {
	fmt.Println("Привіт, Го!")
}
```
Після виконання цього коду на екрані виведеться `"Привіт, Го!"`.

## Поглиблений огляд

У історичному контексті, Go був створений в Google у 2007 році, а вже у 2012 році був офіційно представлений.

Серед альтернатив для Go можна виділити Java, C++ і Python. Вони всі використовуються для створення різноманітних програм, але Go вирізняється легкістю читання коду, простотою і швидкістю роботи.

Створення нового проекту в Go включає в себе підготовку робочого середовища, створення і налаштування файлу `main.go` для запуску вашої програми і роботу з пакетами.
