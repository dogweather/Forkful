## Що і навіщо?

Видобування підрядків - це процес вилучення частини рядка за допомогою вказаних індексів. Це важливо для обробки і аналізу даних у програмах.

## Як робити:

Нижче наведено приклади коду Go, як видобувати підрядки.

```Go
package main

import (
    "fmt"
)

func main() {
    name := "Привіт, Україна!"
    substr := name[8:15] // Видобуваємо підрядок між індексами 8-14
    fmt.Println(substr) // Виводимо підрядок
}
```

Коли ми запустимо цей код, отримаємо такий вивід:

```Go
Україна!
```

## Поглиблений огляд:

Видобування підрядків було визнано важливим чинником для роботи з рядками ще з перших днів комп'ютерного програмування. Воно безпосередньо впливає на обробку і аналіз даних та може суттєво впливати на продуктивність вашого програмного забезпечення.

Натомість, хоч слайси в Go і дозволяють легко здійснювати видобування підрядків, іноді можуть виникнути проблеми з ними, оскільки вони посилаються на ту ж базову значення рядка. Якщо ви робите видобування з великого рядка і потім відкидаєте його, цей великий рядок ніколи не буде видалений сміттєвим збирачем, бо до нього існує активне посилання.

Альтернативою видобування підрядків може бути використання функцій strings.Split або regexp, де ви можете шукати підрядки за шаблоном, що може бути більш гнучким підходом в залежності від вашої задачі.
