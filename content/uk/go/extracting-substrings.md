## Чому

Ви можете зацікавитися екстрагуванням підрядків для полегшення обробки текстових даних або для знаходження певної інформації всередині великого тексту. За допомогою екстрагування підрядків ви можете швидко та ефективно знаходити та використовувати необхідну інформацію.

## Як

Для екстрагування підрядків у Go використовується метод `string[i:j]`, де `i` - індекс початку підрядка, а `j` - індекс, на якому закінчується підрядок.

```Go
str := "Привіт, світ!"
substring := str[0:5]
fmt.Println(substring)
// Вивід: "Привіт"
```

Ви також можете використовувати метод `strings.Split` для розбивання тексту на підрядки за певним роздільником.

```Go
str := "один, два, три, чотири"
arr := strings.Split(str, ",")
fmt.Println(arr)
// Вивід: ["один", " два", " три", " чотири"]
```

## Поглиблене дослідження

У Go є багато різних методів для екстрагування підрядків, наприклад `strings.Index`, `strings.Replace`, `strings.Trim` та багато інших. Кожен з цих методів має свої особливості та використання, тому рекомендуємо ознайомитися з офіційною документацією Go для більш детальної інформації.
