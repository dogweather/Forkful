## Що і чому?

Конвертація дати в рядок - це процес перетворення об'єкта дати в строкову форму, яка зручна для відображення або роботи з нею. Програмісти роблять це, щоби дати могли легко читатися та оброблятися в багатьох сценаріях використання.

## Як це зробити:


```Go
package main

import (
	"fmt"
	"time"
)

func main() {
	t := time.Now()
	fmt.Println(t.Format("2006-01-02"))
}
```
Вивід:

```
2024-05-15
```
Головне тут це формат, який використовується у функції `Format`. "2006-01-02" є стандартним шаблоном в Go для формату року-місяця-дня.

## Поглиблений матеріал

Історично, форматування дати було запроваджено в мові Go ще з її ранніх версій, і було мотивовано потребою виконувати складніше форматування дати.

Альтернатива - використовувати пряме форматування рядків за допомогою `Sprintf`. Але `Format` зазвичай надає чистіший, і більш читабельний код.

Ваш строковий шаблон може включати ще й форматування годин, хвилин і секунд, а також мілісекунд. Наприклад: `2006-01-02 15:04:05.00` виконує повне форматування дати й часу з мілісекундами.
