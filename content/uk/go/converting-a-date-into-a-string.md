## Чому

Конвертування дати в рядок є важливим елементом у програмуванні, оскільки дати зберігаються у вигляді чисел, а не зрозумілих для людини форматів. При конвертуванні дати в рядок, ми можемо легко відображати її у зрозумілому для людини вигляді.

## Як це зробити

```Go
package main

import (
	"fmt"
	"time"
)

func main() {
	// Створення дати
	date := time.Date(2021, time.May, 5, 0, 0, 0, 0, time.UTC)

	// Конвертування дати у рядок за допомогою дефолтного формату
	dateString := date.String()

	// Виведення результату
	fmt.Println(dateString) // Результат: 2021-05-05 00:00:00 +0000 UTC
}
```

Для конвертування дати у рядок, ми використовуємо функцію `String()`, яка повертає рядок у вигляді `YYYY-MM-DD hh:mm:ss timezone`.

Також, ми можемо самостійно вказати формат для конвертування. Для цього, ми використовуємо функцію `Format()` та передаємо їй бажаний формат у вигляді рядка.

```Go
// Конвертування дати у рядок за заданим форматом
dateString := date.Format("02.01.2006")
// Виведення результату
fmt.Println(dateString) // Результат: 05.05.2021
```

## Глибокий погляд

У Go, дата зберігається у вигляді `time.Time`, яка містить ряд корисних методів для роботи з датою та часом. Функція `String()` використовує вбудований формат `time.RFC3339` для конвертування дати у рядок. Цей формат використовується у багатьох урлах та інших протоколах.

Крім цього, в Go також є можливість обробки дат та часу у різних часових зонах за допомогою функції `Parse()`, яка приймає рядок та формат та повертає `time.Time` з відповідною зону.
