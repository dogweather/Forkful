## Чому

Робота з датами та часом є необхідною частиною багатьох програм, особливо з врахуванням різних часових зон та форматів дат. Крім того, часто необхідно перетворити дату на строкове значення для виведення на екран або для збереження у файлі. Elm має вбудовані інструменти для легкого та точного перетворення дат у потрібні формати.

## Як це зробити

Давайте розглянемо приклад звичайного сценарію, де нам необхідно перетворити поточну дату на строкове значення та вивести її на екран:

```Elm
import Time exposing (...) -- потрібні функції для роботи з датами
import Date.Format as Format -- для форматування дати

-- отримуємо поточну дату та час
currentTime : Time.Posix
currentTime =
  Time.now

-- перетворюємо дату у строкове значення з форматом "DD/MM/YYYY"
currentDate: String
currentDate =
  Format.format "%d/%m/%Y" (Time.millisToPosix currentTime)

-- виводимо на екран поточну дату
main =
  Html.text ("Сьогоднішня дата: " ++ currentDate)
```

Вивід на екрані буде виглядати так:

Сьогоднішня дата: 26/07/2021

Ми використали функцію `Time.now`, щоб отримати поточну дату та час у форматі POSIX, який є спеціальним типом дати в Elm. Далі ми скористалися модулем `Date.Format` для форматування дати за допомогою функції `Format.format`, де першим параметром є певний шаблон формату, а другим - Poisx-дата, яку ми отримали раніше.

Модуль `Date.Format` має багато інших функцій для роботи з датами та часом, таких як `Format.dayOfWeek`, `Format.month`, тож ви можете використати їх, щоб отримати необхідну вам інформацію.

## Глибокий аналіз

Коли ми використовуємо функцію `Time.now`, ми отримуємо час у форматі POSIX, який є кількістю секунд, що пройшли з 1 січня 1970 року. Цей формат універсальний для всіх комп'ютерних систем та мов програмування, тому його використовують як стандарт.

У модулі `Date.Format` є функція `Format.fromString`, яка дозволяє перетворити строкове значення у форматі дати на тип `Format.Formatter`. Цей тип має вбудовані методи, наприклад, `Format.monthName`, що поверне назву місяця або `Format.weekdayName`, що поверне назву дня тижня.
