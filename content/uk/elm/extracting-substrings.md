## Чому

Виділення підрядків є важливою частиною багатьох програм, таких як обробка рядків тексту, пошук і заміна певних символів та багато іншого. В Ерлм є потужний спосіб для виконання цієї задачі за допомогою функції `String.slice`, яка дозволяє вибирати підрядки за певними індексами.

## Як використовувати

Для того, щоб виокристати функцію `String.slice` необхідно передати їй рядок, початковий та кінцевий індекси для виділення підрядка. Наприклад:

```elm
let myString = "Це приклад повідомлення."
String.slice 4 10 myString
-- повідом
```

Хоча це приклад з використанням текстового рядка, функція `String.slice` також працює зі списками та масивами символів.

```elm
let myList = ["Е", "р", "л", "м", " ", "є", " ", "м", "о", "в", "н", "и", "й", " ", "м", "о", "в", "а"]
String.slice 2 6 myList
-- рлм є
```

## Глибокий занурення

Функція `String.slice` також приймає негативні значення для початкового та кінцевого індексів, що дозволяє починати вибір підрядка з кінця рядка.

```elm
let myString = "Привіт, Елм!"
String.slice -4 -1 myString
-- Елм
```

Також існує другий параметр `String.slice` - крок, який визначає, який символ буде використовуватися для виділення підрядка. Наприклад, з кроком 2 будуть виділятися кожен другий символ.

```elm
let myString = "Це є перемикач"
String.slice 1 10 myString 2
-- єпри
```
