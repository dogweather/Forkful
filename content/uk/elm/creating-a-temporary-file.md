## Чому

Створення тимчасового файлу є важливою частиною багатьох програм, особливо у веб-додатках. Це дозволяє зберігати тимчасові дані для подальшого використання або пересилання до інших джерел. Для більшої ефективності та кращої організації коду, користувачі Elm можуть використовувати спеціальні функції для створення тимчасових файлів.

## Як

Створення тимчасового файлу в Elm є досить простою задачею. Спочатку ми маємо імпортувати модуль `File` з `elm/core` пакета і дати йому псевдонім для зручності:

```elm
import File as Temp
```

Потім, за допомогою функції `Temp.tempFile`, ми можемо створити тимчасовий файл з заданим префіксом та розширенням:

```elm
Temp.tempFile "myPrefix" "txt"
```

Це поверне новий тимчасовий файл, який можна використовувати для зберігання даних. Давайте подивимось на приклад коду та його виведення:

```elm
import File as Temp

main =
  let
    tempFile = Temp.tempFile "myPrefix" "txt"
  in
    case tempFile of
      Ok file ->
        "Тимчасовий файл було успішно створено!"
      Err error ->
        "Помилка створення тимчасового файлу :("
```

В даному прикладі ми створюємо тимчасовий файл з псевдонімом `tempFile`. Якщо це вдається, ми отримуємо позитивне повідомлення, а якщо ні - негативне.

## Глибока погруження

Також, можна зробити деякі додаткові маніпуляції з тимчасовим файлом після його створення. Наприклад, ми можемо використовувати `Temp.withTempFile`, щоб передати функцію, яка буде використовувати створений тимчасовий файл:

```elm
Temp.withTempFile
  (Temp.write "Hello, world!")
  |> Expect.equal "Hello, world!"
```

Тут ми використовуємо функцію `write` для запису даних у тимчасовий файл, а потім порівнюємо його з оригінальним текстом.
