# Чому

Програмування може здатися складним та викликати багато питань, але однією з його найприємніших сторін є можливість отримувати потрібну інформацію в реальному часі. Одна з таких корисних функцій - отримання поточної дати. Це є необхідним для багатьох програм, таких як календарі, податкові обчислення та інші.

# Як це зробити

В Elm існує спеціальний пакет DateTime, який дозволяє отримати поточну дату з легкістю. Достатньо просто імпортувати його та використати функцію now, яка повертає DateTime об'єкт з поточною датою та часом.

```Elm
import DateTime

currentDate = DateTime.now
```

Також можна використовувати функцію `toIsoString` для отримання дати у форматі ISO:

```Elm
import DateTime

currentDate = DateTime.now |> DateTime.toIsoString
```

Результат буде виглядати приблизно так: `"2018-11-11T13:14:15.000Z"`. Для більш детальної інформації про доступні функції та формати дати, рекомендуємо ознайомитися з [офіційною документацією](https://package.elm-lang.org/packages/elm/time/latest/DateTime#now).

# Детальний розгляд

Перед тим як використовувати пакет DateTime, важливо розуміти, що це "фіктивний" пакет, який працює за допомогою якогось іншого засобу доступу до дати та часу в подвійному обгортці. Таким чином, код, написаний в Elm, запуститься на браузері та Node.js без додаткових налаштувань.
