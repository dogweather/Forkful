## Що це таке та навіщо?
Капіталізація рядка - це перетворення першої букви тексту у велику літеру. Програмісти використовують капіталізацію для форматування виводу тексту, відповідно до граматичних правил або для підвищення читабельності інтерфейсів користувача.

## Як це зробити:
У Elm немає вбудованої функції для капіталізації рядків, тому нижче наведено приклад реалізації:

```Elm
import String

capitalize : String -> String
capitalize str =
    case String.uncons str of
        Nothing ->
            ""

        Just (firstLetter, restOfString) ->
            String.toUpper (String.fromChar firstLetter) ++ restOfString

main =
    String.words "хай живе ельм! найбагатші мови програмування!"
        |> List.map capitalize
        |> String.join " "
        |> Debug.toString
        |> text

-- Вивід: "Хай Живе Ельм! Найбагатші Мови Програмування!"
```

## Занурення в тему:
Капіталізація - давня практика, яка йде з глибини літературних традицій. У програмуванні, важливо розуміти, що різні мови можуть мати власні правила капіталізації. Наприклад, в деяких культурах капіталізується не тільки перша літера речення, але й імена власні та інші слова. В Elm, ми імітуємо цю поведінку з кастомними функціями, тому що стандартна бібліотека мови не надає такої функціональності "з коробки". Щодо альтернатив, можна використовувати зовнішні пакети або власні рішення для більш складної логіки капіталізації, яка включає місцеві мовні особливості.
