## Чому
Чому важливо вирівнювати рядок у програмуванні на Elm? Це може призвести до більш зрозумілої та організованої кодової бази, що полегшує його розвиток та утримання з часом.

## Як
```Elm
import String

String.capitalize "привіт"   -- Виведе "Привіт"
String.capitalize "elm"   -- Виведе "Elm"
```

У цьому фрагменті коду ми імпортуємо модуль String, який надає цінні функції для роботи з рядками. Метод capitalize приймає рядок та повертає копію цього рядка, в якому перша літера велика. Це особливо корисно при форматуванні введення користувача, де слова можуть бути написані з малих букв.

## Глибший розбір
Інші корисні функції для роботи з рядками у модулі String включають toUpper (переведення всіх букв до верхнього регістру), toLower (переведення всіх букв до нижнього регістру) та length (повертає довжину рядка). Ці функції можуть бути корисні при валідації введення користувача або при обробці даних.
