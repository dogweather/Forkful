## Чому

Конвертування рядка до нижнього регістру - це важливий процес в програмуванні, який дозволяє полегшити роботу з текстом. Незалежно від того, чи ви працюєте над веб-додатком, або створюєте програму для мобільних пристроїв, конвертування рядка до нижнього регістру допоможе зробити вашу роботу ефективнішою.

## Як це зробити

Для конвертування рядка до нижнього регістру ви можете скористатися функцією `toLower` з пакету `String`. Ось приклад коду у мові Elm:

```Elm
import String exposing (toLower) -- імпортуємо функцію toLower з пакету String

string = "ELM ПРОГРАМУВАННЯ" -- задаємо початковий рядок

lowerString = toLower string -- конвертуємо рядок до нижнього регістру

-- друкуємо результат
-- буде виведено "elm програмування"
```

## Детальний аналіз

Функція `toLower` приймає рядок і повертає його ж, але вже у нижньому регістрі. Але як це працює насправді? Давайте розберемось.

У мові Elm всі рядки представлені як список символів. Щоб перетворити рядок до нижнього регістру, функція `toLower` проходить по цьому списку і для кожного символу використовує функцію `Char.toLower` для його конвертації до нижнього регістру. Після чого зібрані символи знову об'єднуються у рядок і повертаються як результат.

Також важливо зазначити, що функція `toLower` враховує існуючі правила для конвертації літер різних мов. Наприклад, для українських літер, які мають дві форми - велику та дуже - функція `toLower` використовує нестандартний алгоритм, щоб правильно конвертувати їх у нижній регістр.
