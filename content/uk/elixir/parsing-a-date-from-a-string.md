## Що & Чому?
Парсинг дати зі строки – це процес перетворення текстового представлення дати у структуру дати, яку програма може зрозуміти та використати. Програмісти роблять це, щоб можна було обробляти та порівнювати дати, завантажувати часові параметри з файлів конфігурацій чи користувацьких даних.

## Як це зробити:
У Elixir для парсингу строк з датою використовують модуль `DateTime`. Ось базовий приклад:

```elixir
# Підключімо модуль
import DateTime

# Парсимо дату
{ok, date_time} = DateTime.from_iso8601("2023-03-18T15:30:45Z")
# Результат: {:ok, ~U[2023-03-18 15:30:45Z]}

# Приклад помилки
{error, reason} = DateTime.from_iso8601("не_дата")
# Результат: {:error, :invalid_format}
```

## Поглиблений Огляд
Парсинг дати захоплює свої корені від традиційних мов програмування, де дату зберігали як рядки. Це зручно для показу людям, але не для машинного оброблення. В Elixir ці завдання полегшують модулі, як-от `DateTime`. Вони дозволяють перетворювати строки у дати та навпаки. Альтернативою може бути використання бібліотек, нативних для Ерланга, наприклад `:calendar`. Однак, `DateTime` в Elixir - це дружніший варіант зі зручним інтерфейсом і підтримкою ISO 8601 формату. Робота з часовими зонами та літнім часом також є значними перевагами `DateTime`.
