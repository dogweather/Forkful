## Чому

Генерація випадкових чисел є важливою частиною програмування, особливо якщо ви працюєте з великими даними або обробкою вхідних даних користувачів. Випадкові числа використовуються для створення унікальних ідентифікаторів, шифрування даних та випробування програм.

## Як

```Elixir
# Генерація випадкового цілого числа
random_int = :rand.uniform(100)
IO.puts "Випадкове ціле число: #{random_int}"

# Генерація випадкового числа з десятковою точністю
random_float = :rand.uniform()
IO.puts "Випадкове число з десятковою точністю: #{random_float}"

# Генерація випадкового числа з певним діапазоном
random_range = :rand.uniform(10..20)
IO.puts "Випадкове число в діапазоні від 10 до 20: #{random_range}"
```

Ви можете також використовувати ```Elixir :rand.seed/1``` для встановлення початкового числа для генерації випадкових чисел, щоб маскувати їх поведінку.

## Поглиблене дослідження

Elixir використовує алгоритм Лібченка Мулона для генерації випадкових чисел. Цей алгоритм заснований на генеруванні послідовності чисел, яка повертається при кожному виклику функції ```Elixir :rand.uniform/1```. Це означає, що при виклику функції із тим же самим початковим числом буде повернено ту саму послідовність випадкових чисел.

У Elixir також є більш потужна бібліотека для генерації випадкових чисел, яка називається Random. Вона має більше функцій для роботи з діапазонами, нормальним розподілом та іншими видами генерації випадкових чисел.
