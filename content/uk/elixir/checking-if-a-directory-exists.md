## Чому

Перевірка існування директорії є важливою задачею для багатьох програмістів. Це дозволяє переконатися, що необхідна директорія існує перед виконанням подальших дій з файлами або ресурсами всередині неї. Існування цієї перевірки також може забезпечити код безпекою, недопущуючи збоїв у програмі через неправильні шляхи до файлів або даних.

## Як

Перевірка існування директорії може бути виконана за допомогою функції `File.dir?/1` в бібліотеці Elixir. Ця функція отримує шлях до директорії в якості свого аргументу і повертає булеве значення `true` або `false` в залежності від наявності цієї директорії.

```Elixir
# Перш ніж використовувати функцію, потрібно імпортувати бібліотеку File
import File

# Перевірка існування директорії "/home/user/Documents"
File.dir?("/home/user/Documents")
# Результат: true

# Перевірка існування неправильного шляху до директорії
File.dir?("/home/user/Downloads")
# Результат: false
```

## Глибокий занурення

Під час роботи з функцією `File.dir?/1` важливо пам'ятати, що вона перевіряє лише існування директорії, а не її доступність для читання або запису. Якщо ці дії необхідні, то потрібно використовувати інші функції, такі як `File.readable?/1` або `File.writable?/1`.

Крім того, якщо шлях до директорії є змінною, то перед тим як використовувати його у функції `File.dir?/1`, потрібно перевірити його валідність та отримати повний шлях до директорії.
