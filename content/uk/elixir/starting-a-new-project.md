## Як створити новий проект з використанням Elixir

Для створення нового проекту ми використаємо команду `mix new`, яка автоматично створить папку з нашим проектом та всіми необхідними файлами. Ось приклад коду, який ви можете виконати в терміналі для створення своєї першої програми на Elixir:

```Elixir
mix new my_project
```

Цей код створить папку під назвою "my_project" та додасть в неї файли з фоновим кодом. Тепер відкрийте свою улюблену IDE та почніть кодувати!

Наприклад, створимо просту функцію, яка виводить повідомлення на екран:

```Elixir
defmodule MyProject do
  def say_hello do
    IO.puts "Привіт, світ!"
  end
end
```

Для запуску програми потрібно виконати наступну команду в терміналі:

```Elixir
mix run -e MyProject.say_hello
```

В результаті ми побачимо на екрані повідомлення "Привіт, світ!". Вітаємо! Ви створили перший проект на Elixir!

## Глибоке занурення

Тепер давайте трохи глибше поглянемо на створення нового проекту з використанням Elixir. Однією з ключових особливостей цієї мови програмування є вбудована система управління залежностями - Hex. За допомогою Hex ми можемо швидко та легко імпортувати зовнішні бібліотеки в наш проект. Наприклад:

```Elixir
defp deps do
    [
      {:httpoison, "~> 1.8"}
    ]
  end
```

Цей код вказує на те, що ми хочемо використовувати бібліотеку "httpoison" з версією 1.8. Після цього потрібно виконати команду `mix deps.get` для завантажен
