##  Чому

Використання аргументів командного рядка - важлива частина програмування в Elixir. Це дозволяє передавати дані програмі через командний рядок, що робить їх більш гнучкими та зручними для користувача.

## Як це зробити

Для того, щоб читати аргументи командного рядка в Elixir, ми можемо використовувати модуль `OptionParser`. Нижче показано приклад коду, який демонструє, як це можна зробити в практиці:

```Elixir

# Визначаємо опції та їх значення
opts = OptionParser.parse(
  # Перший аргумент - список опцій, що мають бути прочитані
  [
    { :foo, [ :integer, default: 0 ] },
    { :bar, [ :integer, default: 0 ] }
  ],
  # Другий аргумент - список аргументів командного рядка
  System.argv
)

# Перевірка на відсутність помилок
case opts do
  {:ok, opts} ->
    # Виконуємо дії, використовуючі значення опцій
    IO.puts "Значення опції foo: #{opts[:foo]}"
    IO.puts "Значення опції bar: #{opts[:bar]}"
  {:error, _reason} ->
    # У разі помилки, виводимо повідомлення та приклад використання
    IO.puts "Неправильне використання аргументів командного рядка!"
    IO.puts "Приклад використання: ./elixir_prog --foo 42 --bar 55"
end
```

Приклад відповіді на командний рядок: `./elixir_prog --foo 42 --bar 55`

```
Значення опції foo: 42
Значення опції bar: 55
```

## Глибоке вивчення

Використання аргументів командного рядка може бути корисним не тільки для передачі значень до програми, але й для запуску програми з різними параметрами. Наприклад, можна використовувати прапорці (флаги) для виконання різних дій або налаштування програми в певних умовах.

Детальніше про роботу з аргументами командного рядка у Elixir можна дізнатися у [документації](https://hexdocs.pm/elixir/OptionParser.html) Elixir.
