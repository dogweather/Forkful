## Чому

Іноді при написанні програм у Elixir потрібно виділити окрему частину або підстроку зі строки. Це може бути заміна підрядка, видалення зайвих символів або знаходження певної послідовності у великій строці. В цьому випадку використання функції витягнення підстроки може бути дуже корисним.

## Як

Для витягнення підстроки зі строки в Еліксірі існує кілька способів.

```Elixir
text = "Ласкаво просимо до статті про програмування Elixir"
```

### Метод slice

Один з способів - використання методу `slice`. Цей метод приймає початковий і кінцевий індекси і повертає підстроку між ними.

```Elixir
text.slice(6, 13) # повертає "просимо до"
```

### Метод pattern matching

Інший спосіб - використання взаємодії з функцією із зразком. В цьому випадку ми передаємо у функцію стрічку із шаблоном, що містить підстроку, яку ми хочемо витягнути, і функція повертає лише цю підстроку.

```Elixir
"програмування" <> _ <> " Elixir" = text # знаходить шаблон "програмування [будь-що] Elixir"
```

### Метод split

Також можна витягнути підстроку, розділивши стрічку на список і потім вибравши елементи, які потрібні.

```Elixir
text |> String.split(" ") |> Enum.slice(3, 5) # повертає список слів від "до" до "статті" - ["до", "статті", "про"]
```

## Deep Dive

Фактично, в Еліксірі не існує окремої функції для витягнення підстроки. Замість цього, можна скористатися різноманітними комбінаціями вбудованих функцій для досягнення бажаного результату. Наприклад, `slice` базується на функції `substr`, а функція для взаємодії із зразком використовує `Regex.run`.
