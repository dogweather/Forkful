# Чому

Написання текстового файлу є важливим етапом при програмуванні, оскільки це дозволяє зберегти та організувати дані для подальшого використання. Також це є необхідною частиною створення програм та скриптів.

## Як

Написання текстового файлу з використанням мови програмування Elixir дуже просте. Нижче наведені приклади коду та вихідний результат для створення текстового файлу зі списком елементів.

```Elixir
# Створення файлу
{:ok, file} = File.open("list.txt", [:write])

# Наповнення файлу списком елементів
list = ["продукти", "одяг", "книги"]
Enum.each(list, fn item ->
    IO.write(file, "#{item} \n")
end)

# Закриття файлу
File.close(file)

# Вміст файлу "list.txt"
продукти
одяг
книги
```

## Глибше залицяючись

Крім створення текстового файлу зі списком елементів, мова програмування Elixir також має багато вбудованих функцій для роботи з файлами. Наприклад, можна читати та записувати дані у файлі, шукати певну інформацію у файлі та багато іншого. Розуміння цих функцій допоможе покращити продуктивність та ефективність при написанні програм.
