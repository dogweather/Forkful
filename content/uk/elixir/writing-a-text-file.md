## What & Why?
(Що та Чому?)
Створення текстових файлів полягає у записі даних у файл на диску відповідно до певного формату. Програмісти роблять це для збереження даних, конфігурації, логування подій чи обміну інформацією між програмами.

## How to:
(Як це зробити:)
```elixir
# Створення текстового файлу з вмістом
File.write!("hello.txt", "Привіт, світ!")

# Дописування до файла
File.open!("hello.txt", [:append]) do |file|
  IO.write(file, "\nДоповнення до тексту.")
end

# Читання з файлу для перевірки результату
File.read!("hello.txt")
```
Очікуваний вивід:
```
Привіт, світ!
Доповнення до тексту.
```

## Deep Dive:
(Глибоке занурення:)
Elixir використовує віртуалізовану ерланг-машину (BEAM) для роботи з файлами, набуваючи переваги від її портової системи та розподілених можливостей. У минулому, для запису файлів часто використовували низькорівневі мови як C, але сучасні інтерпретовані мови як Elixir пропонують зручні абстракції з достатньою швидкістю для більшості задач. В Elixir для роботи з файлами є багато модулів, таких як `File` та `IO`, і це робить процес запису в файл спрощеним та ефективним.
