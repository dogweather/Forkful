**## Що та чому?**
Читання текстового файлу - це процес отримання даних, записаних у текстовому форматі, якими машина може керувати. Програмісти виконують цю дію для обробки великих обсягів даних, або для виконання взаємодії з користувачем через прості текстові інтерфейси.

**## Як це зробити:**
Для початку, давайте відкриємо файл за допомогою методу `File.read/1`:

```Elixir
{:ok, content} = File.read("your_file.txt")
IO.puts(content)
```

Вищезазначений код відкриває файл з назвою "your_file.txt" і виводить його вміст у консоль.

**## Поглиблений погляд**
Iсторично, починаючи з перших мов програмування, читання і запис використовувались для обробки даних. Elixir, будучи мовою, орієнтованою на обробку даних, також вершиною цю традицію. Щодо альтернатив, можна використовувати бази даних для зберігання даних, проте для простіших завдань це може бути надмірним. Окрім File.read, Elixir також має метод File.stream, який дозволяє читати файли частинами, особливо корисно, коли працюєте з великими файлами.
