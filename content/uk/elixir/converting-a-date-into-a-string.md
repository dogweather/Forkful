# Чому

Перетворення дати у рядок є важливою навичкою для кожного програміста Elixir. Це дозволяє зручно працювати з датами у вашому коді і виводити їх у зрозумілому форматі для користувача. Якщо ви займаєтесь розробкою веб-застосунків, ця навичка стане незамінною для створення зрозумілих та зручних інтерфейсів.

# Як це зробити

У Elixir існує багато способів конвертування дати у рядок, але найпростіший з них - використання функції `DateTime.to_string/3`. Давайте подивимось на приклади:

```Elixir
DateTime.to_string({{2019, 5, 28}, {13, 30, 0}}, "{YYYY}-{MM}-{DD}")
# "2019-05-28"

DateTime.to_string({{2019, 5, 28}, {13, 30, 0}}, "{YYYY}-{Mo}-{DD}")
# "2019-May-28"

DateTime.to_string({{2019, 5, 28}, {13, 30, 0}}, "{YYYY}-{Do}-{h12}:{m}-{a}")
# "2019-28th-1:30-pm"
```

Зверніть увагу, що при переконвертуванні дати, вам потрібно вказати бажаний формат, використовуючи спеціальні плейсхолдери. Також ви можете вказати часовий пояс, використовуючи другий аргумент функції.

# Глибоке погруження

Якщо вам потрібно більше гнучкості або специфічного форматування дати, ви можете використовувати функції `DateTime.to_iso8601/1` або `DateTime.to_string!/2`. Також існує багато інших функцій, які дозволяють працювати з датами у різних форматах та виконувати різноманітні операції.
