## Що й чому?

Строки в програмуванні - це послідовність символів. "Конкатенація" означає сполучення двох або більше рядків в один. Програмісти користуються цим для об'єднання даних, форматування виведення та менш комплікованого кодування.

## Як це робиться:

Простий приклад конкатенації рядків у Elixir.

```Elixir
str1 = "Привіт, "
str2 = "світе!"
IO.puts str1 <> str2
```

Виведе: "Привіт, світе!"

Також ви можете використовувати інтерполяцію:

```Elixir
name = "Володимир"
IO.puts "Привіт, #{name}!"
```

Виведе: "Привіт, Володимир!"

## Пірнаємо глибше:

Конкатенація рядків - це одна з основних концепцій в будь-якій мові програмування. Вона використовується вже декілька десятиліть і постійно вдосконалюється для кращої продуктивності та зручності використання. В Elixir ви можете використовувати операцію `<>` або інтерполяцію для контакулювання рядків. 

Єдиний момент, який треба пам'ятати: Elixir працює з рядками як з "бінарними" або послідовностями байтів. Це дозволяє Elixir ефективно об'єднувати рядки, але варто бути обережними з не-ASCII символами.
