# Чому

Видобування підрядків є зручним і корисним підходом у багатьох завданнях програмування на Clojure. Наприклад, ви можете використовувати це для знаходження підрядків у великому текстовому рядку або для розбиття тексту на менші частини для подальшої обробки.

# Як це зробити

Щоб видобути підрядки у Clojure, ви можете використовувати функцію `subs`. Наприклад, якщо у вас є рядок "Привіт, світ!", ви можете використовувати цю функцію, щоб видобути підрядок "Привіт":

```Clojure
(subs "Привіт, світ!" 0 6)
```

Результатом буде рядок "Привіт".

Щоб визначити довжину рядка, ви можете використовувати функцію `count`. Наприклад:

```Clojure
(count "Привіт, світ!")
```

Результатом буде число 13.

Можна також використовувати вирази з функцією `subs`, щоб знаходити підрядки на основі певних умов. Наприклад, якщо у вас є рядок з ім'ям та прізвищем, ви можете використовувати цей код, щоб видобути лише ім'я:

```Clojure
(let [ім'я-прізвище "Іван Петров"]
  (subs ім'я-прізвище 0 (count ім'я-прізвище -1)))
```

Результатом буде лише ім'я "Іван".

# Глибше занурення

Функція `subs` приймає три аргументи: рядок, початковий індекс та кінцевий індекс. Індекси починаються з нуля, а кінцевий індекс не включає сам підрядок.

Крім того, ви можете використовувати додаткові функції, такі як `str`, `str/join` та `str/split`, для подальшої обробки видобутих підрядків. Також існує багато інших функцій, які дозволяють вам виконувати різноманітні завдання з обробки рядків у Clojure.
