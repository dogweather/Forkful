## Чому

Написання текстового файлу є важливою частиною програмування в мові Clojure. Це дозволяє зберігати дані для подальшого використання та створювати структуровані файли для зручної роботи з кодом.

## Як

Написання текстового файлу в Clojure досить просте. Нижче наведено приклад коду, що демонструє створення файлу і його заповнення деяким текстом.

```Clojure
;; Імпортуємо бібліотеку для роботи з файлами
(require '[clojure.java.io :as io])

;; Визначаємо шлях до файлу
(def file-path "file.txt")

;; Відкриваємо файл в режимі для запису
(with-open [file (io/writer file-path)]
  ;; Записуємо текст у файл
  (.write file "Це текст, який буде збережено у файлі")
)

```

Після виконання цього коду, у вашій директорії з файлом повинен з'явитися новий файл "file.txt", який буде містити текст "Це текст, який буде збережено у файлі".

## Глибока занурення

Написання текстового файлу також дозволяє зберігати структуровані дані з використанням більш складних структур даних, наприклад, векторів та хеш-мап. Нижче наведено приклад коду, що демонструє створення файлу і записування в нього вектора з хеш-мапами.

```Clojure
;; Імпортуємо бібліотеку для роботи з файлами
(require '[clojure.java.io :as io])

;; Визначаємо шлях до файлу
(def file-path "data.txt")

;; Вектор з хеш-мапами
(def data [{:name "Іван", :age 28}
           {:name "Олена", :age 25}
           {:name "Петро", :age 31}])

;; Відкриваємо файл в режимі для запису
(with-open [file (io/writer file-path)]
  ;; Записуємо дані у файл
  (doseq [d data]
    (.write file (str "Ім'я: " (:name d) ", Вік: " (:age d) "\n")))
)

```

Після виконання цього коду, у вашій директорії з файлом повинен з'явитися новий файл "data.txt", який буде містити наступне:

```
Ім'я: Іван, Вік: 28
Ім'я: Олена, Вік: 25
Ім'я: Петро, Вік: 31
```
