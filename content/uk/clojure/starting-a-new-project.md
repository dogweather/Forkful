## Що та чому? 

Створювання нового проекту - це процес виконання необхідних кроків для початку роботи над новою програмою. Програмісти роблять це, щоб забезпечити правильне оточення для коду, управління залежностями та успішне виконання програми.

## Як то робити:

Створити новий проект у Clojure можна за допомогою Leiningen. Приклад коду та його результат:

```Clojure
$ lein new my-project
```

Ця команда створить новий проект з іменем "my-project" в поточній директорії. Він буде містити каркас коду та інші необхідні файли для початку.

## Ближчий погляд: 

(1) Історичний контекст: Leiningen це автоматизоване середовище Clojure, створено відносно недавно, в 2010 році. Його ціль - значно полегшити розробку Clojure, особливо створення нових проектів. 

(2) Альтернативи: Хоча Leiningen є найпопулярнішим середовищем, альтернативи включають Boot та Clojure CLI. Boot засноване на концепції "программи як конвеєри", тоді як Clojure CLI має менш гнучкі можливості, але забезпечує швидкість і простоту.

(3) Деталі реалізації: При створенні нового проекту Leiningen автоматично створює три основні файли: project.clj, core.clj та test.clj. project.clj містить інформацію про проект і залежності, core.clj - основний файл з кодом, а test.clj - файл для тестових випадків.
