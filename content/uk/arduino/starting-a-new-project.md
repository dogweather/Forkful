## Чому

Starting a new project can be a fun and exciting challenge for any Arduino enthusiast. Not only does it allow you to explore your creativity, but it also helps you learn more about programming and electronics. With an endless array of possibilities, starting a new project with Arduino is a great way to expand your knowledge and skills.

## Як почати

Щоб почати новий проект з Arduino, вам спершу потрібно мати сам апаратний мікроконтролер Arduino та встановлений на комп'ютері Arduino IDE. Після цього ви можете переконатися, що ваш ардуіно правильно підключений до комп'ютера, натиснувши кнопку "Перевірити", щоб переконатися, що код написаний без помилок. Коли ви готові, ви можете натиснути кнопку "Завантажити", щоб відправити свій код на ардуіно та побачити результат.

```Arduino
// Простий приклад вмикальника світла

int LED = 13; // визначення піна, на якому під'єднано світлодіод
int button = 2; // визначення піна, на якому під'єднаний кнопка

void setup() {
  pinMode(LED, OUTPUT); // встановлення піна світлодіоду на вихід
  pinMode(button, INPUT); // встановлення піна кнопки на вхід
}

void loop() {
  if(digitalRead(button) == HIGH){ // перевірка, чи кнопка натиснута
    digitalWrite(LED, HIGH); // увімкнення світлодіода
  }
  else{
    digitalWrite(LED, LOW); // вимкнення світлодіода
  }
}
```

У цьому прикладі, коли кнопка під'єднана до піна 2, натискання кнопки приведе до увімкнення світлодіода, підключеного до піна 13. Це простий, але дуже корисний приклад того, як контролювати зовнішні пристрої за допомогою ардуіно.

## Глибоке погруження

Початок нового проекту з ардуіно також дозволяє вам використовувати різноманітні сенсори, датчики та інші пристрої для створення більш складних інтерактивних проектів. Крім того, ви можете вивчати нові бібліотеки та функції, що допоможуть вам розвивати свої навички програмування.

Наприклад, ви можете використовувати бібліотеку Servo, щоб керувати кутом обертання сервоприводів, або бібліотеку Wire для зчитування даних з модулів сенсорів. Це лише кілька з безлічі можливостей, які можуть бути використані під час створення нового проекту з ардуіно.
