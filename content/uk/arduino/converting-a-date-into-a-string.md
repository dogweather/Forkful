# Чому

Існує багато ситуацій, коли потрібно перетворити дату, яка представлена в програмі, на рядок (строку). Наприклад, для відображення дати на дисплеї або для запису до змінної у форматі, який не підтримує дату і час.

# Як це зробити

Для перетворення дати в рядок можна використати функцію `String()` у бібліотеці Arduino. Нижче показано приклад коду, який демонструє, як це зробити:

```Arduino
int day = 25;
int month = 12;
int year = 2020;

String date = String(day) + "/" + String(month) + "/" + String(year);
```

Після виконання цього коду, змінна `date` буде містити рядок "25/12/2020". Цей приклад використовує цілочисельні змінні, але можна також використовувати значення, які повертають функції `day()`, `month()` та `year()` з бібліотеки `Time`.

# Глибока підкрутка

Хоча функція `String()` багато у чому спрощує процес перетворення дати в рядок, є кілька підводних каменів, про які варто знати. Наприклад, якщо день або місяць має одну цифру (наприклад, "5" замість "05"), то вони будуть відображатись як одна цифра у рядку. Це може викликати проблеми, наприклад, якщо потрібно виконати пошук і порівняння дат. Для попередження цього, можна використовувати функцію `sprintf()`, яка дозволяє задати формат рядка. Наприклад, `sprintf("%02d", day)` поверне рядок, який завжди має дві цифри для дня (наприклад, "05" замість "5").
