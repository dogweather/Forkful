## Чому

Перевірка існування директорії є важливим етапом у програмуванні Arduino. Це дозволяє перевірити, чи існує необхідна директорія перед тим, як програма почне виконувати певні дії з файлами.

## Як це зробити

Для перевірки існування директорії використовується функція `exists()`, яка належить до бібліотеки SD. Приклад коду для перевірки, чи існує директорія "myDirectory":

```Arduino
#include <SD.h>

void setup() {
  // ініціалізуємо SD карту
  SD.begin(10);
  
  // перевіряємо чи існує директорія
  if (SD.exists("myDirectory")) {
    // виконуємо необхідні дії
    Serial.println("Директорія існує!");
  } else {
    // виконуємо інші дії
    Serial.println("Директорія не існує!");
  }
}

void loop() {
  // код для циклічного виконання
}
```

В цьому прикладі ми спочатку ініціалізуємо SD карту за допомогою функції `SD.begin()`, а потім використовуємо функцію `exists()` для перевірки директорії. Після цього можна виконувати певні дії з файлами в залежності від результату перевірки.

## Глибше дослідження

Функція `exists()` перевіряє, чи існує файл або директорія відповідно до заданого шляху. Якщо файл або директорія існують, функція повертає `true`, якщо ні - `false`. Це дозволяє виконувати певні дії з файлами або директоріями тільки у випадку їх наявності.
