## Чому

Чи не варто знати поточну дату? Незалежно від того, чи ви виконуєте проекти з використанням Arduino для школи чи для розваги, ця проста функція може бути дуже корисною. Вона може допомогти вам точно відслідковувати час та дату подій у ваших проектах, а також показувати їх на дисплеї або використовувати для прийняття рішень у вашій програмі.

## Як це зробити

Для початку створимо змінні, в які ми будемо зберігати дані про дату та час. Для цього використовуємо вбудовані функції `day()`, `month()`, `year()`, `hour()`, `minute()` та `second()`. Потім, використовуючи функцію `println()`, виводимо значення цих змінних на монітор порту, щоб перевірити правильність отриманих даних.

```
Arduino Setup
void setup() {
  Serial.begin(9600);
}

Arduino Loop
void loop() {
  // Get current date and time
  int day = day();
  int month = month();
  int year = year();
  int hour = hour();
  int minute = minute();
  int second = second();

  // Print values to Serial Monitor
  Serial.print("Today is ");
  Serial.print(day);
  Serial.print("/");
  Serial.print(month);
  Serial.print("/");
  Serial.print(year);
  Serial.print(" at ");
  Serial.print(hour);
  Serial.print(":");
  Serial.print(minute);
  Serial.print(":");
  Serial.println(second);

  delay(1000);
}
```

В результаті ви повинні побачити на моніторі значення поточної дати та часу, що оновлюються кожної секунди. Тепер ви можете використовувати ці змінні для будь-яких потреб вашого проекту.

## Глибокий занурення

Вбудована функція `day()` повертає цілочисельне значення від 1 до 31, яке відповідає поточному дню. Вона також може приймати один аргумент - вказану дату - і повертає день цієї дати. Аналогічно, функції `month()` та `year()` повертають цілочисельні значення від 1 до 12 та від 2000 до 2099 відповідно.

Функція `hour()` повертає цілочисельне значення від 0 до 23, що відповідає поточному годиннику 24-годинного формату. Функція `minute()` повертає цілочисельне значення від 0 до 59 для поточної хвилини, а функція `second()` повертає цілочисельне значення від 0 до 59 для поточної секунди.
