# Текстові файли та їхнє читання в Arduino

## Чому
Завдання читання текстового файлу може виникнути у багатьох проектах, де Arduino необхідно взаємодіяти з іншими пристроями або зберігати інформацію. Наприклад, при створенні пристрою зберігання даних або при роботі з датчиками, які відправляють дані у текстовому форматі.

## Як
Для читання текстового файлу в Arduino необхідно використовувати бібліотеку SD. Нижче наведений приклад коду, який демонструє читання файлу "data.txt" та виведення його вмісту у моніторі порту:

```Arduino
#include <SD.h>

File myFile;

void setup() {
  // Ініціалізуємо бібліотеку SD та встановлюємо пін для картки пам'яті
  SD.begin(10); 

  // Відкриваємо файл для читання
  myFile = SD.open("data.txt");

  // Зчитуємо та виводимо вміст файлу
  while (myFile.available()) {
    Serial.write(myFile.read());
  }

  // Закриваємо файл
  myFile.close();
}

void loop() {
  // Пустота, бо ми читаємо файл лише раз у функції setup
}
```

Після виконання цього коду, у моніторі порту ви побачите вміст файлу "data.txt", який буде виведений по одному байту на кожній новій стрічці.

## Глибока погруження
Для читання текстового файлу відразу у вигляді строки, можна використовувати функцію `readString()`, або для зчитування числових значень - функцію `parseInt()`. Також можна використовувати різні методи для пошуку конкретної інформації у файлі, наприклад, за допомогою функції `find()`.

Більш детальну інформацію про читання текстових файлів в Arduino можна знайти у [документації з бібліотекою SD](https://www.arduino.cc/en/Reference/SD).
