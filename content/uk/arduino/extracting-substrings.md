## Таблиця. Пояснення програми

Ця стаття пропонує ознайомитися з витягуванням підрядків в Arduino. Ми розглянемо, як це зробити та навіщо це потрібно.

## Причина

Витягування підрядків є корисною властивістю для багатьох проектів, де потрібно працювати з текстовими даними. Наприклад, це може бути потрібно для зчитування дати або часу з текстового сенсору або звернення до імені користувача з клавіатури.

## Як це зробити

Для початку потрібно завантажити бібліотеку `string.h`. Вона дозволяє працювати з рядками і витягувати з них підрядки. Для витягування підрядка з вихідного рядка потрібно використати функцію `substring()`. Давайте подивимося на приклад коду:

```Arduino
#include <string.h> // завантажуємо бібліотеку

void setup() {
  Serial.begin(9600); // виставляємо швидкість передачі даних
  while(!Serial); // чекаємо поки не підключиться монітор порта
}

void loop() {
  String fullString = "Hello Ukraine!";
  
  // Використовуємо функцію substring, щоб витягнути підрядок
  String subString = fullString.substring(6, 12); // в даному випадку, це буде "Ukraine"
  
  Serial.print("Підрядок: ");
  Serial.println(subString);
}
```

Як бачимо, ми завантажили бібліотеку `string.h` та використали функцію `substring()` для витягування підрядка з рядка `"Hello Ukraine!"`. У даному випадку ми задали початок і кінець підрядка, який хочемо витягнути. Не забудьте, що нумерація символів в рядку починається з нуля. Тому для витягування символів з 6 по 11, ми вказали 6 та 12, оскільки останній символ не включається в підрядок.

Також, варто зазначити, що функція `substring()` працює з об'єктом `String`, тому перш ніж використовувати її, необхідно використати ключове слово `String` перед назвою змінної.

## Глибше піре

Функція `substring()` може приймати різні параметри для витягування підрядків. Давайте розглянемо деякі з них:

- `substring(index)`: повертає підрядок починаючи з індексу і до кінця рядка.
- `substring(index, length)`: повертає підрядок з початку індексу заданої довжин
