# Чому

Згаришні числа є невід'ємним елементом в програмуванні на Arduino, оскільки вони дозволяють генерувати випадкову або близьку до випадкової послідовність чисел. Це може бути корисно для створення ігор, симуляцій або будь-яких інших програм, які вимагають випадкових значень.

# Як це зробити

Для генерування випадкових чисел в Arduino використовується функція "random()", яка приймає два аргументи - мінімальне і максимальне значення, між якими потрібно згенерувати число. Наприклад, ви можете використовувати наступний код для генерування випадкового числа від 0 до 100:

```Arduino
random(0, 100);
```

Також, ви можете використовувати функцію "randomSeed()", щоб ініціалізувати генератор випадкових чисел, використовуючи як базове значення який-небудь фізичний вихідний сигнал, наприклад, використовуючи сигнал від сенсора температури. Це може зробити ваші випадкові числа ще більш випадковими.

# Глибше

Один з народжень випадкових чисел на базі software generator є дійсно випадковим, хоча й здатними до багатьох корисних операцій. Випадкові числа на базі hardware generator більш вірогідно будуть по-справжньому випадковими, оскільки вони використовують зовнішні фізичні події, такі як рух зірок або шум радіоперемикачів, як джерело для випадкових дій.
