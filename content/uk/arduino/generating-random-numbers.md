# Генерування випадкових чисел в Arduino 

## Що і навіщо?
Генерування випадкових чисел - це процес створення чисел, які не мають передбачуваних значень. Це важливо для програмістів, оскільки випадкові числа широко використовуються для створення унікальних ідентифікаторів, випадкового вибору елементів, тестування алгоритмів тощо.

## Як зробити:
Для генерування випадкових чисел можемо використовувати функцію `random()` в Arduino. Дивіться приклади нижче:

```Arduino
void setup() {
  // Відкрити серійний порт
  Serial.begin(9600);
}

void loop() {
  // Генерувати випадкове число між 0 і 100
  int randNumber = random(0, 101);
  
  // Друкувати випадкове число
  Serial.println(randNumber);
  
  // Затримка на 1 секунду
  delay(1000);
}
```

Виходом буде вся послідовність випадкових чисел з проміжком 1 секунда між ними.

## Поглиблений взгляд
- Історичний контекст: Генерування випадкових чисел дуже важливе в комп'ютерному програмуванні і датується ще з часів першобутніх комп'ютерів.
- Альтернативи: Окрім `random()`, в Arduino є й інші методи для генерування випадкових чисел, наприклад `randomSeed()`, що використовується для ініціалізації генератора випадкових чисел.
- Виконання: Реальна генерація "випадкових" чисел на комп'ютері є важкою задачею. Arduino `random()` фактично генерує групу перемішану чисел, яка повторюється при кожному запуску до появи нового `randomSeed()` або живлення.
