## Що і Для чого?

Обчислення дати в майбутньому чи минулому - це процес визначення конкретної дати, відстаючи від відомої дати на визначену кількість днів, тижнів, місяців або років. Програмісти роблять це для обробки задачі, що вимагають врахування часових проміжків або обрахунку термінів.

## Як це зробити:

Спробуйте використовувати наступний код для обчислення дати в майбутньому на Arduino:

```Arduino
#include <TimeLib.h>
void setup() {
  setTime(8, 29, 0, 1, 1, 2011); // встановити час 8:29:00 1 січня 2011
}
void loop () {
  time_t t = now(); 
  t = t + 7*24*60*60; // додати 7 днів
  Serial.println(year(t)); // рік
  Serial.println(month(t));// місяць
  Serial.println(day(t));// день
}
```
Приклад виводу:

```Arduino
2011
1
8
```

## У Глибину:

Обчислення дати з'явилося зі створенням перших календарів, що були необхідні для організації сільськогосподарської діяльності. В комп'ютерному програмуванні воно стало необхідним для різноманітних задач, включаючи зберігання дати і часу створення файлу, планування подій та організацію роботи.

Альтернативи для обчислення дати включають використання інших бібліотек, таких як DateTime або використання інтернет-послуг, таких як NTP сервери.

Але у всіх разів основна задача - це правильне обрахування різниці в часі між поточним часом і бажаною датою, і навпаки.
