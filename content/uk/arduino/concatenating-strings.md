## Чому

У програмуванні Arduino можна поєднувати рядки для створення динамічних повідомлень або надсилання даних на екран виводу. Це дозволяє зробити наш код більш універсальним та зручним для користувачів.

## Як це зробити


Для початку створимо дві змінні типу *String*, які ми хочемо об'єднати:

```Arduino
String message = "Привіт,";
String name = "Дмитре!";
```

Далі, застосуємо функцію *concat()*, яка об'єднує рядки та повертає нову змінну:

```Arduino
String finalMessage = message.concat(name); // Результат: "Привіт, Дмитре!"
```

Спробуємо вивести цей результат на екран виводу або відправити посиланням через з'єднання з Інтернетом:

```Arduino
Serial.println(finalMessage); // Результат на екрані: Привіт, Дмитре!
client.println(finalMessage); // Відправляється через мережу
```

## Розглянемо детальніше

Звернімо увагу, що при виклику функції *concat()* оригінальна змінна залишається незмінною, тож ми можемо зберегти її будь-якому іншому проміжному значенню та продовжувати об'єднувати рядки. Крім того, функція *concat()* підтримує не тільки об'єднання тексту, але і чисел, що дозволяє створювати більш складні повідомлення.

## Дживу связке

Для створення динамічних рядків, коли деякі частини повідомлення можуть змінюватися, можна застосувати конструкцію *String concatenation*. Вона дозволяє поєднувати текстові рядки та змінні без використання функції *concat()*:

```Arduino
String voltage = "Заряд батареї: ";
float percentage = 78.9; // Приклад значення
String message = voltage + String(percentage) + "%"; 
// Результат: "Заряд батареї: 78.9%"
```

На цьому ми завершуємо нашу коротку статтю про об'єднання рядків в Arduino. Надіємося, що тепер ви зможете створювати більш корисний та зручний код за допомогою цієї техніки. Дякуємо за увагу!
