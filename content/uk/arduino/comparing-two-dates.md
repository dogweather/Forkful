## Чому

Програмування на Arduino може бути дуже захоплюючим та цікавим хобі для багатьох людей. Але чи ви знали, що ви можете порівнювати дати на вашому Arduino? Ця віддалена проблема може бути цікавою для практикувального програміста або для початківця, який хоче розвиватися у своїх навичках. Давайте розберемось, як це зробити!

## Як це зробити

Для початку, нам потрібно оголосити змінні для двох дат:
```Arduino
int firstDay = 17; // перша дата
int firstMonth = 9;
int firstYear = 2021;
int secondDay = 8; // друга дата
int secondMonth = 7;
int secondYear = 2021;
```

Тепер ми можемо використовувати умови для порівняння двох дат. Наприклад, ми можемо з'ясувати, яка дата є пізнішою:
```Arduino 
if (firstYear > secondYear) {
  // перша дата пізніша
  Serial.print("Перша дата пізніша");
}
else if (secondYear > firstYear) {
  // друга дата пізніша
  Serial.print("Друга дата пізніша");
}
else {
  // роки однакові, перевіряємо місяці
  if (firstMonth > secondMonth) {
    // перша дата пізніша
    Serial.print("Перша дата пізніша");
  }
  else if (secondMonth > firstMonth) {
    // друга дата пізніша
    Serial.print("Друга дата пізніша");
  }
  else {
    // місяці та роки однакові, перевіряємо дні
    if (firstDay > secondDay) {
      // перша дата пізніша
      Serial.print("Перша дата пізніша");
    }
    else if (secondDay > firstDay) {
      // друга дата пізніша
      Serial.print("Друга дата пізніша");
    }
    else {
      // обидві дати однакові
      Serial.print("Обидві дати однакові");
    }
  }
}
```

Виходячи з нашого прикладу, в Serial Monitor виведеться повідомлення “Друга дата пізніша”.

## Поглиблене дослідження

У цій задачі, ми скористалися умовами, щоб порівняти дві дати. Але що робити, якщо необхідно порівнювати більше двох дат? Також можна розглядати всі можливі випадки, такі як початок та кінець місяця, вісімнадцяте лютого тощо.
