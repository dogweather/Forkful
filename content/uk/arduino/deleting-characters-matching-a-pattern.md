## Що та Навіщо?

Видалення символів, що відповідають паттерну - це процес відбору символів з рядка за певними критеріями, які ви вирішили використовувати. Програмісти роблять це для уникнення помилок, як-наприклад, при введенні даних користувачем.

## Як це зробити:

Давайте створимо функцію, яка перебирає кожний символ в рядку і видаляє небажані символи:
```Arduino
char* removeChars(char* str, char target) {
  char* p = str;
  char* q = str;
  while(*p) {
    if (*p != target)
      *q++ = *p;
    p++;
  }
  *q = '\0';
  return str;
}
```
Застосуймо нашу функцію до рядка "Hello, World!":
```Arduino
char str[] = "Hello, World!";
removeChars(str, 'l');
Serial.println(str);
```
Ви отримаєте наступне:
```
Heo, Word!
```

## У глибину:

1. Історичний контекст: Первинний метод видалення символів натрапляє користувачів Arduino на ітерацію кожного символу у рядку. Це випливає з його низькорівневої природи та обмежених мульти-таскінгових можливостей.
 
2. Альтернативи: В інших мовах програмування існують більш прямі методи для цієї задачі. Наприклад, в Python ви можете просто використати метод `replace`.

3. Подробиці реалізації: Наша функція перебирає кожен символ в рядку за допомогою покажчиків `p` і `q`. Якщо символ в `p` не є символом, що потрібно видалити, ми зберігаємо його в `q` і зміщуємо `q`. Якщо він є символом, який потрібно видалити, ми просто зміщуємо `p`.
