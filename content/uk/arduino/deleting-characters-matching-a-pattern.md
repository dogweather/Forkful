## Навіщо

У цій статті ми дізнаємося, як видалити символи, що відповідають певному шаблону, за допомогою Arduino. Це корисна навичка для програмістів, які працюють з текстовими даними та хочуть швидко та ефективно очищати їх від непотрібних символів.

## Як

Насамперед, ми повинні ознайомитися зі спеціальними функціями для роботи з текстом, які надає Arduino. Одна з них - `charAt()`, яка повертає символ за вказаним індексом з рядка. За допомогою цієї функції ми можемо перевірити, чи відповідає символ вказаному шаблону та, у разі позитивної відповіді, видалити його за допомогою функції `remove()`.

```Arduino
String text = "Hello, world!";
char pattern = 'o';
// Перевірка кожного символу тексту
for(int i=0; i < text.length(); i++){
  // Якщо символ відповідає шаблону,
  // видаляємо його за допомогою remove()
  if(text.charAt(i) == pattern){
    text.remove(i);
  }
}
Serial.println(text); // Виводимо результат: Hell, wrld!
```

У цьому прикладі ми видалили всі символи 'o' зі строку `text`. Аналогічно, ви можете видалити і інші символи, які не відповідають вашим потребам.

## Глибокий дайв

Якщо ви хочете налаштувати щось більш складне, ви можете використовувати регулярні вирази для визначення шаблону. Для цього вам знадобиться підключити бібліотеку `Regex.h`. Дивіться приклад нижче:

```Arduino
#include <Regex.h>

String text = "Hello, world!";
String pattern = "l"; // Шаблон - одна буква "l"
String replacement = ""; // Кожна відповідність буде замінена на порожню строку
Regex regex(pattern);
// Використовуємо функцію `String::replaceAll()` для заміни кожної відповідністі на порожню строку
String result = regex.replaceAll(text, replacement);
Serial.println(result); // Виводимо результат: Heo, word!
```

Перш за все, ми підключаємо бібліотеку `Regex.h`. Далі ми визначаємо змінні для тексту, шаблону та заміни. Потім ми створюємо об'єкт `Regex` за допомогою шаблону, який буде використовуватися для пошуку в тексті. Нарешті, ми використовуємо функцію `String::replaceAll()` для заміни кожної відповідності на порожню строку та виводимо результат.
