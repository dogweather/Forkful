## Чому

Чому нам потрібно обчислювати дату у майбутньому або минулому? Один з основних причин - це необхідність планувати події, які відбудуться через певний час, або аналізувати минулі події для досягнення кращого розуміння. Можливо, Ви хочете знати, коли закінчиться дійсність Вашої візи або коли ваші діти відзначатимуть свою наступну день народження. У будь-якому випадку, процес обчислення дати в майбутньому або минулому може бути досить корисним у повсякденному житті.

## Як

Для початку, нам потрібно імпортувати клас "DateTime" з пакету "java.time". Потім ми можемо використовувати функцію "now ()" для отримання поточної дати. Для обчислення дати у майбутньому ми можемо використовувати функцію "plusDays (n)", де "n" відповідає кількості днів, які ми хочемо додати до поточної дати. Наприклад, якщо ми хочемо обчислити дату, яка буде через тиждень, ми можемо викликати "now (). PlusDays (7)". У майбутньому ви побачите, що вихідний формат дати буде "рік-місяць-день". Також, якщо Ви хочете обчислити дату в минулому, Ви можете взяти той же підхід, але використовувати функцію "minusDays (n)". Далі наведений код на мові Котлін для обчислення дати:

```Kotlin
// Імпорт класу DateTime
import java.time.DateTime

// Створення змінної, яка містить поточну дату
val currentDate = DateTime.now()

// Обчислення дати, яка буде через 7 днів
val futureDate = currentDate.plusDays(7)

// Обчислення дати, яка була 30 днів тому
val pastDate = currentDate.minusDays(30)

// Вивід результатів
println("Сьогоднішня дата: $currentDate")
println("Дата через 7 днів: $futureDate")
println("Дата 30 днів тому: $pastDate")
```

Результат виконання коду буде наступним:

```
Сьогоднішня дата: 2021-06-25
Дата через 7 днів: 2021-07-02
Дата 30 днів тому: 2021-05-26
```

Зверніть увагу, що вихідний формат дати може відрізнятися, залежно від налаштувань Вашої системи.
