## Що це таке і навішо?

Отримання поточної дати - це процес, в якому програма визначає актуальний момент у часі (тобто "сьогодні"), часто в форматі день/місяць/рік. Ми використовуємо такі дані для різних потреб - від слідкування за хронологією подій до генерації таємних ключів.

## Як це зробити:

Гадаю, тепер ви готові побачити, як це можна зробити в Kotlin. Ви можете отримати поточну дату наступним чином:

```Kotlin
import java.time.LocalDate

fun main(args: Array<String>) {
    val currentDate = LocalDate.now()
    println("Поточна дата: $currentDate")
}
```

При виконанні цього коду поточна дата буде виведена у вихідний потік. Результат може бути таким:

```
Поточна дата: 2022-02-25
```

## Поглиблено:

Використання дати та часу були винайдені десь на світанку програмування і з того часу займають важливе місце. В Kotlin 1.0 не було вбудованого типу дати/часу, але щоб вирішити цю проблему, ми мали користуватися Java-бібліотеками. Зовсім недавно, в Kotlin 1.8, було вводено нові бібліотеки часу та дати.

Як альтернативу, вам можуть бути цікаві бібліотеки Joda-Time і ThreeTenABP. Вони пропонують широкий спектр функцій для роботи з датою й часом.
