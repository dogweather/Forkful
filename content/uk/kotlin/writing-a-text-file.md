## Чому

Написання текстових файлів є важливою навичкою для багатьох програмістів, оскільки це дозволяє зберігати та перетворювати дані у зручний для розуміння формат. Наприклад, ви можете створити текстовий файл, який містить список ваших завдань або результати обчислень вашої програми.

## Як це зробити

Для того, щоб створити текстовий файл у Kotlin, ви можете використати клас `File` і метод `writeText()`. Давайте розглянемо приклад:

```Kotlin
val file = File("myTextFile.txt") // створення об'єкту файлу
val text = "Це текст, який буде збережений у файлі." // створення змінної з текстом
file.writeText(text) // запис тексту у файл
```

Цей код створить новий файл з назвою "myTextFile.txt" і запише у нього текст. Ви можете використати будь-який текст, який ви хочете зберегти у файлі.

Також, ви можете використати метод `appendText()` для додавання тексту до вже існуючого файлу, або метод `readText()` для отримання змісту файлу у вигляді строки.

## Deep Dive

Написання текстових файлів в Kotlin також дозволяє вам використовувати додаткові функції, такі як встановлення додаткових атрибутів для файлів. Наприклад, ви можете створювати каталоги, копіювати та переміщувати файли з використанням методів класу `Files`.

Також варто зазначити, що ви можете використовувати різні кодування тексту для збереження у файлі, такі як UTF-8 або UTF-16. Для цього ви можете використовувати метод `charset()` і передати йому потрібне кодування.
