## Чому

Напевно, ви знаєте, що якщо у вашій програмі відбуваються проблеми, то найкращим рішенням є виведення на екран різних повідомлень для відладки. Виведення таких повідомлень допомогає відслідковувати, як саме працює ваш код та виявляти проблеми.

## Як

Для того, щоб виводити повідомлення для відладки у вашій програмі на Kotlin, вам потрібно:

1. Використовувати функцію `println()` для виведення повідомлень на екран. Наприклад:

```Kotlin
println("Повідомлення для відладки")
```

2. Додати необхідну інформацію про стан вашої програми. Наприклад, ви можете додати значення змінної, яка використовується у даний момент, або інформацію про роботу певної функції.

```Kotlin
val x = 5
println("Значення змінної x: $x")
```

3. Запустити вашу програму та переглянути виведені повідомлення для відладки. Якщо була знайдена якась проблема, ви можете використовувати цю інформацію для її вирішення.

## Deep Dive

Використання виведення повідомлень для відладки є потужним інструментом для виявлення та вирішення проблем у програмах на Kotlin. Цей підхід допомагає полегшити процес відлагодження і знижує час, необхідний для вирішення проблем.

Важливо також пам'ятати, що виведені повідомлення необхідно видаляти після того, як ви вирішили проблему, оскільки вони можуть спричинити збій у роботі вашої програми.
