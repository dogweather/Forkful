## Чому

Іноді, під час програмування на Kotlin, може виникнути ситуація, коли потрібно видалити певні символи з рядка, які відповідають заданому шаблону. Це може бути необхідно, наприклад, під час роботи зі змінними або при форматуванні введених користувачем даних. В таких випадках, знання того, як видалити символи, які відповідають певному шаблону, є дуже корисним.

## Як це зробити

Мова програмування Kotlin надає нам зручний метод для видалення символів, що відповідають певному шаблону - `replace(Regex, String)`.

```Kotlin
val string = "Привіт, користувачі! Цікаво з вами поговорити."
val output = string.replace(Regex("[ви]"), "")
println(output)
// Результат: Прт, корстуач! Цкести з ми погоорит.

val input = "Hello, world! This is a test."
val output2 = input.replace(Regex("[aeiou]"), "*")
println(output2)
// Результат: H*ll*, w*rld! Th*s *s * t*st.
```

У цьому прикладі, ми використовуємо метод `replace` для видалення символів `в` та `и` із рядка. Також, ми можемо використовувати шаблон `[aeiou]` для видаляння всіх голосних символів.

## Глибинний аналіз

Щоб краще зрозуміти механізм видалення символів за шаблоном в Kotlin, необхідно розглянути деякі важливі аспекти. По-перше, метод `replace` приймає два параметри - `Regex` та `String`. `Regex` описує шаблон для знаходження символів, які потрібно видалити, а `String` вказує на те, на який вид символів їх замінити.

Також, важливо пам'ятати, що ми можемо використовувати будь-які регулярні вирази для опису шаблону, що дозволяє нам більш гнучко налаштувати процес видалення символів.
