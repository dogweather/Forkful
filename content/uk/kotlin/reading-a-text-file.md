## Чому

Кожен програміст заслуговує на ефективні інструменти для читання та обробки текстових файлів. Незалежно від того, чи ви працюєте з веб-скриптами, десктопними додатками або мобільними додатками, читання текстових файлів є необхідною частиною багатьох програм.

## Як

Для початку, нам потрібно отримати доступ до текстового файлу в нашій програмі. Це можна зробити за допомогою вбудованого пакету Kotlin `java.io.FileReader`. Для зручності, ми можемо створити функцію, яка буде приймати ім'я файлу та повертати список рядків, які містяться у цьому файлі.

```Kotlin
import java.io.FileReader

fun readTextFile(fileName: String): List<String> {
  // створюємо порожній список для збереження рядків
  val lines = mutableListOf<String>()
  // створюємо екземпляр класу FileReader, який буде читати файл за допомогою назви файлу
  val reader = FileReader(fileName)
  // створюємо "буфер" для збереження рядків
  val buffer = CharArray(4096)
  // починаємо читати рядки з файлу та додаємо їх до списку
  while (reader.read(buffer) != -1) {
    lines.add(buffer.concatToString())
  }
  // закриваємо з'єднання із файлом
  reader.close()
  // повертаємо список рядків
  return lines
}

// приклад використання
val fileName = "my_text_file.txt"
val lines = readTextFile(fileName)
// виводимо список рядків у консоль
println(lines)
```

Виявляється, у нас є кілька способів читання файлів в Котлін. Ми можемо використовувати `readLines()` метод класу `File`, який повертає список рядків з файлу. Також ми можемо використовувати `forEachLine()` метод, який дозволяє нам обходити кожен рядок файлу та виконувати певну дію над ним.

## Глибина

Якщо ви хочете глибше дізнатися про читання текстових файлів в Котлін, то варто розглянути наступні теми:

- читання та записування в різні типи файлів, такі як CSV та JSON;
- обробка помилок під час читання файлів, таких як відсутність файлу або помилка у форматі;
- оптимізація читання великих файлів шляхом використання `BufferedReader` та `BufferedInputStream` класів.
