## Чому

Створення нового проекту - це шанс розвивати свої навики програмування і вивчати нові технології. Крім того, це може бути великий крок до створення чогось цікавого та корисного для світу.

## Як

У Kotlin є багато властивостей, які роблять його привабливим для розробників. Давайте розглянемо кілька прикладів коду, щоб показати, наскільки легко його вивчити та використовувати.

```Kotlin
val name = "John"
val age = 27
println("$name is $age years old.")
```

У цьому прикладі ми створили дві змінні з типами даних String та Int, та застосували їх у функції `println()` для виведення рядка на екран. Зверніть увагу на використання спеціального символу `$`, що дозволяє вставити значення змінної у рядок.

```Kotlin
val numbers = listOf(2, 4, 6, 8, 10)
for (num in numbers) {
  println(num * 3)
}
```

У цьому прикладі ми створили список чисел с допомогою функції `listOf()`, а потім використали цикл `for` для виведення на екран кожного числа, помноженого на 3. Зверніть увагу на використання оператора `in`, який перебирає елементи списку.
