## Що і чому?
Запуск нового проєкта - це створення нової програми з нуля. Програмісти роблять це, щоб вирішити конкретну проблему або зробити життя людей кращим.

## Як це зробити:
Зараз я покажу, як розпочати новий Kotlin проєкт у середовищі розробки IntelliJ IDEA. По-перше, отримаємо IntelliJ IDEA за посиланням https://www.jetbrains.com/idea/download/ 

```Kotlin
// Створюємо новий проект
fun main(){
    println("Вітаємо у новому проєкті!")
}
```
Якщо все зроблено вірно, то ви отримаєте таке виведення програми:

```
Вітаємо у новому проєкті!
```

## Поглиблений матеріал
1. Історичний контекст: Kotlin був створений у 2011 році командою JetBrains, яка шукала безпечну й продуктивну альтернативу Java.

2. Альтернативи: Java є популярною альтернативою Kotlin до Android розробки. Однак з часом Kotlin стає все популярнішим завдяки своїм безпековими функціями та синтаксисом, який легко читати.

3. Деталі реалізації: Коли ми створюємо новий проект, IntelliJ IDEA автоматично генерує файл build.gradle, де ми можемо налаштувати залежності нашого проекту.
