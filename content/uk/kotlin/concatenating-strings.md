## Чому

З'єднання рядків є важливим аспектом програмування, особливо у мові Kotlin. Це дозволяє поєднувати різні рядки тексту та створювати більш складні повідомлення або дані. Використання з'єднання рядків є незамінним у багатьох ситуаціях, коли потрібно представити інформацію для користувачів чи зберегти дані у програмі.

## Як це зробити

Для з'єднання рядків у мові Kotlin використовується оператор "+" або функція "plus()". Наприклад, у наступному коді показано, як з'єднати два рядки:

```Kotlin
val firstString = "Привіт,"
val secondString = "світе!"
val combinedString = firstString + " " + secondString 

println(combinedString)
```

В цьому прикладі ми використали інтерполяцію рядків, де значення змінних автоматично вставляються у рядок за допомогою символу "$". Наприклад:

```Kotlin
val name = "Олександр"
val message = "Привіт, $name! Як справи?"

println(message)
```

Результат виконання цього коду буде: "Привіт, Олександр! Як справи?".

Функція "plus()" також дозволяє з'єднувати більше двох рядків за один раз. Наприклад:

```Kotlin
val firstString = "Це"
val secondString = "дуже"
val thirdString = "просто"
val combinedString = firstString.plus(secondString).plus(thirdString)

println(combinedString)
```

Результатом виконання буде "Це дуже просто".

## Глибокий пір

З'єднання рядків у Kotlin є дещо більш ефективним, ніж у багатьох інших мовах програмування. Це через те, що воно використовує спеціальні функції, такі як "StringBuilder", які дозволяють додавати рядки до існуючих замість створення нових кожного разу. Це дозволяє покращити продуктивність та ефективність програми.

Також, у Kotlin є можливість додавати інші типи даних до рядків. Наприклад, це може бути числа, булеві значення чи навіть інші рядки. Для цього використовується функція "toString()" для перетворення типів даних на рядки.
