## Чому

Можливо, ви питаєтеся: "Зачому мені потрібно читати аргументи командного рядка в Kotlin?" Це може бути необхідним для багатьох різних програм, наприклад, ви можете використовувати аргументи командного рядка для передачі параметрів у вашу програму або для налаштування її роботи. Далі ми з'ясуємо, як це робити.

## Як це робити

Загалом, для читання аргументів командного рядка в Kotlin використовується об'єкт `args`, який містить масив рядків, які були передані у вашу програму як аргументи. Для доступу до цих аргументів можна використовувати індексування, наприклад, `args[0]` поверне перший аргумент, `args[1]` - другий і так далі.

```Kotlin
fun main(args: Array<String>) {
    println("Перший аргумент: ${args[0]}")
    println("Другий аргумент: ${args[1]}")
    println("Третій аргумент: ${args[2]}")
}
```

Якщо ви запустите цю програму з командної строки з аргументами "Привіт", "світ" і "!", то ви побачите наступний вихід:

```Kotlin
Перший аргумент: Привіт
Другий аргумент: світ
Третій аргумент: !
```

Також ви можете використовувати цикли для обробки всіх аргументів командного рядка.

```Kotlin
fun main(args: Array<String>) {
    for (arg in args) {
        println(arg)
    }
}
```

Якщо запустити з аргументами "Привіт", "світ" і "!", ви побачите наступний вихід:

```Kotlin
Привіт
світ
!
```

## Озираючись в глиб

Для більшої гнучкості, ви можете також використовувати бібліотеку `kotlinx.cli`, яка дозволяє вам описувати аргументи вашої програми у вигляді об'єктів та автоматично читати їх значення. Для цього необхідно додати залежність в `build.gradle` файл вашого проекту та додати наступний код до `main()` функції:

```Kotlin
val parser = ArgParser("my-program")
val output by parser.option(ArgType.String, shortName = "o", description = "Назва вихідного файлу").required()
val input by parser.argument(ArgType.String, description = "Шлях до вхідного файлу").required()

parser.parse(args)

println("Вихідний файл: $output")
println("Вхідний файл: $input")
```

Якщо запустити цю програму з аргументами "-o result.txt" та "input.txt", ви побачите наступний вихід:

```Kotlin
Вихідний файл: result.txt
Вхідний файл: input.txt
```
