## Тому чому

Заміна дати на рядок є часом необхідною процедурою при прогресії проекту. Це може бути корисним для відображення дати у користувальницькому форматі або для збереження дати в файлі. Розуміння того, як зробити це вірно в мові програмування Rust може бути корисною навичкою для вас.

## Як це зробити?

Зроблення конвертації може бути простою задачею в Rust, якщо ви знаєте правильні функції та типи даних для використання. Ось приклад коду, який демонструє, як перетворити дату у рядок:

``` Rust
use chrono::{DateTime, Local, Datelike};

fn main() {
    let date = Local::now();
    let formatted_date = date.format("%B %d, %Y").to_string();
    println!("Today is {}", formatted_date);
}
```

У цьому прикладі ми використовуємо бібліотеку `chrono` для отримання поточної дати та форматування її за допомогою методу `format()`. Рядок `"%B %d, %Y"` визначає бажаний формат дати, де `%B` - повне назва місяця, `%d` - день місяця, а `%Y` - рік.

Виконання цього коду повинно вивести наступне:

```console
Today is August 20, 2021
```

Це лише один з багатьох способів просто виконати конвертацію дати у рядок в мові Rust. Існує також більш гнучкий підхід для форматування дати, що може бути корисним для різних задач.

## Глибше дослідження

Тепер, коли ми вже знаємо, як зробити конвертацію дати у рядок, давайте поглибимося в теорію і побачимо, як це працює під капотом.

У мові Rust, дата зберігається як об'єкт `DateTime`, який містить інформацію про рік, місяць, день, годину, хвилину та секунду. Якщо ми хочемо отримати просто рядок з датою, нам потрібно використовувати метод `to_string()`, який перетворить об'єкт `DateTime` у рядок за допомогою встановленого формату.

Також, якщо потрібно зберегти дату у файл, ми можемо використовувати метод `format()` для зміни формату за допомогою специфікатора `%` і потім зберегти рядок у файл.
