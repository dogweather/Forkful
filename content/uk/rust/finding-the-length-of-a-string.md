## Навіщо

Перед тим як почати програмувати в Rust, ми часто задаємо собі питання "чому?". Одним з найбільш поширених використань мови Rust є робота з рядками. І, щоб ефективно працювати з рядками, нам потрібно знати їх довжину. Тож, сьогодні ми розглянемо, як знайти довжину рядка в Rust.

## Як

Для початку, нам потрібно імпортувати бібліотеку `std::str`, яка містить функцію `len`. Потім, використовуючи цю функцію, ми зможемо знайти довжину будь-якого рядка.

```Rust
use std::str;

fn main() {
    let s = "Привіт, світ!";
    let len = str::len(s);
    print!("Довжина рядка {} дорівнює {} символам", s, len);
}
```

Вивід:

```
Довжина рядка Привіт, світ! дорівнює 13 символам
```

## Глибокий занурення

У мові Rust рядки є об'єктами, тому для знаходження їх довжини ми можемо використовувати метод `len()`:

```Rust
fn main() {
    let s = "Привіт, світ!";
    let len = s.len();
    print!("Довжина рядка {} дорівнює {} символам", s, len);
}
```

Вивід:

```
Довжина рядка Привіт, світ! дорівнює 13 символам
```

Також, важливо пам'ятати, що рядки в мові Rust є індексованими, тобто кожен символ має свій власний індекс, починаючи з нуля. Тож, щоб отримати загальну довжину рядка, ми можемо використовувати метод `chars()`:

```Rust
fn main() {
    let s = "Привіт, світ!";
    let len = s.chars().count();
    print!("Довжина рядка {} дорівнює {} символам", s, len);
}
```

Вивід:

```
Довжина рядка Привіт, світ! дорівнює 11 символам
```
