## Чому

Написання в стандартну помилку (standard error) є важливою складовою програмування в Rust. Це дозволяє зберігати повідомлення про помилки та виводити їх у консоль для зручності налагодження та аналізу програми.

## Як це зробити

Щоб виводити повідомлення в стандартну помилку, можна скористатися функцією `eprintln!()`, яка дозволяє передавати текстові дані та значення змінних, які потрібно вивести. Нижче наведений приклад коду та його вивід:

```Rust
fn main() {
    let name = "Yaroslav";
    let age = 27;
    
    eprintln!("Привіт, моє ім'я - {}, мені {} років", name, age);
}
```

Вивід:

```
Привіт, моє ім'я - Yaroslav, мені 27 років
```

## Глибоке погруження

В стандартній бібліотеці Rust є ще одна функція для виводу в стандартну помилку - `writeln!()`. Вона дозволяє направляти повідомлення в будь-який об'єкт, який має метод `write()`. Це може бути, наприклад, файл або з'єднання з іншим потоком виводу. Для використання цієї функції, потрібно включити модуль `std::io`.

```Rust
use std::io::{self, Write};

fn main() {
    let mut buf = Vec::new();
    let msg = "Це повідомлення буде направлено в буфер";
    
    writeln!(&mut buf, "{}", msg).expect("Помилка запису в буфер");
    
    println!("Вміст буфера: {:?}", buf);
}
```

Вивід:

```
Вміст буфера: [208, 129, 208, 181, 32, 208, 191, 208, 184, 208, 188, 208, 187, 208, 190, 208, 187, 209, 130, 32, 208, 178, 208, 190, 208, 191, 208, 187, 208, 178, 208, 187, 208, 190, 208, 186, 208, 190, 208, 181, 32, 208, 178, 32, 209, 131, 208, 182, 208, 181, 209, 129, 209, 130, 208, 189, 208, 184, 208, 191, 208, 190, 208, 186, 208, 190, 208, 180, 208, 178, 209, 129]
```
