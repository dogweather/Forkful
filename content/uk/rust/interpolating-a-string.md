## Що це таке і навіщо?
Інтерполяція рядка - це процес вставки значень змінних або виразів прямо у рядок. Це робить код більш читабельним і допомагає уникнути помилок при з'єднанні рядків.

## Як це зробити:
В Rust інтерполяція рядків реалізована за допомогою методу `format!()`. Ось приклад:
```Rust
let ім'я = "Василь";
let вік = 30;

let текст = format!("Моє ім'я {}, і мені {} років.", ім'я, вік);
println!("{}", текст);
```
На виході отримаємо:
```
Моє ім'я Василь, і мені 30 років.
```

## Поглиблений розділ
Інтерполяція рядків була введена в багатьох мовах програмування ще в 1960-ті роки, і вона стала стандартом у багатьох сучасних мовах. 

Альтернатива інтерполяції рядків у Rust - це використання оператора `+` для з'єднання рядків, але це може призвести до помилок і знижує читабельність коду.

Внутрішньо, метод `format!` використовує макроси Rust для генерації коду, який динамічно створює новий рядок з вказаними значеннями.
