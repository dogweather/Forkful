# Видалення символів, які відповідають певному шаблону в Rust

## Що і навіщо?

Видалення символів, які відповідають певному шаблону - це процес, коли ми тримаємо рядки, але видаляємо з них певні символи. Програмісти роблять це, щоб очистити дані або зробити інформацію більш зрозумілою для обробки.

## Як це зробити:

Ви можете видалити символи, які відповідають шаблону, використовуючи функцію `replace()` в Rust. Цей приклад показує, як це зробити:

```Rust
fn main() {
    let s = "Привіт, Світе!";
    let result = s.replace("і", "");
    println!("{}", result);
}
```

Виводом буде: `Првт, Свте!`. Заміна `і` на пустий рядок видалила всі входження `і`.

## Поглиблений аналіз 

Рядки в Rust були розроблені так, щоб бути якнайбільш гнучкими і універсальними. Коли ви видаляєте символи, що відповідають шаблону, ви фактично створюєте нову версію рядка без тих символів. 

Альтернативно, ви також можете використовувати функцію `retain()`, яка дозволяє вам видалити символи на основі функції зворотного виклику. 

```Rust
fn main() {
    let mut s = String::from("Привіт, Світе!");
    s.retain(|c| c != 'і');
    println!("{}", s);
}
```

Виводом буде також: `Првт, Свте!`.
