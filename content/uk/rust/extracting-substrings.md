# Зникаємо в Расті: Як Дістати Підрядок

## Що та Навіщо?
Використання підрядків, або "substring," - це коли ви берете частину певного рядка. Це здійснюється програмістами для розбиття більших рядків на менші, простіші об'єкти, або для видалення непотрібних даних.

## Як це зробити:
В Расті видобування підрядка відносно прямолінійне завдання. Використання методу 'slice' може бути допоміжним. Ось як це виглядає:

```Rust
fn main() {
let s = String::from("Привіт, світ!"); 
let hello = &s[0..7];
let world = &s[9..14];
println!("{} {}!", hello, world);
}
```

Виходить так:
```
Привіт світ!
```

## Поглиблений огляд:
**Історичний контекст:** Раст забезпечує простий спосіб роботи з підрядками, що значно відрізняється від більш старіших мов, таких як С. Натомість він використовує концепцію 'slicing', що існує в інших сучасних мовах.

**Альтернативи:** Раст має методи `splitn()` і `split_terminator()`, які розділяють рядок за допомогою роздільника.

**Деталі імплементації:** В расті рядки представляють собою об'єкти UTF-8, тому коли ви використовуєте метод `slice`, вам доведеться пам'ятати про межі UTF-8 символів.
