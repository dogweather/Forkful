## Чому
Екстракція підрядків є важливою частиною програмування, тому що дозволяє легко та ефективно маніпулювати рядками. Використовуючи цей метод, ви можете витягнути певну частину рядка, що спростить ваші завдання та збільшить продуктивність програми.

## Як виконати
Для того, щоб використовувати екстракцію підрядків в мові Rust, вам необхідно знати деякі основи цієї мови програмування. Нижче наведено код, який демонструє простий приклад екстракції підрядків за допомогою функції `get_substring`:

```Rust
fn get_substring(string: &str, start: usize, end: usize) -> &str {
    &string[start..end]
}

fn main() {
  let my_string = "Це приклад рядка для екстракції підрядка.";
  let sub_string = get_substring(&my_string, 17, 24);
  println!("{}", sub_string); // виводить "рядка"
}
```

У цьому прикладі ми використовуємо функцію `get_substring`, яка приймає рядок, початковий та кінцевий індекси та повертає підрядок цього рядка, починаючи з заданого індексу та закінчуючи перед заданим кінцевим індексом.

## Глибоке погруження
Щоб краще зрозуміти, як працює екстракція підрядків в мові Rust, давайте розглянемо більш складний приклад. Нехай у нас є рядок "Привіт, це ще один приклад рядка." Якщо ми застосуємо функцію `get_substring` із початковим індексом `6` та кінцевим індексом `21`, результатом буде "це ще один".

У рядках мови Rust, індекси починаються з нуля, тому позиція букви "ц" буде 6, а позиція букви "о" буде 21 (необхідно врахувати пробіли).
