## Що та навіщо?

Роздрук дебагу - простий та ефективний спосіб відстеження стану програми під час її виконання. Програмісти використовують його для з'ясування, де і як саме в прогорамі виникли помилки.

## Як це зробити:

Щоб вивести значення на консоль для налагодження, використовуйте 'Debug' трейт. 
```Rust
#[derive(Debug)]
struct Test{
    field1: i32,
    field2: String,
}

fn main() {
    let instance = Test {
        field1: 32,
        field2: "Hello!".to_string(),
    };
    // Виведення налагоджувальної інформації
    println!("{:?}", instance);
}
```
Цей код виведе на консоль: 
``` Rust
Test { field1: 32, field2: "Hello!" }
```

## Глибше занурення:

Дебаг роздрук використовувався з перших днів програмування. Він завжди був простим та ефективним шляхом з'ясувати, що відбувається в коді. 

З розвитком інструментів налагодження роздрук дебагу став частиною більш складних систем налаштувань з використанням засобів точок зупинки та перехоплення виключень.

Як альтернативу Rust пропонує значно більш деталізовану версію друку - формат `"{:#?}"`, який відображає дані в більш структурованому виді.

```Rust
println!("{:#?}", instance);
```
В результаті отримаємо більш переглядну структуру:
```Rust
Test {
    field1: 32,
    field2: "Hello!",
}
```
