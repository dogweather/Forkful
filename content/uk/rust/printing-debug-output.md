# Чому

Програмування - це складний і тонкий процес, де виникає безліч помилок і проблем. Щоб розібратися у них, необхідно використовувати відладочний вивід. Він дозволяє перевірити, чи правильно виконується код, який написаний із використанням Rust. А також допоможе знайти та виправити помилки у вашому коді.

# Як використовувати

Щоб вивести відладочну інформацію в Rust, можна використовувати макрос `println!()`, який дозволяє виводити значення змінних та повідомлення про їхній стан, за допомогою спеціальних позначок. Приклад виведення змінної `num` та повідомлення про її значення виглядає так:

```Rust
println!("Значення змінної num: {}", num);
```

Результатом буде виведення повідомлення "Значення змінної num: значення_змінної" у відладочний вивід.

# Глибше занурення

Окрім звичайних способів використання відладчика, ви також можете налаштувати додаткові параметри для макросу `println!()`. Наприклад, використовуючи флаг `e` для форматування, можна вивести повідомлення у вигляді звичайного коду на мові Rust:

```Rust
println!("Значення змінної num: {:?}", num);
```

Результатом буде виведення повідомлення "Значення змінної num: значення_змінної" у вигляді коду на мові Rust.
