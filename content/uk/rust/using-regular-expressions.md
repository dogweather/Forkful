## Чому

Regular expressions (регулярні вирази) - це могутній інструмент для роботи з текстом у програмуванні. Вони дозволяють здійснювати пошук і заміну певних символів або шаблонів у тексті. Завдяки цьому, ви можете ефективно обробляти великі текстові дані і швидко знаходити необхідну інформацію.

## Як використовувати

Коли ви працюєте з текстом у вашій Rust програмі, ви можете скористатися регулярними виразами, щоб зробити роботу з ним більш простою і ефективною. Для цього ви можете використовувати стандартну бібліотеку Rust - Regex (регулярний вираз), яка надає багато корисних функцій для роботи з регулярними виразами.

Давайте розглянемо приклад, де ми хочемо знайти всі слова, що починаються на літеру "а". Перш за все, нам потрібно імпортувати тулкіт Regex у нашу програму, за допомогою рядка ```use regex::Regex;```. Далі, ми можемо створити новий об'єкт Regex, передавши шаблон у форматі рядка в конструктор: ```let re = Regex::new(r"^a\w*").unwrap();``` Тут символ ```^``` вказує на початок слова, а ```\w``` означає будь-яку букву чи цифру. По суті, наше регулярне вираження відповідає всім словам, які починаються з літери "а". 

Тепер ми можемо використати метод ```captures()``` для знаходження всіх співпадінь у тексті, застосованому до нашого шаблону. Наприклад, ми можемо перебрати всі результати та вивести їх у консоль:

```
for cap in re.captures("алгоритм аабснт abc") {
    println!("{:?}", cap.unwrap().as_str());
}
```

Результат виконання цього коду буде таким: 

```
"алгоритм"
"аабснт"
```

## Глибоке занурення

Регулярні вирази можуть бути дуже корисними, але вони також можуть бути досить складними для розуміння. Якщо ви хочете швидше освоїти цей інструмент, ми радимо вам приділити час на вивчення загальних правил та практикуватися на регулярних виразах з різними текстовими даними.

Крім того, рекомендуємо придивитися до різних можливостей та функцій, які надає бібліотека Regex, таких як зворотні посилання, встановлення поверхнев
