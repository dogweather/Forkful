## Що і чому?

Регулярні вирази (regex) - це шаблони для пошуку та маніпулювання текстом. Програмісти використовують їх для ефективної обробки рядків, перевірки форматів даних, парсингу і заміни тексту.

## Як це зробити:

```Rust
use regex::Regex;

fn main() {
    let re = Regex::new(r"\b\w{5}\b").unwrap();
    let text = "Ось простий рядок з декількома словами.";

    for cap in re.find_iter(text) {
        println!("Знайдене слово: {}", cap.as_str());
    }
}

```

Вивід буде:

```
Знайдене слово: простий
Знайдене слово: рядок
```

## Поглиблений погляд

Регулярні вирази беруть початок в теорії автоматів 1950-х років. Альтернативи regex включають парсери та специфічні бібліотеки для синтаксичного аналізу. У Rust робота з регулярними виразами зазвичай відбувається через крейт `regex`, який використовує жадібні алгоритми для забезпечення високої продуктивності.
