## Чому

Створення тимчасових файлів є важливою частиною програмування на Rust. Це дозволяє зберегти тимчасові дані, які можуть бути необхідними для виконання програми.

## Як

Створення тимчасового файлу в Rust дуже просте. Використовуючи метод `std::fs::File::create()`, можна створити файл з самим тимчасовим ім'ям або вказати його власне. Приклад коду та вихідних даних наведено нижче:

```rust
use std::fs::File;
use std::io::prelude::*;

fn main() {
    // Створення тимчасового файлу з ім'ям "temp.txt"
    let mut file = File::create("temp.txt").expect("Не вдалося створити файл");

    // Запис тексту в файл
    file.write_all(b"Привіт, Україно!").expect("Не вдалося записати в файл");
}
```

Тепер, якщо відкрити файл "temp.txt", то буде знайдено текст "Привіт, Україно!". Також можна використати метод `write()` для запису окремих байтів.

## Глибше

Створення тимчасових файлів в Rust корисно, оскільки вони автоматично видаляються після закриття програми або при завершенні блоку коду. Також їх можна використовувати для писання тестів або для збереження тимчасових даних при обробці великих файлів.
