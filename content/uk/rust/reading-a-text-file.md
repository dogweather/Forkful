## Чому

Програмування - це процес, який потребує постійного вдосконалення та вивчення нових мов. І однією з таких мов є Rust. Ця мова програмування стає все більш популярною серед розробників, оскільки вона дозволяє писати ефективний та безпечний код. Одним з важливих елементів програмування є робота зі звичайними текстовими файлами. Тому, якщо ви хочете вдосконалити свої навички програмування на Rust, першим кроком буде вивчення процесу обробки текстових файлів.

## Як

Для початку, необхідно підключити стандартну бібліотеку Rust ```std::fs```, яка містить функції для роботи з файлами. Далі, за допомогою методу ```read_to_string```, ми можемо зчитати вміст текстового файлу і зберегти його у змінну. Зверніть увагу, що цей метод повертає ```Result``` тип, тому ми повинні обробляти можливі помилки зчитування файлу.

```Rust
use std::fs;

let file_content = fs::read_to_string("example.txt");

match file_content {
    Ok(content) => println!("File content: {}", content),
    Err(err) => println!("Error while reading file: {}", err),
}
```

У результаті, ми отримуємо вміст файлу в змінній ```file_content``` та виводимо його у консоль за допомогою методу ```println!```.

## Глибоке дослідження

Окрім основного методу ```read_to_string```, бібліотека ```std::fs``` також містить інші корисні функції для роботи з файлами. Наприклад, ```read``` - це метод, який читає вміст файлу у вигляді байтів. Це може бути корисно, якщо ви працюєте з бінарними файлами, а не з текстовими. Також, є методи для запису в файл та отримання інформації про файл, такі як розмір та час останньої модифікації.

## Див. також

Детальну інформацію про роботу з файлами у Rust ви можете знайти у [документації](https://doc.rust-lang.org/std/fs/). Також, радимо ознайомитися з іншими корисними функціями бібліотеки ```std```, які допоможуть у вашій подальшій роботі з мовою.
