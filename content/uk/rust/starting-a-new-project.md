## Що і чому?

Створення нового проекту - це всього лише запуск нової ідеї або програми у світ. Це роблять програмісти, щоб трансформувати свої концепції в реальний код та надавати корисні рішення для світу.

## Як це зробити:

Новий проект Rust можна створити за допомогою cargo, інструменту створення пакетів Rust. Нижче наведено приклад коду з вашою початковою конфігурацією.

```Rust
// Запустимо команду в терміналі:
cargo new my_project
```

Ви отримаєте виведення подібне до цього:

```Rust
Created binary (application) `my_project` package
```

## Занурення в деталі:

**1. Історичний контекст**

Cargo введено в Rust 0.6.0 у 2013 році. Він став ключовим фактором у високоорганізованій структурі проектів Rust.

**2. Альтернативи**

Ручне створення файлів та директорій можливе, проте це менш продуктивне, і хиби тут легко зробити.

**3. Деталі реалізації**

"cargo new" створює новий проект з файлом "Cargo.toml" (з конфігурацією пакета Rust) і директорією "src" з файлом "main.rs", вашим основним вхідним файлом.
