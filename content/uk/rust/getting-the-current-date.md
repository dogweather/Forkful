## Що це таке і навіщо?
Визначення поточної дати в програмуванні — це процес отримання інформації про поточний час і дату. Програмісти роблять це для логування, таймстемпів, функціоналу з нагадуваннями чи перевірки часових рамок завдань.

## Як робити:
```Rust
use chrono::{Local, Datelike};

fn main() {
    let now = Local::now();
    println!("Сьогоднішня дата: {}", now.format("%Y-%m-%d"));
}
```
Прикладний вивід:
```
Сьогоднішня дата: 2023-04-12
```

## Пірнання у глибину:
Rust використовує `chrono` крейт, бо стандартна бібліотека не має всебічного рішення для роботи з датою та часом. `chrono` надає багаті можливості для парсингу, форматування та обчислення дат і часів. Історично, мови програмування розвивали сильні функції для роботи з часом через його складність і велику роль в багатьох додатках. Як альтернативу `chrono`, можна використовувати бібліотеку `time`, але `chrono` залишається популярнішим варіантом у Rust спільноті через свою гнучкість і велику кількість функцій.
