## Why

У світі програмування існує багато різних функцій та методів, які дозволяють здійснювати різні дії з даними. Одна з таких дій - це конкатенація рядків. Це означає об'єднання двох або більше рядків в один рядок. На перший погляд це може здатися не такою важливою дією, але насправді, конкатенація рядків є незамінною частиною багатьох програм та алгоритмів. В цій статті ми розглянемо, як саме можна здійснити конкатенацію рядків в мові програмування C.

## How To

Для початку, давайте розглянемо приклад простого коду на мові C, який дозволяє здійснювати конкатенацію рядків:

```C
#include <stdio.h>

int main()
{
   char str1[50] = "Привіт";
   char str2[50] = " Україно!";
   char result[100];

   //здійснюємо конкатенацію за допомогою функції strcat()
   strcat(result, str1);
   strcat(result, str2);

   //виводимо результат
   printf(result);

   return 0;
}
```

Виходячи з нашого прикладу, можна помітити, що для конкатенації рядків необхідно мати два чи більше рядків, а також створити третій рядок, в який буде записаний результат конкатенації. Для здійснення самої конкатенації ми використовуємо функцію `strcat()`, яка додає один рядок до іншого.

Виходячи з прикладу вище, виведений результат буде наступним:

```C
Привіт Україно!
```
