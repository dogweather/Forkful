##Чому
Побудовання програмного забезпечення завжди вимагає детального аналізу та відлагоджування. Вивід даних для налагодження допомагає розробникам залишатися на топі проблем та швидко знаходити їх вирішення.

##Як
Для виводу налагоджувальної інформації використовується функція ```printf``` у мові програмування C. Для цього потрібно передати строку з текстом, який потрібно вивести, у якості першого аргументу, а змінні у вигляді решти аргументів. Наприклад:
```C
int num = 10;
printf("Значення num: %d", num);
```
В результаті в консолі з'явиться повідомлення "Значення num: 10".

##Глибоке дослідження
Часто вивід налагоджувальної інформації потрібно додавати в різних частинах програми для прослідковування змін у значеннях змінних. Можна також використовувати комбінацію функцій ```printf``` та ```scanf``` для виводу та вводу даних налагоджувальний цілях.
