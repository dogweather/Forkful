## Чому 

У цій статті ми розглянемо, як зробити першу букву в рядку великою за допомогою програмування на C. Це важлива операція, яка може бути корисною для багатьох проектів, де потрібно форматувати текст.

## Як зробити першу букву великою

Для початку нам необхідно створити змінну, яка буде зберігати наш рядок для форматування. Для цього ми можемо використати тип даних `char`, оскільки нам потрібно буде обробити один символ за раз. Ось приклад коду, який створює змінну та друкує на екрані початковий рядок:

```C
char str[] = "приклад рядка";
printf("Рядок до форматування: %s\n", str);
```
Вихідний результат буде виглядати так:
```
Рядок до форматування: приклад рядка
```
Тепер ми можемо перехід до самого форматування рядка. Щоб першу букву перетворити у велику, нам потрібно зробити наступне:
1. Перевірити, чи перша буква рядка є маленькою.
2. Якщо так, то замінити її на велику літеру за допомогою функції `toupper`.
3. Надрукувати отриманий результат.
Ось як це можна зробити у коді:

```C
if (islower(str[0])) {
    str[0] = toupper(str[0]);
    printf("Результат форматування: %s\n", str);
}
```
Вихідний результат:
```
Результат форматування: Приклад рядка
```

## Deep Dive 

Під капотом функції `toupper` використовується стандартна бібліотека `ctype.h`, яка надає нам інші функції для роботи з символами. Наприклад, функція `islower` перевіряє, чи є символ маленькою літерою. Це важливо знати, оскільки зазвичай ASCII таблиця має різні коди для маленьких та великих літер. Тому перед тим, як замінити першу букву на велику, ми маємо перевірити, чи дійсно вона є маленькою. 

Існує також інший підхід до форматування рядка - використання функції `strlwr`, яка перетворює всі символи рядка у маленькі. У цьому випадку нам потрібно буде зробити додаткові кроки, щоб знову перетворити першу букву на велику. Такий підхід може бути корисним, якщо ми хочемо форматувати не тільки першу букву, а всі слова
