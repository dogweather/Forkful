## Чому

Конвертування дати в рядок є важливою задачею при роботі з даними та функціями дат у програмуванні. Він дозволяє спростити відображення дати у зрозумілому форматі та зробити її більш зрозумілою для користувачів програми. Також, конвертування дати може бути необхідним при роботі з базами даних та збереженні даних у файл.

## Як це зробити?

Існує кілька способів виконати конвертування дати в рядок в мові програмування C. Один з найпоширеніших способів - використання стандартної функції `strftime()`, яка дозволяє вказати необхідний формат виводу дати у рядок. Використовуючи цей спосіб, код буде виглядати наступним чином:

```C
#include <stdio.h>
#include <time.h>

int main() {
    time_t now = time(NULL); //отримуємо поточний час
    struct tm *local = localtime(&now); // конвертуємо у локальний час
    char output[30]; //ініціалізація масиву для збереження рядка
    strftime(output, 30, "%d.%m.%Y", local); //конвертуємо дату у зрозумілий формат
    printf("Поточна дата: %s", output); //виводимо результат
    return 0;
}
```

Вихідним результатом буде: `Поточна дата: 31.12.2021`.

## Занурення в тему

У мові C існує також структура `tm`, яка призначена для збереження інформації про час і дату. Ця структура використовується під час конвертування дати в рядок за допомогою функції `strftime()`. Для отримання більш детальної інформації про цю структуру та функцію ви можете ознайомитися з документацією мови C.
