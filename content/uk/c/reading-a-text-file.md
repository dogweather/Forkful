# Чому

У вас є завдання прочитати текстовий файл в програмі на мові C, але ви не знаєте, як це зробити? Чи це важке завдання? Якщо ви подумали про це, то ця стаття для вас! Тут ми розкажемо про те, чому важливо вміти читати текстові файли та як це зробити.

# Як це зробити

Почнемо з того, як зчитати текстовий файл в програмі на мові C. Для цього нам знадобиться використати стандартну бібліотеку `stdio.h` та функції `fopen`, `fscanf` та `fclose`. Ось приклад коду:

```C
#include <stdio.h>

int main() {
    FILE *file;
    char buffer[255];

    // Відкриття файлу для читання
    file = fopen("file.txt", "r");

    // Зчитування рядка з файлу
    fscanf(file, "%s", buffer);

    // Виведення зчитаного рядка
    printf("%s", buffer);

    // Закриття файлу
    fclose(file);

    return 0;
}
```

Вхідні дані у файлі мають бути у форматі, який ми вказали у функції `fscanf`, у нашому випадку це `%s` для рядків. Якщо у файлі знаходиться більше ніж 255 символів - це не проблема, ми можемо збільшити розмір масиву `buffer`. Також не забудьте перевіряти наявність помилок при відкритті та читанні файлу, щоб програма не зламалась у разі, якщо файл не існує або містить некоректні дані.

# Глибоке погруження

Якщо вам потрібно більше інформації про читання текстових файлів у програмі на мові C, рекомендуємо ознайомитись з офіційною документацією стандартної бібліотеки `stdio.h`. Ви також можете перевірити різні функції для зчитування файлів, наприклад `fgets`, яка дозволяє зчитати цілий рядок з файлу. Також не забувайте про роботу з файлами у режимі запису `w`, яка дозволяє зберегти зчитані дані у новий файл.
