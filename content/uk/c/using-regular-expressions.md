## Чому

Регулярні вирази є потужним інструментом для роботи зі строками в С програмуванні. Вони дозволяють здійснювати широкий спектр операцій зі строками, таких як пошук, заміна та перевірка правильності форматування. Використання регулярних виразів допомагає зменшити кількість коду, необхідного для обробки строк, і робить програми більш ефективними та змінними.

## Як

Для використання регулярних виразів у С потрібно підключити бібліотеку <regex.h>. Далі, ви можете скористатися функціями, такими як regcomp () та regexec (), для компіляції та виконання регулярних виразів відповідно. Наприклад, давайте спробуємо знайти всі слова, які починаються на "пр" у строковому масиві:

```C
#include <stdio.h>
#include <regex.h>

int main()
{
    regex_t regex;
    char pattern[] = "^пр.*";
    char str[] = ["прикладної математики", "програмування", "психологія"];
    int i;
    for(i=0; i<3; i++)
    {
        if(regcomp(&regex, pattern, REG_EXTENDED|REG_NOSUB) != 0)
        {
            printf("Неправильний регулярний вираз\n");
            return 1;
        }
        if(regexec(&regex, str[i], 0, NULL, 0) == 0)
        {
            printf("%s", str[i]);
        }
        regfree(&regex);
    }
    return 0;
}

//Вивід: програмування
```

## Глибокий розгляд

Регулярні вирази мають досить складний синтаксис, але добре розуміння його може значно полегшити обробку строк. Сутність регулярних виразів полягає в тому, що вони дозволяють шукати в певному тексті всі збіги з певним шаблоном. Наприклад, "^пр.*" буде збігатися з будь-яким текстом, що починається на "пр". Є також багато різноманітних метасимволів, таких як "*", "+", "?", тощо, які дозволяють здійснювати більш точні пошуки.
