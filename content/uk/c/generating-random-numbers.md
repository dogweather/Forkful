## Що і навіщо?

Генерація випадкових чисел - це процес створення числових значень без очевидного шаблону. Розробники часто використовують це для встановлення випадкових значень, емуляції випадкових подій, створення унікальних ідентифікаторів тощо.

## Як це зробити:

```C
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main () {
   unsigned int random_number;
   srand(time(0));
   random_number = rand();
   printf("Випадкове число: %u\n", random_number);
   return 0;
}
```

У зразку вище ми використовуємо функції `rand()` та `srand()` де аргумент для `srand()` є постійно змінюється, тому кожна нова програма генерує випадкове число.

## Докладно:

Генерація випадкових чисел в програмуванні істує ще з часів, коли комп'ютери були аналоговими машинами. Різні методи включають випадковий ввід заснований на користувацькій активності (наприклад, швидкість натискання клавіш), алгоритми на основі зовнішньої фізичної активності (наприклад, шумовое живлення) та багато інших. 

Є також псевдовипадкові генератори чисел, які використовуються в більшості сучасних систем, таких як C stdlib's `rand()`. Ці генератори створюють числа, які виглядають випадковими і мають рівномірний розподіл, але вони є детермінованими і можуть відтворити однакову послідовність чисел з однаковим початковим значенням.
