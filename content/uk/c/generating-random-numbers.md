## Чому

Генерація випадкових чисел є важливою частиною багатьох програм і ігор. Такі числа використовуються для створення унікальних ідентифікаторів, шифрування даних і створення випадкового ефекту в іграх. Tому, знання як згенерувати випадкові числа є корисним для будь-якого досвідченого програміста.

## Як

Існує багато способів генерування випадкових чисел в мові програмування C. Один з найпростіших способів - використання функції `rand()` з бібліотеки `stdlib.h`. Нижче наведений приклад коду, який генерує 10 випадкових цілих чисел в діапазоні від 1 до 100 і друкує їх на екран:

```C
#include <stdio.h>
#include <stdlib.h>

int main()
{
    int i, num;
    
    for (i = 0; i < 10; i++)
    {
        num = rand() % 100 + 1;
        printf("%d\n", num);
    }
    
    return 0;
}
```

Вихідним результатом будуть 10 різних цілих чисел від 1 до 100, наприклад:

```
64
3
98
22
51
75
10
45
89
27
```

## Глибокий занурений

У мові програмування C випадкові числа генеруються за допомогою алгоритмів, які використовують математичні формули і деякі початкові значення, відомі як "насіння" (seed). Більш складні алгоритми можуть забезпечити більш випадкові числа, використовуючи тривіальну формулу, необхідно забезпечити більша кількість насіння. 

Помітно, що ця проста генерація випадкових чисел не є повністю випадковою, оскільки кожного разу, коли виконується програма, буде одне і те ж саме випадкове число. Щоб уникнути цього, можна використовувати системний час комп'ютера як насіння для генерування випадкових чисел.

Існує також більш розширений спосіб генерування випадкових чисел - за допомогою криптографічно безпечного генератора. Цей метод забезпечує випадкові числа, які неможливо передбачити і використовувати для криптографічних цілей.
