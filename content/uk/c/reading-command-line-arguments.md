##Чому

Здібності роботи з командним рядком є важливою частиною процесу розробки програмного забезпечення на мові С. Дізнання, як читати аргументи командного рядка може покращити ефективність вашої програми та забезпечити більш гнучкий підхід до взаємодії з користувачем.

##Як

Для того, щоб прочитати аргументи командного рядка в мові С, ви можете скористатися функцією `main()` та подачею двох параметрів: `argc` та `argv`. Давайте розглянемо приклад:

```C
#include <stdio.h>

int main(int argc, char *argv[]) {
    int i;

    printf("Кількість аргументів командного рядка: %d\n", argc);

    for (i = 0; i < argc; i++) {
        printf("Аргумент %d: %s\n", i, argv[i]);
    }

    return 0;
}
```

У цьому прикладі ми використали цикл `for` для виведення кількості аргументів та кожного аргументу окремо. Запустіть цей код, передавши йому якісь аргументи командного рядка, і ви можете побачити результат. Наприклад, якщо запустити програму з аргументами "Hello" і "World", ви побачите наступне:

```
Кількість аргументів командного рядка: 3
Аргумент 0: ./a.out
Аргумент 1: Hello
Аргумент 2: World
```

##Глибоке пікання

Розбираючись з читанням аргументів командного рядка, важливо враховувати декілька особливостей. Перш за все, зверніть увагу, що перший аргумент командного рядка завжди буде шляхом до виконуваного файла. Крім того, також важливо дотримуватися типів даних для переданих аргументів. Наприклад, якщо аргументом є рядок, ви можете використовувати функцію `strcmp()` для порівняння його з іншим рядком.
