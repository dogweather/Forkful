З глибоким зацікавленням багато хто з нас розглядає кількість різних функцій у мові програмування C. Одна з них - це вимірювання підстрок. Це корисний інструмент, який може бути застосований в багатьох сценаріях програмування.

## Чому

Найчастіше, ми потребуємо вирізати частину тексту з великого рядка або масиву символів. Наприклад, можливо, нам потрібно виокремити ім'я конкретного користувача з URL-адреси або отримати певну підстроку зі строки, яка описує дату. У таких ситуаціях нам потрібно використовувати функції для вимірювання підстрок. 

## Як це зробити

Для початку, нам потрібно познайомитися з декількома функціями, які допомагають витягнути підстроку з рядка. Одна з них - це функція "strncpy", яка копіює n символів з однієї строки в іншу. Для прикладу, давайте зробимо так, щоб зрізати перші 5 символів зі строки 'Hello World':

```C
char str1[] = "Hello World";
char str2[6];
strncpy(str2, str1, 5);
str2[5] = '\0';
printf("%s", str2);
```

У нашому випадку, вихідною підстрокою буде 'Hello'.

## Глибоке погруження

Маючи деяке розуміння про функції вимірювання підстрок, давайте розглянемо деякі практичні приклади, як їх можна використовувати. Наприклад, можливо, ми хочемо перевірити чи задана стрічка починається з певної підстроки. Можемо використати функцію "strncmp", яка порівнює n символів двох строк і повертає 0, якщо вони рівні. Наприклад:

```C
char str1[] = "Hello World";
char str2[6];
strncpy(str2, str1, 5);
str2[5] = '\0';
if (strncmp(str1, str2, 5) == 0) {
    printf("Строка починається з 'Hello'");
}
```

Також, важливо пам'ятати, що існують багато інших функцій вимірювання підстрок, які можуть бути використані для різних завдань, таких як пошук індексу підстроки у рядку або заміна одного рядка іншим.
