## Чому

У сучасному світі програмування, порахувати точну дату та час є необхідністю для багатьох програм. Це може бути важливо для подачі заявок, виконання певних завдань або просто для відображення поточної дати для користувача. Навіть якщо ви просто цікавитесь створенням програм, вивчення отримання поточної дати є корисним навичкою.

## Як зробити 

Отримання поточної дати в C-мові може здатися тривіальною задачею, але насправді воно вимагає деяких знань та вмінь. Наприклад, є кілька стандартних функцій, які допоможуть вам отримати дату, такі як `time ()` для отримання епохи часу та `localtime ()` для перетворення епохи у структуру `tm`, яка містить детальну інформацію про дату та час.

Ось приклад коду, який забезпечить вам отримання поточної дати та часу:

```C
#include <stdio.h>
#include <time.h>

int main(void)
{
    // отримуємо епоху часу
    time_t current_time = time(NULL);
    
    // конвертуємо епоху у структуру tm
    struct tm * tm_ptr = localtime(&current_time);
    
    // отримуємо детальну інформацію про дату та час
    int year = tm_ptr->tm_year + 1900; // поточний рік
    int month = tm_ptr->tm_mon + 1; // поточний місяць
    int day = tm_ptr->tm_mday; // поточний день
    int hour = tm_ptr->tm_hour; // поточна година
    int min = tm_ptr->tm_min; // поточна хвилина
    int sec = tm_ptr->tm_sec; // поточна секунда
    
    // виводимо поточну дату та час
    printf("Поточна дата і час: %d/%d/%d %d:%d:%d", day, month, year, hour, min, sec);
    
    return 0;
}
```

Результат виконання цього коду буде відображений так:

`Поточна дата і час: 26/03/2021 10:15:35`

## Поглиблене вивчення

У C-мові, дата та час представлені у вигляді епохи, що є кількістю секунд, що пройшли з 1 січня 1970 року. Це називається Unix Epoch і це широко використовується в різних операційних системах та програмах.
