## Чому

У програмуванні часто виникає потреба порівнювати дві дати. Це може бути корисно, наприклад, для перевірки чи умови виконання коду стосуються певного періоду часу. У таких випадках користувач може застосувати функцію порівняння дат.

## Як

Для порівняння дат використовується функція `difftime()`, яка включена в стандартну бібліотеку `time.h`. Ця функція приймає два параметри - дві дати, які потрібно порівняти. Давайте переглянемо приклад коду, який порівнює дві дати та виводить результат:

```C
#include <stdio.h>
#include <time.h>

int main() {
    // створення структур для збереження дат
    struct tm date_1, date_2;
    
    // заповнення даних для першої дати (24 вересня 2021)
    date_1.tm_year = 121; // 2021 - 1900
    date_1.tm_mon = 8;    // вересень, 8 - 1 (починаємо з 0)
    date_1.tm_mday = 24;  // 24-е число
    
    // заповнення даних для другої дати (1 жовтня 2021)
    date_2.tm_year = 121; // 2021 - 1900
    date_2.tm_mon = 9;    // жовтень, 9 - 1 (починаємо з 0)
    date_2.tm_mday = 1;   // 1-е число
    
    // порівняння дат та збереження результату у змінній diff
    double diff = difftime(mktime(&date_2), mktime(&date_1));
    
    // вивід результату
    printf("Різниця між датами у секундах: %f", diff);
    
    return 0;
}
```

У цьому прикладі ми використовуємо функцію `mktime()` для перетворення структури дати в стандартну форму `time_t`, яку можна використовувати для порівняння. Результатом буде число, яке відповідає різниці між датами у секундах.

## Глибока погляд

Якщо результуюче число дуже велике, значення дат були введені неправильно або між ними дуже велика різниця. Також варто пам'ятати, що функція `difftime()` не враховує перехід на зимовий/літній час, тому результат може бути неточним у таких випадках.
