## Що і чому?
Конвертація рядка в малі літери - це процес, коли всі букви в рядку змінюються на малі літери. Програмісти роблять це для єднання формату, зручності порівняння рядків та уникнення неточностей.

## Як це робити:
Для конвертації рядків у C, існує функція `tolower()`. Ця функція бере одну букву як вхід та повертає її в нижньому регістрі. Використання цього в циклі, ви можете конвертувати весь рядок.

```C
#include <ctype.h>
#include <stdio.h>

void Lowercase(char str[]){  
    int i = 0;
    while (str[i]){
        str[i] = tolower(str[i]);
        i++;
    }
}

int main(){
    char str[] = "HELLO world!";
    Lowercase(str);
    printf("%s\n", str); 
    return 0;
}
```
Виведення:
```C
hello world!
```

## Поглиблений розгляд
1. **Історичний контекст:** В початкових версіях C, не було стандартної функції для конвертації рядків у нижній регістр. Розробники використовували свої методи, які інколи створювали проблеми з сумісністю.

2. **Альтернативи:** Є методи для ручного конвертування рядків у нижній регістр, але вони зазвичай менш надійні та ефективні.

3. **Деталі реалізації:** Функція `tolower()` перевіряє, чи є вхідний символ в верхньому регістрі. Якщо так, вона просто повертає цей символ з посунутим значенням ASCII, що відповідає малому символу.
