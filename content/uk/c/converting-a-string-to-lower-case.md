##Чому

Робота з рядками є невід'ємною частиною багатьох програм у мові С. Часто потрібно змінити рядок на нижній регістр для подальшої обробки. Завдяки цьому ми можемо виконувати пошук, порівняння та інші маніпуляції зі словами в тексті. 

##Як 

Для перетворення рядка на нижній регістр у мові С ми використовуємо функцію `tolower()`. Приклад реалізації можна побачити нижче:

``` C
#include<stdio.h>
#include<ctype.h>

int main() {
	char str[20] = "СЛОВАРЬ";
	int i;
	
	for(i = 0; str[i]; i++){
		str[i] = tolower(str[i]);
	}
	
	printf("%s", str);
    
    return 0;
}
```

##Глибокий аналіз

У мові С символи записуються у вигляді числового значення, використовуючи таблицю ASCII. Верхній регістр латинських літер відповідає значенням від `65` до `90`, а нижній - від `97` до `122`. Функція `tolower()` переводить літеру у нижній регістр, змінюючи числове значення символу. Таким чином, для всіх великих літер відбувається перетворення на малі.
