## Що і чому?
Інтерполяція рядків — це вставка значень змінних безпосередньо в рядок. Робиться це для спрощення створення та редагування рядків, що містять дані.

## Як це зробити:
Ось приклад інтерполяції рядків в мові програмування С.

```C
#include <stdio.h>

int main()
{
   int age = 22;
   printf("I am %d years old.\n", age);

   return 0;
}
```
Після виконання програми ви побачите: 
```C
I am 22 years old.
```

## В глибину:
1. Історичний контекст: Інтерполяція рядків вживалася ще з часів мови програмування Perl, але в С вона стала особливо популярною завдяки простоті та зручності використання.
2. Альтернативи: В С ви також можете скористатися функцією sprintf для реалізації інтерполяції рядків, але її використання мало б ефективне, якщо вам потрібно працювати лише з одним рядком.
3. Деталі виконання: Функція printf в С використовує спеціальні плейсхолдери (наприклад, %d для цілих чисел), і вставляє значення змінних в рядок у відповідних місцях.
