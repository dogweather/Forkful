## Що це і навіщо?
Знаходження довжини рядка - це процес визначення кількості символів у конкретному рядку. Програмісти роблять це, щоб оптимізувати роботу з рядками і підвищити ефективність програм.

## Як це робити:
Використовуючи функцію `strlen()` в мові програмування C, ви можете легко визначити довжину рядка. Ось приклад коду:

```C
#include <stdio.h>
#include <string.h>

int main() {
    char str[] = "Привіт, світ!";
    int len = strlen(str);
    printf("Довжина рядка: %d\n", len);
    return 0;
}
```
Виходь: `Довжина рядка: 13`

## Поглиблений матеріал
Цей метод визначення довжини рядка був запроваджений в мові С з самого його початку. Його можна замінити вручну реалізованим кодом, але використання вбудованої функції `strlen()` є найефективнішим рішенням. Функція працює шляхом підрахунку кількості символів, доки не дійде до NULL-термінатора.
