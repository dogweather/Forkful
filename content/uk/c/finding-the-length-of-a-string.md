## Чому

Якщо ви практикуєте програмування, ви ознайомлені зі строками. Строки використовуються для збереження текстової інформації та мають важливе застосування в багатьох програмах. Однією з найпоширеніших задач з роботою зі строками є знаходження їх довжини. У цій статті ми розглянемо причини, чому так важливо вміти визначати довжину стрічки в програмах на мові C.

## Як це зробити

У мові C існує функція `strlen()`, яка дозволяє знаходити довжину строки. Для того, щоб скористатися нею, потрібно передати у неї в якості аргументу вказівник на початок строки. Наприклад:

```C
char str[] = "Hello World!";
int length = strlen(str);
printf("Довжина строки - %d", length);
```

У цьому прикладі ми змінній `length` присвоїли значення, отримане з функції `strlen()`. На виводі ми отримаємо `Довжина строки - 12`, оскільки у нашій стрічці 12 символів.

Ще один спосіб визначити довжину строки - це використовувати цикл `while`. У цьому випадку, ми будемо перевіряти кожен символ стрічки, доки не досягнемо символу кінця строки `\0`. Наприклад:

```C
char text[] = "Programming";
int i = 0;
while (text[i] != '\0') {
    i++;
}
printf("Довжина строки - %d", i);
```

На цей раз ми отримаємо на виводі `Довжина строки - 11`. Навіть якщо приведений приклад здається складним, варто зрозуміти, що `strlen()` фактично виконує ту саму роботу, але вона є більш ефективною і коротшою для написання.

## Детальний аналіз

Тепер давайте трохи краще розберемося з процесом визначення довжини строки. У C-програмі, строки зберігаються як масиви символів, тому для того, щоб отримати довжину строки, ми повинні вирахувати кількість символів в масиві до зустрічі символу `\0`, який вказує на кінець строки.

Функція `strlen()` просто перевіряє кожен символ масиву, поки не знайде символ `\0`, і повертає кількість пройдених символів.

Використання циклу `while` для визначення довжини строки займає більше часу та ресурсів, адже потрібно перевіряти кожен символ до знаходження символу `\0`.
