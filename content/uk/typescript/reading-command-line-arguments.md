## Чому

Читання аргументів командного рядка є важливим та корисним навичкою у програмуванні на TypeScript. Використання аргументів командного рядка дозволяє вам передати параметри в свою програму під час її запуску, що дозволяє змінювати поведінку програми без необхідності внесення змін у сам код.

## Як

У TypeScript є можливість отримувати та обробляти аргументи командного рядка за допомогою спеціальної змінної `process.argv`. Ця змінна містить масив зі всіма аргументами, які були передані при запуску програми. Наприклад:

```TypeScript
const args: string[] = process.argv;
console.log(args);
```

В результаті виконання цього коду буде виведено масив з аргументами командного рядка, які були передані при запуску програми.

```
["node", "index.ts", "arg1", "arg2", "arg3"]
```

У цьому прикладі аргументи `arg1`, `arg2` та `arg3` були передані безпосередньо після імені файлу з програмою. Для кращого розуміння, давайте розглянемо приклад коду для обробки цих аргументів:

```TypeScript
const args: string[] = process.argv;
const arg1: string = args[2];
const arg2: string = args[3];
const arg3: string = args[4];
console.log(`Argument 1: ${arg1}`);
console.log(`Argument 2: ${arg2}`);
console.log(`Argument 3: ${arg3}`);
```

В результаті виконання цього коду буде виведено:

```
Argument 1: arg1
Argument 2: arg2
Argument 3: arg3
```

Також важливо зазначити, що аргументи командного рядка передаються як рядки, тому може знадобитися перетворення типів даних, якщо ви плануєте використовувати їх у своєму коді.

## Детальний розгляд

Зазначений метод для читання аргументів командного рядка досить простий і добре підходить для більшості випадків. Проте, іноді може бути необхідно більш гнучким чином обробляти аргументи та їх значення. У таких випадках, ви можете використовувати бібліотеку `commander`, яка забезпечує елегантний та зручний інтерфейс для обробки аргументів командного рядка.

Наприклад, використовуючи `commander`, ви можете створити детальнішу список аргументів та їх значень, які програма очікує. Крім того, бібліотека дозволяє додавати параметри за замовчуванням та валідувати вхідні значення аргументів.
