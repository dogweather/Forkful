## Почему

Удаление символов, соответствующих шаблону, может быть полезным, если вы хотите очистить входные данные от ненужных или неверных символов, чтобы можно было обработать их дальше.

## Как это сделать

Для удаления символов, соответствующих шаблону, мы можем использовать метод `replace()` вместе с регулярным выражением.

```TypeScript
const input = "1a2b3c";
const output = input.replace(/[abc]/g, ""); 
console.log(output); // выводит "123"
```

Здесь мы создаем переменную с исходными данными и затем используем метод `replace()` с регулярным выражением `[abc]`, которое означает удаление любого символа `a`, `b` или `c`. Мы также добавляем флаг `g`, чтобы удалить все совпадения, а не только первое.

Если нам нужно удалить все символы, кроме указанных в шаблоне, мы можем добавить символ `^` перед шаблоном. Например:

```TypeScript
const input = "1a2b3c";
const output = input.replace(/[^abc]/g, ""); 
console.log(output); // выводит "abc"
```

Мы также можем использовать метасимволы, такие как `\d` для удаления всех цифр, или `\w` для удаления всех букв и цифр.

## Глубокий погружение

Регулярные выражения - это мощный инструмент, который может помочь нам обрабатывать и очищать данные. Мы можем использовать более сложные шаблоны, чтобы заменить или удалить несколько символов за один раз.

Например, мы можем использовать квантификаторы, такие как `*` или `+`, чтобы указать, что мы хотим удалить все повторяющиеся символы.

```TypeScript
const input = "11a22b33c";
const output = input.replace(/[a-z]+/g, ""); 
console.log(output); // выводит "112233"
```

Используя регулярные выражения, мы также можем проверять наличие символов в строке и даже заменять их на другие значения. Например, мы можем заменить все гласные на символ `*`.

```TypeScript
const input = "hello world";
const output = input.replace(/[aeiou]/g, "*"); 
console.log(output); // выводит "h*ll* w*rld"
```

Также стоит отметить, что в TypeScript мы можем использовать тип `string` для регулярных выражений и указывать флаги (например, `i` для игнорирования регистра) после закрытия шаблона.
