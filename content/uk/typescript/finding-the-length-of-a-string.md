## Чому

Написання програмного забезпечення є складною, але захоплюючою задачею, яка потребує розуміння фундаментальних концепцій та вміння вирішувати завдання за допомогою коду. Одним з таких ключових аспектів є визначення довжини рядка в мові програмування TypeScript. Це дозволяє нам обробляти та маніпулювати рядками даних в нашій програмі. У цьому блозі ви дізнаєтеся, як і чому важливо знати, як визначити довжину рядка в TypeScript.

## Як

Для того, щоб отримати довжину рядка в TypeScript, ми можемо скористатися вбудованою функцією `length`, яка повертає кількість символів у рядку.

```TypeScript
// Приклад коду для визначення довжини рядка
let str: string = "Привіт, світе!";
console.log(str.length); // Виведе 13, оскільки рядок містить 13 символів
```

Також, ми можемо використовувати цю функцію для визначення довжини будь-якого типу даних, який наслідує клас `String`, такий як масив символів.

```TypeScript
// Приклад коду для визначення довжини масиву символів
let arr: string[] = ["Hello", "world"];
console.log(arr.join("").length); // Виведе 10, оскільки рядок "Hello" має 5 символів, а "world" 5 символів, тож разом вони становлять 10
```

## Вглиб (Deep Dive)

Для того, щоб краще зрозуміти, як працює функція `length`, давайте розглянемо її ближче. Вона є методом класу `String`, що робить її доступною для використання на будь-якій змінній, яка має тип `string`. Цей метод просто повертає кількість символів у рядку, не враховуючи пробіли та інші спеціальні символи.

Наприклад, якщо ми оголосимо змінну `hello` з рядком "Hello, world!", то вона буде мати довжину 13. Але якщо розподілити цей рядок на дві змінні, наприклад `hello` та `world`, і злити їх, результатом буде рядок "Hello world!", який вже матиме довжину 11. Це тому, що метод `length` не враховує пробіли та коми у рядку.
