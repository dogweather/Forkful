## Чому
Багато програмістів часто розпочинають новий проект, щоб вдосконалювати свої навички, знайомитися з новими технологіями або просто для розваги.

## Як почати
Щоб розпочати новий проект на TypeScript, спочатку потрібно встановити його. Для цього використовується пакетний менеджер npm, який дозволяє легко встановлювати та оновлювати пакети. Нижче наведений приклад для встановлення TypeScript:

```TypeScript 
npm install -g typescript
```

Після встановлення TypeScript, необхідно створити нову папку для проекту та перейти в неї. Далі, створюємо файл з розширенням .ts та вказуємо початковий код програми:

```TypeScript
console.log("Привіт, світе!");
```

Тепер за допомогою терміналу можна запустити нашу програму:

```TypeScript
tsc <файл з кодом>
```

Це перетворить наш код на JavaScript та створить новий файл з розширенням .js. Для запуску програми використовуємо команду:

```TypeScript
node <файл з кодом>
```

## Глибока занурення
Для того, щоб глибше дослідити TypeScript, можна використовувати його документацію та приклади з Інтернету. Також, можна досліджувати інші пакети та бібліотеки, які підтримують TypeScript, такі як React або Angular.

Щоб полегшити собі роботу з TypeScript, рекомендується використовувати редактори коду, які підтримують його, наприклад Visual Studio Code.
