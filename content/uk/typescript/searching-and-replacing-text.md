## Чому

Можливо, ви вже зіткнулися з ситуацією, коли потрібно знайти і замінити певний текст у своєму програмному коді. Це може бути трудомістким і часом нудним процесом. Однак, за допомогою TypeScript ви можете легко виконати пошук і заміну тексту з декількох рядків коду.

## Як це зробити

Для пошуку і заміни тексту використовується вбудований метод `replace`, який доступний для рядків у TypeScript. Цей метод приймає два параметри - рядок, який потрібно замінити, і рядок, на який буде замінено. Ось приклад коду:

```TypeScript
let text = "Привіт, світ!";
text = text.replace("світ", "Україна");
console.log(text); //Виведе "Привіт, Україна!"
```

У цьому прикладі ми створили рядок `text` і використали метод `replace`, щоб замінити слово "світ" на "Україна". Зверніть увагу, що ми переназначили значення `text` після виконання методу `replace`.

Щоб замінити всі входження певного тексту, можна використати регулярний вираз у другому параметрі методу `replace`, наприклад:

```TypeScript
let text = "Тут багато багато букв a";
text = text.replace(/a/g, "e");
console.log(text); //Виведе "Тут бегете бегете букв e"
```

Для більш складних сценаріїв пошуку і заміни можна використати функцію зворотного виклику (callback function). Ось приклад:

```TypeScript
let text = "Скоро в Україні будуть свята!";
text = text.replace(/(Україні)(.*)(свята)/, (match, p1, p2, p3) => {
  return `${p3} в ${p1} будуть ${p2}`;
});
console.log(text); //Виведе "Свята в Україні будуть скоро!"
```

У цьому прикладі ми використовуємо регулярний вираз, щоб знайти слова "Україні" і "свята" у рядку `text`. У функції зворотного виклику ми використовуємо групи регулярного виразу, щоб перетворити порядок слів і підставити рядок "скоро" між ними.
