## Чому

Генерація випадкових чисел є важливою частиною програмування, оскільки воно дозволяє створювати випадкові значення для різних сценаріїв, таких як тестування програм, розробка ігор або створення унікальних ідентифікаторів. Це також може бути корисним для створення рандомного елементу в дизайні веб-сайтів.

## Як

Для початку, імпортуємо готовий модуль для генерації випадкових чисел у TypeScript:
```TypeScript
import { random } from "lodash";
```

Щоб згенерувати випадкове ціле число від 1 до 10, можна використовувати наступний код:
```TypeScript
let randomNumber = random(1, 10);
console.log(randomNumber); // виведе випадкове число від 1 до 10
```

Або можна використовувати метод ```random``` без параметрів, який буде генерувати випадкове число від 0 до 1:
```TypeScript
let randomNumber = random(); 
console.log(randomNumber); // виведе випадкове число від 0 до 1
```

Можна також використовувати готовий модуль Node.js ```crypto``` для генерації випадкових чисел:
```TypeScript
import { randomBytes } from "crypto";
let randomNumber = randomBytes(4);
console.log(randomNumber); // виведе випадкове число у форматі байтів
```

## Глибокий занурення

Генерація випадкових чисел базується на алгоритмах, які використовують певні початкові значення або ключі для створення "випадковості". Такі алгоритми можуть бути криптографічно безпечними або не безпечними, тому важливо використовувати належні методи і бібліотеки для генерації випадкових чисел залежно від конкретного сценарію.

Також важливо пам'ятати, що випадкові числа не є повністю випадковими і можуть повторюватися. Для створення більш унікальних ідентифікаторів, можна використовувати спеціальні бібліотеки для генерації UUID-ідентифікаторів.
