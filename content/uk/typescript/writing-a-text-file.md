## Зачем?

Написання текстового файлу є невід'ємною частиною програмування, оскільки це дозволяє зберігати дані та налаштовувати їх відповідно до потреб проекту.

## Як це зробити?

```TypeScript
// Створення потоку запису в файл
const fs = require('fs');
const file = fs.createWriteStream('sample.txt');

// Запис даних у файл
file.write('Привіт, Друзі!');

// Закриття потоку
file.end();

// Вивід успішного повідомлення
console.log('Дані успішно записані у файл!');
```

Результат у файлі sample.txt:

```
Привіт, Друзі!
```

## Глибоке вивчення

Написання текстового файлу може бути більш складним процесом, ніж здається на перший погляд. Уникаючи помилок та розпізнавання винятків, важливо приділяти увагу наступним аспектам:

- Використання правильного шляху до файлу (або його створення, якщо він не існує)
- Відкриття потоку для запису у буфері
- Закриття потоку після завершення запису
- Робота з даними: кодування та форматування
- Обробка помилок та винятків при збереженні файлу
