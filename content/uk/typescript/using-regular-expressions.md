## Чому

Застосування регулярних виразів може полегшити написання коду і допомогти вам зберегти час, шукаючи та замінюючи рядки в текстових документах. Вони є потужним інструментом для обробки інформації та виявлення відповідностей в тексті.

## Як це зробити

Найпростіший спосіб використання регулярних виразів у TypeScript - це використовувати вбудовану функцію `match ()`. Для прикладу, давайте подивимося, як ми можемо знайти усі слова, що починаються на "а" у тексті:

```TypeScript
let text = "Адам і Єва вирушили в Австралію";
let matches = text.match(/\bа\w+/g);
```

Результат буде містити масив, кожний елемент якого буде починатися на "а":

```TypeScript
["Адам", "Австралію"]
```

Замість того, щоб використовувати просто літеру, ми можемо також використовувати клас `\w`, який буде збігатися з будь-якою буквою. Також, застосувавши прапорець `g`, ми можемо знайти всі входження, а не тільки перше.

## Глибокий занурення

Регулярні вирази дають нам більше можливостей, ніж просто шукати текст, який відповідає певному шаблону. Наприклад, ми можемо використовувати функцію `replace ()`, щоб замінити певну частину тексту на щось інше.

```TypeScript
let text = "Привіт, Джейн! Як справи?";
let replacedText = text.replace(/Джейн/g, "Джон");
```

Результат буде наступним:

```TypeScript
"Привіт, Джон! Як справи?"
```

Ми також можемо використовувати регулярні вирази для перевірки правильності введених даних, для виявлення електронної пошти або номера телефону, або навіть для створення власних шаблонів для пошуку.
