## Чому

Існує багато ситуацій, коли нам потрібно вилучити певну частину тексту з рядка чи масиву. Наприклад, може бути необхідно отримати дату з формата дати і часу або виокремити колонку даних з таблиці. В таких випадках корисним може бути екстрагування підрядків для швидкого та ефективного розв'язання завдань.

## Як це зробити

Є декілька способів екстрагувати підрядки в TypeScript. Найпростіший спосіб - використання методу `slice()` або `substring()` для рядків і методу `slice()` для масивів. Приклади коду та вихідні дані показані нижче:

```TypeScript
// Вилучення підрядка з рядка
const text = "Це прикладній текст.";
const substring = text.slice(4, 12);
console.log(substring); // виводить "прикладній"

// Вилучення підрядка з масиву
const numbers = [1, 2, 3, 4, 5];
const subarray = numbers.slice(2, 4);
console.log(subarray); // виводить [3, 4]
```

Також можна використовувати регулярні вирази для екстрагування підрядків, використовуючи метод `match()`. Наприклад, якщо нам потрібно отримати всі слова з рядка, можемо скористатися наступним кодом:

```TypeScript
const text = "Це є простий рядок тексту";
const words = text.match(/\w+/g);
console.log(words); // виводить [ "Це", "є", "простий", "рядок", "тексту" ]
```

## Глибокий аналіз

Розглянемо детальніше методи `slice()` та `substring()` для рядків. Вони використовуються для отримання підрядків з рядка, при цьому враховуючи початковий та кінцевий індекси. Основною відмінністю між цими методами є те, що `slice()` може приймати від'ємні індекси, які відлічуються з кінця рядка. Наприклад, `-4` означає 4 символи до кінця рядка.

Також, метод `substring()` не підтримує від'ємні індекси, але він може автоматично коригувати неправильно вказані індекси, щоб початковий був меншим за кінцевий.
