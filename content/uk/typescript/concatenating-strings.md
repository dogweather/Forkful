## Чому

Конкатенація рядків - це важлива техніка програмування, яка дозволяє об'єднувати рядки в більш складні об'єкти. Вона дозволяє створювати більш динамічні інтерфейси та вражаючі візуальні ефекти. Крім того, вона є важливим інструментом для роботи з даними та введення користувача.

## Як використовувати

```TypeScript
let firstName: string = "Андрей";
let lastName: string = "Петренко";
console.log(`Привіт, я ${firstName} ${lastName}!`);
```

Результат: "Привіт, я Андрей Петренко!" 

У цьому прикладі ми використали синтаксис строки шаблону в TypeScript, використовуючи знаки між "`" для вставки змінних у текстовий рядок. Це дозволяє нам зробити код більш читабельним та зручним для редагування.

Окрім цього, конкатенацію рядків можна виконати за допомогою оператора "+" або методу "concat". Наприклад:

```TypeScript
console.log("Hello " + "World!"); // результат: "Hello World!"
console.log("Hello".concat(" ", "World!")); // результат: "Hello World!"
```

Ці приклади демонструють, що конкатенація рядків може виконуватись з різними типами даних, такими як рядки, числа чи булеві значення.

## Глибокий аналіз

Крім зазначених вище, TypeScript також надає можливість використовувати метод "join" для конкатенації рядків масиву. Цей метод дозволяє приєднувати рядки з комами або будь-яким іншим розділовим знаком. Наприклад:

```TypeScript
let fruits: string[] = ["яблуко", "банан", "виноград"];
console.log(fruits.join(",")); // результат: "яблуко,банан,виноград"
```

Крім цього, TypeScript має зручну функцію "replace" для заміни частин рядка. Таким чином, можна створювати більш складні вирази для конкатенації рядків. Наприклад:

```TypeScript
let hobbies: string[] = ["фотографування", "мальовання", "книги"];
console.log(`Мені подобається ${hobbies[0]}. Я не люблю ${hobbies[1]}. Але я читаю ${hobbies[2]}.`.replace("не", "").toUpperCase());
// результат: "МЕНІ ПОДОБАЄТЬСЯ ФОТОГРАФУВАННЯ. Я ЛЮБЛЮ МАЛЬОВАННЯ. АЛЕ Я ЧИТАЮ КНИГИ."
```
