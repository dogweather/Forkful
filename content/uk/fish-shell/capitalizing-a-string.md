# Програмування на Fish Shell: Як привести рядок до верхнього регістру?

## Що це таке і навіщо це потрібно?

Приведення рядка до верхнього регістру це процес зміни символів у рядку з  малого регістру на великий. Програмісти роблять це для урівняння даних, перевірки рівності рядків, а також для поліпшення читабельності.

## Як це робити:

```Fish Shell
echo "привіт, світ" | string upper
```
**Виведення:**
```Fish Shell
ПРИВІТ, СВІТ
```
у Fish Shell, ви можете використовувати `string upper`, щоб перетворити рядок в нижньому регістрі на рядок у верхньому регістрі.

## Поглиблений огляд

Перші скриптові мови, такі як Perl і Python, вже містили функції для роботи з рядками, включаючи приведення їх до верхнього регістру. Fish Shell вважається сучасною оболонкою, що забезпечує багато покращень, включаючи більш зрозумілі команди для роботи з рядками.

Альтернативно, ви можете використовувати команду `tr` в більш старих оболонках для приведення рядків до верхнього регістру. Ось як це виглядає:

```bash
echo "привіт, світ" | tr '[:lower:]' '[:upper:]'
```

Всередині Fish Shell, `string upper` працює за принципом ітерації кожного символа у рядку та заміни його еквівалентом у верхньому регістрі.
