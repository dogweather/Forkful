## ЧомуУ світі програмування необхідно постійно навчатися і розширювати свої знання. Використання регулярних виразів є важливою навичкою, яка дозволяє спростити та прискорити роботу зі строками в коді. Вони дозволяють швидко знаходити, замінювати та перевіряти певні шаблони в тексті. Якщо ви плануєте займатися програмуванням, то навчання регулярним виразам буде безсумнівно корисним.  ## Як цього зробитиУ Fish Shell є вбудований модуль для роботи з регулярними виразами, який дозволяє використовувати їх у командному рядку і в скриптах. Для початку треба додати ключове слово `pcre` до команди `set`, щоб підключити цей модуль. Далі можна використовувати команди `string match` та `string replace` для пошуку та заміни підрядків за допомогою регулярних виразів. Ось приклад використання з командному рядку: ```Fish Shell set -g pcre string replace '([0-9]+)' '$1 dollars' <<< 'I have 10 dollars'``` Результат буде: `I have 10 dollars` Для скриптів можна використовувати `string match -r` для пошуку за допомогою регулярних виразів та `string replace -r` для заміни. Будьте уважні, оскільки регулярні вирази можуть бути досить складними і вимогливими до написання!  ## Розбираємося детальнішеТакож в Fish Shell можна використовувати не тільки стандартні регулярні вирази, але й їх розширення, наприклад, PCRE (Perl Compatible Regular Expressions) або POSIX (Portable Operating System Interface). Для цього достатньо вкласти спеціальну прапорці `-P` або `-p` перед регулярним виразом. Крім того, можна використовувати різні опції, які дозволяють розширити функціонал регулярних виразів, такі як `i` для ігнорування регістру, `g` для знаходження всіх входжень, `m` для багаторядкового пошуку та інші. Регулярні вирази можна також комбінувати і використовувати у складних шаблонах для пошуку та заміни тексту. Можна навіть створювати свої власні підшаблони та використовувати їх у регулярних виразах.
