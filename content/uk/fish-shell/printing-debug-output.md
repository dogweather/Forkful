## Що і чому?
Друкування відладочного виводу - це процес відображення внутрішніх робочих даних вашої програми. Програмісти роблять це, щоб зрозуміти, як працює програма в певний момент часу.

## Як це зробити:
На Fish Shell для друку інформації використовують команду `echo`. Подивимося на приклад виводу "Привіт, світ!":

```fish
echo "Привіт, світ!"
```

Цей скрипт виведе наступне:

```fish
Привіт, світ!
```

Ви також можете використовувати змінні для зберігання і відображення інформації:

```fish
set greeting "Привіт, світ!"
echo $greeting
```

Як ви виведете:

```fish
Привіт, світ!
```

## Поглиблений розбір
Розпочавши свій шлях з Bourne Shell, традиція друку відладочних повідомлень досить стара. Fish Shell також дозволяє робити те саме, але із більш сучасним синтаксисом.

Альтернативою друку відладочного виводу є використання спеціальних інструментів відладки, які дають більше інформації про внутрішній стан програми.

Щодо деталів реалізації, Fish Shell використовує системні виклики операційної системи для відправки тексту в стандартний вивід.
