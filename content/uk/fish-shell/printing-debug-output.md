## Чому

Нерідко під час написання програм на Fish Shell виникають ситуації, коли потрібно вивести на екран певні значення змінних або перевірити, чи правильно працює код. У таких випадках друк відладочної інформації на екран є найшвидшим та найпростішим способом з'ясувати, що саме відбувається у програмі.

## Як це зробити

Для друку відладочної інформації в Fish Shell існує спеціальна команда `echo` з параметром `debug`, який показуватиме додаткову відлагоджувальну інформацію. Наприклад, якщо хочете вивести на екран значення змінної `count`, просто виконайте:

```Fish Shell
echo $count --debug
```

Це дозволить вам переконатися, що змінна `count` має потрібне значення. Також можна використовувати команду `printf` для більш складних друків. Наприклад:

```Fish Shell
printf "Змінна count має значення %d" $count --debug
```

Це виведе на екран повідомлення: `Змінна count має значення [значення змінної count]`.

## Глибше погрузження

Друк відладочної інформації також можна налаштувати за допомогою прапорців команди `echo` і команди `set`. Наприклад, ви можете встановити прапорець `debug_format` для зміни формату виводу відлагоджувальної інформації, або використовувати прапореці `BASH_COMPAT` чи `ZSH_COMPAT` для виведення інформації у форматі, схожому на Bash або Zsh відповідно.

Також можна використовувати команди відлагоджувальних інструментів, наприклад `fish_prompt` або `fish_key_reader`, для створення своїх власних засобів друку відладочної інформації.
