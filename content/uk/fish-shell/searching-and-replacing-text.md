# ## Що і чому?
Заміна та пошук тексту - це процес, коли ви знаходите конкретний фрагмент тексту та замінюєте його на інший. Програмісти роблять це, щоб легко та ефективно внести зміни в код, особливо в великих кодових базах.

# ## Як це зробити:
Подивимося, як виконати пошук та заміну тексту в оболонці риби. 

```Fish Shell
# Декларація змінних
set old "старе слово"
set new "нове слово"

# Вміст файлу до заміни
cat file.txt
> старе слово в файлі

# Заміна тексту
sed -i 's/$old/$new/g' file.txt

# Вміст файлу після заміни
cat file.txt
> нове слово в файлі
```

# ## Поглиблений аналіз
У пошуку та заміні тексту є своя історія. Ця техніка датується з часів, коли комп'ютери були великими як кімнати, і мали дуже обмежені ресурси. Пошук та заміна були ключовими для ефективного використання цих ресурсів.

В оболонці риби, команда `sed` використовується для пошуку та заміни тексту. Sedі є потужною утилітою, яка може працювати з рядами в текстових файлах.

Але є й альтернативи `sed`, такі як `awk`, `perl`, `python`, і `grep`. Вибір різних утиліт вирішується задачами та особистими вподобаннями.
