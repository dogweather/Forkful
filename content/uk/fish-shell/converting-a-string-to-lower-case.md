# З чого почати
Програмування може здаватися досить складною справою, але іноді навіть найпростіші операції можуть викликати запитання. Одним з таких прикладів є конвертація рядка в нижній регістр. Ця операція може бути корисна для багатьох речей, таких як пошук по ключових словах, порівняння рядків або для забезпечення однакового форматування даних. В цьому блозі ми розглянемо, як конвертувати рядок в нижній регістр за допомогою оболонки Fish Shell для наших читачів в Україні.

## Як
Для початку, створіть новий файл з розширенням .fish і відкрийте його у текстовому редакторі. Ми будемо використовувати вбудовану у Fish Shell функцію `string lower`, яка конвертує рядок в нижній регістр. Нижче наведений код, який використовує цю функцію:

```fish
set my_string "Привіт Світе"
echo (string lower $my_string)
```

Ви можете побачити, що рядок "Привіт Світе" було конвертовано в "привіт світе". Звичайно, ви можете змінити значення `my_string` на будь-який інший рядок, який ви хочете конвертувати.

## Глибоке погруження
Крім використання вбудованої функції `string lower`, можна використовувати також команду `tr` для виконання цієї операції. Наприклад,

```fish
set my_string "Привіт Світе"
echo $my_string | tr '[:upper:]' '[:lower:]'
```

Команда `tr` відповідає за заміну однієї частини рядка на іншу. У цьому випадку ми вказуємо, що всі великі літери будуть замінені на відповідні малі. Результат буде аналогічним до першого прикладу.
