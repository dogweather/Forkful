Також, ви можете легко порівнювати дати за допомогою Fish Shell. Це дозволяє проводити більш точні і складні порівняння дат, що є корисним для різних типів програм та скриптів.

## Чому

Порівнювання дат є важливою частиною багатьох програм та скриптів. Це може бути корисно для виконання різних дій в залежності від того, яка дата більша або менша. Наприклад, ви можете відправити сповіщення, якщо дата зустрічі зараз пройшла або перевірити, чи термін дії певного документа вже минув. За допомогою Fish Shell ви можете легко зробити такі порівняння дат, щоб забезпечити правильну роботу вашої програми.

## Як

Для порівняння дат у Fish Shell використовуйте команду `expr`. Нижче наведений приклад, де ми порівнюємо дві дати та отримуємо результат у вигляді числа. Зверніть увагу, що формат дати повинен бути `MMDD` (місяць та день).

```Fish Shell
set date1 0101 # перша дата
set date2 0202 # друга дата
set result (expr $date1 - $date2) # результат порівняння дат
echo $result # виводимо результат
```

У цьому прикладі, результат буде `-31`, оскільки 1 січня (0101) більша за 2 лютого (0202). Ви можете використовувати цей результат умовах та більш складних порівняннях дат.

## Глибока поглибка

Ви також можете використовувати команду `date` для порівняння дат у Fish Shell. Це дозволяє вам отримати відформатовану дату за допомогою різних аргументів та порівняти їх значення. Наприклад, ви можете порівняти роки двох дат, використовуючи аргумент `"+%Y"` для команди `date`.

```Fish Shell
set date1 "2020-01-01" # перша дата
set date2 "2021-01-01" # друга дата
set year1 (date -d "$date1" "+%Y") # рік першої дати
set year2 (date -d "$date2" "+%Y") # рік другої дати
echo $year1 # виводимо рік першої дати
echo $year2 # виводимо рік другої дати
```

Цей приклад дозволяє отримати відформатовану дату у зручному для порівняння форматі. Ви можете використовувати різні аргументи та формати дати, щоб скласти більш складні порівняння.
