## Чому

Використання командного рядка може здатися незвичним та складним. Однак знання основ інтерфейсу командного рядка може значно полегшити виконання різних завдань та зробити роботу з комп'ютером більш ефективною. Одним із найважливіших аспектів роботи з командним рядком є читання аргументів команд.

## Як

Для читання аргументів команд у Fish Shell використовується змінна `argv`. Ця змінна містить масив із всіма аргументами, які були передані в команді. Приклад використання:

```Fish Shell
argv[1] # повертає перший аргумент команди
argv[2] # повертає другий аргумент команди
argv[-1] # повертає останній аргумент команди
```

У випадку, якщо ви хочете отримати всі аргументи команди у вигляді окремого рядка, можна використовувати змінну `argstring`. Наприклад:

```Fish Shell
echo $argstring # виводить усі аргументи команди у вигляді одного рядка
```

## Deep Dive

При використанні командного рядка у більш складних сценаріях, може знадобитися отримати лише деякі аргументи команди. Для цього можна використовувати параментри із зірочкою (`*`) та крапкою (`.`). Наприклад, якщо ви хочете отримати всі аргументи починаючи з третього, але не більше ніж п'ять, можна використовувати такий синтаксис:

```Fish Shell
argv[3*..4] # поверне третій, четвертий та п'ятий аргументи команди
```
