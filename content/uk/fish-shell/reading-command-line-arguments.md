## Що й Навіщо?

Аргументи командного рядка - це значення, які передаються програмі при запуску. Програмісти використовують їх для контролю поведінки програми без необхідності жорстко задавати параметри в коді.

## Як це робити:

```Fish Shell
function greet
    echo "Привіт, $argv[1]"
end
```

Запустіть цю функцію з аргументом:

```Fish Shell
greet Мир
```

Вам має видати:

```Fish Shell
Привіт, Мир
```

Обробка декількох аргументів:

```Fish Shell
function greet
    for name in $argv
        echo "Привіт, $name"
    end
end
```

Запустіть цю функцію з кількома аргументами:

```Fish Shell
greet Анна Ігор Володимир
```

Результат буде:

```Fish Shell
Привіт, Анна
Привіт, Ігор
Привіт, Володимир
```

## Глибше в матеріал

Історично, аргументи командного рядка були основним способом взаємодії з програмами. Вони все ще активно використовуються, особливо в скриптах оболонки, хоча поточні ПЗ мають більш складні інтерфейси.

Іншими способами передачі параметрів могли б бути середовище, конфігураційні файли, інтерактивний інтерфейс.

Щодо подробиць реалізації, argv (або "аргументи вектора") насправді є звичайним списком рядків всередині fish. У fish немає окремих "посиційних параметрів" як в bash, лише один список argv.
