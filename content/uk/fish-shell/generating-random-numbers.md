## Чому

Створення випадкових чисел може бути важливим елементом при програмуванні, так як це допомагає генерувати унікальні дані для тестування або створення випадкової послідовності для використання у грі або іншому програмному продукті.

## Як

```Fish Shell``` має вбудований інструмент ```random``` для генерування випадкових чисел. Цей інструмент приймає два параметри: мінімальне та максимальне значення. Наприклад, якщо нам необхідно згенерувати випадкове число від 1 до 10, ми можемо скористатися наступним кодом:

```
set min 1
set max 10

# згенерувати випадкове число
random -l $min $max
```

Цей код буде виводити в терміналі різні числа між 1 та 10 кожного разу, коли його виконувати.

## Глибоке дослідження

Крім використання вбудованого інструменту, також можна встановити пакет ```fish-random``` для генерування більш складних випадкових чисел, таких як випадкові рядки та бінарні дані. Цей пакет має багато опцій для налаштування генератора випадкових чисел.
