## Що і навіщо?

Генерація випадкових чисел - це процес створення чисел, які не передбачуваються. Програмісти використовують це для непередбачуваного поведінки, наприклад, для тестування, гри, захисту даних і багато іншого.

## Як це робити:

Отже, ось декілька зразків коду Fish Shell для генерації випадкових чисел.

```fish
# Генеруємо випадкове число між 0 та 255.
echo (random 0 255)
```

Вище наведений код повертає випадкове число в діапазоні від 0 до 255.

```fish
# Генеруємо випадкове число від 1 до 10
echo (random 1 10)
```

Цей код генерує випадкове число в діапазоні від 1 до 10.

## Поглиблений аналіз

Генерація випадкових чисел використовувалась з самого початку історії комп'ютерної техніки. Псевдовипадкові генератори чисел, які використовують багато програм, не є дійсно випадковими, але є достатньо непередбачуваними для багатьох застосувань.

В альтернативних мовах програмування, таких як Python або JavaScript, можна також генерувати випадкові числа, але методи та синтаксис відрізняються від Fish Shell.
