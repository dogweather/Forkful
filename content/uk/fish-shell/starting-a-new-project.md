## Що та чому?

Створення нового проєкту в програмуванні - це процес організації коду для нової задачі або програми. Програмісти роблять це, щоб ефективно створювати, відлагоджувати та підтримувати код.

## Як це зробити:

Для створення нового проєкту необхідно створити нову директорію та ініціалізувати її. Ось як це можна зробити в Fish Shell:

```Fish Shell
~> mkdir my_project
~> cd my_project
my_project ~> touch main.fish
```
Після цього у директорії 'my_project' буде створено новий файл 'main.fish'.

## Поглиблений огляд:

На початку термінальне програмування було досить простим. Проте, з розвитком технологій, стало зрозуміло, що потрібен більш гнучкий підхід. Fish Shell - одна з відповідей на цю потребу.

Є інші альтернативи Fish Shell, такі як Bash або Zsh, але Fish Shell вирізняється своєю простотою та зручністю.

Перед тим, як почати новий проєкт в Fish Shell, необхідно встановити його на вашу систему. Його можна встановити використовуючи різні керувальники пакетами, наприклад, для Ubuntu вам підійде 'apt':

```Fish Shell
~> sudo apt-get install fish
```
Після встановлення, ви зможете використовувати Fish Shell наче ваш термінал за замовчуванням.
