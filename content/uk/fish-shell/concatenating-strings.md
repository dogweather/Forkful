## Що & Чому?
Конкатенація рядків - це процес об'єднання двох або більше рядків у один. Програмісти це роблять для створення індивідуальних рядків або для маніпулювання даними без необхідності модифікувати оригінальні рядки.

## Як це зробити:
У Fish Shell сценарії конкатенації рядків не вимагає використання спеціального оператора. Ми можемо об'єднати рядки, просто пишучи їх поряд.

```fish
# Set двох рядків
set string1 "Розмовляймо про "
set string2 "Fish Shell."
# Об'єднайте їх
echo $string1$string2
```
Виходячи з цього, ми отримаємо `"Розмовляймо про Fish Shell."`

## Поглиблення у тему:
Історично, обробка рядків завжди була важливою частиною програмування. Конкатенація рядків, зокрема, допомагає привести в порядок і об'єднати рядки даних, незалежно від мови програмування.

У Fish Shell немає вбудованого оператора для конкатенації, але його простота робить цей процес ефективним. Проте, у якості альтернативи, `string join` може бути використаний для більшії гнучкості.

Реалізація конкатенації рядків в Fish Shell це просто об'єднання рядків, тому немає значних подробиць для обговорення.
