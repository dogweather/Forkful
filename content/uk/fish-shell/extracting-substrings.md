## Чому
Виділення підрядків є корисною навичкою у програмуванні, оскільки вона дозволяє ефективніше обробляти та маніпулювати рядками. Це особливо корисно у веб-розробці, де часто доводиться працювати зі великими обсягами даних.

## Як
Почнемо зі створення змінної з рядком, з яким ми будемо працювати:

```Fish Shell
set input "Це приклад рядка"
```

Тепер, за допомогою команди `string sub`, ми можемо виділити підрядок з нашого вхідного рядка. Наприклад, якщо хочемо виділити слово "приклад", використовуємо наступну команду:

```Fish Shell
string sub $input 4 7
```

Перше число вказує на позицію початку підрядка, а друге - кількість символів, які потрібно виділити. У цьому випадку ми виділили символи з 4 по 7, що відповідає слову "приклад". Вихідний рядок виглядатиме наступним чином:

```Fish Shell
приклад
```

## Глибоке занурення
Крім виділення підрядків за певною позицією та кількістю символів, можна використовувати також регулярні вирази для більш гнучкого виділення. Наприклад, якщо хочете виділити слово "рядка" з будь-якого рядка, використовуйте такий комбінацію:

```Fish Shell
string match -r "рядка" $input
```

Також можна використовувати регулярний вираз для заміни певної частини рядка на інший рядок. Наприклад, якщо хочете замінити слово "приклад" на "демо", використайте наступний приклад:

```Fish Shell
string replace -r "приклад" "демо" $input
```

В результаті отримаємо рядок замість вхідного рядка:

```Fish Shell
Це демо рядка
```
