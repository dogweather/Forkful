## Що і для чого?

Читання текстового файлу - це процес, коли комп'ютер зчитує інформацію з такого файлу. Програмісти це роблять, щоб обробити його вміст та використати в програмах або скриптах.

## Як зробити:

Приклад коду та його виведення в Fish Shell:

```Fish Shell
# Читання файлу з ім'ям myfile.txt
cat myfile.txt
```

Якщо ви запустите цей код, він виведе вміст файлу `myfile.txt`.

```Fish Shell
# Виведення:
Привіт, це просто текстовий файл.
```

## Занурення в тему:

Читання текстових файлів - це базова можливість в будь-якій оболонці, включаючи Fish Shell. Цей процес важливий для багатьох сценаріїв, включаючи обробку файлів, створення тестових даних або сканування логів. Історично команда `cat` використовувалася в Unix для зчитування файлів з 1971 року. 

Альтернатива команді `cat` - команда `less`, яка також може бути використана для читання текстових файлів. Ця команда є більш впорядкованою, приємною для можливості списокування великих файлів.

Процес читання файлу включає відкриття файлу, зчитування даних з нього та закриття файлу. Fish Shell виконує ці завдання за вас, коли ви використовуєте команду `cat` або `less`.
