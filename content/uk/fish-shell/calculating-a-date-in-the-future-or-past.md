## Чому

У програмуванні часові вирахування знаходять широке застосування, особливо коли мова йде про обчислення дати в майбутньому або минулому. Це дуже корисно для створення різноманітних таймерів, а також для знаходження інформації про події, які відбулися у певний період часу. У цій статті ми розглянемо, як за допомогою Fish Shell зручно виконувати такі обчислення.

## Як

Щоб розпочати роботу з обчисленням дат, нам потрібно мати деяку початкову дату. Для цього ми можемо скористатися командою `date`, яка повертає поточну дату та час. Далі, нам потрібно опрацювати цю дату, щоб отримати потрібний результат. Давайте розглянемо кілька прикладів за допомогою коду в Fish Shell.

```Fish Shell
# Отримуємо поточну дату та час
set current_date (date +%s)

# Додавання 1 дня до поточної дати
set future_date (math $current_date + (60*60*24))

# Виведення результату у зрозумілому форматі
echo (date -d @$future_date +'%d-%m-%Y')
```

В цьому прикладі ми отримуємо поточну дату у форматі Unix timestamp, що дозволяє здійснювати математичні операції з датами. При додаванні 1 дня до поточної дати ми використовуємо множення на кількість секунд у дні (60*60*24). Наступною датою буде отриманий результат, який ми можемо вивести у зрозумілому форматі за допомогою команди `date -d` зі спеціальним параметром `%d-%m-%Y`, що показує день, місяць та рік.

## Глибше занурення

Крім базового додавання днів, Fish Shell також дозволяє виконувати більш складні математичні операції з датами. Наприклад, можна додавати тижні, місяці або навіть роки до поточної дати. Для цього потрібно просто змінювати кількість секунд, яку ми додаємо до початкової дати.

```Fish Shell
# Додавання 1 тижня до поточної дати
set future_date (math $current_date + (60*60*24*7))

# Додавання 1 місяця до поточної дати
set future_date (math $current_date + (60*60*24*30))

# Додавання 1 року до поточної дати
set future_date (math $current_date + (60*60*24*365))
```
