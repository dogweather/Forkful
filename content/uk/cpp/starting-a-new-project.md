**## Що і чому?**

Починаючи новий проєкт в програмуванні, ви ефективно створюєте новий софт. Причин багато — можливо, ви розробляєте новий товар, разом з командою вирішуєте проблему або просто прагнете вдосконалити свої навички.

**## Як?:** 

```C++
#include <iostream>
int main(){
    std::cout << "Привіт, світ!";
    return 0;
}
```
Цей код напише "Привіт, світ!" в консолі. Просто це перший крок на шляху до створення реального проєкту в C++.

**## Занурення в деталі:**

1) *Історичний контекст:* Від початку епохи комп'ютерів, програмісти починали нові проєкти з простого "Hello, World!" Програма ставить пріоритети правильно, допомагаючи побачити, що ви робите, перед тим як вони виростуть.

2) *Альтернативи*: Новий проєкт може бути створений з нуля або на базі існуючого шаблону. Компанії часто використовують внутрішні шаблони для роутинних задач.

3) *Деталі реалізації:* Розпочинайте новий проєкт з чіткою ідеєю того, що ви хочете досягнути. Використовуйте інструменти середовища розробки для створення сценарієвого файлу, створення дерева проєкту і налаштування параметрів компіляції.
