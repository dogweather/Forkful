##Чому

Строка, яку ми отримуємо від користувача, може містити слова з неправильним регістром. Для зручності читання і обробки цієї інформації, іноді потрібно перевести її в нижній або верхній регістр. Це може бути корисно для пошуку чи сортування даних. В цьому блозі ми розглянемо як виконати цю операцію в мові програмування C++.

##Як це зробити

Для виконання цієї задачі використовується функція `toupper()` з бібліотеки `<cctype>`, яка конвертує символи в верхній регістр. Нижче наведено приклад коду, який приймає введену користувачем строку та виводить її в верхньому регістрі:

```C++
#include <iostream>
#include <cctype>
#include <string>

int main()
{
	std::string str;
	std::cout << "Введіть рядок: ";
	std::getline(std::cin, str);

	// Використання циклу для перебору символів та виклику функції toupper()
	for (int i = 0; i < str.length(); i++) {
		str[i] = toupper(str[i]);
	}

	std::cout << "Слова з верхнього регістру: " << str << std::endl;
	return 0;
}
```

Приклад введення користувача та виведення результату:

```
Введіть рядок: ПрИкЛаД ТеКстУ
Слова з верхнього регістру: ПРИКЛАД ТЕКСТУ
```

##Глибшим поглядом

Є кілька способів зробити перетворення до верхньої чи нижньої межі регістру за допомогою функції `toupper()`. Наприклад, можна застосувати її до окремих символів, чи використати вбудовані функції для обробки цілого рядка. Також є можливість використовувати спеціальні функції для роботи з кирилицею. Більше про це можна дізнатися з офіційної документації мови C++.
