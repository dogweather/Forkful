## Для чого

Зчитування текстового файлу є важливою складовою частиною у розробці програмного забезпечення і може бути корисним в багатьох сферах, таких як обробка даних, парсинг веб-сторінок або створення звітів. Якщо ви плануєте використовувати мову програмування C++, знання про те, як читати текстові файли, обов'язково стане в пригоді.

## Як це зробити

Почнемо з простого - підключіть стандартну бібліотеку для роботи з файлами `fstream` і використовуйте методи `open()`, `is_open()` та `close()` для відкриття, перевірки відкриття і закриття файлу відповідно. У наступному прикладі ми прочитаємо текстовий файл зі статистикою про зарплати та виведемо дані на екран.

```C++
#include <fstream>
#include <iostream>

int main() {
    // відкриваємо файл для зчитування
    std::ifstream file;
    file.open("salaries.txt");

    // перевіряємо чи вдалося відкрити файл
    if (file.is_open()) {
        // рядок для збереження даних з файлу
        std::string data;

        // зчитуємо дані з файлу та виводимо їх на екран
        while (getline(file, data)) {
            std::cout << data << std::endl;
        }

        // закриваємо файл
        file.close();
    } else {
        // якщо відкриття файлу було невдалим, виводимо повідомлення про помилку
        std::cout << "Не вдалося відкрити файл!" << std::endl;
    }
    
    return 0;
}
```

Маючи файл `salaries.txt` з таким змістом:

```
Ім'я: Олексій Посада: Розробник
Зарплата: 5000 доларів на місяць
```

ми отримаємо поступове виведення цих даних на екран:

```
Ім'я: Олексій Посада: Розробник
Зарплата: 5000 доларів на місяць
```

З поміченої останньої строки видно, що `getline()` збереже рядок до першого символу перенесення рядка, тому ми отримаємо точний вихід на екрані. Також, при зчитуванні чисел можна використовувати метод `>>` у поєднанні з потоком вводу `istream`.
