## Для чого

Не завжди перед ціллю стоїть готовий, жорстко запрограмований виконавець. Іноді потрібно змінити умови під час роботи, або ж взагалі не знати точного набору виконуваних дій. Це особливо корисно, коли працюєте з командною строкою, оскільки вона дозволяє змінювати параметри без необхідності перекомпіляції програми.

## Як

Для того, щоб читати командні аргументи в C++, перш за все потрібно підключити бібліотеку `iostream` і `stdio.h`. Далі можна зчитати аргументи за допомогою вбудованих функцій. Наприклад, функція `argc` вказує на кількість переданих аргументів, а `argv` зберігає їх значення. Ось приклад коду:

```C++
#include <iostream>
#include <stdio.h>

int main(int argc, char *argv[]) {
    // Першим аргументом буде назва програми
    std::cout << "Назва програми: " << argv[0] << std::endl; 
    // Перевіримо, чи передані аргументи
    if (argc > 1) {
        // Виведемо всі аргументи
        for (int i = 1; i < argc; i++) {
            std::cout << "Аргумент " << i << ": " << argv[i] << std::endl;
        }
    } else {
        std::cout << "Не було передано аргументів" << std::endl;
    }
    return 0;
}
```

Приклад виклику програми: `./arguments.exe arg1 arg2`. Вивід буде такий:

```
Назва програми: ./arguments.exe
Аргумент 1: arg1
Аргумент 2: arg2
```

## Глибше

Розглянемо детальніше, як саме зчитуються командні аргументи. При компіляції програми, компілятор копіює рядки аргументів з командної строки до масиву `argv`, а кількість аргументів записує в `argc`, передаючи ці значення у функцію `main()`. Таким чином, ми можемо легко звертатися до цих значень у нашій програмі.

Окрім вбудованих функцій `argc` і `argv`, існує багато інших методів для роботи з командними аргументами, наприклад, використання бібліотеки `getopt()` або обробка їх у вигляді строки.
