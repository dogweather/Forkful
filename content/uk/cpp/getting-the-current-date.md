## Чому

Поточна дата є важливою частиною багатьох програм і проектів. Включення дати допомагає відстежувати час, виконувати планування та створювати звіти. Також це може бути корисним для статистичних аналізів або встановлення термінів дії документів. В цілому, отримання поточної дати є необхідною функцією в програмуванні.

## Як

Найпростішим способом отримати поточну дату є використання стандартної бібліотеки `ctime`. Для цього потрібно підключити заголовочний файл `<ctime>` та використовувати функцію `time` для отримання мітки часу. Приклад коду наведений нижче:

```C++
#include <iostream>
#include <ctime>

int main() {
  // Отримання поточного часу в секундах
  time_t now = time(0);

  // Конвертація часу в локальний час
  char* dt = ctime(&now);

  // Виведення поточної дати та часу
  std::cout << "Поточна дата та час: " << dt << std::endl;

  return 0;
}
```

Виведення програми може бути наступним:

```
Поточна дата та час: Thu Oct 21 22:39:23 2021
```

Існує також багато інших способів отримання поточної дати, таких як використання сторонніх бібліотек або API. Результат може представляти дату у різних форматах, тому важливо розібратися зі своїм конкретним випадком використання.

## Глибоке занурення

Функція `time` повертає мітку часу в секундах, яка представляє кількість секунд, які пройшли з 1 січня 1970 року. Цей метод отримання поточної дати називається "Unix time" і використовується у багатьох операційних системах та програмувальних мовах. Можна використовувати цю мітку для обчислення різниці часу між двома подіями або для конвертації у більш зрозумілий формат за допомогою стандартних функцій.
