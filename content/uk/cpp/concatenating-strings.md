## Для чого

При програмуванні на C++, використання функції конкатенації рядків може бути корисним для з'єднання декількох рядків в один більший рядок. Це може бути корисним при будь-якому проекті, де необхідно об'єднати текст або змінні в один рядок перед подальшою обробкою даних.

## Як це зробити

Для конкатенації рядків використовується "+" оператор. У прикладі нижче, ми створюємо два рядки і з'єднуємо їх в один за допомогою цього оператора. Наприклад:

```C++
#include <iostream>
#include <string>

using namespace std;

int main() {

    // створення двох рядків
    string first = "Привіт,";
    string second = "це мій перший";
    
    // конкатенація рядків за допомогою "+" оператора
    string result = first + " " + second + " C++ проект!";
    
    // вивід результату
    cout << result << endl;
    
    return 0;
}

// вивід: Привіт, це мій перший C++ проект!
```

## Як це працює

Коли ми використовуємо "+" оператор виразу рядків, компілятор розпізнає його як конкатенацію і об'єднує рядки в один. Однак, при використанні цього оператора, необхідно пам'ятати про типи даних - оператор "+" може бути використаний тільки для об'єднання рядків з типом "string".

## Детальніше про конкатенацію рядків

Крім "+" оператора, існує ще кілька інших способів конкатенації рядків в C++. Наприклад, використовуючи функцію `concat()`, яка приймає два рядки як аргументи і повертає результат їх об'єднання. Можна також використовувати функцію `append()`, яка дозволяє додавати ще один рядок до вже існуючого.
