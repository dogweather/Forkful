## Чому
Розрахунок дати в майбутньому або минулому є важливою задачею для багатьох програм, таких як планери подій або таймери. Точне визначення дати є необхідним для правильної роботи цих програм.

## Як
Для початку необхідно підключити бібліотеку <ctime> у вашому коді, щоб мати можливість користуватися функціями для розрахунку дати. Нижче наведено приклад коду для обчислення дати 3 дні в майбутнє від поточної дати у форматі "ДД/ММ/РРРР". 

```C++
#include <iostream>
#include <ctime>

using namespace std;

int main() {
    // поточна дата
    time_t now = time(0);
    tm *ltm = localtime(&now);
    
    // обчислення дати в майбутньому
    ltm->tm_mday += 3; // додаємо 3 дні
    mktime(ltm); // перетворюємо в правильний формат
    
    // виведення в форматі "ДД/ММ/РРРР"
    cout << "Дата через 3 дні: " << ltm->tm_mday << "/" << ltm->tm_mon+1 << "/" << ltm->tm_year+1900 << endl;
    
    return 0;
}
```

**Вихід:**
Дата через 3 дні: 12/10/2021

## Поглиблене вивчення
У програмуванні існує понад 50 різних форматів для представлення дати і часу. У C++, дата зберігається у форматі "struct tm", який включає в себе такі змінні, як день місяця, місяць, рік, години та інші. Для обчислення дати у майбутньому або минулому, необхідно користуватися функціями <ctime>, такими як `mktime()` для перетворення в потрібний формат та `localtime()` для отримання поточної дати.
