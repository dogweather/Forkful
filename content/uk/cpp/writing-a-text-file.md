## Чому

Програмування - це процес створення програм та алгоритмів, який дозволяє нам зробити наше життя простішим та ефективнішим. Одним з важливих аспектів програмування є робота з текстовими файлами. Навіть якщо ви не плануєте стати професійним програмістом, вміння працювати з текстовими файлами може бути корисним у різних сферах життя, наприклад, при написанні звітів або резюме.

## Як

Для того, щоб створити текстовий файл за допомогою мови програмування C++, нам потрібно використати бібліотеку fstream. Спочатку потрібно підключити цю бібліотеку за допомогою директиви ```#include <fstream>```. Потім можна створити об'єкт типу ofstream, який дозволить нам записувати дані у файл. Наприклад, використовуючи методи ```open()``` та ```close()```, ми можемо відкрити та закрити файл:

```C++
    #include <fstream>

    int main()
    {
        ofstream file;
        file.open("example.txt");

        // додамо деякі дані до файлу
        file << "Привіт, світ!" << endl;
        
        // закриємо файл
        file.close();
        
        return 0;
    }
```

Після запуску цього коду, у вашій папці буде створено файл з назвою "example.txt", у якому буде написано "Привіт, світ!".

## Глибше занурення

Не тільки створення, але й редагування та читання текстових файлів є важливою частиною програмування. Для редагування файлів ми можемо використовувати методи ```seekg()``` та ```seekp()```, які дозволяють пересувати позицію в файлі та записувати дані у будь-якому місці. Також, для зручності, можна використовувати цикли для зчитування та записування у файл, наприклад, можна створити програму, яка буде зчитувати введені користувачем дані та записувати їх у файл.
