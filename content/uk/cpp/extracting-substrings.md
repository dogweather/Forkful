## Чому

Зачому ви мали б переглядати підрядних рядків у своїх програмах? Якщо ви працюєте з текстовими даними, ви, напевно, зустрілися з ситуацією, коли вам потрібно було виділити окрему частину рядка. Наприклад, ви можете мати ім'я користувача та електронну адресу, а потім хочете окремо отримати ім'я та поштову скриньку. В такому випадку вам необхідно використовувати витяги підрядків.

## Як виконати

Витяги підрядків використовуються у випадках, коли ви хочете отримати частину рядка за допомогою індексів або певних критеріїв. Це легко зробити за допомогою функції substr () у C++.

```C++
#include <iostream>
#include <string>

int main() {
    std::string str = "Hello, World!";
    
    // отримання підрядка від 0 до 5 символу
    std::string substr = str.substr(0, 5);
    std::cout << substr << std::endl;
    
    // отримання підрядка з 5 символу і до кінця
    substr = str.substr(5);
    std::cout << substr << std::endl;
    
    // отримання підрядка по критерію - з символу "W" і до символу "!"
    substr = str.substr(str.find("W"), str.find("!") - str.find("W") + 1);
    std::cout << substr << std::endl;
    
    return 0;
}
```

Вивід:

```
Hello
, World!
World!
```

## Вивчення поглиблено

Алгоритм функції substr () в C++ досить простий. Вона приймає два аргументи, перший - це позиція початку витягу, і другий - кількість символів, яку необхідно вибрати. Також ви можете використовувати функції find () та rfind () для знаходження бажаних символів у рядку та отримання позиції для витягування.
