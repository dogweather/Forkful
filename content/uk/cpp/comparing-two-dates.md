## Чому

З кожним днем все більше людей вчаться програмувати на мові C++, яка є однією з найбільш широко використовуваних мов у світі. І час від часу у програмах необхідно порівнювати дві дати, наприклад, для перевірки часових меж або для сортування об'єктів за датою. У цьому блозі ми розглянемо як порівнювати дві дати у C++ та як це може бути корисно у вашому програмуванні.

## Як

Для порівняння двох дат у C++, ми можемо скористатися класом `std::chrono::time_point`. Нижче наведений приклад коду, що демонструє порівняння двох дат.

```C++
#include <iostream>
#include <chrono>

int main() {
    // Стовримо часові точки для двох дат
    // Тут ми задаємо дати 25 грудня 2021 і 1 січня 2022
    std::chrono::time_point<std::chrono::system_clock> date1(std::chrono::minutes(1640426860));
    std::chrono::time_point<std::chrono::system_clock> date2(std::chrono::minutes(1641027560));

    // Порівняємо дати
    if (date1 < date2) {
        std::cout << "Перша дата раніше за другу дату." << std::endl;
    }
    else if (date1 > date2) {
        std::cout << "Перша дата пізніше за другу дату." << std::endl;
    }
    else {
        std::cout << "Обидві дати однакові." << std::endl;
    }

    return 0;
}
```

Вивід:

```
Перша дата раніше за другу дату. 
```

Цей приклад показує, що ми можемо використовувати оператори порівняння `<` та `>` для порівняння дат. Ми також можемо використовувати інші оператори порівняння, такі як `<=` та `>=`. 

## Огляд

У C++ дати можна порівнювати за допомогою класу `std::chrono::time_point` з використанням стандартних операторів порівняння. Цей клас може бути корисним у багатьох ситуаціях, коли необхідно порівняти дати, наприклад, при роботі з об'єктами, які містять часові дані.
