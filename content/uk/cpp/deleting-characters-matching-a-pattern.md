#Чому

Часто під час програмування нам потрібно видалити з рядка певні символи, які відповідають деякому шаблону. Це може бути зроблено за допомогою функції `erase` в C++ і це дозволяє нам легко і швидко обробляти рядки за певними вимогами або правилами.

#Як це зробити

Для того, щоб видалити символи відповідно до заданого шаблону, ми будемо використовувати цикл `for`, який буде перевіряти кожен символ рядка. Якщо символ відповідає шаблону, то ми будемо використовувати функцію `erase` для його видалення з рядка. Ось приклад коду, де ми видаляємо всі цифри з рядка:

```C++
string str = "Hello123";
for (int i = 0; i < str.length(); i++) {
  if (isdigit(str[i])) {
    str.erase(i,1);
  }
}
cout << str; // виведе "Hello"
```

Як бачимо, ми можемо використовувати функцію `erase` в циклі для видалення символів за певним шаблоном. Також варто зазначити, що ми вказуємо кількість символів, які потрібно видалити (1) в якості другого аргументу функції `erase`.

#Глибше

Якщо потрібно видалити більше одного символу за один раз, можна замість цього використовувати функцію `replace`, яка приймає три аргументи: індекс початку, кількість символів, які потрібно замінити, та рядок, який буде замінювати вибрану частину. Таким чином, ми можемо зробити заміну символів замість їх видалення.

Також варто пам'ятати, що функція `erase` та `replace` є частиною бібліотеки `<algorithm>`, тому можна використовувати їх разом з іншими функціями цієї бібліотеки для більш зручного та ефективного програмування.
