## Що та Навіщо?
Видалення символів що відповідають зразку - це техніка, яка дозволяє видаляти конкретні символи з рядків. Програмісти використовують це, щоб очистити дані чи працювати з бажаними частинами тексту.

## Як це зробити:
```C++
#include<iostream>
#include<string>
#include<regex>

int main() {
    std::string str = "Привіт 123 Світ456";
    std::regex r("\\d");
    std::string new_str = std::regex_replace(str, r, "");
    std::cout << new_str;
    return 0;
}
```
Вихід: “Привіт Світ”

## Глибше занурення:
Видалення символів за допомогою зразків виявилося процедурою, яка є корисною в багатьох мовах програмування з моменту розвитку регулярних виразів у 60-ті роки XX століття. Для цієї задачі в C++ використовується `std::regex_replace()`. 

Є альтернативи цьому підходу. Наприклад, можна використовувати ітератори чи створювати власні функциї, але regex виявилися найбільш універсальними і спрощують код.

Однак важливо враховувати, що регулярні вирази можуть бути менш ефективними за ємнісними ресурсами в порівнянні з іншими підходами, особливо на великих даних.
