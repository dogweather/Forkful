## Чому

При програмуванні в С++, іноді нам потрібно перевірити, чи існує директорія нашої системи. Це важлива складова для роботи з файлами та збереженням даних. Дізнаємося, як це зробити.

## Як це зробити

Попередньо створимо змінну, де буде зберігатися шлях до нашої директорії.

```C++
string path = "./sample_directory";
```

Тепер давайте напишемо функцію, яка буде перевіряти, чи існує ця директорія. Для цього будемо використовувати клас `std::filesystem`. Для початку, оголосимо його за допомогою директиви `using`. Імпортуємо модуль `<filesystem>` для цього класу.

```C++
using namespace std::filesystem
#include <filesystem>
```

Тепер за допомогою функції `exists` перевіримо, чи існує наша директорія. Ця функція поверне булеве значення - `true`, якщо директорія існує та `false` в противному випадку.

```C++
if(exists(path)){
    cout << "Директорія існує!" << endl;
}else{
    cout << "Директорія не існує!" << endl;
}
```

В результаті ми отримаємо наступну вивід на консоль:

```C++
Директорія існує!
```

## Глибше

Якщо ми хочемо більш детально розібратися з перевіркою директорії, можемо використовувати інші функції з класу `std::filesystem`. Наприклад, функція `is_directory` поверне `true`, якщо передана їй шлях веде до існуючої директорії, а функція `is_regular_file` перевірить, чи є переданий шлях файлом.
