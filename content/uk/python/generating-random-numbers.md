## Чому

Генерація випадкових чисел є одним із найбільш захоплюючих аспектів програмування Python. Вона може бути використана для створення випадкових даних для тестування, симуляцій, або ж просто для отримання випадкових результів у вашій програмі. Це також допомагає зробити код більш цікавим і непередбачуваним.

## Як

Існує кілька способів генерації випадкових чисел в Python, і ми розглянемо деякі з них тут.

### Використання модуля `random`

Найпростішим способом генерації випадкових чисел є використання вбудованого модуля `random`. Для цього нам потрібно імпортувати цей модуль у нашому коді, після чого ми можемо використовувати його функції, як показано нижче:

```Python
import random

# Генеруємо випадкове ціле число в діапазоні від 0 до 10
random_num = random.randint(0, 10)
print(random_num)

# Генеруємо випадкове дійсне число від 0 до 1
random_float = random.random()
print(random_float)
```

Вивід:

```text
5
0.482716973418
```

### Використання модуля `numpy`

Модуль `numpy` має багато функцій для роботи з числами, включаючи генерацію випадкових чисел. Для використання цього модуля нам спочатку потрібно його імпортувати, а потім ми можемо використати функції `numpy.random`, як показано нижче:

```Python
import numpy as np

# Генеруємо масив з 10 випадкових цілих чисел в діапазоні від 0 до 100
rand_int_arr = np.random.randint(0, 100, 10)
print(rand_int_arr)

# Генеруємо масив з 5 випадкових дійсних чисел від 0 до 1
rand_float_arr = np.random.random(5)
print(rand_float_arr)
```

Вивід:

```text
[86 50 40 59 33 10 95  7 38 13]
[ 0.17861117  0.92690369  0.15039564  0.52622704  0.5897479 ]
```
