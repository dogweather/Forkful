## Чому

Конкатенація рядків - це важлива концепція в програмуванні, що дозволяє об'єднати два або більше рядки в один. Це може бути корисно, коли ви хочете створити власні повідомлення або форматувати дані для виведення.

## Як це зробити

Це легко зробити в Python, використовуючи символ "+" для об'єднання рядків. Давайте подивимося на приклад коду:

```Python
first_name = "Олександр"
last_name = "Довбуш"
full_name = first_name + " " + last_name
print(full_name)
```

Вивід: Олександр Довбуш

У цьому прикладі ми створили дві змінні з іменем і прізвищем, а потім об'єднали їх, використовуючи символ "+" та додавши пробіл між ними. Результатом є новий рядок, що містить повне ім'я.

## Глибше погляньмо

У Python конкатенація рядків виконується за допомогою оператора "+" або методу "format". Але важливо пам'ятати, що рядки можуть бути об'єднані лише з іншими рядками, тому, якщо вам потрібно об'єднати рядок та іншого типу даних, такого як ціле число, потрібно перетворити його на рядок за допомогою функції "str ()".

Наприклад:

```Python
age = 28
print("Мені " + str(age) + " років")
```

Вивід: Мені 28 років

Також варто зазначити, що у Python існує спеціальний метод "join", який дозволяє об'єднати багато рядків в один. Цей метод може бути корисним при роботі зі списками рядків.
