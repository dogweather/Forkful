## Що і чому?
Читання текстового файлу - це процес отримання даних з файлу та їх подальшого використання в програмі. Програмісти використовують це для збереження або завантаження даних, конфігурацій, тексту для користувачів тощо.

## Як це зробити:
Отже, ось простий спосіб читання текстового файлу в Python:
```Python
# Відкриваємо файл на читання
file = open('file.txt', 'r')

# Читаємо файл
data = file.read()

# Закриваємо файл
file.close()

# Вивід прочитаного файлу
print(data)
```
Виконання цього зане запису вам виведе вміст файлу `file.txt`.

## Поглиблений огляд
Раніше програмісти використовували бібліотеку `os`, щоб працювати з файлами. Python 3 замінив його бібліотекою `io`, що дозволило працювати з файлами в асинхронному режимі. Існують також інші методи читання файлів: `readline()` для читання рядка файлу, `readlines()` для читання всіх рядків у вигляді списку.
