## Чому

Написання коду - це захоплююче і творче заняття, яке дозволяє здійснити ваші ідеї і творити щось нове. Робота над новим проектом може стати стимулом для вас, а також документація для майбутніх проектів.

## Як

```Python
# Створюємо список користувачів
users = ["Володимир", "Олексій", "Анна"]

# Виводимо кожного користувача в списку
for user in users:
  print("Привіт, " + user + "!")
```

Вихідний код виведе:

```Python
Привіт, Володимир!
Привіт, Олексій!
Привіт, Анна!
```

Код прикладу показує, як створити список користувачів і вивести привітання для кожного з них за допомогою циклу for.

## Глибокий дослідження

Перш ніж почати новий проект, важливо чітко сформулювати його мету і завдання. Також необхідно визначити які інструменти та технології будуть використовуватися в процесі розробки. Для кращого розуміння та уникнення проблем у майбутньому, корисно скласти план роботи та намітити кроки для досягнення поставленої мети.
