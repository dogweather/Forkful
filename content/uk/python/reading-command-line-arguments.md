# Чому

Програмування має багато аспектів, але часом, над проектами потрібно працювати самостійно та без графічного інтерфейсу. Це той час, коли командні аргументи стають корисним інструментом для передачі інформації до програми. Їх читання є важливою навичкою для кожного програміста, який хоче працювати з командними інтерфейсами.

# Як
```Python
import sys

# отримуємо список аргументів, переданих з командного рядка
arguments = sys.argv
# перевіряємо, чи існує аргумент
if len(arguments) >= 1:
    # виводимо перший аргумент
    print("Перший аргумент:", arguments[0])
```
Приклад виводу:
```
$ python arguments.py hello world
Перший аргумент: hello
```
Модуль `sys` містить функцію `argv`, яка дозволяє отримати всі аргументи, передані з командного рядка. Це зручний спосіб для передачі параметрів в програму, особливо при її автоматизації. У прикладі, ми виводимо перший аргумент, але ви можете робити з ним, що завгодно.

# Детальний огляд

Командні аргументи є рядками, які передаються з командного рядка при запуску скрипта python. Їхнє читання може бути корисним для інтерактивної роботи з програмами та їх конфігурації. Використовуючи модуль `sys` та функцію `argv`, ви можете отримати доступ до всіх аргументів і обробити їх відповідним чином.
