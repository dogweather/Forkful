# Чому

Вивчення довжини рядка є важливим аспектом програмування в Python, оскільки це дозволяє зробити багато корисних операцій з текстом. Наприклад, ви можете перевірити, чи вміщується певний рядок в обмежену кількість символів або знайти найбільш часто зустрічаючуся літеру. Далі ми розглянемо докладніше, як здійснити це завдання в Python.

## Як

```Python
# Приклад коду для знаходження довжини рядка
text = "Привіт, світ!"
print(len(text))
```

Вищенаведений код виведе на екран число 13, яке є довжиною рядка "Привіт, світ!". Це виконується за допомогою вбудованої функції `len()`, яка підраховує кількість символів у переданому їй аргументі. Відомо, що рядки є незмінними об'єктами в Python, тому їх довжина не може бути змінена.

```Python
# Приклад коду для знаходження кількості символів у введеному користувачем рядку
text = input("Введіть рядок: ")
print(len(text))
```

Цей код дозволяє користувачеві ввести будь-який рядок, а потім виводить його довжину на екран. Це розширює можливості програми, оскільки дозволяє обробляти різні варіанти введених даних.

## Vertoslud

За кулісами, функція `len()` використовує внутрішній метод `__len__` об'єкта, який повертає довжину рядка у вигляді цілого числа. У поєднанні з іншими методами для роботи з рядками, такими як `split()` та `join()`, ви можете створити потужний інструмент для маніпулювання текстом.
