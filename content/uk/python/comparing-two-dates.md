# Чому

Існує багато ситуацій, коли потрібно порівняти дві дати в Python, наприклад для визначення старшої дати або для перевірки, чи минув певний час з моменту останньої події. У цьому пості ми розглянемо, як можна порівнювати дати в Python та надати докладнішу інформацію про цей процес.

## Як це зробити

Для порівняння дат в Python можна використовувати різні методи, такі як `datetime`, `timedelta` та `dateutil`. Одним з найпростіших способів є використання операторів порівняння, таких як `>`, `<` або `==`. Наприклад, щоб порівняти дві дати за допомогою оператора `>`, можна скористатися таким кодом:

```Python
date1 = datetime.datetime(2021, 1, 1)
date2 = datetime.datetime(2021, 1, 15)

if date1 > date2:
    print("Дата 1 є більшою за дату 2.")
else:
    print("Дата 2 є більшою за дату 1.")
```

Отримаємо наступний результат:

```
Дата 2 є більшою за дату 1.
```

Щоб врахувати і час, не лише дату, можна використовувати об'єкти `datetime` з методом `timedelta`, який дозволяє додавати або віднімати певну кількість часу. Наприклад, щоб перевірити, чи минуло 24 години з моменту останньої події, можна використати такий код:

```Python
now = datetime.datetime.now()
last_event = datetime.datetime(2021, 1, 1, 12, 0, 0)

if now - last_event > datetime.timedelta(hours=24):
    print("Від здійснення події минуло більше 24 годин.")
else:
    print("Від здійснення події не минуло 24 годин.")
```

Результат буде залежати від поточного часу, але за умови, що ми запускаємо цей код після 24-го години з моменту події, отримаємо такий результат:

```
Від здійснення події не минуло 24 годин.
```

## Глибокий аналіз

У Python дати можна представляти за допомогою різних об'єктів, наприклад `time.struct_time`, `datetime.datetime`, `datetime.date` чи `dateutil.parser.parse`. Кожен з цих об'єктів має свої особливості і може використовуватися для різних потреб.

Окрім того, багато датасетів укладається в діапазон між двома датами, тому корисно мати набір інструментів для роботи з датами в Python. Є багато пакетів
