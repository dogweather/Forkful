## Навіщо

Екстракція підрядків є корисною технікою в програмуванні Python, оскільки дозволяє вам вибирати певну частину тексту зі строки. Це особливо корисно при обробці великих обсягів даних або при роботі зі строковими даними, які містять багато інформації.

## Як виконати

Для екстракції підрядків в Python використовуються методи `find()` та `split()`. Наприклад, якщо ви хочете витягнути підрядок зі строки “Привіт, світ!” ви можете використати наступні команди:

```Python
s = "Привіт, світ!"
print(s.find("віт")) # виведе 3
print(s.split(",")) # виведе ['Привіт', 'світ!']
```

Наведені приклади показують, що `find()` повертає індекс першого входження підрядка, а `split()` розділяє строку за вказаним роздільником і повертає список підрядків.

Щоб вибрати конкретний підрядок зі строки, використовуйте індексацію. Наприклад, якщо ви хочете витягнути слово “світ” зі строки “Привіт, світ!”, ви можете використати наступне:

```Python
s = "Привіт, світ!"
print(s[7:11]) # виведе "світ"
```

Зверніть увагу, що індексація починається з нуля і закінчується на останньому символі перед двокрапкою.

## Глибоке вивчення

Для того щоб краще зрозуміти екстракцію підрядків в Python, необхідно знати деякі деталі про рядки в цій мові програмування. Рядки є незмінними об'єктами, тобто їх значення не можна змінити після створення. Також, кожен символ у рядку має свій власний індекс, починаючи з нуля.

Для конкатенації рядків відповідає оператор `+`, але пам'ятайте, що він створює новий об'єкт рядка, а не модифікує уже існуючий. Також, Python має багато вбудованих методів для роботи з рядками, таких як `lower()`, `upper()`, `replace()` тощо.

Тепер, коли ви знаєте основні речі про рядки в Python, екстракція підрядків стане для вас ще простішою.
