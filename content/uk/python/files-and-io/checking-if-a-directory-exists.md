## Як це зробити:
Python надає рідні способи перевірки існування директорії за допомогою модулів `os` та `pathlib`. Ось приклади для обох:

### Використовуючи модуль `os`
```python
import os

# Вказати шлях до директорії
dir_path = "/path/to/directory"

# Перевірити наявність директорії
if os.path.isdir(dir_path):
    print(f"Директорія {dir_path} існує.")
else:
    print(f"Директорія {dir_path} не існує.")
```

### Використовуючи модуль `pathlib`
```python
from pathlib import Path

# Вказати шлях до директорії
dir_path = Path("/path/to/directory")

# Перевірити наявність директорії
if dir_path.is_dir():
    print(f"Директорія {dir_path} існує.")
else:
    print(f"Директорія {dir_path} не існує.")
```

### Бібліотеки третіх сторін
Хоча стандартна бібліотека Python достатня для перевірки наявності директорії, бібліотеки, такі як `pathlib2`, можуть бути альтернативами для забезпечення узгодженості між версіями Python або для додаткових функцій.

***Примітка:*** Станом на останні версії Python, `pathlib` достатньо потужний для більшості випадків використання, роблячи бібліотеки третіх сторін менш необхідними для цього конкретного завдання.
