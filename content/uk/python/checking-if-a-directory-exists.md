## Що і чому?

Перевірка наявності каталогу в Python - це процес перевірки, чи існує папка на диску вашого комп'ютера. Програмісти роблять це, щоб запобігти помилкам, які можуть виникнути при спробі роботи з неіснуючим каталогом.

## Як це робити:

Ось декілька прикладів коду:

```Python
import os

# Перевірка наявності каталогу
if os.path.isdir('/path/to/directory'):
    print("Каталог існує")
else:
    print("Каталог не існує")
```

Виведення:

```Python
Каталог не існує
```

## Пірнення глибше:

1. **Історичний контекст:** Функція `os.path.isdir()` була частиною стандартної бібліотеки Python ще від самого початку, що відбиває необхідність в управлінні файловою системою.
2. **Альтернативи:** Можна також використовувати `pathlib`, модуль, отриманий у Python 3.4, який пропонує об'єктно-орієнтований інтерфейс для роботи з файлами і каталогами:
```Python
from pathlib import Path

# перевірка наявності каталогу
if Path('/path/to/directory').exists():
    print("Каталог існує")
else:
    print("Каталог не існує")
```
3. **Деталі реалізації:** Функція `os.path.isdir()` використовує системні виклики операційної системи для перевірки реальної наявності каталогу, тому цей метод є ефективним і точним.
