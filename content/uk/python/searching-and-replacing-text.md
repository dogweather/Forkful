## Чому

Заміна тексту є важливою функцією у програмуванні, оскільки вона дозволяє легко змінювати вміст великих текстових файлів або швидко оновлювати дані бази даних. Це допомагає уникнути дублювання коду та полегшує процес редагування.

## Як це зробити

Для початку, нам потрібен текстовий файл, у якому ми хочемо виконати пошук та заміну. Наприклад, у нашому випадку це може бути файл зі списком імен користувачів. Для того, щоб замінити усі входження одного рядка на інший рядок, ми можемо використати метод `replace()`:

```Python
with open("users.txt", "r") as file:
    data = file.read()

new_data = data.replace("John", "Ivan")

with open("users.txt", "w") as file:
    file.write(new_data)
```

При цьому коді, ми відкриваємо файл, зчитуємо всі дані у змінну `data`, замінюємо слово "John" на "Ivan" у цій змінній, а потім зберігаємо нові дані у нашому файлі. Тепер усі входження слова "John" будуть замінені на "Ivan".

## Детальніше про заміну тексту

У Python є також багато інших способів заміни тексту, таких як регулярні вирази або бібліотеки, які дозволяють працювати з текстом більш ефективно. Наприклад, бібліотека `re` дозволяє виконувати заміну за допомогою регулярних виразів:

```Python
import re

with open("users.txt", "r") as file:
    data = file.read()

new_data = re.sub(r"John\D", "Ivan", data)

with open("users.txt", "w") as file:
    file.write(new_data)
```

Такий код замінить усі входження слова "John" на "Ivan", але тільки якщо за ним не буде розділового знаку (наприклад, коми або крапки). Таким чином, ми можемо виконувати більш точну та гнучку заміну, використовуючи регулярні вирази.
