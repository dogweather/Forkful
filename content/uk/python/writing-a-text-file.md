## Чому

Програмування - це потужний інструмент, що дозволяє створювати різноманітні програми та скрипти для автоматизації завдань. Одним з важливих аспектів програмування є робота з текстовими файлами. Написання текстового файлу дозволяє зберігати дані, створені вами, для подальшого використання, а також дозволяє упорядковувати та аналізувати великі обсяги інформації.

## Як

Існує кілька способів створення текстових файлів в Python, але найпоширенішими є використання вбудованих функцій `open()` та `write()`.

```Python
# Відкриття текстового файлу для запису
f = open("my_text_file.txt", "w")

# Запис текстового рядка до файлу
f.write("Це мій перший текстовий файл!")

# Закриття файлу
f.close()
```

Тепер, якщо ви відкриєте файл "my_text_file.txt", то побачите, що до нього доданий текстовий рядок. Також, ви можете використовувати ці функції для додавання даних до вже існуючого файлу, використовуючи параметр "a" замість "w".

```Python
# Відкриття додаткового файлу для запису
f = open("my_text_file.txt", "a")

# Запис додаткового текстового рядка до файлу
f.write("\nУ мене є ще багато ідей для цього файлу!")

# Закриття файлу
f.close()
```

## Глибоке вивчення

Написання текстових файлів також дає вам можливість керувати форматуванням та розміщенням даних у вашому файлі. Наприклад, ви можете використовувати функцію `format()` для додавання змінних у ваш рядок і встановлення потрібного порядку.

```Python
# Відкриття текстового файлу для запису
f = open("my_text_file.txt", "w")

# Запис текстового рядка із змінними до файлу
name = "Олександра"
age = 25
f.write("Привіт, мене звати {}, мені {} років.".format(name, age))

# Закриття файлу
f.close()
```

Детальніше про роботу з текстовими файлами у Python ви можете дізнатися у [документації](https://docs.python.org/3/tutorial/inputoutput.html#reading-and-writing-files). Також, варто вивчити основи роботи з рядками у Python, оскільки вони використовуються для запису даних у текстові файли.
