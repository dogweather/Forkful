## Для чого

Переважно, коли працюємо з рядками в Python, ми хочемо змінювати їх, додаючи або видаляючи певні символи. Також часто до задачі належить і капіталізувати або робити прописними всі літери в рядку. Це може бути необхідно для виведення інформації у відповідному форматі, для порівняння рядків або для інших операцій з виводом даних. 

## Як це зробити

Найпростіший спосіб капіталізувати рядок використовує метод `.capitalize()`. Потрібно лише викликати цей метод на змінній, вказавши її назву і зберегти результат. Переглянемо приклад:

```Python
x = "привіт світ!"
y = x.capitalize()
print(y)
```

Результат виконання програми буде виглядати наступним чином:

```Python
Привіт світ!
```

Якщо потрібно капіталізувати всі слова в рядку, можна використати метод `.title()`. Він робить першу літеру кожного слова великою, а всі інші - маленькими. Переглянемо приклад застосування цього методу:

```Python
x = "моя улюблена країна - україна"
y = x.title()
print(y)
```

Результат:

```Python
Моя Улюблена Країна - Україна
```

Також можна використовувати методи `.upper()` або `.lower()` для зміни всіх літер у рядку на великі чи маленькі відповідно.

## Глибше занурення

У Python, рядки є незмінними об'єктами, що означає, що ми не можемо змінювати їх напряму. Тому, коли ми капіталізуємо або змінюємо рядки методами, ми створюємо новий об'єкт рядку. Це важливо пам'ятати, особливо при роботі з великими обсягами даних, оскільки це може вплинути на продуктивність програми.

Також, варто зазначити, що капіталізація рядків залежить від поточної кодування. Наприклад, кирилицю можна капіталізувати, щоб отримати готові ініціали, але кодування ASCII не містить властивості для цього. Тому, перед застосуванням методів капіталізації, варто переконатися, що ми працюємо з правильним кодуванням.
