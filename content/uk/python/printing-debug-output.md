## Чому?

Друк відладочного виводу є важливим інструментом для програмістів, який допомагає зрозуміти, що відбувається в коді, коли він виконується. Він може бути корисним для виявлення помилок і вдосконалення роботи програми.

## Як це зробити?

Для друку відладочного виводу використовуйте функцію `print()` у своєму коді. Наприклад, якщо вам потрібно вивести значення змінної `x`, ви можете написати наступний код:

```Python
x = 5
print(x)
```

Це призведе до наступного виводу:

```
5
```

Ви також можете використовувати форматування рядків для виведення більш складної відладочної інформації. Наприклад, щоб вивести значення двох змінних `x` і `y`, ви можете написати наступний код:

```Python
x = 5
y = 3
print("Значення x: {}, значення y: {}".format(x, y))
```

Тоді вивід буде виглядати так:

```
Значення x: 5, значення y: 3
```

## Глибокий погляд

Можна також використовувати відладочне повідомлення для виведення додаткової інформації про кожен крок у виконанні програми. Наприклад, якщо ви використовуєте цикл `for` для обробки списку, можна додати відладочне повідомлення, щоб побачити, які значення проходять через цикл.

```Python
my_list = [1, 2, 3, 4, 5]

for num in my_list:
  print("Змінна num має значення: {}".format(num))
```

Це призведе до наступного виводу:

```
Змінна num має значення: 1
Змінна num має значення: 2
Змінна num має значення: 3
Змінна num має значення: 4
Змінна num має значення: 5
```

Також, для зручності, можна використати спеціальний параметр `end` у функції `print()` для виведення повідомлення на тому ж рядку.

```Python
print("Вивід повідомлення", end="")
print("на тому ж рядку.")
```

Результат буде виглядати так:

```
Вивід повідомлення на тому ж рядку.
```
