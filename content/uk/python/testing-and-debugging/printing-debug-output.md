## Що це та навіщо?
Виведення дебаг інформації - це коли ви друкуєте в консоль дані, щоб зрозуміти, що відбувається у вашій програмі. Програмісти це роблять, аби швидко знайти помилки та переконатись, що все працює як задумано.

## Як це робити:
Використайте `print()` для друку в консоль. Це найпростіший спосіб побачити, що відбувається з вашим кодом.

```python
# Приклад простого друку в консоль
x = "Привіт, світ!"
print(x)  # Виводить: Привіт, світ!

# Використання дебагу з форматуванням
name = "Олексій"
age = 30
print(f"Ім'я: {name}, Вік: {age}")  # Виводить: Ім'я: Олексій, Вік: 30
```

## Пірнаємо глибше:
Друкування в консоль є найстарішим методом для дебагінгу. Його альтернативи інклюдують логування до файлу або використання інтегрованих середовищ розробки (IDE) з детальними налагоджувачами. Логування краще для вирішення серйозних проблем та збору даних у продакшені, а налагоджувачі дають більше контролю, але `print()` є найшвидшим способом отримати базове розуміння коду.
