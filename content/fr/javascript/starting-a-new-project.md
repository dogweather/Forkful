## Quoi & Pourquoi ?
Commencer un nouveau projet de programmation, c'est comme établir une nouvelle colonie sur une planète vierge. C'est une façon pour les programmeurs de donner vie à leurs idées, de résoudre des problèmes et de créer de nouvelles applications.

## Comment faire :
Créer un nouveau projet JavaScript est simple. Vous pouvez vous servir de votre éditeur de texte préféré ou utiliser un environnement de développement intégré (IDE) pour un travail plus structuré. Voici à quoi ressemble le tout début d'un projet JavaScript :

```Javascript
// Créez un nouveau fichier JavaScript, par exemple `monProjet.js`
// Écrivez le code suivant :

console.log("Bienvenue dans mon nouveau projet!");

// Enregistrez le fichier et exécutez-le dans votre environnement de développement
// Vous devriez voir affiché : "Bienvenue dans mon nouveau projet!"
```

## La totale
Historiquement, JavaScript a été créé pour rendre les sites web plus interactifs. Aujourd'hui, ce langage a évolué pour atteindre des plateformes hors navigateur comme Node.js.

Il existe des alternatives à JavaScript. Par exemple, TypeScript (qui est un sur-ensemble strictement typé de JavaScript), Dart, ou Elm. Mais, JavaScript reste le choix de prédilection pour de nombreux programmeurs en raison de sa flexibilité, son énorme communauté et son support omniprésent dans les navigateurs.

Lorsque vous commencez un nouveau projet JavaScript, vous pouvez choisir de partir de zéro ou d'utiliser des modèles de projets, aussi appelés "boilerplates" ou des générateurs de projets comme `create-react-app` pour les applications React ou `vue-cli` pour les applications Vue.js.
