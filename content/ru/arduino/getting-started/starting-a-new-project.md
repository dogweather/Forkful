## Как это делается:
```Arduino
// Создайте простой скетч Blink, чтобы начать новый проект на Arduino

void setup() {
  pinMode(LED_BUILTIN, OUTPUT); // Установите встроенный LED как выход
}

void loop() {
  digitalWrite(LED_BUILTIN, HIGH); // Включите LED
  delay(1000);                     // Подождите секунду
  digitalWrite(LED_BUILTIN, LOW);  // Выключите LED
  delay(1000);                     // Подождите еще секунду
}
```

Подключите вашу плату Arduino, загрузите скетч и наблюдайте, как встроенный LED мигает каждую секунду.

## Подробный разбор
Когда вы начинаете новый проект на Arduino, вы следуете по стопам бесчисленного количества изобретателей и любителей. Arduino был запущен в 2005 году в Иврее, Италия, как инструмент для студентов без знаний в области электроники и программирования. С тех пор он стал основой в DIY электронике, создании прототипов и образовательном кодировании.

Существуют альтернативы началу проекта с нуля. Вы можете модифицировать существующий код или использовать библиотеки, чтобы добавить сложные функции, не изобретая велосипед – но ничто не сравнится с волнением от создания чего-то уникально вашего.

Скетч начинается с функции `setup()`, которая выполняется один раз для настройки вашего оборудования, за ним следует функция `loop()`, которая выполняется непрерывно, позволяя вам контролировать поведение вашего проекта. Освойте использование и структуру этих функций, и вы будете на правильном пути к тому, чтобы стать профессионалом Arduino.
