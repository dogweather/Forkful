## Что и Зачем?

Получение текущей даты в Go включает захват даты настоящего момента из системы. Программисты отслеживают время для того, чтобы ставить временные отметки на события, планировать задачи или просто показывать пользователям текущую дату.

## Как это сделать:

Вот как можно получить текущую дату в Go:

```Go
package main

import (
	"fmt"
	"time"
)

func main() {
	currentTime := time.Now()
	fmt.Println("Текущая Дата:", currentTime.Format("2006-01-02"))
}
```

Если вы запустите это, вы получите вывод, похожий на:
```
Текущая Дата: 2023-04-05
```
Формат `"2006-01-02"` может показаться странным, но в Go этот конкретный порядок является справочным форматом для форматирования даты.

## Подробнее

До появления Go, языки программирования имели каждый свои странные способы управления датами и временем. Go это упрощает, в основном. Почему числа `2006-01-02 15:04:05`? Это мнемоника — числа от 1 до 7 в порядке возрастания, соответствующие `год-месяц-день час:минута:секунда`.

Есть альтернативы, такие как использование времени Unix (секунды с 1 января 1970 года), но это менее удобочитаемо. Пакет `time` в Go предоставляет утилиты для обоих форматов и более того, учитывая часовые пояса и переход на летнее время. Под капотом, `time.Now()` связывается с часами вашей системы для получения текущей даты и времени.
