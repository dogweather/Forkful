## Что и почему?
Сравнение двух дат означает проверку их взаимоотношений: одна раньше, позже или совпадает с другой? Программисты делают это для управления сроками, планирования событий или отслеживания длительности.

## Как это сделать:

```Go
package main

import (
	"fmt"
	"time"
)

func main() {
	// Определение двух дат
	date1 := time.Date(2023, time.April, 1, 0, 0, 0, 0, time.UTC)
	date2 := time.Now()

	// Сравнение дат
	if date1.Before(date2) {
		fmt.Println("Date1 раньше Date2")
	} else if date1.After(date2) {
		fmt.Println("Date1 позже Date2")
	} else {
		fmt.Println("Date1 совпадает с Date2")
	}
	
	// Получение продолжительности между датами
	duration := date2.Sub(date1)
	fmt.Printf("Продолжительность между датами: %v\n", duration)
}
```

Пример вывода для запуска 2 апреля 2023 года:

```
Date1 раньше Date2
Продолжительность между датами: 24ч0м0с
```

## Подробнее
В старые времена сравнение дат в программировании было головной болью — представьте сложные расчёты и постоянную отладку ошибок. Go упрощает этот процесс с помощью пакета `time`. Методы `Before()`, `After()` и `Equal()` позволяют легко сравнивать объекты `Time`.

У вас есть альтернативы. Вы могли бы вручную сравнить год, месяц и день, но это значит больше кода для достижения того же результата. Или вы могли бы использовать сторонние библиотеки, хотя стандартная библиотека Go обычно вполне достаточна.

Технически, `Sub()` возвращает тип `Duration`, который вы можете преобразовать в секунды, минуты, часы или даже наносекунды. Помните, часовые пояса могут вас подвести; всегда учитывайте их при сравнении дат.
