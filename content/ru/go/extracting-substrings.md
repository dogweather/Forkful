## Что и Почему?
Извлечение подстрок означает вырезание частей из строки. Программисты делают это, чтобы изолировать, анализировать или манипулировать конкретными фрагментами данных внутри более крупной строки.

## Как это сделать:
Go делает это легко благодаря стандартной библиотеке и срезам. Вот краткое руководство:

```Go
package main

import (
	"fmt"
	"strings"
)

func main() {
	phrase := "The quick brown fox jumps over the lazy dog"
	
	// Использование срезов
	part := phrase[4:9]
	fmt.Println(part) // Вывод: quick
	
	// Использование пакета strings
	start := strings.Index(phrase, "brown")
	end := start + len("brown")
	substring := phrase[start:end]
	fmt.Println(substring) // Вывод: brown
}
```

## Подробное изучение
Краткий исторический экскурс: Go появился на сцене в 2009 году как проект с открытым исходным кодом, чтобы сделать программирование более веселым и продуктивным. В нем работа со строками была максимально упрощена — для простых задач не требовались регулярные выражения. В других языках, например, в Python, есть похожие механизмы срезов.

Конечно, существуют альтернативы, такие как пакеты `regexp` и `bytes` для более сложных задач. Однако базовая функция `Index` и срезы покрывают большинство нужд без усложнений. Под капотом строки в Go — это просто срезы байтов. Так что, когда вы делаете срез строки, вы на самом деле создаете новый заголовок среза, указывающий на исходный массив строки. Это делает извлечение подстрок в Go быстрым и эффективным в плане использования памяти.
