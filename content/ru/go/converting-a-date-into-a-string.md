## Что и Почему?
Преобразование даты в строку означает изменение формата даты с того, который используется компьютером, на тот, который легко могут прочитать люди. Программисты делают это для отображения дат на интерфейсах или для форматирования их для отчетов и логов.

## Как это сделать:
В Go преобразование даты в строку довольно простое с помощью пакета `time`.

```go
package main

import (
	"fmt"
	"time"
)

func main() {
	currentTime := time.Now()
	fmt.Println("Отформатированная Дата:", currentTime.Format("2006-01-02 15:04:05"))
}
```

На выходе может получиться что-то вроде этого:
```
Отформатированная Дата: 2023-04-07 14:21:34
```

Метод `Format` использует специальную опорную дату: Пн Янв 2 15:04:05 MST 2006. Вы сопоставляете желаемый формат с раскладкой этой опорной даты. Хитрый трюк, правда?

## Глубокое Погружение
Пакет `time` в Go обрабатывает операции с датой и временем. Метод `Format` из структуры `time.Time` является рабочей лошадкой.

Почему такая странная опорная дата "2006-01-02 15:04:05"? В ранние дни Go этот шаблон был выбран потому, что числа (от 1 до 7) уникальны и увеличиваются на 1, так что каждое представляет разный компонент формата времени. Это делает его причудливым, но интуитивно понятным, как только вы это поймете.

Альтернативы? Конечно, у нас есть сторонние библиотеки, такие как `timeparse` или `strftime`, которые имитируют обработку времени в других языках. Но для большинства из нас стандартная библиотека прекрасно справляется с задачей.

За кулисами, форматирование включает в себя анализ оформления опорного времени и замену частей соответствующими значениями из фактически форматируемого времени. Оно также обрабатывает преобразование часовых поясов – обязательный аспект для глобальных приложений.
