## Что и Почему?
Преобразование дат в строки изменяет объект даты на текстовый формат. Программисты делают это для того, чтобы отображать даты удобным для пользователя способом или для подготовки их к хранению в текстовом формате, например, CSV или JSON.

## Как это сделать:
Python упрощает преобразование дат в строки. Используйте метод `strftime`, доступный у объектов даты. Вот как это делается:

```Python
from datetime import datetime

# Получить текущую дату и время
now = datetime.now()

# Преобразовать в строку в формате: Месяц день, Год
date_string = now.strftime("%B %d, %Y")
print(date_string)  # Вывод: 29 марта, 2023 (или текущая дата)

# Формат: ГГГГ-ММ-ДД
iso_date_string = now.strftime("%Y-%m-%d")
print(iso_date_string)  # Вывод: 2023-03-29 (или текущая дата)
```

## Подробнее
Исторически преобразование даты в строку всегда было важной частью программирования из-за необходимости представления дат в формате, понятном человеку.

Альтернативы `strftime` включают использование метода `isoformat` для формата ISO 8601 или сторонние библиотеки, такие как `arrow` и `dateutil`, которые предлагают более гибкие варианты разбора и форматирования.

С точки зрения реализации, `strftime` означает "string format time" (формат времени строки) и имеет корни в программировании на C. `strftime` в Python интерпретирует форматирующие коды, такие как `%Y` для года и `%m` для месяца, что позволяет настроить вывод почти бесконечным числом способов.
