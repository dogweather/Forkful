## Что и Зачем?

Получение текущей даты означает выяснение сегодняшней даты в соответствии с внутренними часами системы. Программисты делают это для маркировки логов, валидации событий и добавления временных меток к данным.

## Как это сделать:

Вам нужно будет подключить `time.h`, чтобы работать со временем в C.

```C
#include <stdio.h>
#include <time.h>

int main() {
    time_t t = time(NULL);
    struct tm tm = *localtime(&t);
    
    printf("Текущая дата: %02d-%02d-%d\n", tm.tm_mday, tm.tm_mon + 1, tm.tm_year + 1900);
    
    return 0;
}
```

Пример вывода:
```
Текущая дата: 15-04-2023
```

## Подробнее

Исторически, работа со временем в C ведет свое начало с ранних дней UNIX благодаря сильным системным возможностями C. Для получения текущих дат мы полагаемся на библиотеку `time.h`, которая существует с тех пор, как C был стандартизирован ANSI.

Тип `time_t` хранит текущее время с момента Epoch (00:00:00 UTC 1 января 1970 года) в секундах. Функция `localtime` преобразует это время в `struct tm`, которая содержит календарную дату и время, разбитые на составляющие.

Альтернативы? Существуют и другие способы манипулирования и представления времени в C. Например, `gmtime` конвертирует `time_t` в координированное всемирное время (UTC) вместо местного времени, которое обрабатывает `localtime`. Используя `strftime`, вы можете широко настраивать формат даты и времени.

Что касается деталей, `time_t` обычно является целочисленным или вещественным типом. Реализация может различаться на разных системах, но стандарт не требует точного типа, лишь того, чтобы он был способен представлять времена.

Когда вы используете функции, связанные с временем, помните, что нужно учитывать летнее время и данные, специфичные для локали, если ваше приложение к этому чувствительно.
