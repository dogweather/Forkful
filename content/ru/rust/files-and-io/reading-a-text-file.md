## Что и Почему?
Чтение текстового файла заключается в получении текстового содержимого из файла с расширением .txt на вашем диске. Программисты делают это для работы с данными, такими как конфигурация, пользовательский ввод, или для обработки больших объемов текста.

## Как:
Стандартная библиотека Rust делает процесс чтения файлов простым.

```Rust
use std::fs::File;
use std::io::{self, Read};

fn main() -> io::Result<()> {
    let mut file = File::open("example.txt")?;
    let mut contents = String::new();
    file.read_to_string(&mut contents)?;
    println!("Содержимое файла:\n{}", contents);
    Ok(())
}
```
Этот код открывает "example.txt", читает его и печатает содержимое.

Пример вывода:
```
Содержимое файла:
Привет, Rustaceans!
```

## Подробнее
Исторически, ввод-вывод файлов может быть сложным, но Rust упрощает это. Существуют альтернативы `read_to_string`, например, использование `BufRead` для построчной обработки, что эффективнее для больших файлов. Внутри Rust чтение файлов использует системные вызовы уровня ОС, буферизуя данные для повышения эффективности.

После выпуска Rust 1.0 язык подчеркивает безопасные системные взаимодействия — чтение файла не является исключением. Тип `Result` инкапсулирует потенциальные ошибки, делая Rust устойчивым к распространенным проблемам, таким как отсутствие файлов или проблемы с разрешениями, без необходимости прибегать к панике.
