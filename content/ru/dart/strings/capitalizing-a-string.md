## Что и Почему?

Преобразование строки в заглавные буквы включает в себя изменение первой буквы слова или всего предложения в верхний регистр, при этом оставляя остальные символы без изменений. Программисты часто используют эту технику при форматировании ввода пользователя или отображении текста, чтобы обеспечить единообразие или соблюдение грамматических правил в пользовательских интерфейсах.

## Как это сделать:

### Использование встроенных методов Dart

Dart предоставляет простые и понятные методы для манипуляций со строками. Чтобы преобразовать слово или предложение в заглавные буквы, обычно берут первый символ, преобразуют его в верхний регистр, а затем конкатенируют его с остальной частью строки. Вот как это можно реализовать:

```dart
String capitalize(String text) {
  if (text.isEmpty) return text;
  return text[0].toUpperCase() + text.substring(1).toLowerCase();
}

void main() {
  var example = "hello world";
  print(capitalize(example)); // Вывод: Hello world
}
```

### Преобразование каждого слова

Чтобы сделать первую букву каждого слова в строке заглавной, можно разделить строку на слова, преобразовать каждое слово в заглавные буквы, а затем снова соединить их:

```dart
String capitalizeWords(String text) {
  return text.split(' ').map(capitalize).join(' ');
}

void main() {
  var example = "hello dart enthusiasts";
  print(capitalizeWords(example)); // Вывод: Hello Dart Enthusiasts
}
```

### Использование сторонних библиотек

Хотя стандартная библиотека Dart покрывает основные потребности, некоторые задачи могут быть выполнены более удобно с использованием сторонних пакетов. Популярным выбором для расширенных возможностей манипуляции со строками, включая преобразование в заглавные буквы, является пакет [`recase`](https://pub.dev/packages/recase). После его добавления в `pubspec.yaml` вашего проекта, вы можете легко преобразовывать строки в заглавные буквы среди других функциональностей:

```dart
import 'package:recase/recase.dart';

void main() {
  var example = "hello world";
  var rc = ReCase(example);

  print(rc.titleCase); // Вывод: Hello World
}
```

Используя `recase`, вы можете делать первую букву заглавной в отдельных словах, целых предложениях или даже следовать другим соглашениям об оформлении без ручной обработки преобразований строк.
