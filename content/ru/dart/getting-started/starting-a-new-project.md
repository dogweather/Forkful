## Как это сделать:
1. **Установите Dart**:
   Убедитесь, что Dart установлен на вашей системе. Если нет, вы можете скачать его с [https://dart.dev/get-dart](https://dart.dev/get-dart). Проверьте установку с помощью:

   ```shell
   dart --version
   ```

2. **Создайте новый проект Dart**:
   Используйте Dart CLI для генерации нового проекта:

   ```shell
   dart create hello_dart
   ```

   Эта команда создает новую директорию `hello_dart` с простым образцом веб- или консольного приложения в зависимости от вашего выбора.

3. **Изучите структуру проекта**:
   
   Перейдите в директорию вашего проекта:

   ```shell
   cd hello_dart
   ```

   Типичный проект Dart включает в себя следующие ключевые файлы и директории:

   - `pubspec.yaml`: Файл конфигурации, который содержит зависимости вашего проекта и ограничения SDK.
   - `lib/`: Директория, где находится большинство кода Dart.
   - `test/`: Директория для тестов проекта.

4. **Добавьте зависимости**:
   Отредактируйте `pubspec.yaml`, чтобы добавить зависимости. Для веб-проектов рассмотрите возможность добавления `http`, популярного пакета для выполнения HTTP-запросов:

   ```yaml
   dependencies:
     flutter:
       sdk: flutter
     http: ^0.13.3
   ```

   После редактирования получите зависимости:

   ```shell
   dart pub get
   ```

5. **Напишите ваш первый код на Dart**:
   
   В директории `lib/` создайте новый файл Dart, `main.dart`, и добавьте простой код на Dart:

   ```dart
   // Подключаем основную библиотеку Dart
   import 'dart:core';

   void main() {
     print('Привет, Dart!');
   }
   ```

6. **Запустите ваше приложение Dart**:

   Выполните вашу программу на Dart с помощью:

   ```shell
   dart run
   ```

   Вывод должен быть:

   ```
   Привет, Dart!
   ```

Следуя этим шагам, вы успешно начали новый проект на Dart, от установки до запуска вашего первого кода на Dart. Эти основы знаний подготавливают почву для более глубокого изучения богатой экосистемы Dart и его возможностей для создания масштабируемых приложений.
