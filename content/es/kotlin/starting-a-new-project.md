## ¿Qué y Por Qué?
Iniciar un nuevo proyecto en programación significa construir una aplicación desde cero. Los programadores inician nuevos proyectos para resolver problemas o crear nuevas funcionalidades utilizando código.

## Cómo hacerlo:
Para empezar un nuevo proyecto en Kotlin, puedes hacerlo a través de IntelliJ IDEA, que es un IDE popular para Kotlin. Aquí hay un ejemplo de cómo hacerlo:
```Kotlin
// Paso 1: Abre IntelliJ IDEA y selecciona "Nuevo Proyecto".
// Paso 2: Selecciona "Kotlin" en el menú y luego "JVM | IDEA".
// Paso 3: Nombra tu proyecto y selecciona la ubicación en tu sistema.
// Paso 4: ¡Haz clic en "Finalizar" y tu nuevo proyecto de Kotlin está listo para ser codificado!
```
Este es un ejemplo de cómo tu "Hola Mundo" se verá en Kotlin:
```Kotlin
fun main() {
   println("Hola Mundo!")
}
```
Si ejecutas el código anterior, verás el siguiente output:
```
Hola Mundo!
```

## Profundizando
Históricamente, Kotlin fue lanzado en 2011 por JetBrains, la compañía detrás de IntelliJ IDEA, como una forma más segura y cómoda de codificar para la JVM que Java. Dado que Kotlin es interoperable con Java, puedes comenzar un nuevo proyecto de Kotlin incluso si tu código base existente es en su mayoría Java.

Unas alternativas a Kotlin pudieran ser Java, Scala o Groovy, pero lo que destaca a Kotlin es su simplicidad y seguridad de compilación.

Al iniciar un nuevo proyecto de Kotlin, debes pensar en cuál será la arquitectura de tu software. ¿Usarás una arquitectura basada en capas o te inclinarás por una arquitectura más modular? ¿Qué patrones de diseño te servirán mejor?
