## Hoe:
De standaardbibliotheek van Rust maakt deze taak eenvoudig met `std::path::Path` en `std::fs`:

```rust
use std::path::Path;

fn main() {
    let path = Path::new("/some/directory");

    if path.exists() && path.is_dir() {
        println!("Map bestaat!");
    } else {
        println!("Map bestaat niet.");
    }
}
```

Voorbeelduitvoer, als de map bestaat:
```
Map bestaat!
```

Voorbeelduitvoer, als de map niet bestaat:
```
Map bestaat niet.
```

## Diepgaand:
Historisch gezien hadden bestandsbewerkingen meer uitgebreide foutafhandeling nodig, wat de code log maakte. Rust vereenvoudigt dit met beknopte methoden die "gewoon werken". Alternatieven, zoals shell-commando's of andere bibliotheken gebruiken, bestaan maar zijn niet zo gestroomlijnd. De `exists()` methode controleert alleen op het bestaan, niet of het een map of een bestand is; combineer het met `is_dir()` voor mappen.
