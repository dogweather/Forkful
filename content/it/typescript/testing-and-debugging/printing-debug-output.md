## How to:
```TypeScript
function somma(a: number, b: number): number {
    console.log(`Sommando ${a} + ${b}`);
    return a + b;
}

const risultato = somma(5, 7);
console.log(`Risultato: ${risultato}`);
```
Output:
```
Sommando 5 + 7
Risultato: 12
```
