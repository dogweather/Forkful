## How to:
Elm usa funzioni come `String.slice` per estrarre sottostringhe. Ecco come funziona:

```Elm
import String

-- Estrai una sottostringa dalla posizione `start` alla posizione `end`.
substring : String -> Int -> Int -> String
substring text start end =
    String.slice start end text

-- Esempio d'uso:
main =
    let
        text = "Ciao, mondo!"
    in
    substring text 0 4  -- Risultato: "Ciao"
```

Output:
```Elm
"Ciao"
```

## Deep Dive
Il concetto di estrarre sottostringhe risale ai primi giorni della programmazione. Prima di funzioni dedicate, gli sviluppatori dovevano iterare carattere per carattere. In Elm, oltre a `String.slice`, puoi usare `String.left` o `String.right` per ottenere parti di una stringa dal principio o dalla fine.
