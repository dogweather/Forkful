## วิธีการ:
ใน Bash, คำสั่ง `date` คือเครื่องมือหลักของคุณสำหรับการเรียกดูวันที่และเวลาปัจจุบัน นี่คือตัวอย่างบางส่วนของวิธีใช้:

1. **รับวันที่และเวลาปัจจุบันในรูปแบบเริ่มต้น:**

```bash
date
```

*ตัวอย่างผลลัพธ์:*
```
Wed Apr 5 14:22:04 PDT 2023
```

2. **กำหนดรูปแบบผลลัพธ์:** คุณสามารถระบุรูปแบบผลลัพธ์โดยใช้ `+%` ตัวกำหนดรูปแบบ (format specifiers) ตัวอย่างเช่น เพื่อแสดงวันที่ในรูปแบบ YYYY-MM-DD:

```bash
date "+%Y-%m-%d"
```

*ตัวอย่างผลลัพธ์:*
```
2023-04-05
```

3. **รับเวลา UNIX timestamp ปัจจุบัน:** UNIX timestamp เป็นจำนวนวินาทีนับตั้งแต่ Unix Epoch (1 มกราคม 1970) นี่เป็นประโยชน์สำหรับสคริปต์ที่ทำการคำนวณอิงตามความแตกต่างของเวลา

```bash
date "+%s"
```

*ตัวอย่างผลลัพธ์:*
```
1672877344
```

ปกติไม่มีการใช้ไลบรารีของบุคคลที่สามที่ได้รับความนิยมสำหรับการดำเนินการพื้นฐานนี้ใน Bash เนื่องจากคำสั่ง `date` ที่มีมาให้นั้นมีฟังก์ชันความสามารถอย่างครอบคลุม เหตุการณ์ทว่า, สำหรับการจัดการวันที่และเวลาที่ซับซ้อนกว่านี้ โปรแกรมเมอร์อาจใช้ภาษาโปรแกรมหรือเครื่องมืออื่นที่เสนอไลบรารีสำหรับการคำนวณและการวิเคราะห์วันที่ เช่น โมดูล `datetime` ของ Python.
