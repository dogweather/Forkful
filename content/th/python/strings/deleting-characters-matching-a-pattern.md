## วิธีการ:
```Python
import re

# ตัวอย่างสตริง
text = "Hello, World! 1234"

# ลบตัวเลขทั้งหมด
no_digits = re.sub(r'\d', '', text)
print(no_digits)  # ผลลัพธ์: "Hello, World! "

# ลบเครื่องหมายวรรคตอน
no_punctuation = re.sub(r'[^\w\s]', '', text)
print(no_punctuation)  # ผลลัพธ์: "Hello World 1234"

# ลบสระ
no_vowels = re.sub(r'[aeiouAEIOU]', '', text)
print(no_vowels)  # ผลลัพธ์: "Hll, Wrld! 1234"
```

## ลึกลงไป
การลบอักขระที่ตรงกับรูปแบบในข้อความมีรากฐานลึกในวิทยาการคอมพิวเตอร์ ย้อนกลับไปถึงเครื่องมือ Unix ต้นๆ เช่น `sed` และ `grep` ใน Python, โมดูล `re` ให้ความสามารถนี้ โดยใช้ regular expressions—เครื่องมือที่ทรงพลังและหลากหลายสำหรับการประมวลผลข้อความ

ตัวเลือกอื่นๆ นอกเหนือจากโมดูล `re` ประกอบด้วย:
- วิธีการสตริงเช่น `replace()` สำหรับกรณีง่ายๆ
- ไลบรารีของบุคคลที่สามเช่น `regex` สำหรับรูปแบบที่ซับซ้อนขึ้นและการสนับสนุน Unicode ที่ดีขึ้น

ภายใต้ฉากหลัง, เมื่อคุณใช้ `re.sub()`, Python interpreter จะคอมไพล์รูปแบบเป็นชุดของ bytecode, ได้รับการดำเนินการโดยเครื่องจักรสถานะที่ทำการจับคู่รูปแบบโดยตรงบนข้อความนำเข้า การดำเนินการนี้อาจใช้ทรัพยากรมากสำหรับสตริงขนาดใหญ่หรือรูปแบบที่ซับซ้อน ดังนั้นการพิจารณาประสิทธิภาพจึงเป็นสิ่งสำคัญสำหรับการประมวลผลข้อมูลขนาดใหญ่
