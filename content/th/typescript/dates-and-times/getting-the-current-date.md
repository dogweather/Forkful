## วิธีการ:
ใน TypeScript, คุณสามารถใช้ `Date` object เพื่อรับข้อมูลวันที่และเวลาปัจจุบัน นี่คือวิธีที่คุณสามารถทำได้:

```typescript
const currentDate = new Date();
console.log(currentDate);
```

ผลลัพธ์ตัวอย่าง:
```
2023-04-12T07:20:50.52Z
```

ส่วนของโค้ดนี้สร้าง `Date` object ใหม่ที่ประกอบด้วยวันที่และเวลาปัจจุบัน ซึ่งจากนั้นจะถูกพิมพ์ออกมาในคอนโซล คุณยังสามารถจัดรูปแบบวันที่โดยใช้ toLocaleDateString() เพื่อรูปแบบที่อ่านง่ายขึ้น:

```typescript
const currentDate = new Date();
console.log(currentDate.toLocaleDateString());
```

ผลลัพธ์ตัวอย่าง:
```
4/12/2023
```

### การใช้ date-fns
สำหรับการจัดการและจัดรูปแบบวันที่ที่ซับซ้อนยิ่งขึ้น ไลบรารี `date-fns` เป็นทางเลือกที่ได้รับความนิยม ขั้นแรก ติดตั้งผ่าน npm:

```bash
npm install date-fns
```

จากนั้น คุณสามารถใช้มันเพื่อจัดรูปแบบวันที่ปัจจุบัน:

```typescript
import { format } from 'date-fns';

const currentDate = new Date();
console.log(format(currentDate, 'yyyy-MM-dd'));
```

ผลลัพธ์ตัวอย่าง:
```
2023-04-12
```

ตัวอย่าง `date-fns` นี้จัดรูปแบบวันที่ปัจจุบันเป็นสตริงในรูปแบบ "YYYY-MM-DD" ไลบรารีนี้เสนอฟังก์ชันมากมายสำหรับการจัดการวันที่ ทำให้เป็นเครื่องมือที่หลากหลายสำหรับโปรแกรมเมอร์ TypeScript ที่ทำงานกับวันที่
