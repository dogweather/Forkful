# למה: 
בש משמש כלי מצוין לניתוח ועיבוד של קבצי טקסט בסביבת המערכת של לינוקס. המאמר הבא יספק הנחיות ודוגמאות כיצד לקרוא קובץ טקסט בעזרת שפת התכנות הזו.

## איך לעשות זאת:
קריאת קובץ טקסט בש נעשית בעזרת הפקודה ```cat```, שמאפשרת להדפיס את תוכן הקובץ למסך. ניתן להשתמש למשל בפקודה הבאה כדי להדפיס את תוכן הקובץ "text.txt": 
```Bash 
cat text.txt 
```

ניתן גם להשתמש בפקודה ```head``` להדפיס את השורות הראשונות בקובץ או פקודת ```tail``` להדפיס את השורות האחרונות. לדוגמה, כדי להדפיס את השורות הראשונות בקובץ "text.txt" ניתן להשתמש בפקודה הבאה:
```Bash
head text.txt
```

בנוסף, ניתן להשתמש בפקודת ```grep``` כדי לחפש טקסט מסוים בקובץ ולהדפיס את השורות שמכילות אותו. לדוגמה, כדי לחפש את המילה "ברכה" בקובץ "text.txt" ניתן להשתמש בפקודה הבאה: 
```Bash
grep "ברכה" text.txt
```

## כיוונון עמוק:
כדי לקרוא קובץ טקסט מסוים בעזרת שפת בש, ניתן להשתמש בלולאות שמאפשרות לעבור על כל השורות בקובץ ולעבד אותן באופן עצמאי. ניתן גם לשלב את בש עם פקודות אחרות כדי לבצע פעולות מתקדמות יותר על הקובץ, כגון שינוי תוכן או סינון נתונים מיותרים. כמו כן, ניתן גם להשתמש בפקודות נוספות כגון ```awk``` ו- ```sed``` לעיבוד טקסט מורכב יותר.
