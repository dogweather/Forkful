בכתיבת הוראות להפעלת מחשב רבים דורשים לכתוב קבצי מלל בהתאם לפקודות של תוכנה מסוימת. לעיתים מוצאים את עצמם צורכים ליצור ביד קובץ כדי להשתמש בו בתהליך התקנת התוכנה או כדי לשמור בו הערות ומידע נוסף. למרבה המזל, באשר לשרת הבאש כתוביות, יש אפשרות לכתוב קובץ החיוות נתונים בדי נתוני טור, הבנייה עם מחולל פקודות נון גרפי מוסף.

## למה

כמו כן, כתיבת תסריטי פעולה או תוכניות עבור מחשבי קוד פתוח מעניקה כישורים רבים, בעוד כי גם משתמשים בכך. באתר שלי יש ביטחון בכך בעצמי שפקודות כתיבה למחשב נייד וכתיבת תגובות קודים במסמכים לפני ליצור פקודות מענקות אותנטיות שונות.

## איך

כדי לכתוב קובץ מלל, נתחיל בפתיחת הבאש התזמוני שלנו ולמצוא את הקובץ הנדרש (או ליצור כמות היקפית שלך). כדי ליצור קובץ, נחפש את הפקודה 'touch' ונצוות עבור השם (path) המושתת אליו:

```Bash
touch my_file.txt
```

עכשיו, נפתח את הקובץ בעזרת פקודת 'nano' (נלקחת מאתר אינטרנט חינם יחד עם שרת הבאש היו כביכול בצורה מודולת שרה, כך מצוות הפשרות כתובות הכתב:

```Bash
nano my_file.txt
```

ניתן לכתוב כל נתון שנרצה להכניס לקובץ בעזרת שורות נתונים ולשמור אותם על ידי לחיצה על ctrl+s (בניתוח עם אחר), ולצאת מהקובץ על ידי לחיצה על ctrl+x.
