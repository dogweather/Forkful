## מה זה ולמה?: 

שינוי מחרוזת לאותיות גדולות הוא פעולה בה כל האותיות משתנות מהתווים הקטנים שלהם להתווים הגדולים שלהם. מתכנתים מבצעים פעולה זו למגוון סיבות, לדוגמה, להפוך קוד לקריא יותר, או ליצור מחרוזת מתאימה לתהליך של נתונים.

## איך לעשות:

ניתן להמיר את המחרוזות לאותיות גדולות ב-Bash באמצעות הפקודה `tr`. ראו בדוג מטה:

```Bash
$ טקסט="אני שיפור תוכנה"
$ אותיותגדולות=$(echo $טקסט | tr '[:lower:]' '[:upper:]')
$ echo $אותיותגדולות
# תוצאה: אני שיפור תוכנה
```

במקרה זה, המשתנה `$אותיותגדולות` יכיל את המחרוזת שבו כל האותיות השטוחות הומרו לאותיות גדולות.

## שילוף עמוק:

אלטרנטיבה לשימוש ב-`tr` היא שימוש בעיון שילוף ב-Bash. Bash החל מגרסה 4.0 תמיכה בעיון שילוף להמרת אותיות.

```Bash
$ טקסט="אני שיפור תוכנה"
$ אותיותגדולות=${טקסט^^}
$ echo $אותיותגדולות
# תוצאה: אני שיפור תוכנה
```
