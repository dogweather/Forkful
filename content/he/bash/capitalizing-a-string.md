## למה

הפונקציונליות של התכנית הזו היא לשנות את האותיות במחרוזת לרישיות בגודל האותיות המקבילות. לדוגמה, אם המחרוזת היא "hello world", היא תשתנה ל-"HELLO WORLD". זה רלוונטי במגוון תחומים כגון תיעוד טכני, פריסת אפליקציות או צורך בהצגת טקסט בכותרת.

## איך להשתמש

המדריך לשימוש שלנו הוא פשוט מאוד וניתן ליישם בעזרת השפת Bash. כדי לשנות מחרוזת לאותיות רישיות, יש לבצע את הפקודה הבאה:

```Bash
echo "hello world" | sed 's/.*/\U&/'
```

צריך לשים לב שהפרמטר "\U" משנה את המחרוזת לאותיות רישיות והפרמטר "&" מציין את המחרוזת המקורית.

התוצאה שתתקבל היא:

```Bash
HELLO WORLD
```

אם ברצונך לשנות את האותיות לכתיב קפיטלי, תוכל להשתמש בפקודה הבאה:

```Bash
echo "hello world" | sed 's/.*/\L&/'
```

המקום שבו "L" במקור היה "U" משנה את האותיות לכתיב קפיטלי. הפלט יהיה:

```Bash
hello world
```

דבר נוסף שצריך לשים לב הוא שהשינוי ייחשף רק לחדו"א (למה לתרגם לעברית). כלומר, אם תרצה לשנות את המחרוזת לאותיות רישיות טרם יציאה מהסקריפט או התפריט שעובד על המחרוזת שלך, אתה תצטרך לשלב את הפקודה לאחר השינוי הרלוונטי.
