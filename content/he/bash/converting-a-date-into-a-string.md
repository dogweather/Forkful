## למה
המרה של תאריך למחרוזת היא חלק חשוב של כתיבת קוד בבאש. היא יכולה לאפשר לנו להציג נתונים בצורה יותר קריאה ומובנת, ולכן כדאי ללמוד איך לעשות זאת.

## כיצד לעשות זאת
המרה של תאריך למחרוזת מתבצעת באמצעות הפקודה `date` בבאש. ניתן להשתמש בתצורת הפקודה הבאה:

```Bash
date +נתב +ןונשיו
```

כאן נציין פורמט של התאריך שברצוננו להציג בתור מחרוזת. לדוגמה, אם נרצה להציג את התאריך הנוכחי בפורמט של "יום-חודש-שנה" נשתמש בפקודה הבאה:

```Bash
date +%d-%m-%Y
```

ואם נרצה להציג את השעה הנוכחית בפורמט של "שעות:דקות:שניות" נשתמש בפקודה הבאה:

```Bash
date +%H:%M:%S
```

כמו כן, ניתן להוסיף טקסט נוסף לפני או אחרי התאריך באמצעות הצבת הטקסט בתוך גרשיים. לדוגמה, אם נרצה להציג את התאריך הנוכחי עם הוספת הטקסט "היום הוא" נשתמש בפקודה הבאה:

```Bash
date +"היום הוא %d-%m-%Y"
```

כאשר נפעיל את כל אלו בבשלילה, נקבל את התוצאות הבאות:

```
13-10-2021
12:33:45
היום הוא 13-10-2021
```
