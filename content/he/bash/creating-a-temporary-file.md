## מה ולמה?
ערכת הקוד ב-Bash מיועדת ליצירת קובצים זמניים. יצירת קובץ זמני מאפשרת למתכנת לאחסן מידע שאינו נדרש לאחר השימוש הנוכחי. בנוסף, היא מקנה לך את היכולת לאחסן ולהפעיל מספר רב של פקודות במהלך הזמן בצורה מסודרת.

## כיצד ליצור:
הפקודה `mktemp` ב-Bash משמשת ליצירת קובץ או ספריה זמניים. הנה דוגמה של שימוש בפקודה:

```Bash
# יצירת קובץ זמני
tempFile=$(mktemp)
echo "הקובץ הזמני שנוצר הוא $tempFile"
```

אתה יכול לדעת את הנתיב של הקובץ הזמני מהפלט של הדוגמא הנ"ל.

## צלילה עמוקה 
יוצר של Bash, Brian Fox, הקנה למשתמשים את היכולת ליצור קבצים זמניים כדי לייצר ולנהל קבצים באופן זמני.

היתרונות בשימוש במנגנון זה הם:
1. **עקרוניות** : קבצים זמניים הם גשמיים בזמן הריצה של התוכנית.
2. **מסדר** : ניתן להשתמש בשמות של קבצים זמניים מרובים לצורך מדיניות של שמות קבצים מתוכנתת ומסודרת.
3. **אבטחה**: היות וקבצים אלה הם זמניים, הם מוחקים עצמם לאחר סיום התוכנית, מה שמקטין את הסיכון להצפה במידע רגיש.
