## מדוע:

יצירת קובץ זמני משמשת כדי לאכסן מידע שנצטרך רק בזמן קצר ואין לנו צורך בשמירה עליו לתקופה ממושכת. זהו כלי מאוד שימושי כאשר אנו רוצים לכתוב קוד בשפת Bash ולחסוך מקום באחסון.

## איך לעשות את זה:

```Bash
# שנית נשתמש בפקודת touch כדי ליצור קובץ זמני בשם temp.txt
touch temp.txt
```
כעת נוכל לבצע פעולות שונות על הקובץ הזמני, כגון כתיבה אליו, קריאה ממנו ועריכה שלו.

לדוגמה, נוכל לכתוב לקובץ הזמני תוכן באמצעות הפקודה echo ולהדפיס אותו למסך:

```Bash
echo "זהו קובץ זמני מכיל טקסט כלשהו" > temp.txt
cat temp.txt
```

תוצאה:

```
זהו קובץ זמני מכיל טקסט כלשהו
```

וכעת נוכל למחוק את הקובץ הזמני באמצעות הפקודה rm:

```Bash
rm temp.txt
```

## חפר לעומק:

ניתן ליצור קובץ זמני על ידי שימוש בפקודת mktemp כדי ליצור קובץ בשם רנדומלי שיחכה לאחסון המידע הזמני שלנו. הפקודה תחזיק את הפקודה הרלוונטית ליצירת קובץ זמני, כלומר touch או echo:

```Bash
mktemp -t temp.XXX
```

ניתן להשתמש בסימבול XXX במקום בשם הקובץ הזמני באופן שניתן לבחור באופן עצמאי. לדוגמה, ניתן להשתמש ב- XXX בתור השם של הקובץ temp בפקודה הקודמת:

```Bash
touch temp.XXX
```

ואז נדבר לשימוש הקובץ הזמני כרגיל:

```Bash
echo "זהו קובץ זמני מכיל טקסט כלשהו" > temp.XXX
cat temp.XXX
rm temp.XXX
```
