# למה
אכן, למה לכתוב בש אם ישנן שפות תכנות אחרות כמו פייתון וג'אווה שיכולות לעשות את כל מה שניתן לעשות בש? אולי אתם יכולים להסביר לחברים ולמשפחה את המינימום הזה, שניתן לעשות עם כלי הבנייה ידנית החדשה של שלא היה ניתן לעשות בכלל?

# כיצד לעשות זאת
אם אתם מעוניינים להתחיל לגלוש בעולם של התכנות בש, אנחנו רוצים להכיר לכם שימוש אחד מעניין של תכנות בש - מחיקת תווים שמתאימים לתבנית מסוימת.

לדוגמה, אם תנסו למחוק את התווים \ בתוך שבאל כגון / Library / Applications, תזהו שהסדר יתקלקל כי מסתבר שתו הנפח כבד מאוד ושהוא צריך להיות בראש התווים האחרונים בכל מקום. כדי לפתור את הבעיה, תוכלו להשתמש בכלי של שטיפת כל התווים שמתחברים אחד לשני, כתווים שמתאים לתבנית מסוימת.

הכנו עבורכם דוגמאות קוד המדגימות כיצד פקודת הניהול הזו עובדת:

```ש
## מעלית / מבנה
שטיפת קלקוד אני_אני / תוכניות ברביעים **רשמי** אותי
כאשר אני_אני דלתי אני_ספת דרך = {- אני_אני משהי —} אני=רפורמטה או =ניקוברוא קא
- גלוש דרך מערך *?
```

לכל התווים במשפט "שטיפת קלקוד אני_אני / תוכניות ברביעים **רשמי** אותי", יושבו כל התווים שמתאים לתבנית "אני_אני" או "אני_ספת", כך שהמשפט יהיה "שטיפת קלקוד תוכניות ברביעים **רשמי**".
