## מה ולמה?
מדובר בפקודת Bash שמאחזרת את התאריך הנוכחי. תכנותים משתמשים בזה לתיעוד, רישום לוגים, ותזמונים.

## איך לעשות:
כאן תלמדו כיצד להשתמש בפקודות שונות ב-Bash כדי לקבל את התאריך הנוכחי.

```Bash
# להצגת התאריך והשעה הנוכחיים
date

# להצגת התאריך בפורמט מותאם אישית (YYYY-MM-DD)
date +%F

# להצגת השעה בפורמט של 24 שעות
date +%R
```

דוגמאות פלט:

```
Tue Mar 21 14:03:26 IST 2023
2023-03-21
14:03
```

## צלילה עמוקה:
הפקודה `date` ב-Bash היא עתיקה כמו היוניקס עצמו, מופיעה כבר מהגירסאות הראשונות של המערכת. בעולם עם אלטרנטיבות כמו פקודת `strftime` בשפות תכנות אחרות, `date` ב-Bash מאפשרת לנו להשיג תוצאות מיידיות עם פחות מאמץ ובגישה ישירה בליין הפקודה. פרמטר ה-%F שראינו למעלה הוא קיצור של `%Y-%m-%d`, שמוציא את התאריך בסטנדרט הבינלאומי ISO 8601. פרמטר ה-%R מציג את השעה בפורמט של שעה דו-ספרתית, דקות דו-ספרתיות, המתאים לשעון 24 שעות הנפוץ בישראל וברחבי העולם.
