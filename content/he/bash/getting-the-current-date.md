## למה

לומר שתמיד יעיל כאשר אנו רוצים להוסיף את תאריך היום הנוכחי לקובץ או לתוכנית שלנו. בשפת התכנות Bash, אנחנו יכולים לעשות זאת בקלות עם פונקציית `date`. למרבה המזל, ישנן כמה דרכים שונות לבצע זאת, כך שבמקום להידרש לבדוק בשעת קריאת התעודה, אנו נקוד תוכניות מצופות בשפת התכנות הזו.

## איך לעשות זאת

לקבלת תאריך היום הנוכחי בשפת התכנות Bash, יתיאם פקודת `date` לעולם עם בחירות שונות. למשל, אם אנו רוצים להדפיס תאריך עם השנה המלאה, נוכל להשתמש בהבחירה `+% d /% m /% Y`. הנה דוגמאות לנוסחאות נוספות כמו גם התוצאה המיידית:

```Bash
date +%d/%m/%Y
20/06/2021
```

```Bash
date +%B %d, %Y
June 20, 2021
```

כמובן, נוכל גם להשתמש בפרמטרים שונים עבור שעה ואזור הזמן, להיות זמינים לעבוד עם עוד ערכים אחרים כמו שניתן להשתמש בהם. לדוגמה, כאן הנה כמה דוגמאות נוספות שאנו יכולים לשנות את קוד הקודים ל:

```Bash
date +%I:%M:%S%p
04:49:15PM
```

```Bash
date --rfc-3339=ns
2021-06-20 16:51:30.35650836+03:00
```

כדי לראות יותר אפשרויות, אנו יכולים להיעזר בעזרה של [מדריך באש](https://www.gnu.org/software/coreutils/manual/html_node/Calendar-Example.html#Calendar-Example) ארוך יותר שנמצא באתר של [גניו הירבעדינו פרולקטס](https://www.gnu.org/software/coreutils/).
