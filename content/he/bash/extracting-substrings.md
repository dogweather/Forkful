##למה
כאשר אנו מטפלים בקבצים וטקסטים בתכנית הַבַּשׁ בַּלְשׁוֹן, ייתכן שנצטרך לחלץ מילים או תת־מחרוזות מתוך טקסט נתון. זה עשוי להיות נחוץ בשביל לבצע חיפוש או עיבוד מידע באופן מונעזר או באופן אוטומטי.

##כיצד לעשות זאת
ניתן לחלץ תתי־מחרוזות מתוך מחרוזת גדולה באמצעות שתי פקודות בסיסיות: `cut` ו־`grep`. בכדי לחלץ את תת-המחרוזת המבוקשת, נצטרך לתת לשתי הפקודות את הפרמטרים המתאימים. נוכל להשתמש בפקודת `cut` על מנת לחלץ שורה מסוימת מתוך קובץ טקסט כדי לקבל את תת-המחרוזת שצריך. למשל, בכדי לחלץ את התא הראשון של מגבלת תכנית הַבַּשׁ בַּלְשׁוֹן, ניתן להשתמש בפקודה הבאה:

```Bash
cut -f 1 <file_name>
```

פקודת `grep` מאפשרת לנו לחפש ולהתאים מחרוזות מתוך טקסט, כולל תת־מחרוזות מסוימות. נוכל להשתמש בפקודה זו כדי לחפש את כל המחרוזות המכילות את התווים "b a s h". במקרה זה, נרצה להתאים את התת-מחרוזת של "bash" עצמו. הנה דוגמה פשוטה:

```Bash
grep "bash" <file_name>
```
