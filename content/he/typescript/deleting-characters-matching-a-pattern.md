## למה

מתוך כוח הרצון האישי והסקרנות שלך, יכול להיות מטרה למחוק תווים התואמים דפוס מסוים. המשימה הזאת יכולה להיות מבוצעת בכדי לסדר או לנקות נתונים, ליצור טקסט חדש או לממן חיפושים מתקדמים.

## איך לבצע

על מנת למחוק תווים התואמים דפוס מסוים בטיפוסקריפט, ניתן להשתמש בפונקציות כמו `replace` ו- `match` כדי למצוא את התווים המבוקשים ולהחליף אותם בתווים ריקים או בתוכן אחר. הנה דוגמאות לכמה קודים שיכולים לעזור לך למחוק תווים על פי דפוס מסוים:

```TypeScript
// מחליף את התווים המתאימים במחרוזת עם תווים ריקים
let newString = "1a2b3c4d".replace(/[a-z]/g, "");

console.log(newString); // פלט: "1234"

// מחזיר את התווים המתאימים במערך חדש
let matches = "Hello World".match(/[aeiou]/g);

console.log(matches); // פלט: ["e", "o", "o"]
```

כיצד הפונקציות הללו פועלות? הפונקציה `replace` מחפשת את כל התווים התואמים לדפוס המצוי בין הסוגריים המרובעים (`/ /`) ומחליפה אותם עם התווים שכותבים לאחר הפסיק השמור (`""` בדוגמה שניתנה). בנוסף, הפונקציה `match` מחפשת את כל התווים התואמים לדפוס ומחזירה אותם במערך חדש.
