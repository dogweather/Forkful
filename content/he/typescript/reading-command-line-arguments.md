## למה
מדוע אנשים יעניינו לקרוא את פרמטרי השורת פקודה?

## איך לעשות זאת
היכן שניתן לקבל פרמטרים מהשורת הפקודה? למה אנחנו צריכים לקרוא פרמטרים מהשורת הפקודה? המהנות ביותר של פרמטרי השורת הפקודה היא שהם מאפשרים לנו לבצע קוד מתוך סביבת הפקודה במקום להתעסק עם ממשק משתמש גרפי. להלן מספר דוגמאות קוד טיפוסקריפט והפלט שלהם כדי להמחיש איך לקרוא פרמטרים מהשורת הפקודה:

```typescript
// המשתנה process הוא חלק מספרייה בתקופת הריצה של טיפוסקריפט
// ניתן להשתמש ב process.argv כדי לקבל את כל הפרמטרים המועברים מהשורת הפקודה

console.log(process.argv);
```

פלט:

```
[ '/usr/local/bin/node', '/Users/myUsername/Documents/code/index.js' ]
```

בדוגמה זו, אנחנו רואים שישנם שני פרמטרים המועברים מהשורת הפקודה: נתיב לפקודה שנמצאת בשימוש ונתיב לקובץ הסקריפט שאנחנו ברצינו להריץ. אם נצליח לתכנת בצורה נכונה, בדרך כלל אנחנו רוצים להתעלם מהפרמטר הראשון ולהשתמש רק בפרמטר השני. להלן דוגמה נוספת עם פונקציה קטנה שמדפיסה את הפרמטר השני:

```typescript
// אנחנו מעבירים את המנעד השני בתוך הפונקציה process.argv כדי להתחל את החיפוש מן הפרמטר השני
function printParam(paramIndex) {
    console.log(process.argv[paramIndex]);
}

// פלט: 'myParam'
printParam(2);
```

שימו לב שאנחנו משתמשים בפרמטר מספרי שמתאים למספר הפרמטר בשורת הפקודה כדי לגשת לפרמטר הרצוי
