## למה

למה צריך למחוק תווים התואמים לתבנית? בדרך כלל, פעולה זו מתבצעת כאשר נמצאת במצב של לקרוא נתונים מקובץ או מעקוב אחר תהליך ורוצים להסיר כל תווים שאינם מתאימים לתנאים ספציפיים.

## איך לבצע פעולה זו?

ניתן למחוק תווים התואמים לתבנית בעזרת הפונקציה `re.sub()` של הספריית המובנית `re` בפייתון. למשל, ננסה למחוק את כל התווים הגדולים מן האלפאבית ממחרוזת נתונה:

```Python
import re

string = "abc123DEF"
new_string = re.sub("[A-Z]", "", string)
print(new_string)
```

התוצאה שנקבל היא: `abc123`. הפונקציה מחליפה את כל התווים הגדולים בתווים ריקים, משמע שהם נמחקים מהמחרוזת.

## מכירות עמוק יותר

כדי להבין יותר טוב את פעולת מחיקת תווים התואמים לתבנית, נדבר קצת על התבנית עצמה. תבנית היא מערך של תווים או מספרים שמתארים מאפיינים מסוימים. כאשר משתמשים בפונקציה `re.sub()`, ניתן להעביר תבנית כארגומנט ראשון ואת התווים המחליפים כארגומנט שני. בדוגמה שניתן למעלה, תבנית היא `[A-Z]`, שמייצגת את כל התווים הגדולים באלפאבית. כל תו שתואם לתבנית יוחלף עם התו הריק כאשר נשתמש בפונקציה `re.sub()`.
