## Why 

למה: נרצה לתאר מדוע לכתוב קובץ טקסט הוא חשוב ושימושי בתחום התכנות בפייתון. כתיבת קבצי טקסט היא דרך יעילה ונוחה לאחסון דאטה ומידע וכן לשתף מידע בין תוכניות שונות. 

## How To 

כיצד לבצע: על מנת לכתוב קובץ טקסט, ניתן להשתמש בפעולת כתיבה לתוך הקובץ. נבצע זאת באמצעות שני ארגומנטים - שם הקובץ ואופציה לפתיחה כדי לכתוב לקובץ. לדוגמה, אם נרצה ליצור קובץ טקסט חדש ולכתוב טקסט בתוכו, נשתמש בפעולה הבאה: 

```Python
file = open("my_file.txt", "w") # 'w' תופעלות פתיחת תוכן חדיש לקובץ
file.write("כתוב טקסט המיועד לקובץ שאני יוצר.") # כאן נכתוב את הטקסט הרצוי
file.close() # נסגור את הקובץ לאחר שנגמר כתיבת הטקסט
```

רצוי גם לזכור לסגור את הקובץ לאחר כל פעולת כתיבה כדי להבטיח שהנתונים נשמרו ולמנוע שגיאות בעת הפתיחה של הקובץ בעתיד. 

## Deep Dive 

כיוון שכתיבת קבצי טקסט היא נפוצה ונעשית באופן תכליתי, חשוב להבין מה קורה מאחורי הקלעים בעת כתיבת טקסט. כאשר נבצע פעולת כתיבה לקובץ, הנתונים מועברים לעמוד RAM במחשב, ומשם נשלחים למערכת הקבצים כדי לאחסן אותם בקובץ בפועל. מאחורי הקלעים, מחשבנו משתמש בכמה אלגוריתמים כדי לכתוב את הנתונים לקובץ בצורה
