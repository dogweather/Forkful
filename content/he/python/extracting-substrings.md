## למה

בעולם התכנות, יתכן שנתקלתם במצב שבו צריך לאתר מחרוזת מסוימת בתוך מחרוזת ארוכה יותר. כדי למצוא את המחרוזת הרצויה, ניתן להשתמש בכלי עזר שנקרא "חיתוך מחרוזת" או "extracting substrings" באנגלית. מדוע זה משתלם? כי אנו יכולים להעלות דרסתיות לקוד ולשמור על קוד נקי ונגיש.

## איך לבצע

בשפת פייתון, ישנם מספר דרכים לחתוך מחרוזות. אחת הדרכים היא להשתמש בשיטה המצויה במחלקת סטרינג (string) בשפת פייתון. למשל, אם נרצה לחתוך את המחרוזת "Hello World" כדי להשיג את הכתבות "Hello" ו"World", נוכל לעשות כך:

```Python
my_string = "Hello World"
print(my_string[0:5])
print(my_string[6:])
```

פלט:

```
Hello
World
```

אנו משתמשים בסוגריים מרובעים כדי לציין טווח של אלמנטים במחרוזת. המספר הראשון מייצג את האינדקס של האלמנט הראשון שאנו רוצים לכלול בחיתוך, והמספר השני מייצג את האינדקס של האלמנט האחרון שנרצה לכלול. אם רוצים להשתמש בכל האלמנטים מהאינדקס הנתון והלאה, ניתן להשאיר את המספר השני (הגבול העליון) ריק. בנוסף, ניתן להשתמש באינדקסים השליליים, כך שהמחרוזת תחתוכה מהסוף שלה. לדוגמה:

```Python
my_string = "Hello World"
print(my_string[-5:])
```

פלט:

```
World
```
