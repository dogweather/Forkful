תמיד ישנם מקרים בתכנות שבהם רוצים להשתמש במחרוזת באותיות קטנות לטובת ניהול נתונים. זה חיוני כאשר מתייחסים למחרוזות שונות כאשר אותיות גדולות וקטנות מסומנות כשונות. לכן בפוסט הזה נלמד כיצד להמיר מחרוזת לאותיות קטנות בלשונית פייתון.

## מדוע

הסיבה העיקרית להמרת מחרוזת לאותיות קטנות היא לניהול נתונים וחיבורות בקוד שונות. כאשר משתמשים באותיות קטנות ישנו ייצוג ייחודי וקל יותר לניהול של המחרוזות ולהשוואתן.

## איך לעשות זאת

תחילה, נשתמש בפקודה בשם `lower()` שתשים את כל האותיות במחרוזת לאותיות קטנות.

```Python
string = "HeLLo WorLd"
new_string = string.lower()
print(new_string)
```

פלט:

```
hello world
```

כמו כן, ניתן להשתמש בפונקציות מובנות כמו `str.lower()` או `str.casefold()` להמרת מחרוזת לאותיות קטנות גם הן. כמו כן, ניתן להשתמש במתודות כמו `capitalize()` או `upper()` להמרת מחרוזת לאותיות גדולות אם צריך.
