## איך לעשות:
קוד פשוט לקרוא מקובץ:

```Python
with open('example.txt', 'r', encoding='utf-8') as file:
    content = file.read()
    print(content)
```

פלט לדוגמה:
```
שלום, עולם!
דוגמה לטקסט בקובץ.
```

לקרוא כל שורה לתוך רשימה:

```Python
with open('example.txt', 'r', encoding='utf-8') as file:
    lines = file.readlines()
    print(lines)
```

פלט לדוגמה:
```
['שלום, עולם!\n', 'דוגמה לטקסט בקובץ.\n']
```
