## למה

קריאת קובץ טקסט היא חלק חשוב מתהליך התכנות בשפת פייתון. זה מאפשר למשתמש לטעון נתונים מקובץ טקסט ולעבד אותם במספר שורות הוראות קצרות וקלות להבנה.

## איך לעשות זאת

כדי לקרוא קובץ טקסט בפייתון, יש לפתוח את הקובץ עם ההדפסה של פונקציית פייתון "open" ולהשתמש בפעולת קריאה לבצע קריאה נתונים מהקובץ. הנה דוגמא פשוטה:

```Python
file = open("myfile.txt", "r")
print(file.read())
```

בקוד זה, אנו קוראים את הקובץ "myfile.txt" ומדפיסים את התוכן שלו. ניתן גם להשתמש בפונקציות כמו "readline" לקרוא שורה אחת בלתי נראית כדי לעבוד עם קבצים גדולים יותר.

כאשר משתמשים בפעולת קריאה, חשוב לסגור את הקובץ בסוף על מנת למנוע מכשולים לפייתון. כדי לעשות זאת, אנו משתמשים בפונקציה "close":

```Python
file = open("myfile.txt", "r")
print(file.read())
file.close()
```

## החקירה העמוקה

כאשר בוחנים קריאת קובץ טקסט, חשוב לזכור כי כדי לעבוד בצורה אפקטיבית עם הנתונים, עלינו לנהל את הקובץ בצורה נכונה. כך מנותן לנו את הגמישות לעבוד עם מספר קבצים ונתונים שונים באופן יעיל.

כמו כן, חשוב לזכור שקריאת קובץ טקסט אינה רק משמעותה טעינה של נתונים מקובץ חוץ ותצורתם גם מאפשרת כמויות גדולות של מידע להיות ממוזגות יחד בקוד.
