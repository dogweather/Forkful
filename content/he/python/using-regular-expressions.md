## למה

ישנם המון כלים שקיימים בפייתון, אבל יש כלי אחד שימושי במיוחד להתמודדות עם מחרוזות - הביטויים הרגולריים. בעזרתם ניתן לחפש, להחליף ולהתאים מחרוזות וגם לאתר התאמות דפקות בטקסטים.

## איך לעשות זאת

הכירו את כלי הביטויים הרגולריים באמצעות קודים ולדוגמה פלט:

```Python
import re

# חיפוש התאמת מחרוזת
text = "ברוכים הבאים למדור הטכנולוגי של פייתון!"
match = re.search(r"פייתון", text) 
print(match.group()) # פלט: פייתון

# חיפוש אחד או יותר מתאמים
matches = re.findall(r"[א-ת]+", text)
print(matches) # פלט: ['ברוכים', 'הבאים', 'למדור', 'הטכנולוגי', 'של', 'פייתון']
```

## חפירה עמוקה

ניתן להשתמש בביטויים הרגולריים כלי לפתירת בעיות מגוונות, כגון מציאת תבניות מסוימות, תיקונם של מחרוזות ופילוח טקסטים לפי דרישות מסוימות. למעשה, כלי הביטויים הרגולריים מאפשרים לנו לנצל את הייחודיות של המחרוזות ולעבוד איתן בצורה חכמה יותר.
