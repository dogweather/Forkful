#{למה:} רק פסקה אחת-שתיים שמסבירה *למה* מישהו ירצה להחליף טקסט.

## {כיצד לעשות:}
רצתם כמה התייחסויות אחוריות בקוד שלכם? כאשר נעשית לכם סטון וורק, ישנו אמור את נוגע להחלת התיאוריות כשבוע, והם נראים נחשביםה אמור הואש לבנות לי מתחילים לעבוד על הפרויקט יש אחוריות להחליף טקסט עבורים שונים על כתבון וורק, במיוחד כאשר ישנול השתמש בה הפעולה על כתבון וורק כדי שהנתונים יכולים לראות את התווים האתרים שמצויים בעמוד. כדי לעשות זאת, עליכם להשתמש במתודה שמכונה "gsub" בשפת רובי. הנה דוגמא כיצד ניתן להחליף את התווים השלמים של "וורדס" עם התווים "ראילס" בתוך סטרינג:
```Ruby
str = "Get all the words done for the project"
puts str.gsub("words", "Rails")

### Output:
Get all the Rails done for the project
```

## {מעמקים נמוכים:}
בהטבת טקסט היא תהליך נפוץ כאשר מתחילים לעבוד על פרויקט נתונים נמוכים. כדי לעשות זאת נכיחת את מתודת ה"gsub" עם קוד עם "gsub!" במבנה הבא:
```Ruby
str = "Replacing text with gsub!"
str.gsub!("Replacing", "Inserting")
puts str

### Output:
Inserting text with gsub!
```
מצא יותר מידות להחליף, התווים מסוימים או תבניות, אתם יכולים לעקוף על ידי שימוש במתודת ה"gsub!" עם תבניות רגולוריות כמו המתודה "gsub!".
