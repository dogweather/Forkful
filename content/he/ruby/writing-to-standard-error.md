## למה

כל פרוגרמזט מתחיל בכתיבת קוד ובבדיקתו. אבל מה קורה כשהקוד שלי לא מתפעל כמו שצריך ואני לא מצליח להבין למה? למרבה המזל, קיים דרך שיכולה לעזור לנו לאתר את הבעיה בקלות ולזה משמשים כתיבה לתחריתות שגיאות. כתיבה לתחריתות שגיאות היא כלי חשוב מאוד בתכנות, שמאפשר לנו לראות מיד איפה הטעות נמצאת ולתקן אותה במהירות. במאמר הבא תגלו כיצד ניתן להשתמש בכתיבה לתחריתות שגיאות, כדי לייעל את תהליך התכנות שלכם.

## איך לעשות זאת

כדי לכתוב לתחריתות שגיאות בשפת רובי, יש להשתמש בפקודה `STDERR.puts`. זאת פקודה שיעזור לנו להדפיס תגית שגיאה בזמן ריצת התכנית. ניתן להשתמש בפקודה זו כדי להדפיס את הערה כלשהי או לתאר את הבעיה שאנו מנסים לפתור. ניתן להשתמש בכל סוגי המשתנים בשפת רובי, כמו מחרוזות, מספרים ומערכים.

לדוגמה, ננסה להדפיס שגיאה של פעולת חיבור תקינה עם מחרוזת ריקה:

```ruby
# קוד תכנית
string1 = "Hello"
string2 = ""
result = string1 + string2
# קוד תחריתות שגיאות
STDERR.puts "שגיאה: לא ניתן לחבר מחרוזת ריקה"
```

כאשר נריץ את התוכנית ונגיע לפקודה תחריתות השגיאות, נקבל את הפלט הבא:

```
שגיאה: לא ניתן לחבר מחרוזת ריקה
```
