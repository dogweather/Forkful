# למה

כאשר מפתחים תוכניות בשפת פיש של, ייתכן שתצטרכו להמיר תאריך למחרוזת. המאמר הזה ילמד אותכם כיצד לעשות זאת על מנת שתוכלו להשתמש בזה בתוכניות שלכם.

## כיצד לעשות זאת

שפת פיש של מציעה פקודה כדי להמיר תאריך למחרוזת בפורמט שתרצו. לדוגמה:

```Fish Shell
set date (date +%Y-%m-%d)
echo $date
```

תוצאה:

```
2020-05-07
```

## רטיבה משוכללת

כאשר משתמשים בפקודה זו, נתון התאריך יהיה בפורמט שנבחר ע"י המשתמש. תוכלו לבחור מהרשימה הבאה, או לייצר אפילו פורמט משלכם:

- `%Y` - שנה בפורמט שנת 4 ספרות
- `%m` - חודש בפורמט של 2 ספרות
- `%d` - יום בחודש בפורמט של 2 ספרות
- `%H` - שעה בפורמט של 24 שעות
- `%M` - דקה בפורמט של 2 ספרות
- `%S` - שניות בפורמט של 2 ספרות
