# למה

כתיבת קובץ טקסט עשוי לגעול כמו משימה קשה, אך בפועל היא מאפשרת לנו לשמור קוד ומידע בצורה מובנת ומסודרת. זה יכול להפוך את העבודה שלנו כמפתחי תוכנה לפשוטה יותר ומאפשר מתן הבנה טובה יותר של קוד לאחרים.

## איך לעשות זאת

תיבת הקוד הבאה תציג כמה דוגמאות לכתיבת קובץ טקסט בתוך Fish Shell:

```
echo "שלום עולם" > file.txt
cat file.txt
```

פלט של הקוד הנ"ל יהיה:

```
שלום עולם
```

כאשר אנחנו משתמשים בפקודה `echo`, אנו יכולים להוסיף טקסט כדי לשנות אותו ולהוסיף מידע נוסף לקובץ. ניתן גם להשתמש בפקודת `cat` כדי להדפיס את תוכן הקובץ.

## רכיב מקיף

כדי לכתוב קובץ טקסט מקיף יותר עם יותר הגדרות ותכנים, ניתן להשתמש בפקודת `printf` כדי ליצור תבנית ולמלא את התוכן הרצוי בעזרת פרמטרים. לדוגמה:

```
printf "שלום %s, האתר שלך הוא %s" "יוסי" "www.example.com" > file.txt
cat file.txt
```

הפלט של הקוד הנ"ל יהיה:

```
שלום יוסי, האתר שלך הוא www.example.com
```

כאן אנחנו משתמשים בפקודת `printf` כדי ליצור תבנית ואנו מציינים את הפרמטרים בעזרת הסימון `%s`.
