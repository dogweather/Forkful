# למה

כשמדברים על פיתוח תוכניות וסקריפטים בכלל ועל שימוש בכלי פיתוח כגון Fish Shell בפרט, הסיבה העיקרית להיכנס לפרוייקט חדש היא כדי לשפר את היכולות שלך כמתכנת ולהבין יותר על הפלטפורמה. התנסות בכלים חדשים תעזור לך להתפתח כמתכנת עם יכולות רחבות יותר.

# כיצד לתחיל

מתחילים פרוייקט חדש ב-Fish Shell הוא מאוד פשוט. ניקח למשל ביצוע פעולה פשוטה של הדפסת הודעה למסך. נתחיל עם יצירת קובץ חדש באמצעות פקודת `touch` ונסמן את הקובץ כקובץ script על ידי הוספת `.fish` לסיומת שלו. למשל, `touch hello.fish`.
בקובץ נכתוב את הפקודה הבסיסית:

```
echo "Hello World!"
```

נפעיל את הסקריפט על ידי השימוש בפקודת `source` ונראה את הפלט הרצוי בטרמינל:

```
Fish Shell > source hello.fish
Hello World!
```

כעת, יש לנו סקריפט ראשון בפיתוח שלנו ב-Fish Shell.

# חפירה עמוקה

התחלת פרוייקט חדש מאפשרת לנו להתנסות ביכולות המתכנת שלנו ולהתחיל להבין יותר על הפלטפורמה והכלים החדשים שאנחנו משתמשים בהם. לכן, עוד כמה דברים שניתן לעשות בפרוייקט המתחיל הזה הם:

- לדעת יותר על הפקודות הבסיסיות של Fish Shell כמו `ls`, `cat`, `cd` וכו'.
- ללמוד על משתנים ופונקציות ב-Fish Shell וכיצד להשתמש בהם.
- לנסות לכתוב סקריפטים מתקדמים יותר כדי ליישם יכולות גבוהות יותר של השפה.
