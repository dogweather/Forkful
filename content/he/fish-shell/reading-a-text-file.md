## למה

קובץ טקסט הוא כלי חשוב לתקשורת עם המחשב. קריאת קובץ טקסט עוזרת לנו להבין מידע, לעבד אותו וליצור מוצרים חדשים ממנו. על פניו, קריאת קובץ טקסט יכולה להיראות כמשימה פשוטה אבל היא מאפשרת לנו לשלוט על המאמץ הבנייה שלנו ולכן היא חשובה לכל אדם שמתעסק עם תכנות.

## לאיך לקרוא קובץ טקסט בעזרת Fish Shell

באמצעות שימוש בפקודות Fish Shell ניתן לקרוא קובץ טקסט באופן מהיר ויעיל. כדי לקרוא קובץ טקסט באמצעות Fish Shell נדרש לבצע שלושה צעדים פשוטים:

1. פתחו את Fish Shell ונווטו לתיקיית הקובץ שברצונכם לקרוא.
2. השתמשו בפקודה "cat" כדי לראות את התוכן של הקובץ מתוך השורת הפקודה.
3. השתמשו בפקודת "less" כדי לאפשר גלישה בקובץ באמצעות החצים העליונים במקלדת ולסגור באמצעות הפקודה "q".

תוכלו לראות את הפקודות והפלט המתאים שלהן בקטע הקוד הבא שהוא כתוב בתחביר של Markdown כדי להציג את הקוד כמו באמת:

```Fish Shell
cat file.txt

output:
This is a sample text from file.txt
```

```Fish Shell
less file.txt

output:
This is a sample text from file.txt
```

## לעשות צלילה עמוקה

כעת שאתם יודעים איך לקרוא קובץ טקסט בעזרת Fish Shell, ניתן לעשות צעדים נוספים ולצלול עמוק יותר בעולם התכנות. אינכם חייבים להשתמש רק בפקודות "cat" ו"less" כדי לקרוא קובץ טקסט, תוכלו להשתמש גם בפקודות נוספות
