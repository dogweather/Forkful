## למה

למה לחוקרים ומתכנתים צריכה להיכנס לתהליך מציאת אורך המחרוזת? לאחד הסיבות הפשוטות ביותר, שמספר את האורך של המחרוזת הוא חיוני לשימוש בהמון מטרות של תכנות שונות. היית רוצה לדעת כמה כתובות URL יש לך? אולי ברצונך לבדוק אם המחרוזת שלך עובדת כראוי? החישוב של אורך מחרוזת עשוי לשמור על הכיוונים מטקסט או לעשות תקשורת עם מערכת אחרת. לכן, למצוא את אורך המחרוזת הוא פעולה הנחוצה בתהליך התכנות.

## איך לעשות

ניתן להשתמש בשפת התכנות Fish Shell כדי למצוא את אורך המחרוזת בקלות ובמהירות. כדי לעשות זאת, ניתן להשתמש בפקודה `string length` ולהזין את המחרוזת כארגומנט. לדוגמה:

```Fish Shell
string length "זהו דוגמה"
```

תוצאה צפויה: `11`

ניתן גם להשתמש בסימן סוכם כדי למצוא את האורך של מספר מחרוזות בו זמנית. לדוגמה:

```Fish Shell
string length "ארבעה" "עשרה" "שלושים"
```

תוצאה צפויה: `4 10 6`

## Deep Dive

למרבה המזל, בשפת התכנות Fish Shell ישנם מספר פונקציות שמסייעות בחישוב אורך מחרוזת. ניתן להשתמש בפונקציות `wordcount` ו- `math round` כדי לקבל את אורך המחרוזת בצורה חדשנית ומדוייקת יותר. הפונקציה `wordcount` מחשבת את מספר המילים במחרוזת והפונקציה `math round` מעגלת את המספר למספר שלם בלבד. לדוגמה:

```Fish Shell
math round (wordcount "זהו אורך מחרוזת?")
```

תוצאה צפויה: `5`
