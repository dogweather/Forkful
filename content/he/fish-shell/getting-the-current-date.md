## למה:

למה לדאוג לתאריך הנוכחי בתכנות בשפת העט?

בשפת העט סקריפטס קל ונוח להשתמש בתאריך הנוכחי ביצירת תסריטים ותוכניות. לדוגמה, בכדי לשמור תיעוד מתעדכן בפעם אחת ביום, או ליצירת קובץ חדש עם תאריך היום בשמו. יכול להיות גם שתאריך הנוכחי ישמש כהרתעות בכתיבת דואר אלקטרוני אוטומטי, כך שהוא יהיה תמיד מדויק ומעודכן.

## איך לעשות זאת:

לפניכם נמצא דגם לקוד שישמור את התאריך הנוכחי במשתנה וידפיס אותו במסך.

```Fish Shell
#!/usr/bin/env fish

# יצירת משתנה המכיל את התאריך הנוכחי
set current_date (date +%Y-%m-%d)

# הדפסת התאריך במסך
echo "התאריך הנוכחי הוא: $current_date"
```

תוצאה:
```
התאריך הנוכחי הוא: 2021-12-23
```

ניתן כמובן להשתמש גם בפקודת `date` ללא הפרמטרים כדי לקבל את תאריך ושעה מדויקים יותר.

## ירידה לעומק:

במידה ואתם מעוניינים ליצור קובץ חדש עם שם הקובץ המכיל גם את התאריך הנוכחי, תוכלו להשתמש בפקודת `touch` יחד עם הפרמטר `date '+%Y-%m-%d'` בכדי ליצור את הקובץ בשם הנחוץ.

בנוסף, ניתן להשתמש בפקודת `date` גם לצורך מטריית טורי אחרת, כגון כתיבת פקודות אלגוריתם או תוכניות שתבצע חישובים מתבססים על התאריך הנוכחי.
