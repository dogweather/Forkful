# למה

אחת העקרונות החשובים בתכנות הוא ניהול נתונים וטיפול בהם בצורה יעילה ויעילה. האפשרות להקפיטליזציה של מחרוזות ב- Fish Shell היא דוגמה נהדרת לכך, מה שמאפשר למשתמש לשתף בקלות נתונים במבנה ספרות נראה וקריא ביותר.

# איך להשתמש

לקבלת הפלט של מחרוזת רגילה תוך ניצול את יכולת פיש לקפיטליזציה, ניתן להשתמש בפונקציית "upper" כדי להמיר את כל האותיות הקטנות במחרוזת לאותיות גדולות.

```
Fish Shell

upper "שלום עולם"
```

הפלט המתוחזק יהיה "שלום עולם".

# טיפול נמקד

התمודדות עם נתונים בתכנות היא תהליך מורכב וכדאי להישאר פתוח לאפשרויות שונות. פיש מאפשר למשתמשים לקפיטליזציה של מחרוזות גם באמצעות הפונקציות "capitalize" ו-"title", המאפשרות להחזיר מחרוזת שהראיית ההתחלה תמיד תהיה גדולה ושאר האותיות יהיו קטנות.
