# למה

הדבג אאוטפוט (debug output) הוא כלי חיוני לכל מתכנת Swift. הוא מאפשר לנו לראות בצורה ברורה מה קורה בתוך התוכנית שלנו ואיפה כבר אנו נתקלים בבעיות. בעזרת הדבג אאוטפוט ניתן למצוא ולתקן באופן מהיר ויעיל את השגיאות המופיעות בתוכנית שלנו.

# כיצד לעשות זאת

דוגמאות הקוד המופיעות להלן יראו לנו איך להשתמש בדבג אאוטפוט בשפת Swift.

```Swift
// נגדיר משתנה מסוג String
var name = "Maya"

// נדפיס את תוכן המשתנה היוזם
print(name)

// נוסיף נקודות קידוד לתוך הדבג אאוטפוט כדי להציג ערך באופן מתאים לחברות
print("The value of name is \(name)")

```

המספרים הרצים שמופיעים בין הסוגריים המכוונים נקראים "משתנה מתאמים" (placeholders) ונועדו להציג ערכים שונים בהתאמה לסיטואציות שונות. בדוגמא הנ"ל, הערך שנדפס יהיה "The value of name is Maya".

# לחקור עוד

עכשיו שאנחנו יודעים איך להשתמש בכלי הדבג אאוטפוט, בואו נראה מה זה בעצם עושה מתחת לקלעים.

שפת Swift מאפשרת לנו להשתמש במספר שיטות שונות להדפסת חלקים ספציפיים של הקוד. לדוגמא, ניתן להשתמש ב print() על מנת להדפיס ערכים שונים לפי דרישה, או להשתמש ב debugPrint() שמאפשר להדפיס גם ערכים בסגנון "פורמט לנדרואיד" עם פרטים נוספים כגון סוג המשתנה וערכו.

עוד מיהוי הדבג אאוטפוט הוא באפשרות להכניס אותו לתוך מצבי בדיקה (debug mode) או השמה לתוך Logger כדי לשמור
