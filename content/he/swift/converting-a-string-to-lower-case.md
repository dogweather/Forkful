## למה:

מי שעובד עם מחשבים ותוכניות לעיתים קרובות מתקל במצב שבו יש צורך לשנות את האותיות של מחרוזת מסוימת. לדוגמה, אולי אתם רוצים להשוות בין שתי מחרוזות כאשר אתם רוצים שהשוואתם תהיה תלויה באותיות קטנות בלבד. במקרה כזה, יעיל להמיר את המחרוזת לאותיות קטנות. יתכן גם שאתם רוצים לייצר פלט שונה בלבד כאשר התרגיל הנכון הוא להמיר את המחרוזת לקטנה.

## כיצד לעשות זאת:

תחילה, נצטרך להשתמש בפונקציה המתאימה להמיר את המחרוזת לאותיות קטנות - `lowercased()`. נשתמש באותה פונקציה על המחרוזת שכבר יצרנו ונדפיס את הפלט:

```Swift 
let string = "היי, עולם!" 
print(string.lowercased()) 
```

פלט: `היי, עולם!`

כעת נראה כיצד ניתן להשתמש בפונקציה זו במקומות אחרים בקוד. לצורך כך, ניתן להשתמש בפונקציה `lowercased()` על אינסטנס של מחרוזת כלשהי:

```Swift 
let string = "ברוכים הבאים לבלוג שלי!" 
let lowercaseString = string.lowercased() //יצירת מחרוזת חדשה עם אותיות קטנות 
print(lowercaseString) 
```

פלט: `ברוכים הבאים לבלוג שלי!`

למעשה, ניתן גם להשתמש בפונקציה זו על מחרוזת שמכילה תווים מיוחדים כמו רווחים או מקפים:

```Swift 
let string = "בעברית יש מקפים" 
print(string.lowercased()) 
```

פלט: `בעברית יש מקפים`
