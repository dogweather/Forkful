## מדוע

מצבי חשיפת גוף ויקאשי מכל התפתחויות בשלבים הקדומים ביותר הם אלה שאמנם מאפשרים את ההתחברות לעולם האבולוציוני שבו נמצאים אך אינם ניתנים להגישה ישירה ומיידית. זהו המצב גם בתחום התכנות עם ארדואינו, כשמקרים מסוימים דורשים מאיתנו לקרוא ולעבוד עם קבצי טקסט שיתווספו כחלק מן הקוד שלנו. לכן, יש צורך ללמוד כיצד לקרוא ולעבוד עם קבצי טקסט על מנת לשפר את היכולת שלנו לפתח יישומים מתקדמים יותר עם ארדואינו.

## איך לעבוד עם קבצי טקסט בארדואינו

כדי לקרוא קובץ טקסט בארדואינו, ניתן להשתמש בפונקציית “File.read()” ולהעביר לה את השם של הקובץ שרוצים לקרוא. ניתן גם לשים לב שכדי לעבור בין המיקום הנוכחי בקובץ למיקום הבא, ניתן להשתמש בפונקציות “File.available()” ו-“file.seek()”. במקרה של מיקום המידע המעוניין לקרוא בקובץ, יש להשתמש בפונקציית “File.readBytesUntil()” ולעבור על כל תו בלי לחזור על התהליך מחדש מהתחלה.

ניתן להתאר מערכת תכנון שמייצגת את המידע מתצוגת זו׃

```Arduino
File file = SD.open("textfile.txt");

if (file) {
  while (file.available()) {
    char text = file.read();
    Serial.print(text);
  }
  file.close();
}
else {
  Serial.println("Error opening text file.");
}
```
