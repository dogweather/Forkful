## למה

למה לבדוק את הקיום של ספרייה? בדיקה של קיום ספרייה יכולה לעזור לנו לוודא כי תיקייה זו קיימת ואם לא, נוכל להתאים את הקוד שלנו כך שיכיל בדיקה נוספת על מנת ליצור את התיקייה החסרה בכדי להיות מוכנים לכל תרחישי הפעולה.

## כיצד לבדוק את הקיום של ספרייה

בשביל לבדוק קיום של ספרייה, אנחנו נשתמש בפונקציה ```exists()``` המובנית של ארדואינו. הפונקציה הזאת מקבלת כפרמטר את שם הספרייה המבוקשת ומחזירה ערך בוליאני ( ```true``` או ```false```) כדי לציין אם הספרייה קיימת או לא. הנה דוגמא קוד שמבצע את הבדיקה:

```Arduino
// קוד שבודק את קיום הספרייה "Wire" ומדפיס את התוצאה
if (exists("Wire")) {
    Serial.println("הספרייה קיימת!");
} else {
    Serial.println("הספרייה לא קיימת!");
}
```

כאשר תריצו את קוד זה על יחידת ה-ארדואינו שלכם, תיראו את הפלט הבא בקונסול:

```
הספרייה קיימת!
```

דוגמא נוספת, אם נרצה לבדוק את קיום של ספרייה שאינה קיימת, הנה מה נקבל:

```Arduino
// קוד שבודק את קיום הספרייה "SD" ומדפיס את התוצאה
if (exists("SD")) {
    Serial.println("הספרייה קיימת!");
} else {
    Serial.println("הספרייה לא קיימת!");
}
```

פלט:

```
הספרייה לא קיימת!
```
