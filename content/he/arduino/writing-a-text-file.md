## מדוע

החלוקה של קוד על פני מספר קבצים יכולה להיות משמעותית לקוראי קוד וגם לכתבי קוד. כך, כאשר המתכנת כותב קובץ טקסט, הוא יכול לבצע את זה בצורה ארגונית ומסודרת יותר, על מנת לקבל קוד נקי ואחראי באמת, כפי שראוי לכל כתב קוד טוב.

## איך לעשות זאת

הנה כמה דוגמאות כדי להדגים את השימוש בקבצי טקסט ב-Arduino:

```Arduino
// קוד לכתיבת טקסט לקובץ חדש

// ייבוא הספריות המתאימות
#include <SD.h>
#include <SPI.h>

// פתיחת קובץ חדש לכתיבה
File myFile = SD.open("newfile.txt", FILE_WRITE);

// בדיקה אם הקובץ נפתח
if (myFile) {
  // כתיבת טקסט לקובץ
  myFile.println("זהו הטקסט שיהיה נכתב בקובץ החדש!");
  myFile.close();
}

// ניתוב הודעה בפלט ()Serial Monitor
Serial.println("קובץ חדש נוצר וניתן לכתוב טקסט אליו.");
```

פלט בפלט ()Serial Monitor יהיה:

```
קובץ חדש נוצר וניתן לכתוב טקסט אליו.
```

וכאשר אנו נגלה את קובץ הטקסט, המידע הבא יוצג בו:

```
זהו הטקסט שיהיה נכתב בקובץ החדש!
```

## חקירה עמוקה

כפי שניתן לראות מהדוגמאות שנתבארו, כתיבת טקסט לקובץ באמצעות קוד Arduino נעשה באמצעות פתיחת קובץ על ידי הספריות SD ו-SPI ושימוש בפעולות כמו println כדי לכתוב טקסט לקובץ. ניתן להשתמש בפקודות נוספות כדי לקרוא או למחוק מידע מקובץ ולהתאים את הקוד לצורך שלכם.
