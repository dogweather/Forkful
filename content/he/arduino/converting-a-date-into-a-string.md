## למה

המרת תאריך למחרוזת בקוד ארדואינו יכולה להיות מועילה במגוון דרישות תכנותיות, כגון יצירת רשומות מונות כך שהן יכולות להיות מוצגות בצורה קצרה וברורה, או בדיקת תקינות של תאריכים בקלות. כמו כן, המרת תאריך למחרוזת יכולה לקלט את התאריך בצורה שנוחה יותר לקריאת האדם.

## איך לצור תאריך ארוך בארדואינו

מטרת הקוד הבא היא ליצור תאריך בפורמט יום/חודש/שנה ולהדפיסו בתור מחרוזת. נשתמש בספריה "TimeLib.h" כדי לגשת לפונקציות של ניהול תאריכים.

```Arduino
#include <TimeLib.h>

void setup() {
  Serial.begin(9600);               // יצירת חיבור סיריאל
  setTime(15, 30, 00, 4, 12, 2021); // יצירת תאריך 15:30 יום רביעי, 4 לחודש דצמבר 2021
}

void loop() {
  Serial.print("היום יום ");
  Serial.print(day());            // קריאת יום
  Serial.print("/");
  Serial.print(month());          // קריאת חודש
  Serial.print("/");
  Serial.println(year());         // קריאת שנה
  delay(1000);
}
```

**פלט:**

```
היום יום 4/12/2021
```

ניתן להשתמש בפונקציות נוספות כדי להוסיף פרטים נוספים לתאריך, כמו שעה ודקות. ניתן למצוא רשימה מלאה של הפונקציות הקיימות בספריה "TimeLib.h" כאן: https://playground.arduino.cc/Code/Time.
