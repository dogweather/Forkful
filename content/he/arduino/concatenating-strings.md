## למה

בתכנות Arduino, קריאה לחלק מידע מאוסף של תווים והתאמה למתחברות היא הרבה פעמים חיונית. התווים יכולים להיות הנתונים שאנו מצפים למתחברת לספק, או גם מילים נוספות לתיאור הנתון.

## איך לעשות

ראשית, נצטרך להגדיר משתנה של סוג מחרוזת. נשתמש בפקודת `String` כדי ליצור משתנה חדש. נוסיף תוכן למשתנה על ידי חברת שתי מחרוזות כדלקמן:

```Arduino
String name = "מרים";
String sentence = "שלום " + name;
```

כעת, נוכל להדפיס את המשתנה `sentence` כדי לקבל את התוצאה הבאה:

```Arduino
Serial.println(sentence); // פלט: "שלום מרים"
```

## חקירת מעמיקה

מחרוזות הן סדרת תווים ותכונותיהן דומות למערך של תווים אחד או יותר. אפשר להוסיף תווים באופן אוטומטי למשתנה `String` על ידי חיבור מחרוזת למשתנה קיים. כדי לחבר יותר ממחרוזת אחת, ניתן להשתמש בפקודות כמו `concat()` או `+=`.

ניתן גם להשתמש בפקודה `charAt()` כדי לקבל או לשנות תו ספציפי במחרוזת. ניתן גם למצוא את אורך המחרוזת עם הפקודה `length()`. כדי לקבל תת-מחרוזת ממחרוזת קיימת, ניתן להשתמש בפקודה `substring()` עם מיקום התווים הרצויים.

לדוגמה, נבחן את הקוד הבא שמציג תוכנית נפרדת:

```Arduino
String sentence = "Hello, World!";
Serial.println(sentence.charAt(7)); // פלט: "W"
```
