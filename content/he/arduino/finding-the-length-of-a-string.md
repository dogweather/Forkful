## למה
רובנו מתחילים ללמוד תכנות בארדואינו כדי ליצור פרוייקטים מהנים ומופלאים. פעולות יסודיות כמו מציאת האורך של מחרוזת יכולות להיות מאגר יסודי בקוד שלנו. פרסום קוד ארדואינו יכול להיות מאתגר, אך עם כמה שההתחלה תופעה מפחידה, יש לנו המון מאמץ כדי לקצר את הפעולות השונות שאנו רוצים לבצע. אז למה כדאי ללמוד איך למצוא את האורך של מחרוזת בקוד ארדואינו?

## איך לעשות זאת
העלאת תכניות ארדווינו היא בסגנון של Java, ולכן אם יש לכם ניסיון עם זה, תוכלו להיות די מסודר בארדווינו. למרות זאת, אתם עדיין תצטרכו ללמוד כמה דברים חדשים כדי לתמוך בתכניות הקטנות שתרצו לכתוב. אחד הדברים המאתגרים היא מציאת אורך של מחרוזת. עם זאת, אל חשש. ניתן לבדוק את האורך של מחרוזת בקלות באמצעות פקודה פשוטה:

```Arduino
String name = "ארדווינו";
int length = name.length(); // ניתן להעביר את המחרוזת המבוקשת בתוך סוגריים לצד המחרוזת
Serial.println(length); // אפשרות להדפיס את האורך של המחרוזת
```

התוצאה שאנחנו מקבלים היא 8, כמו כן, יש עוד כמה דרכים לעשות את המשימה באותה הפשטות:

```Arduino
char name[] = "ארדווינו";
int length = strlen(name); // יעזור לצורך חסכון בזיכרון
Serial.println(length);
```
