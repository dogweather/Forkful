## מדוע

ניתן להשתמש במספרים רנדומליים בתוכניות ארדואינו כדי ליצור אירועים אקראיים ולהפעיל פעולות שונות בפרויקטים שלנו. הם יכולים לשמש כדי לייצר תנועה, אורות, צלילים ועוד ולתת פאזל נוסף לפרויקט שלכם.

## איך לעשות זאת

באמצעות פקודות פשוטות בארדואינו, ניתן ליצור מספרים רנדומליים בטווח מסוים. למשל, הקטע הבא של קוד ייצור מספר רנדומלי מ100 עד 500 וידפיס אותו לעזרה בנוחיות:


```Arduino
int randomNumber = random(100, 501); // יצירת מספר רנדומלי בין 100 ל-500
Serial.println(randomNumber); // הדפסת המספר למסך הקונסול
```

תוצאה:

`327`


ניתן גם ליצור תנועה רנדומלית בעזרת שימוש במספרים רנדומליים. לדוגמה, בקוד זה נפעיל מנוע ונחכה זמן רנדומלי לפני שנכבה אותו:


```Arduino
int randomTime = random(100, 1000); // מספר רנדומלי כדי להשלים את התנועה
digitalWrite(9, HIGH); // הפעלת מנוע
delay(randomTime); // התעכבות רנדומלית לפני כיבוי המנוע
digitalWrite(9, LOW); // כיבוי מנוע
```
