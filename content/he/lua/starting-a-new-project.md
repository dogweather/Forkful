## What & Why?
מה זה פרויקט חדש ולמה כל כך הרבה מתכנתים נלהבים להתחיל אחד? בסיסית, זה הרגע בו אנחנו יוצרים משהו מאפס. אנשים עושים את זה כדי לפתור בעיות, ללמוד משהו חדש, או לבנות מוצר שיעשה שינוי.

## How to:
Lua היא שפה פשוטה ללמידה ולשימוש. הנה כמה שורות קוד שמראות איך להתחיל פרויקט חדש:

```Lua
-- כדי להתחיל, אנחנו יוצרים קובץ חדש בשם main.lua

-- כעת מתחילים לכתוב קוד:
print("ברוכים הבאים לפרוייקט החדש שלי ב-Lua!")

-- מגדירים פונקציה פשוטה
function greet(name)
    print("שלום " .. name .. ", נעים להכיר!")
end

-- קוראים לפונקציה
greet("אלי")
```

פלט לדוגמה:
```
ברוכים הבאים לפרוייקט החדש שלי ב-Lua!
שלום אלי, נעים להכיר!
```

## Deep Dive
Lua נוצרה ב-1993 בברזיל. היא נועדה להיות פשוטה לשילוב ביישומים אחרים. בשנים האחרונות, היא הפכה לפופולרית בתעשיית המשחקים ובאוטומציה של בדיקות. כלים חלופיים כמו Python או JavaScript עשויים להיות בעלי פונקציונליות דומה, אבל Lua מצטיינת במהירות וקלות השילוב.

ברוב המקרים, התחלת פרויקט ב-Lua לא תדרוש ממך יותר מלכתוב קוד פשוט בקובץ טקסט ולהריץ אותו עם מתרגם Lua. אין צורך בסביבות פיתוח מסובכות או בתכנון מורכב. זה מאפשר למתחילים לקפוץ ישר לעומק המים.
