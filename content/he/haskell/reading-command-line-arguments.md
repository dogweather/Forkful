## למה
התכנות ב־Haskell הוא אחד מהתחומים המוכרים והמעניינים ביותר בתחום התכנות פונקציונלי. כדי לעבור מעט יותר מעבר לתכנית בסיסית, אתה צריך ללמוד איך לקרוא ארגומנטים משורת הפקודה. זה מאפשר לך לשפר את היכולות שלך כתכנת ולהתאים את התכנית שלך לצרכים המדויקים שלך.

## כיצד לקרוא ארגומנטים משורת הפקודה
ישנם שלושה צעדים בסיסיים לקריאה של ארגומנטים משורת הפקודה ב־Haskell: ייבוא מודול, קבלת הארגומנטים ועיבוד הארגומנטים.

```Haskell
import System.Environment

main = do
    args <- getArgs
    putStrLn ("ארגומנטים: " ++ show args)
```
כאן, אנחנו מייבאים את המודול System.Environment שמאפשר לנו לקבל גישה לפונקציה getArgs. תחילה, אנחנו משתמשים בפונקציה main כדי לכתוב את התכנית. בשורת הפקודה השנייה, אנחנו משתמשים בפונקציה getArgs כדי לקבל את כל הארגומנטים שהועברו לתכנית בשורת הפקודה. לבסוף, אנחנו מדפיסים את הארגומנטים בעזרת הפונקציה putStrLn.

## טיפים לקריאת ארגומנטים משורת הפקודה
- אם תרצה להשתמש בארגומנטים בתוך התכנית שלך, תקדים להכניס אותם לתוך רשימה שתהפוך ליהיות חלק מהמקרוים של הפונקציה main.
- תמיד התרגל לעבוד עם סוגים מסוימים של ארגומנטים כמו מחרוזות, מספרים וכו'. פונקציות קריאת משתנים הנמצאים במקרוים רבים אינן מציבות את סוג המשנה במקרה
