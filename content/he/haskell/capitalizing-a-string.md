## למה:

שלום לכולם וברוכים הבאים לבלוג הפרוגרמותצ'י הקזואלי המיועד לקהל הקוראים בעברית. הפעם אני רוצה לדבר איתכם על הצורך להפיק אותיות גדולות ביותר במחרוזת בשפת האסקל. זהו פעולה פשוטה שעשויה להיות מעט מבלבלת בתחילה, אך לאורך הפוסט הזה אני אסביר לכם מדוע וכיצד לעשות זאת בקלות.

## כיצד לעשות זאת: 

השימוש בפונקציית `toUpper` יכול להיות הפתרון הטוב ביותר להפיק את האותיות הגדולות במחרוזת. לפני שנתחיל עם דוגמאות, נטמע את המודול `Data.Char` שכולל את הפונקציה `toUpper`. נוכל להשתמש בה כדי להמיר את כל האותיות במחרוזת לגדולות. לדוגמא:

```Haskell
import Data.Char

capitalizeString :: String -> String
capitalizeString str = [toUpper c | c <- str]

capitalizeString "hello, world" -- הפלט יהיה "HELLO, WORLD"
```

מבוא פשוט לעקוב אחר הקוד להבנתו: בשורה הראשונה אנו מייבאים את המודול `Data.Char`. בשורה השנייה אנו מגדירים פונקציה שמקבלת מחרוזת כפרמטר ומחזירה מחרוזת שכוללת את כל האותיות המקוריות שהופכות לגדולות באמצעות הפונקציה `toUpper`. בשורת הקריאה האחרונה אנו קוראים לפונקציה על מחרוזת ספציפית כדי להפעיל אותה.
