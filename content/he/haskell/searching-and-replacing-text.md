# למה

לברר טקסט הוא כלי מאוד חשוב עבור מתכנתי Haskell וזה מאפשר להם לאתר ולהחליף טקסט לפי צורכי קוד. לכן, אנחנו צריכים ללמוד איך לעשות זאת בצורה נכונה ויעילה.

# איך לעשות זאת

לפניכם תמצאו כמה דוגמאות של כיצד לבצע חיפוש והחלפה של טקסט בשפת Haskell. כל אחת מהדוגמאות מלווה בהסברים ובכמה תוצאות לדוגמה.

```Haskell
-- דוגמה 1: חיפוש והחלפה של מילה בטקסט
main = do
  let text = "שלום לכולם, שלום עולם!"
  putStrLn $ replace "שלום" "היי" text
  -- התוצאה תהיה: "היי לכולם, היי עולם!"

-- דוגמה 2: הבהרת החלק הראשון של מחרוזת
main = do
  let text = "מלון חמסה"
  putStrLn $ take 6 text
  -- התוצאה תהיה: "מלון "

-- דוגמה 3: החלפת תווים ספציפיים במחרוזת
main = do
  let text = "255 שעות ביום"
  putStrLn $ map (\c -> if c == 'ש' then '-' else c) text
  -- התוצאה תהיה: "255 תעות ביום"
```
