## למה

למה לעסק בהשגת התאריך הנוכחי בכלל? התאריך הנוכחי הוא מרכיב מרכזי ביותר ביישומי התוכנה שלנו ומשמש כשעון ביותר לתוכניותינו היומיומיות. כשאנו מפעילים את התוכנה שלנו ורוצים לצפות בתאריך של היום, זה דבר נח שנמצא בסמוך ובזמן הנוכחי. אבל מה בדיוק קורה מאחורי הקלעים כשאנו מבקשים את התאריך הנוכחי? בעזרת המדריך הזה, אנחנו נלמד כיצד לשכור את הקוד ונראה מה קורה בתוך המכונה.

## איך לעשות זאת

ישנם כמה דרכים לקבל את התאריך הנוכחי בשפת תכנות גו. אחת הדרכים היא להשתמש בפונקציה time.Now() המגיעה עם החבילה של time. לדוגמה:

```Go
package main

import (
    "fmt"
    "time"
)

func main() {
    currentTime := time.Now()
    fmt.Println(currentTime)
}
```

בקוד הזה, הערך המחזיק את התאריך הנוכחי מאוחסן בקובץ currentTime ומודפס באמצעות פונקציית fmt.Println(). אם הייתם מריצים את הקוד הזה, התאריך הנוכחי יודפס בתוך תוכנית הצפיה שלכם.

הדרך השנייה לקבל את התאריך הנוכחי היא על ידי שימוש בפונקציות של החבילה time כמו Month(), Day(), Year() וכו '. כל אחת מאלה תחזיר לכם את הערך המתאים לתאריך הנוכחי, כך שניתן להשתמש בהם כדי לבנות תבניות שונות של תאריך לפי הצורך שלכם.
