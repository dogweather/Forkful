## מה ולמה?
קביעת תאריך בעתיד או בעבר היא פשוט החישוב של תאריכים לאחר או לפני תאריך נתון. תכנותים עושים את זה בדרך כלל כדי לנהל לוחות זמנים, תזכורות, או לחשב מועדים חשובים.

## איך ל:
```Go
package main

import (
	"fmt"
	"time"
)

func main() {
	// השגת תאריך נוכחי
	now := time.Now()
	fmt.Println("היום:", now.Format("02-01-2006 15:04:05"))

	// חישוב תאריך בעתיד
	future := now.AddDate(0, 1, 10) // תוסיף חודש אחד ועשרה ימים
	fmt.Println("עתיד:", future.Format("02-01-2006 15:04:05"))

	// חישוב תאריך בעבר
	past := now.AddDate(0, -2, -5) // תחסיר שניים חודשים וחמישה ימים
	fmt.Println("עבר:", past.Format("02-01-2006 15:04:05"))
}
```
פלט לדוגמה:
```
היום: 28-03-2023 14:06:27
עתיד: 07-05-2023 14:06:27
עבר: 23-01-2023 14:06:27
```

## ניתוח מעמיק
כשנתקלים בשאלת חישובי תאריכים, היסטורית זה היה מורכב. לפני שהיו מחלקות תאריך וזמן, כמו `time` ב-Go, היית צריך לחשב ערכי זמנים באמצעות פונקציות נמוכות. כיום, החבילה `time` של Go מספקת יכולות לינאריות להוספה והפחתה של תאריכים - פשוט ויעיל.

ישנן אלטרנטיבות ל-Go כמו Joda-Time ב-Java או Arrow ב-Python, אבל Go מציעה גישה נקיה ומינימליסטית לטיפול בזמן.

ביצוע החישוב מתבצע באמצעות שיטת `AddDate` של מופע `time.Time`. היא מקבלת שלושה פרמטרים: שנים, חודשים, וימים, שיכולים להיות חיוביים (לעתיד) או שליליים (לעבר). `time` מתייחסת לעובדה שתאריכים הם לא פשוט מספרים ומאפשרת לטפל במורכבויות של חודשים שונים ושנים מעוברות.
