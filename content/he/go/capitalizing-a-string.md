## למה

כאשר אנחנו מפתחים אפליקציות תוכנה בשפת תכנות גו, ייתכן כי נצטרך לשנות את הצורה של מחרוזות מסוימות. דוגמה נפוצה לכך היא להפוך אותן לאותיות גדולות או קטנות. המטרה של זה היא לייצר תוצאה נכונה וקריאה יותר למשתמשים ולמכונות שמעבדות את האפליקציה. בכתבה זו נלמד כיצד לחשב מחדש את המחרוזות האלו וכיצד להשתמש בפונקציות כדי להשיג את התוצאה הנדרשת.

## איך לעשות זאת

תחילה נצטרך ליצור משתנה המכיל את המחרוזת שאנחנו רוצים לשנות, למשל "golang". לאחר מכן, נשתמש בפונקציה strings.ToUpper() כדי להמיר את המחרוזת לאותיות גדולות. הנה דוגמה של קוד בשפת גו:

 ```Go
// מגדירים את המחרוזת שצריך לשנות
stringToChange := "golang" 

// משתמשים בפונקציה כדי להמיר לאותיות גדולות 
capitalizedString := strings.ToUpper(stringToChange)

// מדפיסים את התוצאה הסופית 
fmt.Println(capitalizedString)  

// פלט: GOLANG
 ```

כמו כן, אפשר גם להשתמש בפונקציה strings.ToLower() כדי להמיר את המחרוזת לאותיות קטנות במקום. כעת אנחנו יכולים להשתמש במחרוזת שהוחלפה בקוד שלנו כדי להדפיס תוצאה זהה או לשנות אותה באופן קבוע בקוד שלנו.
