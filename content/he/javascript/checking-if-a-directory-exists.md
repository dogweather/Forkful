## מדוע

בעולם התכנות, ישנם מצבים שונים בהם יש צורך לבדוק אם יישומון מסוים מכיל קובץ או תיקייה מסוימים. אחד הדוגמאות הנפוצות היא לבדוק אם התיקייה המיועדת לאחסון קבצים קיימת כדי למנוע תקלות נוספות בתהליך הכתיבה או הקריאה של קבצים מהתיקייה הזו. לכן, יש חשיבות רבה בלבדוק אם תיקייה קיימת לפני שממשיכים בתהליך התכנות.

## איך לבדוק אם תיקייה קיימת

בכדי לבדוק אם תיקייה קיימת באמצעות ג'אווהסקריפט, ניתן להשתמש בפונקציה שנקראת `fs.existsSync ()`. הפונקציה תקבל כפרמטר את הנתיב לתיקייה ותחזיר ערך בוליאני, שיציין האם התיקייה קיימת או לא. ניתן להשתמש בפונקציה זו בצורה הבאה:

```Javascript
if (fs.existsSync('path/to/directory')) {
  console.log('התיקייה קיימת!');
} else {
  console.log('התיקייה לא קיימת.');
}
```

בדוגמא זו, אנו משתמשים בתנאי `if` כדי לבדוק האם הערך שמוחזר על ידי הפונקציה הוא `true` או `false`. אם הערך הוא `true`, זה אומר שהתיקייה קיימת ואנו מדפיסים הודעת הצלחה. אחרת, אם הערך הוא `false`, אנו מדפיסים הודעת שגיאה שתייגזר ממנה שהתיקייה לא קיימת.

**הערה:** כדי להשתמש בפונקציות קבצים בג'אווהסקריפט, יש לייבא את המודול `fs` כך: `const fs = require('fs');`.
