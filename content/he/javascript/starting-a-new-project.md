## מָה ולָמָּה?

הפתיחה של פרויקט חדש היא את התהליך בו אמנים יוצאים לדרך לפתור בעיה מסויימת באמצעות קוד. אנחנו עושים את זה כדי להגשים רעיונות שלנו, לשפר משהו שקיים או ללמוד שפת תכנות חדשה.

## כיצד:

הנה דוגמה ליצירת פרויקט חדש בעזרת create-react-app:

```Javascript
npx create-react-app my-app
cd my-app
npm start
```

הפלט של קוד זה מציג אפליקציית React מראש, הפועלת בלוקלהוס, בערוץ 3000. 

## צלילה מעמיקה:

איננו יודעים תמיד היכן אנו רוצים ללכת כשאנחנו מתחילים פרויקט, ההיסטוריה מלאה בבעיות שנפתרו על-ידי יצירת פרויקט חדש. העבודה מראש על פרויקט חדש מאפשרת לנו להתמקד בבעיות מסוימות מבלי להתרכז יותר מדי בקוד המרכזי של האפליקציה. חלופה ל-Create React App הוא Next.js, שמאפשר יכולת של Server-Side Rendering.

## ראה גם:

כדאי לעיין במקורות הבאים עבור מידע הרחבה:

- דוקומנטציה של [create-react-app](https://create-react-app.dev/docs/getting-started/) 
- מדריך למשתמש של [Next.js](https://nextjs.org/learn)
- בלוג של [Dan Abramov](https://overreacted.io/) , איש שלמד הרבה למפתחי React.
