# למה: 
יצירת מספרים אקראיים הוא כלי חשוב בתכנות ויסודי כל פרוגרמה. מספרים אקראיים משמשים במגוון רב של יישומים, כגון מבחני יחסים, משחקים, וכלים הדרישים בשביל ליצור תנאים כילופים.

## איך לעשות זאת: 
בדרך כלל, תוכניות צריכות ליצור מספרים אקראיים לפי צורך. למרבה המזל, דרך פשוטה לעשות זאת בפתרון קוד Javascript. ניתן להשתמש בפונקציות מובנות כמו Math.random () או ליצור פונקציות משלנו שמגבילות את המספרים לטווח מסוים ועוברות על סדרת המספרים האקראיים בצורה נכונה. להלן כמה דוגמאות פשוטות של יצירת מספרים אקראיים בשפת JavaScript באמצעות סוגי קוד המסומנים ב"```javascript ... ```":

### יצירת מספר אקראי בין 0 ל-1: 
```javascript
Math.random(); // 0.365293501894
```

### יצירת מספר אקראי בין 0 ל-10: 
```javascript
Math.random() * 10; // 7.65324701953
```

### יצירת מספר אקראי בין 1 ל-100: 
```javascript
Math.floor(Math.random() * 100) + 1; // 53
```

כפי שניתן לראות, בכל דוגמה נראית הפעלה של פונקציית Math.random () ושינוי קל שלה כדי לעשות את זה נכון. ניתן להשתמש גם בפונקציות אחרות כמו Math.round () ו Math.ceil () כדי לעשות זאת עוד יותר קל.
