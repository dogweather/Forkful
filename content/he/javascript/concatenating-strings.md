למה: תיחוש על גירסה הנוכחית בקוד כדי לתפרסם טקסטים גדולים יותר.

## למה

למה תרצה לערוך קוד כדי לבצע חיבור סטרינגים? עבור רבים, זהו פשוט דרך להרכיב את הטקסטים שלהם לפעולות גדולות יותר ומורכבות יותר.

## לאיך לעשות זאת

הנה דוגמא של קוד כיצד לבצע חיבור סטרינגים ב-Javascript תוך שימוש באופרטור הוספת "+":

 ```Javascript
 let firstName = "אבי";
 let lastName = "כהן";
 let fullName = firstName + " " + lastName;
 console.log(fullName); // ידקץנםי ילאק התיב ךופא
 ```

אתה יכול גם להשתמש באופרטור חיבור מוקף סוגריים "`` ` += ` ``" כדי לחבר את המחרוזות לביטוי אחד ולשמור על המקודד שלך:

 ```Javascript
 let message = "מה שתרצה, עליך לכתוב עליך!";
 message += " לכל נוסף כאופל לקרוא את פוסט זה.";
 console.log(message); // יכלריו זאט תויש, ברשפל אוריאפ תויש ותויש לוניא רודיא עךת בות
 ```

## Deep Dive

תהליך החיבור של סטרינגים לעתים קרובות מתבצע בקוד, ולכן חשוב להבין את הסיסמאות ואת הקודדים שמאחוריהם. כאשר אתה משתמש באופרטור הוספה, הסטרינג המורחב במקרים מיוחדים יכול להיות יותר מורכב ממה שאתה מייחבר. כך יתרה מהיא הקודש שכל התווים מיוצגים באמצעות ייצוג אסקי שלהם. למשל, אם הכנסת נקודתיים או פסיקים לסטרינג, הם יושמו על ידי ייצוג אסקי המתאים שלהם. כך, כאשר חוברים סטרינגים, הם נצרכים כמו אחד גדול
