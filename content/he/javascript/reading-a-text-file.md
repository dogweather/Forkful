בלוג פורמט נוח לכתיבה בקוד Javascript

## מדוע

בעולם התכנות, קריאת קבצי טקסט היא פעולה נפוצה וחיונית. כתיבת קוד שיעבוד מידע מקבצי טקסט נתונים היא אחת ממנהגי המדוברים ביותר בתעשייה הטכנולוגית. האפשרויות ללימוד, עריכה ובדיקה של מידע מובנות בתכנות טקסט נוחה ואידיאלית

## איך לעשות זאת

בשפת Javascript, קריאת קובץ טקסט היא פעולה פשוטה ונפוצה. נציג לכם המדריך קוד פשוט שלאחריו תפעיל את הקוד, הוא יקרא את הקובץ וידפיס את הנתונים שמצא בו:

```Javascript
fs = require('fs');
fs.readFile('data.txt', 'utf8', (err, data) => {
  if (err) {
    console.error(err);
    return;
  }
  console.log(data);
});
```

*Output:*

```
Hello World!
```

נוכל לראות ששורת קוד ראשונה מפעילה את הספריה fs (קובץ מערכת). השורה הבאה מפעילה את הפעולה readFile שמקבלת כארגומנטים את שם הקובץ וקבוע שמציין את הקודד של הטקסט (utf8). התוצאה שתחזור במקרה של הצלחה תהיה שמעוניינים להדפיס עבורה. עוד קוות אצל לבדוק האם יש תקלות בקריאת הקובץ, ואם כן לפונות להדפסת אותו.
