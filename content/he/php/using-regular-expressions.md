## למה

השימוש בביטויים רגולריים הוא חלק חשוב מתהליך התכנות בשפת PHP. באמצעות ביטויים רגולריים, ניתן לחפש ולהחליף טקסט בצורה מהירה ויעילה, וליישם שכבת ביטחון נוספת לאתר או לאפליקציה.

## איך לעשות זאת

נוכל לכתוב רגולריים בפתיחת רווח וסביבהם שָׁכוּת, כדי להגדיר את הביטוי שנרצה לחפש. ניתן להשתמש בתווים מיוחדים כדי להגביל את החיפוש לתווים מסוימים, וניתן להוסיף תנאים סגורים כגון "כלום או יותר" או "אחד או יותר" כדי להתאים לתבניות שונות יותר של הטקסט. להלן כמה דוגמאות של ביטויים רגולריים בשפת PHP:

```PHP
// בדיקת תואמים למספר פלאפון
if (preg_match("/^[05][0-9]{8}+$/", $phone_number)) {
    echo "מספר הטלפון תקין.";
}

// מציאת כתובת מייל חוקית
if (preg_match("/^[a-z0-9._%+-]+@[a-z0-9.-]+\.[a-z]{2,}$/", $email)) {
    echo "המייל שהוכנס תקין.";
}

// החלפת מילה במחרוזת
$text = "היי, אני אוהב לתכנת ב-PHP!";

$new_text = preg_replace("/PHP/", "JavaScript", $text);

echo $new_text; // היי, אני אוהב לתכנת ב-JavaScript!
```

## צלילה עמוקה

השתמשנו כאן במספר דוגמאות פשוטות לביטויים רגולריים, אך באמצעותם ניתן לבצע הרבה יותר מאשר את הדוגמאות הקטנות שלנו. ניתן לסנן ולמיין טבלאות של מידע, לבצע תיקון וניקוי של דפים וחומות אש, ועוד.
