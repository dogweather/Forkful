## מה ולמה?

התחלת פרויקט חדש בתכנות זו הקמת אפליקציה, באמצעות כתיבת קוד מתכנת מאפס. מתכנתים מתחילים פרויקט חדש בשביל לגשת לרעיון חדש או למענה על בעיה מסוימת.

## איך לעשות:

נתחיל ביצירה של קובץ PHP חדש:

```PHP
<?php
// This is your new project!
?>
```
אז מה ראינו כאן? קיצור דרך שהמחשב משתמש בו כדי לפענח את הקוד שלך. מסיינים התחלה וסוף של קובץ ה PHP.

## Deep Dive: צלילה לעומק

(1) בהקשר היסטורי, PHP היא שפת תכנות שפותחה בשנת 1995. היא מתאימה במיוחד לפיתוח אפליקציות אינטרנט.
(2) ישנן חלופות ל PHP כמו JavaScript, Python, או Ruby. כל אחת מהן מגיעה עם יתרונות וחסרונות משלה.
(3) בפרטים של היישום, PHP מתומך ברוב שרתים, מערכות הפעלה, ומסדים נתונים. היא מאפשרת שפת ממשק של משתמש אינטואיטיבית.
