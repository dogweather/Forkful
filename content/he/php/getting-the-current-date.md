# למה

למה לצרף תאריך נוכחי בתכנות PHP? פעולה זו חיונית ליישום מגוון משימות, כגון כתיבת וילדצס, ביצוע פעולות חישוביות והצגת תאריך על פני דפי אינטרנט.

# איך לצרף תאריך נוכחי בקוד PHP

```PHP
<?php
// שימוש בפונקציית date() לקבלת תאריך נוכחי
echo "היום הוא: " . date("d/m/Y") . "<br>";

// כתיבת וילדצ'ס עם תאריך ושעה
echo "עכשיו הוא: " . date("d/m/Y h:i:sa") . "<br>";

// משיכת תאריך וילדתי מכתיבת שני מספרים
$d = strtotime("01 January 2021");
echo "תאריך וילדי ל-Rosh Hashanah: " . date("d/m/Y", $d) . "<br>";
?>
```
התוצאות יהיו:
היום הוא: 16/11/2021
עכשיו הוא: 16/11/2021 04:22:27צהריים
תאריך וילדי ל-Rosh Hashanah: 06/09/2021

כאן אנו משתמשים בפונקציית date() ליצירת תאריך נוכחי מתקן ושנותר לפי בחירתנו. אנו מגדירים את הפורמט הרצוי של התאריך בתוך סדרת התווים שמתחילה בלוח הזכרונות. הכנסת איבר זה בין גרשים יצירת המשייכת את התאריך שנציג. לדוגמה, "Y" מייצג את שנת התאריך, "d" מייצג את היום בתאריך ו"m" מייצג את החודש.

אנו יכולים גם לשנות את הפורמט של השעה בעזרת הניבוב "h" ליצירת פורמט שעון דווקאי, ו- "i" עבור דוקדן.
