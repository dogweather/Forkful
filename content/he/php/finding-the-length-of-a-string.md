## למה
בתכנות ב-PHP ישנו מגוון רחב של פעולות לעיבוד מחרוזות (Strings). אחת מהן היא למצוא את אורך המחרוזת, כלומר מספר התווים או התווים המורכבים במחרוזת. אנו נצליח להרחיב את היכולות שלנו כתכנתנים על ידי לימוד כיצד למצוא את אורך מחרוזת בפעולת פשוטה זו.

## איך למצוא את אורך המחרוזת
כדי למצוא את אורך המחרוזת ב-PHP, אנו נשתמש בפעולה המובנת `strlen()`. היא תקבל מחרוזת כארגומנט ותחזיר לנו את אורךה ביחידות `bytes` של התווים במחרוזת. הנה דוגמא של קוד פשוט שמציג כיצד להשתמש בפעולה זו:

```PHP
<?php
$string = "שלום עולם";
echo strlen($string); // יחזיר 10
?>
```

כאן אנו משתמשים בפעולה `strlen()` כדי למצוא את אורך המחרוזת העברית "שלום עולם". ניתן לראות שהמחרוזת כוללת 10 תווים, כולל רווחים.

כדי להדגים שהמחרוזות אומרות דבר אחד ברור עבור המחשב ועוד אחד לשימושנו ניצור משתנה פשוט המכיל מחרוזת ארוכה. נעבוד עם המחרוזת "Hello World!" כמו דוגמא:

```PHP
<?php
$string = "Hello World!";
echo strlen($string); // יחזיר 12
?>
```

כמו כן, אנו יכולים להשתמש בפעולה `strlen()` על משתני מחרוזת בטקסט צפוי:

```PHP
<?php
$word1 = "Hello";
$word2 = "World";
echo strlen($word1 . $word2); // יחזיר 10
?>
```

ניתן גם לקלוט מחרוזת מהמשתמש ולאחר מכן למצוא את אורךה:

```PHP
<?php
$string = readline("נא להכניס מחרוזת:");
echo strlen($string); // יחזיר את האורך של המחרוזת שהוקלדה על ידי המשתמש 
?>
```
