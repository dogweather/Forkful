## למה

קריאת קובץ טקסט היא כלי חשוב בתוך עולם התכנות של PHP. זה מאפשר לנו לקרוא ולעבוד עם קבצי טקסט בפורמטים שונים, כגון CSV או JSON. בנוסף, קריאת קובץ טקסט היא חיונית ליצירת קשר עם מסדי נתונים וליישומים ווב בכלל.

## איך לעשות זאת

בכדי לקרוא קובץ טקסט ב-PHP, נשתמש בפונקציית `fopen` כדי לפתוח את הקובץ עבור קריאה. ניתן לציין את האופציות הנדרשות, כגון "r" עבור קריאה בלבד. לאחר מכן, נשתמש בפונקציה `fread` כדי לקרוא את התוכן של הקובץ ולהציג אותו בתוך משתנה. לבסוף, נסגור את הקובץ באמצעות הפונקציה `fclose`.

```PHP
$file = fopen("file.txt", "r");
$content = fread($file, filesize("file.txt"));
fclose($file);
echo $content;
```

הפלט יחזיר את כל התוכן של הקובץ, כולל תוכן שורה ותווים מיוחדים.

## חפירה עמוקה

עכשיו שאנחנו יודעים איך לקרוא קובץ טקסט, נראה כיצד אנחנו יכולים לעבוד איתו. באמצעות פונקציה `fgets`, אנחנו יכולים לקרוא כל שורת טקסט בנפרד ולטפל בה כראוי. ניתן גם להשתמש בפונקציות נוספות כדי להציג תווים מיוחדים בקובץ טקסט, כגון האטמות או מיילרים.
