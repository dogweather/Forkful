# למה
למה עומד אדם לבחון אם תיקיית קיימת.

כשאנחנו מתכנתים ב-PHP, קבלת קלט מהמשתמש היא חלק בלתי נפרד מהתהליך. אם המשתמש מזין שם של תיקייה, חשוב לוודא שהתיקייה הזו קיימת כדי למתן את התוצאה הנכונה בפעולות שלנו.

# איך לבדוק אם תיקייה קיימת
בכדי לבדוק אם תיקייה קיימת באמצעות קוד PHP, נשתמש בפונקציה שנקראת `file_exists`. פונקציית זו בודקת האם הנתיב המצויין קיים ומחזירה תשובה חיובית או שלילית. נציג דוגמאות לשימוש בפונקציה זו עם תיקיות שונות:

```PHP
<?php
// בדיקת תיקייה קיימת
if (file_exists('תיקייה_קיימת')) {
    echo "תיקייה זו קיימת!";
} else {
    echo "התיקייה לא נמצאת.";
}
// ניתן להשתמש בכתובת מלאה של התיקייה
if (file_exists('/home/user/תיקייה_קיימת')) {
    echo "תיקייה זו קיימת!";
} else {
    echo "התיקייה לא נמצאת.";
}
// בדיקת תיקייה שאינה קיימת
if (file_exists('תיקייה_לא_קיימת')) {
    echo "תיקייה זו קיימת!";
} else {
    echo "התיקייה לא נמצאת.";
}
?>
```

פלט:
```
תיקייה זו קיימת!
תיקייה זו קיימת!
התיקייה לא נמצאת.
```

ניתן גם להשתמש בפונקציה `is_dir` כדי לבדוק אם הנתיב המצויין הוא באמת תיקייה.

# העיון העמוק
