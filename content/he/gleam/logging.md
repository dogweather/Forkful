## מה ולמה?
לוגינג הוא בעצם הדרך בה אנו מתעדים מה קורה בתוכניות שלנו. זה כמו להיות עם קופסה שחורה קטנה; כשדברים הולכים לא נכון (ותאמין לי, זה יקרה), לוגים הם בלתי נחוצים להבנת מה קרה, אבחון בעיות, ושיפור ביצועים.

## איך לעשות:
ב-Gleam, לרוב תשתמש בספריית לוגינג - אין מנגנון לוגינג מובנה מראש. נניח שאנחנו משתמשים ב-'crate' היפותטי `gleam_logger`. הנה איך אתה יכול לשלב אותו:

```gleam
import gleam_logger

pub fn main() {
  gleam_logger.info("האפליקציה מתחילה!")
  let result = intense_calculation()

  case result {
    Ok(value) -> 
      gleam_logger.debug("החישוב הצליח", value)
    Error(err) -> 
      gleam_logger.error("החישוב נכשל", err)
  }
}
```

הפלט הצפוי בלוגים שלך ייראה פחות או יותר כך:

```
INFO: האפליקציה מתחילה!
DEBUG: החישוב הצליח 42
ERROR: החישוב נכשל סיבה: חלוקה באפס
```

## טבילה עמוקה
אומנות הלוגינג קיימת מראשית ימי התכנות. מפעילי מערכת היו אכן מקבלים לוגים מהמחשב - ודאי כי הכל פועל חלק. קדימה בזמן, והלוגינג עבר לדיגיטלי, והפך לחלק בלתי נפרד מפיתוח התוכנה.

למרות ש-Gleam, שהוא שפה יחסית צעירה המכוונת לאקוסיסטם של Erlang, אין לה מסגרת לוגינג מובנית, אפשר לנצל את מתקני הלוגינג המבוגרים והבשלים של Erlang או ספריות אחרות שניתנו על ידי הקהילה. לכל אחת מהן יש תכונות ותמורות שונות: חלקן יוכלו לספק לוגינג מובנה, בעוד אחרות מיועדות יותר לפלט טקסט פשוט.

עכשיו, שאלת יישום של מתקן לוגינג: האם זה פשוט? במבט ראשון, כן. אבל אם תקלף את השכבות, תתבונן בעיסוק במתחמות, בבעיות עומס קלט/פלט, סיבוב לוגים, תקנון פורמט (למשל JSON ללוגינג מובנה), סינון רמות, ואולי אפילו מעקב מתפרסם. בנוסף, בפרדיגמה פונקציונלית, בדרך כלל רוצים שהשפעות חיצוניות (כמו לוגינג) יהיו נשלטות וברות תחזית.

## ראה גם
הנה היכן אפשר למצוא עוד על פרטי המכניקה של הלוגינג ב-Gleam ובאקוסיסטם שלו:
- [תיעוד :logger של Erlang](http://erlang.org/doc/apps/kernel/logger_chapter.html): מכיוון ש-Gleam מתקמפל ל-Erlang, זה רלוונטי באופן ישיר.
- [מסמכי ספריית התקנים של Gleam](https://hexdocs.pm/gleam_stdlib/): לעדכונים על כלי לוגינג חדשים שעשויים להתווסף.
- [Awesome Gleam](https://github.com/gleam-lang/awesome-gleam): רשימה מסודרת של משאבים, שעשויים לכלול ספריות לוגינג ככל שהן יהפכו לזמינות.
