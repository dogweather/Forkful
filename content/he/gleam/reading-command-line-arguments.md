# למה?

כדי להסביר בפוסט זה את הצורך והיתרונות שבקריאת פרמטרי שורת הפקודה בשפת גלים.

## איך לעשות זאת?

כדי לקרוא פרמטרי שורת הפקודה בשפת גלים, ניתן להשתמש בפונקציה `System.argv()` ולשלוח לה את הפרמטרים הרלוונטיים. לדוגמה:

```Gleam
let params = System.argv()
```

בפונקציה זו, המשתנה `params` יכיל את הפרמטרים שנשלחו לתוכנית דרך שורת הפקודה. ניתן להדפיס את הפרמטרים הללו באמצעות הפונקציה `IO.puts()`. לדוגמה:

```Gleam
let params = System.argv()

IO.puts("הפרמטרים שנשלחו לתוכנית הם:")
IO.inspect(params)
```

כאשר נפעיל את התוכנית עם הפרמטרים "א", "ב" ו-"ג", נקבל את הפלט הבא:

```
הפרמטרים שנשלחו לתוכנית הם:
["א", "ב", "ג"]
```

ניתן גם לקרוא פרמטרים ספציפיים בעזרת הפונקציה `List.nth()` ולכן להשתמש בהם בתהליך תהליך התכנות.

## שטח מצומצם

כאשר יש צורך לכתוב תכניות ניתן לתת למשתמש התוכנית אפשרות להכניס פרמטרים ספציפיים בטיפול שורת הפקודה. זה יכול לגרום לתהליך ההרצה להפוך לקל יותר ולאפשר התאמה אישית של התוכנית לפי הצורך של המשתמש.

כמו כן, השימוש בפרמטרים אלו יכול לגרום לתוכנית לרוץ במהירות יותר ולחסוך במשאבי מעבד.
