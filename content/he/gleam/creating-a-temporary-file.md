## למה
יצירת קובץ זמני יכול להיות כלי עזר מאוד חשוב בתהליך תכנות, מאפשר אחסון ושימוש בנתונים באופן זמני עד שהם ייכנסו למקומם הבקבוקי בקוד. בכתיבת קוד בשפת Gleam, יצירת קובץ זמני תיהיה אחד הכלים המומלצים לשימוש בתהליך הפיתוח.

## איך לבצע
ליצירת קובץ זמני בשפת Gleam ניתן להשתמש בפונקציה `File.Temp.make()`, שתחזיר אובייקט של קובץ זמני חדש. ניתן לקרוא לפונקציה זו עם שם הקובץ כפרמטר, כך שניתן יהיה להשתמש בשם זה בהמשך בקוד. לדוגמה:

```
Gleam
let { Ok, Error } = File.Temp.make("example")
```

כדי לכתוב או לקרוא מתוך הקובץ הזמני, ניתן להשתמש בפונקציות `File.write()` ו`File.read()` עם האובייקט שנוצר מפונקציה `File.Temp.make()` כמו בדוגמה הבאה:

```
Gleam
let file = File.Temp.make("example")
File.write(file, "Hello World!")
let content = File.read(file) // תוצאה יהיה "Hello World!"
```

## מעמקים
כאשר ניתן ליצור קובץ זמני בהנחה שהקובץ קיים ונמצא במסלול ההרצה של הקוד, מסלול זה יוכן בקובץ המקורי. אם לא יש לנו זיכרון לקובץ זמני, הוא יימחק כאשר הפונקציה `File.Temp.make()` תסתיים והקוד ייגמר. ייצירת קובץ זמני מספקת גם את היכולת לבדוק האם הקובץ נמצא באותו מסלול ולפתור בעיות של גישה לקבצים ונתיבים בפיתוח ואיחסון של נתונים מזמניים.
