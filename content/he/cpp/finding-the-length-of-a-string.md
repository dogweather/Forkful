## מדוע: בלבד 1-2 משפטים כדי להסביר *למה* מישהו ישתתף במציאת אורך של מחרוזת.

מציאת אורך של מחרוזת הינו תרגול חשוב ותפקידי בכל תכנות קוד. זה מאפשר למשתמש לדעת כמה תווים יש במחרוזת כדי להתמודד עם רווח ואינפורמציה נוספת.

## איך לעשות זאת: דוגמאות תכנותיות ופלט מחוץ לבלוקי קוד "`` `C++ ... ``" קוד.

```
#include <iostream>
#include <string>

using namespace std;

int main() {
  string myString = "שלום עולם!";
  int length = myString.length();
  cout << "אורך המחרוזת הוא: " << length << endl;
  return 0;
}
```

פלט:
```
אורך המחרוזת הוא: 12
```

## טעימה עמוקה: מידע נוסף על מציאת אורך של מחרוזת.

כאשר אנו מדברים על אורך של מחרוזת, אנו מתכוונים למספר התווים במחרוזת, כולל רווחים ותווים מיוחדים. בדרך כלל, פונקציית האורך של מחרוזת תחזיר ערך מספרי שהוא אינדקס האחרון של התו האחרון במחרוזת. זה אומר שאם תחזיר את האורך של המחרוזת "Hello", תקבל את הערך 5, למרות שיש רק 5 תווים.

בנוסף, כאשר אנו משתמשים בפונקציה זו על מחרוזת שהוא תו אחד בלבד ("H" למשל), היא תחזיר את תוי המרווח הנואם קודמו או אחריו. זה נובע מהעובדה שתו הרווח מחלק מקוד המחמד שנמצא בעיצוב של המחרוזות.

רצוי לסמן שזוהי פונקציה בסיסית ונגישה לכל תכנית כיוון שהיא חלק מסטנדרט השפה *C++*.
