## למה

חישוב תאריך בעתיד או בעבר הוא פעולה חשובה בתכנות בשפת C++, כיוון שהוא מאפשר לנו ליצור תוכניות יעילות יותר שמתאימות לצרכים שונים. החישוב של תאריך נכון עם אלגוריתמים מתאימים יכול לעזור לנו לחישב את התאריך הדיוק לחלוקת משימות או לקביעת תאריך משלוח.

## איך להשתמש

כדי לחשב תאריך בעתיד או בעבר ב-C++, ניתן להשתמש בפונקציות קיימות בספריית סטנדרטית של השפה. לדוגמה, הפונקציה time.h מאפשרת לנו לחשב את התאריך הנוכחי ולהוסיף או להחסיר ימים באמצעות פעולות חשמל. כדי להשתמש בפונקציות אלה, צריך להכין את הסביבה בה נרצה להשתמש בהם, על ידי כתיבת הפקודות הבאות:

```C++
#include <iostream>
#include <ctime>
```

אחרי כדי שיש לנו את הסביבה הנכונה, ניתן להשתמש בפונקציות לחישוב תאריכים בקלות. לדוגמה, ניתן לכתוב תוכנית המדפיסה את התאריך אחד מעבר לתאריך הנוכחי:

```C++
#include <iostream>
#include <ctime>

int main()
{
  // ימי השבוע 
  char weekdays[7][10] = {"ראשון", "שני", "שלישי", "רביעי", "חמישי", "שישי", "שבת"};
  
  // מונה ימי הקיץ מחודש 1 
  int days_counter = 153;
  
  // מבצע חישוב והדפסה  
  std::time_t now = time(0);
  std::tm *ltm = localtime(&now);
  
  // החישוב  
  ltm->tm_mday += days_counter;
  ltm->tm_isdst = 1;
  std::mktime(ltm);

  std::cout << "התאריך ב 153 ימים בעתיד הוא: ";
  std::cout << ltm->tm_mday << "/" << 1 + ltm->tm_mon << "/" << 1900 + ltm->tm_year << std::endl;
  std::cout << "היום בשבוע הוא: " << weekdays[ltm->tm_wday] << std::endl;
  
  return 0;
}
```
