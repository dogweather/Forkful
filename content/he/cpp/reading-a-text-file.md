## למה
כתיבת קוד היא כמו ניסוי ומסע רוחני יחד. קריאת קבצי טקסט היא אחת המשימות הבסיסיות והחשובות בתוך תהליך התכנות. כשמתחילים ללמוד C++, להיפגש ולבצע פעולות על קבצי טקסט הוא דבר מרתק וגורם לתוצאות מעניינות. 

## איך זה עושים
```c++
#include <iostream>
#include <fstream> //ספריה המאפשרת קריאה מקבצים

using namespace std;

int main() {
  // יצירת אובייקט קובץ
  ifstream myFile("sample.txt"); 

  // בדיקת אם נמצאים בתוך הקובץ
  if (myFile.is_open()) {
    // קריאת התוכן הטקסטי של הקובץ
    string line;
    while (getline(myFile, line)) {
      // אתחול נתון מהקובץ
      int data = stoi(line);

      // פעולות נוספות..

      // הדפסת התוצאה
      cout << "מספר מוזר: " << data << endl;
    }
    // סגירת הקובץ
    myFile.close();
  } else {
    // השורה הבאה מודפסת כאשר הקובץ לא נמצא
    cout << "לא נמצא קובץ מתאים";
  }
  return 0;
}
```

בדוגמה הזו, אנחנו משתמשים בספריה "fstream" שמאפשרת לנו ליצור אובייקט קובץ ולבצע פעולות עליו, כמו קריאה וכתיבה. במשתנה "line" אנחנו מקריאים כל שורה בקובץ ומאתחלים את המשתנה המספרי "data" עם הערך הטקסטי הרלוונטי. לאחר מכן, אנחנו מבצעים פעולות נוספות על המספר ומדפיסים את התוצאה. אם הקובץ לא נמצא, נודיע למשתמש על כך.
