# למה
נתחייב לעבוד עם טקסט כמתכנתים כל הזמן, ולפעמים זה כולל צורך בשינויי טקסט גם. לדוגמא, אם יש לנו רשימת משתנים בקוד שלנו ואנחנו רוצים לשנות את השם שלהם כולם במקום שהרשימה מוגדרת כך.

# איך לעשות זאת
הפתרון הפשוט ביותר הוא להשתמש בפונקציה "replace" שנמצאת בסטרינג קלאס שבספריית הסטנדרט של C++.

```C++
#include <iostream> 
#include <string> 
using namespace std; 

int main() 
{ 
    // יצירת רשימת משתנים 
    string names = "דניאל,נעה,תמר,אורי,שלומי"; 
    
    // החלפת נעה עם נורית 
    names.replace(names.find("נעה"), 6, "נורית"); 

    // מבצע את אותו ההחלפה כמה פעמים שיש 
    while(names.find("אורי") != string::npos) 
    { 
        names.replace(names.find("אורי"), 4, "אריאל"); 
    } 
    
    // הדפסת הרשימה המעודכנת 
    cout << names << endl; 
    
    return 0; 
} 
```

פלט של הקוד ליצירת רשימת משתנים חדשה המוחלפת עם השמות המעודכנים:

דניאל,נורית,תמר,אריאל,שלומי

# העומק של הנושא
כאשר אנחנו משתמשים בפונקציית "replace" נעבור על הפרמטרים בכיוון הבא: מיקום של המילה המקורית, מספר התווים של המילה המקורית ומילה חדשה שאנחנו רוצים להחליף את המילה המקורית בה. אם נרצה להחליף עבור מספר תווים של המילה המקורית שונה ממספר התווים של המילה החדשה, אז נצטרך לשנות את הפרמטר השני לתווית אחרת כדי לכוון את ההחלפה הנכונה.
