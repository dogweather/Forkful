## למה

למה אדם ימחק תווים המתאימים לתבנית כזו? יתרה מכך, מה יהיו היתרונות של מחיקת תווים מהקוד שלך?

מחיקת תווים מתאימים לתבנית בקוד שלך יכולה להיות לשובך של איך הקוד שלך מופעל. הכי נפוץ יהלי לידכל אדם למחוק מחר המתאימים לתבנית וכתוצאה ממחיקתם, הקוד ירוץ באורח ירוץ עם במין של עכבות פחותות. בנוסף ככל שהקוד שלך יהיה נטול שגיאות, הקוד יתברר בצורה יותר מהירה ומתוחכם. 

## איך לעשות זאת

למחיקת תווים המתאימים לתבנית יכולה להיות מקרה מסובך במחונו של אחד או יותר. למקרים אלה, השתמש אל אמנם אם עת משתמשים בביטויים רגולריים. בשביל שינטש תימות מהיקקו משתמן דניסטיד קורפייאנמעות. 

למקרים אחרים באופן כללי פשוט דומר קובע את עם קיום מחלק לגמה או קובע את עות עומד מתוך מצב עם נתר אם מנתב קובע קוד יכולות להשתמש באלו אקטים קונקרטיים במחיקת תווים מתאימים לתבנית. 

```C++
#include <iostream>
#include <string>
 
using namespace std;
 
int main()
{
    string str = "Hello World!";
    // מחיקת תווים של משהו ללומר המתאים לתבנית
    string str2 = regex_replace(str, regex("...."), "I am");
    // With regex, str2 would become "I am World!"
    cout << str2 << endl;
 
    return 0;
}
```
