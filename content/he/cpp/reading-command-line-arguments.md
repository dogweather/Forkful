## למה?

קרוא ארגומנטים משורת הפקודה היא כלי חשוב בתוכנות שלנו. היא מאפשרת לנו לתת ערכים ופרמטרים שונים לתוכנית, מאפשרת להיות קל יותר ומותאמת <Organization>. כמו לתת לה מידע זה הוא מאוד מועיל כשאנחנו מפתחים תוכניות מוחזקות גם כן את האפשרות להטמיע את הפרמטרים לקוד.

## איך לעשות זאת?

קריאה לרשימת ארגומנטים שנשלחו לתוכנית שלנו היא קלה תודות למשתנה "argc" (או "משתנה מספר הארגומנטים") והמערך "argv" (או "מערך הארגומנטים"). בכדי לקבל את המידע שנשלח אליהם, אנו צריכים להשתמש במשתנים הללו בתוכנית שלנו.

```C++
#include <iostream>

int main(int argc, char* argv[])
{
    // אתחול משתנה עם ערכים משלנו
    int num1 = 0;
    int num2 = 0;
    
    // בדיקה אם נשלחו מספיק ארגומנטים
    if(argc > 2)
    {
        // קריאה לאינדקס שני של מערך הארגומנטים והמרה למספר שלם
        num1 = std::atoi(argv[1]);
        num2 = std::atoi(argv[2]);
        
        // הדפסת הנתונים שנקלטו
        std::cout << "Num1: " << num1 << std::endl;
        std::cout << "Num2: " << num2 << std::endl;
    }
    else
    {
        // הודעת שגיאה אם לא נשלחו מספיק ארגומנטים
        std::cout << "Error! Not enough arguments provided." << std::endl;
    }
    
    return 0;
}
```

### פלט:

אם נקלטו שני מספרים משורת הפקודה, התוכנית תדפיס את הערכים הנתונים, כמו כן כל הערכים שנקלטו יהיו מספרים שלמים ולא מחרוזות כמו בדוגמה שלנו.

```
Input: ./program 10 20

Output:
Num1: 10
Num2: 20
```
