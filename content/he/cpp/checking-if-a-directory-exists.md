## למה: 
קוראים יקרים, אין ספק שעובד עם קבצים ותיקיות הוא חלק בלתי נפרד מתהליך התכנות בשפת C++. אחת המשימות הקשות ביותר היא לוודא אם תיקייה קיימת או לא. אם אתם מתחילים ללמוד תכנות ב-C++ ונתקלתם בשאלה זו, או אם פשוט מתעניינים לדעת כיצד לבצע פעולה זו בצורה יעילה, קראו הלאה ללמוד ייתר על כך.

## כיצד לבדוק אם תיקייה קיימת באמצעות קוד C++:
בסעיף זה, אני אסביר כיצד לבדוק אם תיקייה קיימת באמצעות קוד C++. הנה דוגמאות קוד ופלטים כדי להדגים את התהליך.

```C++
#include <iostream>
#include <filesystem>

namespace fs = std::filesystem;

int main() 
{
    std::string dir_path = "/home/user/documents";  // הגדרת נתיב לתיקייה

    if (fs::exists(dir_path))  // בדיקה האם התיקייה קיימת במערכת הקובץ
    {
        std::cout << "התיקייה קיימת" << std::endl;
    }
    else
    {
        std::cout << "התיקייה לא קיימת" << std::endl;
    }
    
    return 0;
}
```

פלט:

```
התיקייה קיימת
```

ניתן לראות שאם התיקייה קיימת, התוכנית תדפיס "התיקייה קיימת", אחרת תדפיס "התיקייה לא קיימת".
