## מה ולמה?

אינטרפולציה של מחרוזת היא  תהליך שבו נתונים משולבים בתוך מחרוזת. אנחנו, מתכנתים, עושים זאת כדי להימנע מהצורך לשלב מחרוזות ומשתנים באמצעות ביצוע פעולות על מחרוזות.

## איך זה נעשה:

Elm איננה תומכת באינטרפולציה של מחרוזת ישירות בלשון עצמה. ניתן להשיג את התוצאה הרצויה על ידי שילוב פונקציות של מחרוזת. דוגמה לקוד:

```Elm
import String

hello name = "שלום " ++ name ++ ", כיצד אתה מרגיש היום?"

main =
    hello "יהודה"
```

במקרה זה, הפלט המוחזר הוא:

```Elm
"שלום יהודה, כיצד אתה מרגיש היום?"
```

## נצילה

אינטרפולציה של מחרוזות היא גישה מקובלת בעולם התכנות, אך המימוש של השפה עשוי להשתנות. ניתן למצוא חלופות בשפת Elm כמו `++` ו-`String.concat`. זווית ההיסטוריה שלנו משאירה את ההיכרות שלנו עם שיטות מרכזיות אלטרנטיביות של חיבור מחרוזת.
