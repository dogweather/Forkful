## מדוע

פונקציית ה- string.capitalize בשפת אלמ משמשת להפיכת האותיות הראשונות במחרוזת לאותיות גדולות. הפעולה הזו יכולה להיות שימושית במגוון מקרים, כגון עיצוב מחרוזות בצורה נכונה או לטיפול בקלט משתמש כדי להבטיח שהאותיות לא יהיו מערבולות.

## כיצד להשתמש 

נתחיל עם מספר דוגמאות פשוטות של כיצד להשתמש בפונקציית ה- string.capitalize בשפת אלמ:

```
Elm.string.capitalize "אלממיממימה" == "אלממיממימה"
Elm.string.capitalize "elm" == "Elm"
Elm.string.capitalize "Elm" == "Elm"
```

ניתן לראות שבכל מקרה, האות הראשונה של המחרוזת מוחזרת כאות גדולה, והאותיות הנוספות נשארות כמו שהן.

נסתכל עכשיו על דוגמאות מעט יותר מורכבות עם מחרוזות שכוללות גם אותיות גדולות נוספות:

```
Elm.string.capitalize "טקלסטן" == "טקלסטן"
Elm.string.capitalize "ELM פרוגרמינג" == "Elm פרוגרמינג"
Elm.string.capitalize "קוד ב- elm שולחן עבודה" == "קוד ב- elm שולחן עבודה"
```

ניתן לראות שהפונקציה מקיימת את מטרתה המבוקשת ומאפשרת הפיכת האות הראשונה לאות גדולה, תוך שימוש נכון גם באותיות גדולות נוספות שיכולות להיות חלק מהמחרוזת.
