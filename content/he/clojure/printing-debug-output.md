## למה

נתחבר לעולם של Clojure, שפת תכנות פונקציונלית וחסרת מצבי פגיעה. אחד הכלים שפותחים שלה מכילים הוא תפקיד המקלדת. לכן, העיקרון היסודי בתכנות הוא ליצור תכניות לעיבוד נתונים. אם אין לך מחלקות לשגרתי, כנראה שתצטרך להפעיל בדיקה על הקוד שלך כדי לבדוק האם הכל תקין. כלי אחד שמסייע לנו לבצע דיבוג בקוד הוא התפתחות בעזרת הפלט. המאמר הזה ילמד אותכם כיצד להדפיס את הפלט שלכם ולהשתמש בו בדיקת קוד.

## איך לעשות

מכאן נוכל לראות את דוגמאות של קוד עם הדפסת פלט וכיצד להשתמש בתוצאות כדי לבדוק את הקוד שלנו.

```Clojure
(defn sum [x y]
  (println "The first number is:" x)
  (println "The second number is:" y)
  (+ x y))
```

כאן, אנו משתמשים בפלט להדפיס את הערכים שלנו ואחר כך לרוא כי הם משמשים לחישוב הסכום בין שני המספרים. כך, נוכל לוודא שהערכים שלנו הועברו נכונים כדי לבצע את החישוב.

```Clojure
(sum 5 7)
```

בדוגמה זו, נכנס את הפונקציה שלנו עם כמה ערכים ונקבל כתוצאה את הפלט המודפס שלנו:

```
The first number is: 5
The second number is: 7
=> 12
```

כפי שניתן לראות, הפלט שהודפס מראה לנו את הערכים שהועברו לפונקציה ואת התוצאה הסופית של החישוב.
