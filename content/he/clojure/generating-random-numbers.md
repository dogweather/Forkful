# למה

יצירת מספרים אקראיים היא כלי חשוב בתחום התכנות. זה מסייע לנו ליצור נתונים מקריים שיכולים לשמש כקלט לאלגוריתמים, מבחנים ושיטות אחרות.

# איך לעשות זאת

הנה דוגמא ליצירת מספרים אקראיים בשפת Clojure באמצעות הפונקציה `rand`:

```Clojure
(rand) ; פלט אפשרי: 0.4323858794620854
(rand)
; פלט אפשרי: 0.20446752435908772
```

ניתן להשתמש בפרמטרים נוספים לפונקציה `rand` כדי לקבוע טווח נתונים ולהגביל את התוצאות של המספרים האקראיים. לדוגמה, ניתן להשתמש בקוד הבא כדי ליצור מספר אקראי בין 1 ל-10:

```Clojure
(rand 1 10) ; פלט אפשרי: 7.082341378612111
(rand 1 10)
; פלט אפשרי: 4.257938741067841
```

כדי ליצור מספרים אקראיים שלמים, ניתן להשתמש בפונקציה `rand-int`, כמו בדוגמאות הבאות:

```Clojure
(rand-int 10) ; פלט אפשרי: 5
(rand-int 10) ; פלט אפשרי: 9
```

ניתן גם להשתמש בפונקציה `shuffle` כדי ליצור רשימה אקראית של ערכים מתוך רשימה קיימת, כפי שמתואר בקוד הבא:

```Clojure
(shuffle ["אפשרות 1" "אפשרות 2" "אפשרות 3"])
; פלט אפשרי: ("אפשרות 2" "אפשרות 3" "אפשרות 1")
```
