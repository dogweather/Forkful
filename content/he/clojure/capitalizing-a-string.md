## למה

תכנות בשפת Clojure הופך תהליך התכנות לפשוט ויעיל עבור מתכנתים. כשאנו מכיבים מחרוזת ב-Clojure, נתונה המחרוזת תוחזר כפי שהייתה, ללא שינויים. אבל לפעמים יש צורך להגדיר את המחרוזת כך שהאות הראשונה תהיה גדולה. כזה לדוגמה שימושי הוא כאשר אנו מדביקים ערכים מתוך מחרוזת לשדות במסמך HTML. לעתים, כדי לכוון את כוחו של שאילתה, נתון שם סדרת אופציונלי של אופשיית התחלואות סופיות שמחזירה חדשות עם נתוני האיש.

## כיצד להפעיל

השתמש בפונקציה `(clojure.string/capitalize str)` כדי להפנות את המחרוזת בשם `str` לכתיב ראשוני, כך שהאות הראשונה תהיה גדולה. לדוגמה:

```Clojure
(clojure.string/capitalize "hello world")
```

הפלט יהיה:

```
"Hello world"
```

או, אם יש לך רשימה של שמות פרטיים, תוכל להשתמש בלולאת `map` על הערך של המחרוזת כדי להחזיר רשימה חדשה של שמות הקבוצה שלך כפי שכתבת אותם.

```Clojure
(map clojure.string/capitalize
  ["john" "joe" "jane"])
```

הפלט יהיה:

```
("John" "Joe" "Jane")
```

## להתעמק
