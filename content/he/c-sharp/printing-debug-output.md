## למה

תכנות הוא תהליך מורכב ומאתגר, וזהו מה שעושה אותו כיף! אבל לעתים קרובות, אנו מתקשים לתת מענה לשאלה התמיהה שמתה לנו שיש למה למעשה אנו עובדים. אחת התכונות החשובות בתכנות היא שימוש ב- debug output בזמן כתיבת הקוד. במאמר זה, אנחנו נלמד למה זה חשוב וכיצד מתבצעת הדפסת debug output בשפת סי-שארפ.

## כיצד לבצע

אם אתם מתכנתים מקצועיים או סטודנטים בכירים, כנראה שאתם מכירים את המשפט "כתוב את התכשיט לפני שנשים את השרשור". אם הינכם רוצים לכתוב קוד ברמה הגבוהה ביותר, עליכם להבטיח כי הקוד שלכם ישתמש באמצעי נבחר שיאפשר לכם לבחון את זה אם תחלתו. פתחו טיפוס חד פעמי שיכול לקבל קלט וקבלו את ספריות ההכרזה שלהן באמצעות הצבעוניות המצויינת:

```c#
static void Main(string[] args)
{
    string input = Console.ReadLine();
    Console.WriteLine($"Input: {input}");
}
```

כתבו את הקוד הזה לכדי תכונת שינוי צבעי המסך כך שתוכלו לזהות את הקלט שנמסר על ידי המשתמש. פתחו את התכונה הזו לתוצאה לפני שתפעילו אותו עצמכם:

```c#
Input: Hello World
```
