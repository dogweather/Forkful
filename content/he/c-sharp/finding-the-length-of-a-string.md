# למה

בתכנות ישנן מספר פעולות נפוצות שיש לבצע כמעט בכל קוד, וחישוב אורך מחרוזת הוא אחת מהן. ניתן לצפות שישתמשו בפעולה זו בכדי לאמת שהמחרוזת מכילה את המידע הנדרש או כדי לסדר ולעבד את הנתונים במדויקות וביעילות.

# כיצד לעשות זאת

בכדי למצוא את אורך מחרוזת בשפת C#, יש להשתמש בפעולת "Length". בדוגמאות הקוד הבאות, נראה איך לכתוב פעולה זו ומה הפלט שלה יהיה.

```C#
string myString = "שלום עולם";
int stringLength = myString.Length;
Console.WriteLine(stringLength); //הפלט יהיה "9"
```

כפי שניתן לראות, הפעולה "Length" מחזירה את האורך המדויק של המחרוזת, כולל רווחים ותווים מיוחדים.

בנוסף, לאורך מחרוזת ניתן להשתמש גם בעזרת פעולה נוספת נקראת "GetStringForEncoding". בדוגמאות הקוד הבאות, נראה כיצד להשתמש בפעולה זו ומה הפלט שלה יהיה.

```C#
string myString = "כיתוב בעברית";
int stringLength = Encoding.UTF8.GetByteCount(myString);
Console.WriteLine(stringLength); //הפלט יהיה "21"
```

הפעולה "GetByteCount" מחזירה את האורך של המחרוזת בבתים ולא בתווים. זה נחוץ במקרים שבהם נרצה ליצור מחרוזת בעבודה עם קידודים שונים.

# מכניסים לעומק

כאשר מחשבים את אורך מחרוזת בשפת C#, מתקבלת תוצאה מדויקת ומבוצעת ביעילות. פעולת "Length" מחשבת את האורך על ידי ספירת התווים במחרוזת. בנוסף, פעולת "GetByteCount" מחשבת את אורך המחרוזת על ידי ספירת הבתים בקידוד הנבחר.
