## למה
למה לכתוב קובץ טקסט בכלל? כתיבת קבצים נוחה ומאפשרת הבנה פשוטה יותר של הקוד המכיל את המידע. כמו כן, קבצי טקסט ניתנים לקריאה בכל מכשיר ומערכת הפעלה, מה שהופך אותם לכלי חשוב בבניית יישומי תוכנה.

## איך לעשות זאת
כדי לכתוב קובץ טקסט בשפת סי-שארפ, יש צורך לממש מחלקת File ושיטת יצירת הטקסט WriteAllText. כאשר מפעילים את השיטה עם פרמטרים המתאימים, היא יוצרת קובץ טקסט חדש או מעדכנת את הקובץ הקיים בתוכנה.

```C#
// דוגמה ליצירת קובץ טקסט חדש
File.WriteAllText("example.txt", "זהו קובץ טקסט חדש!");

// דוגמה לעדכון קובץ טקסט קיים
File.WriteAllText("example.txt", "זהו קובץ טקסט מעודכן!");
```

ניתן גם להשתמש בשיטות נוספות כגון WriteAllLines עבור כתיבת מספר שורות לקובץ, או AppendAllText להוספת טקסט לקובץ קיים.
