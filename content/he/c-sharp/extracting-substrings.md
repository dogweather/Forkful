## למה

מחרוזות (strings) הן חלק חשוב מתוך כל תכנית מחשב, ולעיתים קרובות נתקלים בצורך להפריד מחרוזת לחלקים קטנים יותר. כך נוצר הצורך בשימוש בפעולת חיתוך substring על מחרוזות. המאמר הזה יסביר לכם את היישומים השונים של חיתוך מחרוזות ואיך לממש אותו בשפת תכנות C#.

## איך לבצע חיתוך מחרוזות בשפת C#

הנה דוגמא פשוטה של יישום פעולת חיתוך substring בשפת C#:

```C#
string str = "מחרוזת לדוגמה";

// חיתוך המחרוזת ממיקום 2 עד סוף המחרוזת
string substr = str.Substring(2);
Console.WriteLine(substr); // הדפסת חלק המחרוזת: חרוזת לדוגמה"

// חיתוך המחרוזת ממיקום 4 באורך 5 תווים
string substr2 = str.Substring(4, 5);
Console.WriteLine(substr2); // הדפסת חלק המחרוזת: רוזת"

// חיתוך המחרוזת מהתו השביעי לסוף המחרוזת
string substr3 = str.Substring(6);
Console.WriteLine(substr3); // הדפסת חלק המחרוזת: לדוגמה"
```

בכל פעם שנפעיל את פעולת החיתוך substring על מחרוזת, אנו מכתיב את המיקום של התו הראשון שאנו רוצים להתחיל ממנו ואת האורך של החלק שאנו רוצים לחתוך. ניתן גם להשתמש במיקום של התו האחרון במחרוזת (-1) במקום לכתוב את האורך.
