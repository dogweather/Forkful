# למה

גנרטורי מספרים אקראיים הם כלי חשוב בתחום התכנות, שמאפשרים לנו ליצור מספרים רנדומליים בצורה פשוטה ומהירה. ניתן להשתמש בגנרטורי מספרים אקראיים במגוון רחב של אפליקציות תכנותיות, כגון משחקים, טסטים, ועוד.

# איך לעשות זאת

למטה תוכלו לראות כמה דוגמאות לקוד שמדגימות כיצד ליצור מספרים אקראיים בשפת C ולהדפיס אותם למסך.

```C
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(){
    // הגדרת משתנה לשמירת המספר האקראי
    int num;
    
    // הגדרת הזמן הנוכחי כזרע לגנרטור האקראי
    srand(time(NULL));
    
    // הפעלת פונקציית rand על מנת ליצור מספר אקראי בין 1 ל-10
    num = rand() % 10 + 1;
    
    // הדפסת המספר האקראי למסך
    printf("המספר שנוצר הוא: %d", num);

    return 0;
}
```
כדי ליצור מספר אקראי בטווח שבין min ל-max, ניתן להשתמש בפונקציית srand עם תוכן של rand() % (max-min+1) + min.

```C
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main(){
    // הגדרת משתנה לשמירת המספר האקראי
    int num;
    
    // הגדרת הזמן הנוכחי כזרע לגנרטור האקראי
    srand(time(NULL));
    
    // הפעלת פונקציית rand על מנת ליצור מספר אקראי בין 10 ל-30
    num = rand() % 21 + 10;
    
    // הדפסת המספר האקראי למסך
    printf("המספר שנוצר הוא: %d", num);

    return 0;
}
```

בכל הדוגמאות הוספנו גם את הפונקציה srand על מנת לגרום לגנרטור האקראי להשתנות בכל רציון, ולא רק ליצור את אותו המספר האקראי בכל
