## למה

למה חשוב לדעת את אורך המחרוזת? בתכנות בשפת C, ייתכן שיהיו תרגילים או פרויקטים שבהם יהיה צורך להשתמש באורך של מחרוזת. כדי לפתור את התרגיל או להשלים את הפרויקט, יהיה חיוני לדעת כמה תווים יש במחרוזת.

## איך לעשות זאת

כדי למצוא את אורך המחרוזת בשפת C, יש כמה אפשרויות. נבחר בדוגמא הפשוטה ביותר - באמצעות הפונקציה `strlen()` שקיימת בספריית המחרוזות המובנת בשפת C.

```C
#include <string.h>
#include <stdio.h>

int main() {
  char string[] = "שלום עולם"; // מחרוזת באורך של 10 תווים
  int length = strlen(string);
  printf("אורך המחרוזת הוא: %d", length); // תצוגת תוצאה: אורך המחרוזת הוא: 10
  return 0;
}
```

נראה כאן שאת אורך המחרוזת אנחנו מציבים למשתנה חדש ואז מדפיסים את התוצאה. חשוב לציין שאורך המחרוזת הוא מספר שלם (integer).

## חקירה מעמיקה

בנוסף לפונקציה `strlen()` שראינו בדוגמא, בשפת C קיימות עוד דרכים למצוא את אורך המחרוזת. למשל, ניתן לרוץ על כל התווים במחרוזת עם לולאת `for` ולהגדיל באופן פשוט את מספר התווים במחרוזת עד שנתקלנו בתו ייחודי שמסמל סוף המחרוזת - `\0`. אם אתם רוצים ללמוד עוד על כל האפשרויות, אתם מוזמנים לחקור יותר בעמוד העזרה של שפת התכנות C.
