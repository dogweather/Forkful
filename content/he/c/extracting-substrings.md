## למה

בכניסה זו אביא כמה דוגמאות מהתחום שלי - תכנות בשפת C. המאמר הזה מתאים לכלים - תוכנתיים, מתכנתים, אנשי מחשב, סטודנטים, וכל אחד שרוצה ללמוד ולהתאמן על הנושא הזה.

מהיכן אני מפרטי במאמר הזה אני מסביר מהם לכתוב וכיצד כדאי לעשות זאת בתכנית C.

## כיצד לעשות זאת

השתמשו בקטע הקוד הבא כדי להבין מה אני מנסה להסביר:

```C
#include<stdio.h>
#include<string.h>

int main() {
    char str1[] = "זה קטע טקסט המשמש כדוגמא עבור תת עולם";
    char extracted[10];
    strncpy(extracted, str1 + 13, 3);
    extracted[3] = '\0';
    printf("%s\n", extracted);
    return 0;
}
```
 
רואים בשורת ה־`#include` שאני משתמש בספריה התקינה `string.h`. ישנן מספר פונקציות בחבילה זו המשמשות לעבודה עם מחרוזות (strings). בדוגמא הזו, אני משתמש בפונקציית `strncpy` שמקפיצה על מחרוזת מסוימת מבפנים. קטע הקוד הזה יחתוך 3 תווים מהמחרוזת הגדולה ישר לתוך `extracted` ולאחר מכן ידפיס אותם.

`מילת`

כדי להבין קצת יותר בהרחבה על הפונקציה `strncpy` ניתן לצפות במקור הידע בוויקיפדיה [כאן](https://en.wikipedia.org/wiki/Strncpy).
