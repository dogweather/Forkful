## למה

מדוע מומלץ לך לקרוא על ארגומנטים שנאספים מקו הפקודה.

## כיצד לעשות זאת

כיצד ניתן לקרוא את הארגומנטים שנשלחו כתפריטים בקו הפקודה בעזרת שפת התכנות Cׁ. בהמשך תמצא דוגמאות קוד ותצוגה לחיפוש הארגומנטים והדפסתם במסך.

```C
#include <stdio.h>

int main(int argc, char *argv[])
{
	// בדיקת אם הוזן מספר מסויים של ארגומנטים
	if (argc == 1) 
	{
		printf("Usage: ./program_name [argument1] [argument2] ... \n");
		return 1; // הוצאה מהתוכנית בעזרת ערך 1
	}
	else
	{
		// הדפסת הארגומנטים לפי סדרם
		for (int i = 1; i < argc; i++)
		{
			printf("%s ", argv[i]);
		}
		printf("\n");
	}
	return 0;
}
```

לדוגמא, אם הרצת את הקובץ המצורף לעיל עם הארגומנטים "Hello" ו-"World", התוכנית תדפיס: `Hello World`.

כמו כן, ניתן להשתמש בארגומנט `argv[0]` בשביל לקבל את שם התוכנית עצמה. דוגמא:

```C
printf("The name of the program is %s \n", argv[0]);
```

## עיון מעמיק

כעת שאת/ה כבר מכיר/ה את הדרך לקרוא ולהשתמש בארגומנטים מקו הפקודה ב-C, יתכן שתרצה/י ללמוד עוד על הנושא. במקום לקבל תעודתית קטנה כפי שמופיעה בתוכנית הדוגמא למעלה, כדאי להשתמש בספריה מוכנה שכזאת: `getopt`. היתרונות של הספריה הם ייעודיות יותר ואפשרות לבדיקת חוקיות הארגומנטים. כמו כן, ניתן למצוא מידע נוסף וטיפים בכתבות ובפורומים באינטרנט.
