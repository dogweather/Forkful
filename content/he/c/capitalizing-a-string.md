## למה

זהו פוסט על שפת תכנות C עבור קוראי העברית שמתחשק לדעת כיצד ניתן להפוך מחרוזת לכיתוב רישיות. הנושא יתבסס על קוד קצר ופשוט כדי למצות את הרעיון בצורה ברורה וקלה להבנה.

## כיצד לעשות את זה

על מנת להפוך מחרוזת לכיתוב רישיות בשפת C, ישנן מספר דרכים שונות לבצע את זה. נדגים שתי דרכים פשוטות באמצעות קוד והדגמה של פלט אפשרי.

```C
#include <stdio.h>
#include <string.h>

// דרך 1: באמצעות הפונקציה strupr
int main() {
   char string[] = "כתוב פה מחרוזת כלשהי";
   printf("המחרוזת מכילה: %s\n", string);
   
   strupr(string);
   printf("המחרוזת עם כיתוב רישיות: %s\n", string);
   
   return 0;
}

// דרך 2: באמצעות לולאה לעבוד על כל אותיות המחרוזת ולהמירן לכיתוב רישיות בעצמה
int main() {
   char string[] = "כתוב פה מחרוזת כלשהי";
   printf("המחרוזת מכילה: %s\n", string);
   
   // בתוך הלולאה, אנו משתמשים בפונקציית toupper הממירה את התו לכיתוב רישי בודד
   for (int i = 0; i < strlen(string); i++) {
      string[i] = toupper(string[i]);
   }
   
   printf("המחרוזת עם כיתוב רישיות: %s\n", string);
   
   return 0;
}
```

פלט (לא משנה אייזה מה תיבחרו):

```
המחרוזת מכילה: כתוב פה מחרוזת כלשהי
המחרוזת עם כיתוב רישיות: כתוב פה מחרוזת כלשהי
```
