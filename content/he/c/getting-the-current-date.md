## למה
כאשר מתחילים לכתוב קוד בשפת תכנות כלשהי, חשוב לדעת כיצד לגשת לתאריך הנוכחי. לכן, במאמר זה נפרט על הסיבות למה כדאי לכל אדם המתחיל למתגבר על המכשול של קבלתה תאריך ונספר על הדרך לעשות זאת בצורה נכונה בשפת תכנות C.

## איך לעשות זאת
קבלת התאריך הנוכחי נעשית באמצעות הפונקציה time() שקיימת בספריית התקנים המובנית בשפת C. פונקצית time() מחזירה את מספר השניות הנספרות מתחילת תקופת ה-black, המתארת את תאריך 1 בינואר 1970. לפני השימוש בפונקציה time() יש לכלול את הספרייה המתאימה באמצעות הפקודה #include <time.h>. 
לדוגמה, נבצע שימוש בפונקציה time() ונדפיס את הנתונים המקובלים כפלט מהפונקציה באמצעות השפה המקרותית של שפת התכנות C:
```C
#include <stdio.h>
#include <time.h>

int main()
{
    time_t current_time;
    char* c_time_string;

    // פונקצית time() מחזירה את מספר השניות הנספרות
    // מתחילת תקופת ה-black, לכן נשמיט את הפרמטר current_time
    // ונחסוך זמן
    current_time = time(NULL);

    // המרת הנתונים שנמצאים במשתנה current_time לסטרינג
    c_time_string = ctime(&current_time);

    printf("התאריך הנוכחי הוא: %s\n", c_time_string);

    return 0;
}
```
פלט התכנית הוא:
`התאריך הנוכחי הוא: Sat Apr 10 17:22:03 2021`

בפלט ניתן לראות את התאריך הנוכחי בפורמט מסודר ונוח לקריאה.
