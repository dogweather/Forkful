## למה
כתיבת קודים בשפת C נחשבת למשימה קשה ומאתגרת, אבל לעתים מציאת הפתרון הנכון לבעיה יכולה להיות מאתגרת ומסיחת דעת. פעולה פשוטה כמו להמיר מחרוזות לאותיות קטנות עשויה לקדם את הקוד שלך ולפתור בעיות מורכבות. במאמר זה, אנו נתרכז בכיצד להמיר מחרוזות לאותיות קטנות בשפת C ונחקור את התהליך הנדרש.

## איך לעשות זאת
```C
#include <stdio.h>
#include <string.h>

// פונקציה הממירה מחרוזות לאותיות קטנות
char *to_lower_case(char *str){
    // קבלת אורך המחרוזת
    int len = strlen(str);

    // פונקצית לולאה עבור כל אות במחרוזת
    for(int i = 0; i < len; i++){
        // בדיקה האם התו הוא אות גדולה
        if(str[i] >= 'A' && str[i] <= 'Z'){
            // המרת התו לאות קטנה על ידי הוספת 32 לערך התו
            str[i] = str[i] + 32;
        }
    }

    // החזרת המחרוזת המומרת לאותיות קטנות
    return str;
}

// הדפסת פלט דוגמה
int main(){
    char str[50] = "HELLO WORLD";
    printf("%s", to_lower_case(str));

    return 0;
}
```

פלט: `hello world`

הסבר: נתחיל עם כלל פשוט - אותיות קטנות בשפת C מיוצגות על ידי ערכי ASCII העולים מ-97 ומעלה, בעוד אותיות גדולות מיוצגות על ידי ערכים נמוכים יותר שמתחלקים ב־32. לכן, כדי להמיר מחרוזת לאותיות קטנות, השתמשנו בלולאת for כדי לעבור על כל תו במחרוזת ובדקנו האם הוא אות גדולה או לא. אם כן, עשינו המרת הערך של התו כדי להפוך אותו לאות קטנה. לבסוף, החזרנו את המחרוזת המומרת לאותיות קטנות.
