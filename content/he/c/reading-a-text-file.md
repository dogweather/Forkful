# קריאת קובץ טקסט בשפת C
## מה ולמה?
קריאת קובץ טקסט בשפת C היא פעולה בה מסופקת מידע מקובץ טקסט לתוכנה. מתכנתים משתמשים בה כדי לשלוט על נתונים חצוניים, לשמור על המידע מאורגן, ולאפשר שיתוף מידע בין תוכנות שונות.

## כיצד:
עיינו בקוד הבא כדי לראות דרך לקרוא קובץ טקסט בשפת C:

```C
#include <stdio.h>

int main() {
    FILE *fp;
    char buffer[255];
    
    fp = fopen("test.txt", "r");
    
    fgets(buffer, 255, (FILE*)fp);
    printf("%s\n", buffer);
    
    fclose(fp);
    
    return 0;
}
```

Исполняя этот код, мы читаем первую строку из файла "test.txt". Входные данные будут отображены на консоли.

## בהקשר עמוק יותר:
### 1. היסטוריה:
היכולת לקרוא קבצים טקסט הייתה חלק משפת C מאז שנוצרה, ומאז זה נותרה כחלק בלתי נפרד מהשפה.

### 2. חלופות: 
ישנן שיטות אחרות לטעון מידע מקובצים, כולל שימוש בספריות צד שלישי, אך המתודה המזוהה מעלה היא המקורית לשפת C.

### 3. פרטי מימוש:
הפונקציה "fopen" שמשמשת לפתיחת הקובץ מחזירה מצביע לקובץ. אם הפונקציה מצליחה, היא מחזירה מצביע לקובץ. אם יש שגיאה, היא מחזירה NULL.
