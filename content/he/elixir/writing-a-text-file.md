## למה

כיצד נהנים מכתיבת קובץ טקסט.

כדי לתת אפשרות למשתמשים ליצור ולאחסן מידע בקובץ ישנם מספר דרכים. אחת מהדרכים הנפוצות היא כתיבת קובץ טקסט, שמאפשרת למשתמשים לכתוב ולקרוא מידע בצורה ברורה ומנומסת. הליכים הבאים יסבירו כיצד לכתוב קובץ טקסט ב-Elixir ואיך להציג את הפלט בקוד.

## כיצד

```Elixir
# יצירת קובץ טקסט חדש עם שמו המוגדר על-ידי המשתנה file_name
File.write(file_name, "תוכן הקובץ") 

# קריאה של קובץ טקסט קיים עם השם המוגדר על-ידי המשתנה file_name
File.read(file_name)

# הצגת הפלט של הקובץ עם המידע הנכתב בקוד
" התוכן של #{file_name} הוא: #{File.read(file_name)} "
```

כמו שניתן לראות בקוד שהוצג לעיל, כתיבת קובץ טקסט ב-Elixir היא פשוטה ומהירה. הכנסת הנתונים וקריאת הנתונים נעשים באמצעות פונקציות מובנות כדי לספק נוחות ויעילות למשתמש. ניתן להשתמש בקלות גם בפונקציות אחרות כדי לעצב את התוכן של הקובץ כפי שתרצו.

## חפירה עמוקה

כמו כל שפת תכנות אחרת, ב-Elixir ישנם מספר יישומים לכתיבת קבצי טקסט. כתיבת קובץ טקסט יכולה לשמש כדי ליצור *דוחות*, *תיעוד* או אפילו לשמור נתונים כגון חדרי צ'אט או רשימת משימות. בקצרה, כתיבת קובץ טקסט היא דרך נוחה ויעילה לכתוב, לאחסן ולקרוא מידע ב-Elixir
