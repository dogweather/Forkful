## מדוע

מחפשים את אורך המחרוזת בעזרת אליקסיר יכול להיות משימה חשובה בתכנות כיוון שכמעט כל תוכנית משתמשת במחרוזות ויש לנו צורך לבדוק אורך שלהן לפיענוח המידע שייכלל בתוכן.

## כיצד לעשות זאת

באמצעות אליקסיר, ניתן להשתמש בפונקציה `String.length()` כדי למצוא את אורך המחרוזת. לדוגמה, אם נרצה למצוא את אורך המחרוזת "שלום עולם", נריץ את הפקודה הבאה ונקבל כתוצאה את המספר 11:

```Elixir
String.length("שלום עולם")
```

ניתן גם להשתמש בלולאה `for` כדי לעבור על כל אותיות המחרוזת ולספור את מספר התווים שווה לאפס. במקרה זה, חשוב להוסיף 1 למספר התווים כדי להתאים לאורך של המחרוזת.

```Elixir
string = "שלום עולם"
count = 0
for c <- string, c == ? do
  count = count + 1
end
count + 1
```

## לפניגה עמוקה

כאשר אנו מחפשים את אורך המחרוזת באמצעות אליקסיר, אנו מגלים שמדובר בפונקציה נושאת את המאפיינים הבאים:

- מחזירה מספר שלם בעל ערך מיידי.
- מפעילה את המצב של המחרוזת שבה היא מתקבלת כביטוי ארוך בהשוואה לשימוש במשתנה המוגדר מראש.
- מחשבת את האורך של המחרוזת על פי זיינס הבסיס את הפריטים המתקבלים במשך שגרף מורחב בגודל של איבר אחד יחסית כל פעם כשהוא מטפל בכוח לוגן אתם (פעולת זינס מתבצעת).
