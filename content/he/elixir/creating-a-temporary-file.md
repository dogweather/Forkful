## Why

תיקיית קבצים זמנית היא כלי עזר חיוני בפיתוח באליקסיר. היא מאפשרת לנו ליצור תיקייה במטרה לאחסן נתונים שלא נשמרים לטווח ארוך. כאשר אנחנו יוצרים קוד או מתבצעת מחקר וניסיון, תיקיית קובץ זמנית מספקת דרך נוחה כדי לאחסן ולשמור נתונים זמניים.

## How To

הנה דוגמא פשוטה שמראה איך ליצור תיקיית קבצים זמניים באליקסיר:

```Elixir

file = Path.temp_file()         # יצירת קובץ זמני
IO.write(file, "Hello world!")  # כתיבת נתונים לקובץ
File.read(file)                 # קריאת הנתונים שנשמרו בקובץ
File.rm(file)                   # המחיקה של הקובץ הזמני

```

הקוד נוצר את קובץ זמני עם נתונים של "Hello world!" ואז קרא אותם חזרה ומחק את הקובץ. תיקיית קצבת קבצים זמנית מספקת דרך פשוטה כדי לבחון נתונים ולנסות קוד בלי לצורך ליצור ולמחוק קבצים באופן ידני.

## Deep Dive

לצורך היצירה וניהול תיקיות קבצים זמניות, אנחנו נזכור את הפונקציה Path.temp_file/2. הארגומנט הראשון הוא עיקרת השם של הקובץ, והשני הוא התיקייה בה שמירת הקובץ. אם לא נספק ארגומנט שני, אליקסיר ידאג ליצור את קובץ הזמני בתיקיית הקובץ המקורית של הקוד. אם נעביר את השם של תיקייה קיימת כארגומנט שני, אליקסיר ינסה ליצור את התיקייה בתוך התיקייה הזאת. אם התיקייה לא קיימת, אז אליקסיר ישתמש בתיקיית הבית של המשתמש.
