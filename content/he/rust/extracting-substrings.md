## למה

כתיבת קוד בספריית Rust יכולה להיות מאתגרת לפעמים, אך יעילותה וביצועיה המהירים משווה את המאמץ. אחת התכונות החזקות ביותר של Rust היא היכולת להפעיל פונקציות ניהול מחרוזות בצורה יעילה וממוקדת. אחת מתכונות הניהול שאנו נכנסים לעומק במאמר הזה הינה יכולת החיתוך של מחרוזות.

## איך לעשות זאת

לביצוע חיתוך של מחרוזת ב-Rust נדרשים שני משתנים - המחרוזת המקורית והאינדקסים של התתי מחרוזות שנרצה לחתוך. ניתן להשתמש בפונקציות כגון `slice` או `get` כדי לקבל את התתי המחרוזות המבוקשות. לדוגמה:

```Rust
let full_string = "Hello, my name is Rust!";
let first_word = full_string.slice(0..5);
// first_word = "Hello"
let last_word = full_string.slice(16..20);
// last_word = "Rust"
```

ציון האינדקסים נעשה על פי המספר הממורה לכל תו במחרוזת, לכן יש לקחת בחשבון שהאינדקסים מתחילים מ-0. כמו כן, ניתן להשתמש בתווים שליליים כדי לקבל חיתוך מסתיר על הסופיים של המחרוזת. לדוגמה, הפעולה: `full_string.slice(-5..)` תחזיר את התת-מחרוזת "Rust!".
