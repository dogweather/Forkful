## למה:

טכנולוגיית פיענוח הביטויים הרגילה, או גם היכולת לחיפוש והתאמת תבניות בטקסט, היא חלק בלתי נפרד מהתכנות בשפת Rust. על פניו, השימוש ברגולר אקספרשנסס יכול להיראות מסובך, אך בפועל זו כלי חזק וחשוב בעבור כל מפתח שרוצה לכתוב קוד יעיל ויציב. כתבוי פוסט זה נכתב בכדי לספק לכם לפנייה חלקה לעולם המרתק של הרגולר אקספרשנסס בפיתוח עם Rust.

## איך להשתמש:

כדי להשתמש ברגולר אקספרשנסס בRust, יש כמה אפשרויות לבחירה. אחת הדרכים הפשוטות ביותר היא להשתמש במודול "regex" המגיע עם התקנת Rust. כדי להתחיל, נציג קטע קוד פשוט שמדגים כיצד ניתן לחפש תבנית בטקסט:

```rust
use regex::Regex;

fn main() {
  let text = "מה נשמע, עולם?";

  let re = Regex::new(r"עולם").unwrap();

  if re.is_match(text) {
    println!("נמצאה תבנית!");
  } else {
    println!("לא נמצאה תבנית.");
  }
}
```

בקוד זה, אנו מייצרים אובייקט של המודול "Regex" ומשתמשים בו לחיפוש של תבנית מסוימת בטקסט שניתן כארגומנט. עם השימוש בפונקציה "is_match", אנו בודקים אם התבנית נמצאת בטקסט ומדפיסים תוצאה רלוונטית.
