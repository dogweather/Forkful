## למה

מחשבת לעצמכם לחשב תאריך בעתיד או בעבר? בפוסט הזה נלמד איך לעשות זאת בעזרת רוסט ונכיר כמה טכניקות שיכולות לסייע לנו בכך.

## כיצד לפעול?

תחילה נדע כיצד למצוא את התאריך הנוכחי, ולאחר מכן נושא את זה בחישובים עבור התאריכים בעתיד או בעבר. בקטע קוד הבא תוכלו לראות דוגמאות ופלט תוצאות:

```
Rust // הפעלה של המודול לניהול תאריכים
fn main() {
  use chrono::{Local, DateTime, Duration};
  
  // מציאת התאריך הנוכחי
  let now = Local::now();
  println!("התאריך הנוכחי הוא: {}", now);
  
  // חישוב תאריך בעתיד
  let future = now + Duration::days(7);
  println!("תאריך בעתיד: {}", future);
  
  // חישוב תאריך בעבר
  let past = now - Duration::weeks(2);
  println!("תאריך בעבר: {}", past);
}
```
**תוצאות:**

```
התאריך הנוכחי הוא: 2021-06-01 17:45:00.123456789+03:00
תאריך בעתיד: 2021-06-08 17:45:00.123456789+03:00
תאריך בעבר: 2021-05-18 17:45:00.123456789+03:00
```

## חפירה עמוקה

החישובים של תאריך בעתיד או בעבר יכולים להיות מורכבים יותר ממה שנראה במבחר הדוגמאות בהם נעזרנו כאן. ישנם מספר טכניקות שיכולות לעזור לנו לבצע חישובים ביעילות עבור תאריכים מורכבים יותר.

כדי לחשב תאריך בעתיד, נוכל להשתמש בפונקציות כגון `Duration::weeks()` או `Duration::days()` עבור חישוב כמה ימים או שבועות אנו רוצים להוסיף או להחסיר מהתאריך הנוכחי. נוכל גם להשתמש בקבוצת הפונקציות שמתחת למודול `Duration` כדי לבצע חישובים מורכבים יותר כגון רבעים של
