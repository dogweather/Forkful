## מה ולמה?
הצמדת מחרוזות היא הפעלה שלאורחה נמרגים שניים או יותר מחרוזות בקובץ טקסט אחד. מתכנתים עושים את זה כדי להפוך קוד לקריא יותר ולשלב קטעים של מידע טקסטואלי.

## איך?
הנה דרכים שונות להצמיד מחרוזות בשפת תכנות ראסט:

```Rust
// ?שיטה 1: משתמשים במתודה "push_str" של String
let mut s = String::from("שלום ");
s.push_str("עולם");
println!("{}", s);  // Prints: שלום עולם

// ?שיטה 2: משתמשים ב-rust מקרו 'format!'
let s1 = String::from("שלום ");
let s2 = "עולם";
let s = format!("{}{}", s1, s2);
println!("{}", s);  // Prints: שלום עולם
```

## צלילה מעמיקה
אם אתה רוצה לדעת עוד, ישנם נושאים בהם ניתן להעמיק. ראה את הרקע ההיסטורי של קישור מחרוזות, החלופות האפשריות ופרטים על המימוש שלך.

1. ברקע ההיסטורי של הצמדת מחרוזות, זה הפך להיות התקן הנקי שמונח בנוסחאות של שפות תכנות אחרות כמו Java, Python, JavaScript, ועוד. בראסט, הצורך בביצועים ובטיהור מחרוזת אמצעות המתודה 'push_str' והמקרו 'format!'.

2. לבנייה ושינוי מחרוזות, ראסט מציע גם שיטות אחרות. כל אחת מהן יש לה יתרונות וחסרונות משלה מבחינת הביצועים, הניקיונות והמימוש.

3. עד שאתה לא מדפיס קוד בשפה חופשית שניתן לשנות אותה-a, פרטי המימוש שלך יכולים להשפיע על הביצועים שלך. תחשוב על איך המחרוזות נשמרות בזיכרון ואיך הן משתנות.
