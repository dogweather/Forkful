# למה

למה לעסוק בהשוואת שתי תאריכים? השוואת תאריכים היא כלי חשוב בתכנות המאפשר לנו לבצע פעולות שונות בתכניות שלנו בהתאם לתאריכים שאנו משתמשים בהם. למשל, אם אנו כותבים אפליקציה לניהול מלאי, נצטרך לבדוק את ההבדלים בין תאריכי הגעה של מוצרים שונים כדי לדעת איזה מוצרים נמכרו יחדיו.

# איך לבצע שוואה בין שתי תאריכים

השוואת שתי תאריכים נעשית באמצעות השוואת מערכי התאריכים של שני התאריכים, כך שאנחנו יודעים איפה הם נמצאים בעבור מערכי התאריכים הסטנדרטיים. לדוגמה, אם אנו רוצים לבדוק אם תאריך מסוים הוא לפני תאריך אחר, נשתמש בפונקציית compareTo.

```Kotlin
val date1 = LocalDate.of(2021, 10, 1)
val date2 = LocalDate.of(2022, 1, 1)

if (date1.compareTo(date2) <= 0) {
    // תאריך 1 הוא לפני תאריך 2
} else {
    // תאריך 1 הוא אחרי תאריך 2
}
```

בנוסף לשוואת מערכי תאריכים, ניתן גם להשוות תאריכים בין עצמם באמצעות פונקציות נוספות כמו isAfter ו-isBefore.

```Kotlin
val date1 = LocalDate.of(2022, 1, 1)
val date2 = LocalDate.of(2021, 10, 1)

if (date1.isAfter(date2)) {
    // תאריך 1 הוא אחרי תאריך 2
}

if (date2.isBefore(date1)) {
    // תאריך 2 הוא לפני תאריך 1
}
```
