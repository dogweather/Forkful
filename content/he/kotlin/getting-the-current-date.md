טיולוגית קוטלין: כיצד לקבל את תאריך התקשורת הנוכחי בקוד

## למה

כאשר אתם מפתחים בקוטלין, ייתכן שתצטרכו להשתמש בתאריך נוכחי בקוד שלכם. זה משמש למצבים רבים, כמו ליישום טסטים או לשימוש בתכנית תאימות ומייצר נתונים דינמיים באופן שוטף. אין שום סיבה להימנע מהשתמשות בתאריך נוכחי, כך שהיא יכולה להיות יעילה לכם כמפתחים.

## כיצד לעשות זאת

עם קוד קוטלין פשוט, קל ואינטואיטיבי, קל לקבל את התאריך הנוכחי באמצעות התכונה "תאריך". בשביל לזהות את התאריך, עליכם ליצור משתנה שמכיל את התאריך הנוכחי ולהשתמש בפעולות הנכונות עליו.

```Kotlin
val currentDate = Date() // Create a Date object
println(currentDate) // Output will be in Tue Feb 25 11:02:24 IST 2020 format
```

את התאריך הנוכחי ניתן לקבל גם בפורמט אחר, כגון תאריך חסר ו- מלא, תצוגת התאריך בפורמט מסוים או עוד פורמטים שהם נוחים לכם.

```Kotlin
val sdf = SimpleDateFormat("dd/M/yyyy")
val currentDate = sdf.format(Date())
println(currentDate) // Output will be in 25/2/2020 format
```

כפי שאתם רואים, קבלת התאריך הנוכחי פשוטה וקלה עם קוד קוטלין.

## לחקור עומק

בחלק זה תוכלו לתת לעצמכם מידע נוסף על כמה פעולות שונות ומעניינות שניתן לבצע עם תאריך נוכחי בקוד קוטלין. כאשר ייצרתם משתנה שמכיל את התאריך הנוכחי, ניתן לגשת בקלות למידע נוסף כמו שעה, דקה, חודש, שנה ועוד.
