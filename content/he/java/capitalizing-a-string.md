##כיצד לעשות
העמוד הוא העםכו

```Java
String str = "hello world";
String capitalizedStr = str.toUpperCase();

System.out.println(capitalizedStr);
```

הפלט המוצג יהיה:
```
HELLO WORLD
```

הפעולה של הפתעה שולל לאך כונב יימה האל הזה לס
