## מדוע

למה לחפש את אורך המחרוזת? ייתכן שתרגישו שזה פעולה פשוטה ומיותרת, אבל ישנם מקרים בהם יהיה חשוב לקבל את גודל המחרוזת כחלק מתהליך שליטה על הנתונים שלכם. אם אתם עובדים עם טקסטים בתוכנות הפיתוח שלכם, או עם בסיסי נתונים, אתם תצטרכו להשתמש בפונקציה זו לעתים קרובות.

## כיצד

הנה דוגמא פשוטה של כיצד למצוא את אורך המחרוזת בשפת ג'אווה:

```Java
String s = "שלום עולם";
int length = s.length();
System.out.println(length);
```

כתוצאה מהקוד הנ"ל, יודפס המספר 9, מכיוון שהמחרוזת "שלום עולם" מכילה תווים בן 9. ניתן גם להשתמש במתודה `length()` כדי לקבל את אורך המחרוזת ישירות בפקודת ההדפסה, כמו בדוגמא הבאה:

```Java
String s = "שלום עולם";
System.out.println(s.length());
```

התוצאה תהיה זהה, אך כעת אנחנו מנתחים את המחרוזת ישירות בפקודת הדפסה.

## עיון מעמיק

אם נבצע עיון מעמיק יותר, נגלה שמתחת לכל עורף ומחרוזת נמצאת מערכת הבינארית של הטקסט. כאשר משתמשים במתודת `length()` נגיע לשימוש בספריית המחרוזת המובנת בג'אווה, שבתוכה נמצאת המתודה המטפלת בביטויים רגולריים ובפונקציות ייעודיות לעיבוד וניתוח טקסט. למשל, בפונקציות ייעודיות ניתן למצוא טכנולוגיות כגון פורמט של תאריך, כתובת אימייל, מספר טלפון ועוד.
