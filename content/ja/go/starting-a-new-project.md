## なぜ

新しいプロジェクトを始めるのに、なぜ誰かが参加するのか、その理由をご紹介します。

## はじめ方

Goプログラミングを始めるためのコーディングの例やサンプル出力を、"```Go ... ```"コードブロック内にてご紹介します。

```
package main

import "fmt"

func main() {
  fmt.Println("こんにちは、世界！")
}
```

出力：こんにちは、世界！

## 詳細を深く
