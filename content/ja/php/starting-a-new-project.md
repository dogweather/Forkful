## What & Why? (何となぜ？)

新しいプロジェクトの始め方は、PHPファイル作成から。新しいアイデア実現やスキル向上を目的として始めるものだ。

## How to: (方法)

```php
<?php
// 新しいプロジェクトのスタートポイント
echo "こんにちは！新しいPHPプロジェクトへようこそ。";

// シンプルな機能の実装例
function greet($name) {
    return "こんにちは、{$name}さん！";
}

// 関数のテスト出力
echo greet("世界");
?>
```

出力:
```
こんにちは！新しいPHPプロジェクトへようこそ。
こんにちは、世界さん！
```
