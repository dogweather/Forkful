## इसका कारण

गाइड में से फ़ाइल पढ़ने का कामायाब तरीका सीखकर आप स्विफ्ट प्रोग्रामिंग में अधिक कूशल हो सकते हैं और इससे आपको अपने प्रोजेक्ट्स में अधिक उपयोगी काम करने में मदद मिल सकती है।

## कैसे करें

अगर आप एक टेक्स्ट फ़ाइल को स्विफ्ट में पढ़ना चाहते हैं तो आपको निम्नलिखित चरणों का पालन करना होगा:

```Swift
// 1. फ़ाइल पथ बनाएं
let filePath = "अपनी_फ़ाइल_का_पूरा_पथ_यहां_लिखें"

// 2. स्ट्रीम ओपन करें
guard let fileStream = InputStream(fileAtPath: filePath) else {
    fatalError("फ़ाइल खोलने में त्रुटि!")
}
fileStream.open()

// 3. फ़ाइल में से डेटा पढ़ें
let data = try Data(readingFrom: fileStream)

// 4. डेटा को स्ट्रिंग में परिवर्तित करें
guard let fileContent = String(data: data, encoding: .utf8) else {
    fatalError("डेटा को स्ट्रिंग में परिवर्तित करने में त्रुटि!")
}

// 5. फ़ाइल से पढ़ी गई सामग्री दिखाएं
print(fileContent)
```

आप इस तरह से सीख सकते हैं कि कैसे स्विफ्ट में एक टेक्स्ट फ़ाइल को पढ़ा जाता है और उसका आउटपुट कैसा होता है।
