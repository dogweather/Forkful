## वजह

तकनीक युक्तियों के साथ लिखित फ़ाइल बनाना आपको अपने स्किल और कॉडिंग प्रैक्टिस को सुधारने में मदद कर सकता है।

## कैसे करें

आप छोटे से कोड उदाहरण के साथ निम्नलिखित कार्यान्वित कर सकते हैं:

```Swift
let text = "मेरा पहला टेक्स्ट फ़ाइल।" 

do {
    try text.write(to: URL(fileURLWithPath: "/Users/Documents/myFile.txt"), atomically: true, encoding: String.Encoding.utf8)
} catch {
    print("एक समस्या हुई। \(error)")
}

```

इस संग्रह में, आप कोड के साथ कुछ वस्तुतः भी जान सकते हैं:

- `text` एक स्ट्रिंग जो आप अपनी फ़ाइल में लिखना चाहते हैं।
- `try` की। यह आपको परिभाषित कोड ब्लॉक के अंदर किसी भी त्रुटि की जांच करने देता है।
- `catch` की। यह त्रुटि हैंडलिंग कोड ब्लॉक है। यह आपको कोड ब्लॉक के भीतर एक उपशीर्षक देता है।
- `URL` यूआरएल आपको फ़ाइल के स्थान को दर्शाता है।
- `atomically` एटॉमिक फ़ोल्डर को अमल्दमसंगत।

यदि आप अपनी पसंदीदा स्ट्रिंग में से एक टेक्स्ट फ़ाइल बनाना चाहते हैं, तो आप ऑट्पुट के रूप में निम्नलिखित फ़ाइल को पाएँगे:

```
मेरा पहला टेक्स्ट फ़ाइल।
```
