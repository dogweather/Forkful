## क्यूँ
एक निर्देशिका के अस्तित्व को जांचने में आपको क्यों रुचि हो सकती है, इसके पीछे कई कारण हैं। यह सुनिश्चित करना आवश्यक हो सकता है कि आपका प्रोग्राम सही संदर्भों या फ़ाइलों से डाटा पढ़े या संग्रह करे। यह भी सुनिश्चित कर सकता है कि आपके प्रोग्राम में कोई त्रुटियां न हों।

## कैसे करें
एक निर्देशिका का अस्तित्व जांचने के लिए, आपको सबसे पहले `FileManager` कक्षा का इस्तेमाल करके सिस्टम निर्देशिकाओं को प्राप्त करना होगा। फिर आप `fileExists(atPath:)` विधि का इस्तेमाल करके निर्दिष्ट निर्देशिका के अस्तित्व को जांच सकते हैं। यदि निर्दिष्ट निर्देशिका मौजूद होगा, तो विधि `true` को वापस देगी और अन्यथा `false` को वापस लौटाएगी। नीचे दिए गए कोड ब्लॉक में एक उदाहरण है:

```Swift
// एक सीधा निर्देशिका में अस्तित्व को जांचना
let directoryPath = "/Users/username/Documents"
if FileManager.default.fileExists(atPath: directoryPath) {
    print("निर्देशिका मौजूद है")
} else {
    print("निर्देशिका नहीं मौजूद है")
}
```

निर्देशिका के अस्तित्व को जांचने के लिए आप `fileExists(atPath:)` के साथ विभिन्न विकल्पों का भी उपयोग कर सकते हैं, जैसे कि निर्देशिका का अस्तित्व जांचने से पहले उसके अंतर्गत फ़ाइलों या निर्देशिकाओं को भी जांच सकते हैं।
