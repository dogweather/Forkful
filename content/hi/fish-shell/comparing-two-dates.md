## क्यों

दो तारीखों को तुलना करने का काम करने के पीछे आपको क्या कारण है? अक्सर हमारे पास दो अलग तारीखों को तुलना करने की आवश्यकता होती है, जैसे कि कोई ईवेंट की तारीख को दोबारा चेक करना, संगठनिक निर्णयों के लिए समय के रूप में दो अलग तारीखों को तुलना करना या शेड्यूल बनाने के लिए दो अलग तारीखों को तुलना करना। दो तारीखों को तुलना करने के लिए Fish Shell का उपयोग करने से आपको स्पीड और सुविधा मिलती है।

## कैसे 
```Fish Shell
# दो अलग तारीखों को तुलना करें
 set date1 (date -f "%Y-%m-%d" "2021-06-01")
 set date2 (date -f "%Y-%m-%d" "2021-07-01")

# दो तारीखों के बीच अंतर की गणना करें
 set diff (math "(" $date2 - $date1 ") / (24 * 60 * 60)")

# तुलना के परिणाम को प्रिंट करें
 printf "पहली तारीख: %s\nदूसरी तारीख: %s\nदोनों तारीखों के बीच अंतर: %s दिन\n" $date1 $date2 $diff
```

आउटपुट:

```
पहली तारीख: 2021-06-01
दूसरी तारीख: 2021-07-01
दोनों तारीखों के बीच अंतर: 30 दिन
```
