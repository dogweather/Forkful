## क्यों?

एक प्रोग्रामर होने के नाते, आप लगातार अपनी कोडिंग कौशल को बढ़ाने की कोशिश करते हैं। स्ट्रिंग जोड़ना इस दिशा में बहुत महत्वपूर्ण है। यह आपको कैसे समझाता है कि आप महत्वपूर्ण फ़ाइलों या डेटा संरचनाओं को एक साथ रख सकते हैं।

## कैसे करें?

```Fish Shell
set first_name "आदित्य"
set last_name "सिंह"
set full_name $first_name" "$last_name
echo $full_name
```

रनिंग यह कोड हमें "आदित्य सिंह" नाम की स्ट्रिंग छापेगा। यहां, हमने पहले दो अलग-अलग स्ट्रिंग बनाईं और फिर इनको जोड़कर एक नयी स्ट्रिंग बनाई। इस तरह से, आप अपनी जरूरत के हिसाब से अनेक स्ट्रिंग्स को जोड़ सकते हैं। 

## गहराई में जाओ

जब आप स्ट्रिंग्स को जोड़ते हैं, तो आपको ध्यान रखना चाहिए कि वे एक ही प्रकार की हों। अन्यथा, आपको अपेक्षित परिणाम नहीं मिलेगा। साथ ही, स्पेशल चरित्र जैसे अपोस्ट्रोफ़ी और दोहरे टिपण्णी उपयोग करने से पहले उन्हें एस्केप करना भी जरूरी है।

इसके अलावा, आप स्ट्रिंग जोड़ने के लिए `string join` भी उपयोग कर सकते हैं। इसका उपयोग अतिरिक्त रिसोर्स के रूप में नहीं होता है, लेकिन इसे पढ़ने और समझने में थोड़ा ज्यादा समय लग सकता है।
