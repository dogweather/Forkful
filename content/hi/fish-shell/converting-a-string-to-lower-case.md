## क्यों

कोई व्यक्ति एक स्ट्रिंग को लोअर केस में रूपांतरित करने में शामिल होने के लिए क्यों में रुचि रखता होगा:

स्ट्रिंग को लोअर केस में रूपांतरित करने से पहले, यह अधिकतर प्रोग्रामिंग भाषाओं में मामूली एक प्रक्रिया है। इससे स्ट्रिंग को मैनिपुलेट करना आसान होता है और उसे योग्य रूप से मानव पठनीय बनाया जा सकता है।

## कैसे करें

यदि आपके पास Fish Shell भाषा का उपयोग करने के लिए कोई अनुभव नहीं है, तो इसका सबसे आसान उपाय है उदाहरण के साथ सीखना। नीचे दिए गए कोड ब्लॉक का उपयोग करके आप स्ट्रिंग को लोअर केस में रूपांतरित करने के लिए प्रयास कर सकते हैं:

```Fish Shell
set str "HELLO WORLD"
set lower_str (string tolower $str)
echo $lower_str
```

आउटपुट:

`hello world`
