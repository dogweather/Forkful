## क्यों

क्या आप सोच रहे हैं कि आपको किसी भी काम के लिए आपको रैंडम नंबर की जरूरत क्यों है? ध्यान देने लायक है कि रैंडम नंबर अलग-अलग कंप्यूटर ने और भागीदारों द्वारा प्राप्त किए गए होते हैं और ऐसे हमेशा समान नहीं होते हैं। इसका मतलब है कि आप उन्हें अन्य प्रकार के आँकड़ों से पृथक कर सकते हैं। इसलिए, रैंडम नंबर आपके डेटा एनालिसिस, गेम डेवलपमेंट और अन्य तरह के प्रोग्रामिंग परियोजनाओं के लिए बहुत ही उपयोगी है।

## कैसे करें

बिना शेष बातों के, यह है कुछ सरल C# कोड जो रैंडम नंबर उत्पन्न करने के लिए है।

```C#
// 1 से 100 के बीच से एक संख्या उत्पन्न करें
int randomNumber1 = Random.Next(1,100);

// दोनों सीमाओं को शामिल करते हुए संख्याओं के बीच से एक संख्या उत्पन्न करें
double randomNumber2 = Random.NextDouble();

// एक 32-बिट सत्यापन नंबर उत्पन्न करें
int randomNumber3 = Random.NextInt32();

// एक 64-बिट सत्यापन नंबर उत्पन्न करें
long randomNumber4 = Random.NextInt64();
```

जैसा कि आप देख सकते हैं, हम `Random` क्लास का उपयोग करके रैंडम नंबर उत्पन्न कर सकते हैं। आप अपने आवश्यकतानुसार इन कोड स्निपेट्स को परिवर्तित कर सकते हैं और उनका उपयोग कर सकते हैं। अगर आप रैंडम नंबर प्राप्त करने के लिए अधिक विस्तृत समझना चाहते हैं, तो आप `Random` क्लास के डॉक्यूमेंटेशन को द
