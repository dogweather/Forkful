## क्यों

अभी भी लिखने का समय आ गया है क्योंकि प्रोग्रामिंग आजकल सबसे प्रसिद्ध और समृद्ध क्षेत्रों में से एक है। सेखते समय आप सिर्फ कुछ बुनियादी टॉपिक पर ध्यान देने की सीमा को छोड़ देंगे, पूरे प्रोग्रामिंग की दुनिया आपके सामने खुलती है।

## कैसे करें

आज हम देखेंगे कि C# में एक टेक्स्ट फ़ाइल कैसे लिखा जाता है। यह एक आसान प्रक्रिया है और आपको कुछ मिनटों में इसे समझने में मदद मिलेगी। हम सबसे पहले "System.IO" नामक नामस्थान को अपने प्रोग्राम में शामिल करेंगे। तो चलिए चलते हैं!

```C#
using System.IO;

...

// नया फ़ाइल बनाएं
string fileName = "sample.txt";

// StreamWriter का प्रयोग करके फ़ाइल खोलें
// यह आपको फ़ाइल में टेक्स्ट लिखने की सुविधा प्रदान करता है
using (StreamWriter writer = new StreamWriter(fileName))
{
    // एक वाक्य लिखें
    writer.WriteLine("हैलो सारे!");

    // कुछ और लिखें
    writer.WriteLine("मैं आपका स्वागत करता हूँ।");

    // फ़ाइल बंद करें
    writer.Close();
}

// फ़ाइल सफलतापूर्वक लिखा गया है!
```

फ़ाइल "sample.txt" को खोलने पर आप निम्नलिखित कोड के साथ निस्सन्देह संदेश देख सकते हैं:

```
हैलो सारे!
मैं आपका स्वागत करता हूँ।
```
