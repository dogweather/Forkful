## क्यों

हैलो दोस्तों! आज मैं आपको C# में Substrings (उपश्रृंखला) निकालना सिखाऊंगा। एक सभी समस्याओं को हल करने के लिए उपश्रृंखला निकालना एक उपयोगी और महत्वपूर्ण कौशल है। आपको निकाले गए उपश्रृंखला का प्रयोग अन्य प्रोग्रामों में भी किया जा सकता है जैसे कि पाठ, ईमेल नामों या उपयोगकर्ताओं की ऊंची पदों को निकालने के लिए।

## कैसे

सबसे पहले, हम समझेंगे कि उपश्रृंखला क्या है। यह एक भिन्न स्ट्रिंग (स्ट्रिंग के साथ संपर्क में) है जो किसी मूल स्ट्रिंग से निकला गया है। C# में हम उपश्रृंखला का प्रयोग `Substring()` फंक्शन के साथ कर सकते हैं। यह फंक्शन स्ट्रिंग के माध्यम से दो इंडेक्स (शुरुआती और अंत भिन्न) स्थानों को लेकर निकालती है और उपश्रृंखला उत्पन्न करता है। नीचे दिए गए कोड ब्लॉक में आप इसका उपयोग कर सकते हैं:

```C#
string str = "हैलो दुनिया";
string subStr = str.Substring(6, 6); // दुनिया
Console.WriteLine(subStr); // दुनिया
```

समझ आ गया? अब आप भी उपश्रृंखला निकालने के लिए तैयार हो गए हैं। हालांकि, ध्यान दें कि स्ट्रिंग के अंत को दाएं से शुरू करके हम संख्या 0 का उपयोग कर सकते हैं।
