Hindi में Gleam Programming के लिए एक आम ब्लॉग पोस्ट

## क्यों

आप सोचते हो के आपको अपने सिस्टम में जांचना है के क्या एक निर्देशिका मौजूद है। समस्या हो सकता है के आप किसी प्रोग्राम को लिख रहे है जो एक निर्देशिका में काम करने के लिए डालेगा। इस तरह के स्थिति में आपको पहले से ही जानना होगा के क्या निर्देशिका मौजूद है ताकि आप आगे बढ़ सके।

## कैसे करें

ऐसा करने के लिए, हमें Gleam फ़ंक्शन `exists?` का उपयोग करके जांच करना होगा। यह फ़ंक्शन दो पैरामीटर लेता है - निर्देशिका का पथ और एक `Boolean` की वापसी। अगर निर्देशिका मौजूद होता है, तो परिणाम `true` होगा जो हमारी अनुमति देता है के हम आगे बढ़ सके।

```Gleam 
exists?("/path/to/directory")
// परिणाम: true
exists?("/path/to/nonexistent/directory")
// परिणाम: false
```

## गहराई से जानें

जाँच निर्देशिका मौजूद होते हुए करना आसान हो सकता है, लेकिन यह काफी महत्वपूर्ण है। यदि आप अपने सिस्टम में निर्देशिकाओं का उपयोग कर रहे है, तो आपको यह जानने के लिए प्रतिदिन बहुत कुछ आ सकता है के क्या चल रहा है और कौनसे निर्देशिका हैं। Gleam में `exists?` फ़ंक्शन का प्रयोग करके, आप अपने सिस्टम को और अधिक बेहतर बना सकते है!
