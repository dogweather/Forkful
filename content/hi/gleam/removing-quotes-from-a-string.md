## क्या और क्यों?
एक स्ट्रिंग से उद्धरण चिह्न (quotes) हटाना अर्थात्‌ आपके पाठ डाटा से उन अतिरिक्त परतों – उद्धरण चिह्नों – को छीलना होता है। प्रोग्रामर इसे इनपुट को सैनिटाइज़ करने, स्ट्रिंग्स को प्रोसेसिंग के लिए तैयार करने या बस अपने एप्लिकेशन्स में चीजों को व्यवस्थित और संगति प्रदान करने के लिए करते हैं। अंतत: यह सब साफ़, उपयोग में लायक डाटा के बारे में है।

## कैसे:
Gleam में उद्धरण चिह्नों को हटाना सरल है। हम पैटर्न मिलान या निर्मित स्ट्रिंग फ़ंक्शन्स का उपयोग कर सकते हैं। यहाँ एक सरल उदाहरण दिखाने के लिए दिया गया है:

```gleam
pub fn remove_quotes(text: String) -> String {
  let without_quotes = string.trim(text, "\"")
  without_quotes
}

pub fn main() {
  let text_with_quotes = "\"Hello, World!\""
  let cleaned_text = remove_quotes(text_with_quotes)
  io.println(cleaned_text)
}
```

नमूना आउटपुट:
```
Hello, World!
```

## गहराई से जानकारी
ऐतिहासिक रूप से, स्ट्रिंग्स में उद्धरण चिह्नों के साथ डील करना पाठ प्रक्रिया और स्क्रिप्टिंग भाषाओं में एक आम कार्य रहा है। स्ट्रिंग्स अक्सर उपयोगकर्ता इनपुट होती हैं या फाइलों से पढ़ी जाती हैं, जो विभिन्न कारणों जैसे कि डेटाबेस डालने या फॉर्मेटिंग के लिए उद्धरण चिह्नों के साथ आ सकती हैं, जिनकी हटाने की आवश्यकता होती है।

Gleam में, हम `string.trim` फ़ंक्शन का उपयोग करके उद्धरण चिह्नों को हटा देते हैं। विकल्प हैं! हम स्ट्रिंग के माध्यम से लूप कर सकते हैं या नियमित अभिव्यक्तियों का उपयोग कर सकते हैं, लेकिन `string.trim` इस कार्य के लिए आपका सुविधाजनक उपकरण है क्योंकि इसकी संक्षेपता और प्रदर्शन के कारण।

यदि हम कार्यान्वयन विवरणों में गोता लगाएँ, तो `string.trim` प्रदत्त पैटर्न से मेल खाने वाले अक्षरों को स्ट्रिंग के आरंभ और अंत से हटाकर काम करता है। इसलिए, यदि आपकी स्ट्रिंग के दोनों छोरों पर उद्धरण चिह्न हैं, तो वे एक ही बार में कट जाते हैं। ध्यान रखें कि यह केवल उद्धरण चिह्नों को हटाता है यदि वे किनारों पर हैं; आपके पाठ के बीच में अच्छी तरह से बैठे उद्धरण चिह्न वहीं रहेंगे।
