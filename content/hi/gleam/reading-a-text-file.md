## What & Why? (क्या और क्यों?)

पाठ फाइल को पढ़ना का मतलब है डिस्क पर मौजूद फाइल से डेटा को प्रोग्राम में लाना। प्रोग्रामर्स ऐसा क्यों करते हैं? क्योंकि कभी आपको कॉन्फ़िगरेशन पढ़नी हो, या डेटाबेस के बिना डेटा की प्रोसेसिंग करनी हो, तो यह ज़रुरी हो जाता है।

## How to (कैसे करें):

Gleam में एक पाठ फाइल पढ़ना आसान है। नीचे एक सरल उदाहरण दिया गया है:

```gleam
import gleam/io

pub fn main() {
  let result = io.read_file("hello.txt")

  case result {
    Ok(contents) -> io.println(contents)
    Error(error) -> io.println(error)
  }
}
```

यदि "hello.txt" में लिखा है "नमस्ते दुनिया!", तो आउटपुट होगा:

```
नमस्ते दुनिया!
```

## Deep Dive (गहराई में जानकारी):

पाठ फाइल पढ़ना एक पुरानी परन्तु आज भी महत्वपूर्ण प्रक्रिया है। इतिहास में, यह कमांड लाइन इंटरफेस और स्क्रिप्टिंग भाषाओं में बहुत प्रयोग की जाती थी। ग्लीम में io मॉड्यूल `read_file` फंक्शन का प्रयोग करके, हम एक साधारण रूप से फाइल का पाठ प्राप्त कर सकते हैं।

इसके विकल्प के रूप में, हम स्ट्रीमिंग या बफर्ड रीड का उपयोग करके लार्ज फाइल्स को भी पढ़ सकते हैं, जिससे मेमोरी का बेहतर इस्तेमाल होगा। `read_file` आमतौर पर UTF-8 एन्कोडेड टेक्स्ट के लिए है। यदि फाइल अलग एन्कोडिंग में है, तो आपको अलग तरीके से पढ़ना और डीकोड करना होगा।
