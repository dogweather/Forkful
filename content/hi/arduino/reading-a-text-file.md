## क्यों
तेकयों करना चाहिए जब आप किसी टेक्स्ट फाइल को पढ़ना चाहते हैं? इससे आप अपने आर्डुइनो इलेक्ट्रॉनिक्स प्रोजेक्ट में अधिक गंभीरता और सुविधा प्राप्त कर सकते हैं।

## कैसे करें
टेक्स्ट फाइल पढ़ने के लिए, सबसे पहले हमें अपने आर्डुइनो बोर्ड को सीखना होगा कि वह कैसे संबंधित स्ट्रिंग ऑब्जेक्ट का उपयोग कर सकता है। नीचे दिए गए उदाहरण में, हम स्ट्रिंग ऑब्जेक्ट को सीखेंगे और टेक्स्ट फाइल से पढ़ने के लिए उसका उपयोग करेंगे।

```Arduino
// स्ट्रिंग ऑब्जेक्ट बनाएं
String fileName = "फ़ाइल_नाम.txt";

// फ़ाइल खोलें
File file = SD.open(fileName);

// फाइल से डेटा पढ़ें
while (file.available()) {
  String data = file.readStringUntil('\n'); // स्ट्रिंग ऑब्जेक्ट में डेटा स्टोर करें
  Serial.println(data); // सीरियल मॉनिटर पर डेटा प्रिंट करें
}

// फ़ाइल बंद करें
file.close();
```

जैसा कि आप उपरोक्त कोड से देख सकते हैं, हमने स्ट्रिंग ऑब्जेक्ट बनाया है और फिर उसका उपयोग करके टेक्स्ट फाइल से डेटा पढ़ा है।

## गहराई से जाएं
टेक्स्ट फाइल से डेटा पढ़ने के लिए, हमें सीखना होगा कि आपको अपने आर्डुइनो स्केच में SD लाइब्रेरी को इम्पोर्ट करना होगा। इसके लिए, स्केच नेविगेशन बार में "सेर्वो लाइब्रेरियों" के लिए ऑप्शन का चयन करें और फिर SD लाइब्रेरी को सेलेक्ट करें।
