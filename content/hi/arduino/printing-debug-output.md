# क्यों

आजकल अर्डुइनो बोर्ड बहुत लोकप्रिय हो गया है। लेकिन कभी-कभी हम सभी कोड सही से काम नहीं करता है। ऐसे में हम कोड के दुर्गमांशों का पता लगाने के लिए डीबग आउटपुट प्रिंट करने के लिए प्रोग्रामिंग करते हैं। इससे कोड में हुई गलतियों को पता लगाना और ठीक करना बहुत आसान हो जाता है।

# कैसे करें

```Arduino
int sensorPin = A0; // एनालॉग सेंसर का पिन नंबर
int sensorValue = 0; // सेंसर से मापा गया मूल्य

void setup(){
  Serial.begin(9600); // सीरियल मॉनिटर शुरू करें
}

void loop(){
  sensorValue = analogRead(sensorPin); // सेंसर से मूल्य पढ़ें
  Serial.println(sensorValue); // सीरियल मॉनिटर में मूल्य प्रिंट करें
  delay(100); // 100 मिलीसेकंड के लिए प्रतीक्षा करें
}
```

ऊपर दिए गए कोड ब्लॉक में हमने एक सेंसर से मूल्य पढ़ने और सीरियल मॉनिटर पर उसे प्रिंट करने का कोड दिखाया है। आप अपने कोड में भी ऐसे ही डीबग आउटपुट प्रिंट कर सकते हैं। सीरियल मॉनिटर पर मूल्य प्रिंट होते हुए आप कोड के दुर्गमांशों को आसानी से पता लगा सकते हैं।

# गहराई में जाने

डीबग आउटपुट प्रिंट करने के लिए `Serial.println()` फंक्शन का इस्तेमाल किया जाता है। इस फंक्शन के परामीटर में हम उस वैरिएबल या मूल्य को भेजते हैं जिसको हम प्रिंट करना चाहते हैं। इसके अलावा हम `Serial.print()` फंक्शन भी इस्तेमाल कर सकते हैं जो कि उसी तरह काम करता है। फर्क सिर
