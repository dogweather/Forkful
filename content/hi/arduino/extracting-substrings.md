## क्यों
एकाधिक टेक्स्ट स्ट्रिंग से उपग्रह निकालने का होना कुछ स्वतंत्र मनोरंजन पर अधिक संवेदनशील और भी गहरा प्रभाव डालता है.

## कैसे करें
एक उदाहरण के साथ दिखाया गया कोड ब्लॉक के साथ "```Arduino ...```":
```
String myString = "Hello World"; // टेक्स्ट स्ट्रिंग घोषित करें
int length = myString.length(); // टेक्स्ट स्ट्रिंग की लंबाई की गणना करें
String substring = myString.substring(0, 5); // यहाँ firstIndex से lastIndex तक स्ट्रिंग के उपग्रह को घोषित करें
Serial.print(substring); // सीरियल मॉनिटर में उपग्रह प्रिंट करें
```

उपरोक्त कोड उदाहरण, "Hello World" टेक्स्ट स्ट्रिंग के पहले 5 अक्षरों को उपग्रह के रूप में प्रिंट करेगा। यहाँ, आप टेक्स्ट स्ट्रिंग में दिए गए firstIndex से lastIndex तक की कोई भी संख्या दे सकते हैं, जो कि आपको अपनी ज़रूरत के अनुसार उपग्रह की लंबाई को नियंत्रित करने का अधिकार देता है।

## गहरी जाँच
एक टेक्स्ट स्ट्रिंग के साथ काम करने के दौरान, कई बार हमें उससे कुछ खास भाग निकालने की आवश्यकता होती है। उदाहरण के लिए, यदि हमें एक इमेल पता दिखाना हो, तो हमें उस इमेल पते की दोनों सरनेम और डोमेन नाम को छोड़कर, अन्य सब को प्रिंट करना होगा। इसके लिए, हम टेक्स्ट स्ट्रिंग से उपग्रहित पहली आंकड़ी या स्पेस का समर्थन कर सकते हैं। यह उपग्रह को लंबाई के आधार पर तय करेगा।
