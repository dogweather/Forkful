## क्यों

आर्दूइनो कोडिंग एक मजबूत और उपयोगी कौशल है जो एलेक्ट्रॉनिक्स प्रोजेक्ट्स को बनाने और स्केच करने में मदद करता है। यह आपको आपके रोबोट्स, स्मार्ट होम या सिंसक्स को स्केच और बनाने में सहायता प्रदान कर सकता है। आर्दूइनो कोडिंग के साथ खेलने में मज़ा भी हो सकता है, जो आपके मनोरंजनीय प्रोजेक्ट्स को भी नया जीवन दे सकता है।

## कैसे

```Arduino
// सादा आर्दूइनो स्केच

void setup() {
  // यहां आप अपने सेंसर्स और मोटर्स को कनेक्ट कर सकते हैं
}

void loop() {
  // अपनी सेंसर्स का डेटा पढ़ने के लिए
  int value = analogRead(A0);

  // कुछ कॉन्डिशन के आधार पर अपनी मोटर को कंट्रोल करने के लिए
  if (value > 500) {
    digitalWrite(12, HIGH); // मोटर को चालू करने के लिए
  }
  else {
    digitalWrite(12, LOW); // मोटर को बंद करने के लिए
  }

  // वेटिंग पीरियड के बाद पुनः करने के लिए
  delay(100);
}
```

यह आर्दूइनो स्केच आपको एक सादा स्मार्ट होम सिस्टम बनाने में मदद कर सकता है। सेंसर के आउटपुट को पढ़ने के लिए analogRead () फंक्शन का उपयोग किया जाता है और दिए गए कंडिशन के अनुसार मोटर को ऑन या ऑफ करने के लिए digitalWrite () फंक्शन का उपयोग किया जाता है।
