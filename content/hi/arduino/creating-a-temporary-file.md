# क्यों

क्या आपने कभी सोचा है कि अपने Arduino प्रोजेक्ट में कैसे एक अस्थायी फ़ाइल बनाने से आपको कई लाभ हो सकते हैं? अस्थायी फ़ाइलें जीवन की मान्यता को सम्बोधित करती हैं जब आपको प्रोजेक्ट को अवरोधित करने की जरूरत होती है।

# कैसे

अस्थायी फ़ाइल बनाना बहुत ही आसान हो सकता है। सबसे पहले, आपको `File` लाइब्रेरी को इन्क्लूड करना होगा। फिर, एक अस्थायी फ़ाइल ऑब्जेक्ट बनाने के लिए, `open()` फ़ंक्शन का उपयोग करें। आप अपनी फ़ाइल का नाम चुन सकते हैं और उसे बदल सकते हैं जब आप अपना प्रोजेक्ट पूरा कर लें। अस्थायी फ़ाइल को बंद करने के लिए `close()` फ़ंक्शन का उपयोग करें।

```
## Arduino कोड उदाहरण:
#include <File.h>

File tempFile;
tempFile.open("temp.txt", "w");
tempFile.println("Hello World!");
tempFile.close();
```

इस प्रोग्राम का आउटपुट "Hello World!" लिख कर `temp.txt` नाम की फ़ाइल में सहेजा जाएगा।
