# क्यों

एक आर्द्रिनो प्रोग्रामिंग में स्ट्रिंग कैपिटलाइज़ करने का कार्य क्यों किया जाता है? स्वयं को इस चुनौतीपूर्ण काम में शामिल करने के लिए कुछ उदाहरण हैं।

# हाउ टू

स्ट्रिंग को कैपिटलाइज़ करने के लिए आर्द्रिनो कोड का उपयोग कैसे किया जाता है? उम्मीद है कि आपको हमारे साथ एक रोगलगान जवाब मिल सकता है। यहाँ हमारे पास कुछ उदाहरण हैं:

``` Arduino
// स्ट्रिंग को कैपिटलाइज़ करने के लिए कोड उदाहरण

String str = "hello world";
Serial.println("लowercase स्ट्रिंग: " + str); // hello world

String capStr = str.toUpperCase();
Serial.println("अपरकेस स्ट्रिंग: " + capStr); // HELLO WORLD
```

# डीप डाइव

आपने हमारे साथ कैसे स्ट्रिंग कैपिटलाइज़ करने के लिए आर्द्रिनो कोड देखा है। लेकिन क्या आपको पता है कि इसकी सहायता से कैसे स्ट्रिंग के आरंभिक अक्षरों को कैपिटलाइज़ किया जाता है? आपको स्ट्रिंग को चर परिवर्तित करने के लिए "toupper" फ़ंक्शन का उपयोग करना होगा। इसके अलावा, आप डेटा टाइप को भी स्ट्रिंग से चर तक परिवर्तित कर सकते हैं। आप आर्द्रिनो कोड संग्रह ("library") में अधिक जानकारी प्राप्त कर सकते हैं।
