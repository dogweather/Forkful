## क्यों

संवादी C प्रोग्रामिंग के साथ जुड़ने के लिए, आपको अपने कोड को टेस्ट करने की आवश्यकता होती है। टेस्टिंग आपको सुनिश्चित करता है कि आपका कोड सही ढंग से काम करता है और आपको सुनिश्चित करता है कि कोड के साथ कोई गड़बड़ी नहीं है। यह आपको बैकग्राउंड में उन समस्याओं को सुलझाने में मदद कर सकता है जो आपने नहीं सोचा था कि वे हो सकते हैं।

## कैसे करें

"```C
#include <stdio.h> 

int main() {
    // आपकी फ़ंक्शन कोड यहाँ लिखें
    return 0; 
}

// आपका टेस्ट कोड यहाँ लिखें 
// आप कैसे आपके कोड को टेस्ट कर सकते हैं
```

टेस्ट करने से पहले, आपको सुनिश्चित करना होगा कि आपका कोड कंपाइल हो रहा है और दूसरे प्रोग्राम के साथ ठीक से संयोजित है। फिर, आपको विभिन्न प्रकार की टेस्ट के सेट बनाने हो सकते हैं, जैसे कि इनपुट का अक्षरिक (character) विभिन्न टेस्ट का उपयोग करके। यदि आप प्रोग्राम के साथ कोई गलतियां पाते हैं, आप इन टेस्ट कोड को उसी चीज़ को ठीक करने के लिए प्रयोग कर सकते हैं।
