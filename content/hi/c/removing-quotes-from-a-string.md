## क्या और क्यों?

एक स्ट्रिंग से उद्धरण चिह्नों को हटाने का मतलब उसकी सामग्री के हिस्से के रूप में शामिल किसी भी उद्धरण चिह्नों—चाहे वह एकल ('') हो या दोहरा ("")—को छाँटना है। प्रोग्रामर इसे इनपुट को सैनिटाइज करने, आगे प्रक्रिया के लिए डाटा तैयार करने, या उन भाषाओं में फाइल पथ और कमांड से निपटते समय सिंटैक्स त्रुटियों से बचने के लिए करते हैं जो स्ट्रिंग्स को डिमार्केट करने के लिए उद्धरण चिह्नों का उपयोग करते हैं।

## कैसे करें:

यहाँ एक C फंक्शन है जो आपकी स्ट्रिंग्स से उन परेशानी वाले उद्धरण चिह्नों को साफ कर देगा:

```c
#include <stdio.h>
#include <string.h>

void remove_quotes(char *str) {
    char *p_read = str, *p_write = str;
    जब तक (*p_read) {
        अगर (*p_read != '"' और *p_read != '\'') {
            *p_write++ = *p_read;
        }
        p_read++;
    }
    *p_write = '\0';
}

int main() {
    char str[] = "He said, \"Hello, 'world'!\"";
    printf("ओरिजिनल: %s\n", str);
    remove_quotes(str);
    printf("सैनिटाइज़ड: %s\n", str);
    return 0;
}
```

नमूना आउटपुट:

```
ओरिजिनल: He said, "Hello, 'world'!"
सैनिटाइज़ड: He said, Hello, world!
```

## गहन विवेचन

एक स्ट्रिंग से उद्धरण चिह्नों को हटाने का काम प्रोग्रामिंग के शुरूआती दिनों से है, जहाँ डाटा स्वच्छता त्रुटियों से बचने (जैसे कि SQL इंजेक्शन हमले) या यह सुनिश्चित करने में कुंजी थी कि एक स्ट्रिंग को सुरक्षित रूप से उन सिस्टमों को पास किया जा सकता है जो उद्धरण चिह्न को नियंत्रण वर्ण के लिए भ्रमित कर सकते हैं।

ऐतिहासिक रूप से, विभिन्न भाषाएँ इस कार्य को अलग तरह से संभालती हैं—कुछ में बिल्ट-इन फंक्शंस होते हैं (जैसे `strip` पाइथन में), जबकि दूसरों, जैसे कि C, डेवलपर्स को निचले-स्तरीय नियंत्रण देने के उनके फोकस के कारण मैनुअल कार्यान्वयन की आवश्यकता होती है।

विकल्पों में `strpbrk` जैसे लाइब्रेरी फंक्शन का उपयोग करना शामिल है ताकि उद्धरण चिह्नों को खोजा जा सके या अधिक जटिल पैटर्न के लिए नियमित अभिव्यक्तियों (PCRE जैसी लाइब्रेरियों के साथ) का उपयोग करना, हालांकि यह केवल उद्धरण चिह्नों को हटाने के लिए अत्यधिक हो सकता है।

ऊपर दिया गया कार्यान्वयन सिर्फ हर वर्ण में स्कैन करता है, स्ट्रिंग में लिखने वाले पॉइंटर स्थान पर केवल गैर-उद्धरण वर्णों की प्रतिलिपि बनाता है। यह कुशल है क्योंकि यह परिणामी स्ट्रिंग के लिए अतिरिक्त मेमोरी की आवश्यकता के बिना स्थान पर किया जाता है।
