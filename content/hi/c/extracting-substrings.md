## लोग आखिरकार अपनी मनपसंद भाषा में प्रोग्रामिंग क्यों करते हैं?

कंप्यूटर भाषाओं में से सबसे लोकप्रिय भाषाओं में से एक, सी सी प्रोग्रामिंग आज भी बहुत लोकप्रिय है। इसका कारण है कि सी सी काफी शक्तिशाली है और न केवल स्थिरता के साथ काम करती है, बल्कि सभी प्रकार के प्रोग्राम भी लिख सकती है। इसमें कई सब्राइनग्स प्राप्त करने के तरीके हैं, इसलिए आइए जानें कि सुबस्ट्रिंग्स को प्राप्त करने का सबसे आसान तरीका क्या है।

## कैसे सी प्रोग्राम में सबस्ट्रिंग निकालें?

यदि आप एक स्ट्रिंग में से कुछ अक्षर निकालना चाहते हैं, तो आपको स्ट्रिंग के स्थानांतरण ने आसान तरीके से सबस्ट्रिंग को प्राप्त करने में मदद कर सकता है। आइए एक उदाहरण देखें:

```C
#include <stdio.h>

int main() {
  char str[] = "Hello World";
  char substr[6];

  // 3 से शुरू होकर 8 तक के अक्षर सबस्ट्रिंग में कॉपी करें
  for (int i=3; i<8; i++) {
    substr[i-3] = str[i];
  }

  printf("सबस्ट्रिंग: %s", substr);

  return 0;
}

// Output: सबस्ट्रिंग: lo Wo
```

यहां, हमने सबस्ट्रिंग के लिए एक नया बफर बनाया है और उसमें आदेश 3 से शुरू करके पंचम अक्षर तक के अक्षरों को कॉपी किया है। आप आदेश को अपनी आवश्यकतानुसार बदल सकते हैं।
