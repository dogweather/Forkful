# सी प्रोग्रामिंग में अस्थायी फ़ाइलें: क्या, क्यों और कैसे?

## क्या और क्यों?

अस्थायी (temporary) फ़ाइल एक ऐसी फ़ाइल होती है जिसे कार्यक्रम की कार्यावधि के दौरान उत्पन्न किया जाता है और बाद में हटा दी जाती है। यहां प्रोग्रामर्स इसे उपयोग करते हैं बीच में उत्पन्न डाटा को संचित करने के लिए, जैसे की बड़े गणनाओं के परिणाम या संचारण के दौरान किसी डाटा की अस्थायी कॉपी।

## कैसे करें:
```C
// C कोड

#include <stdio.h>

int main() {
    char tempname[L_tmpnam];
    char* filename = tmpnam(tempname);

    FILE* tmpf = fopen(filename, "w");
    fprintf(tmpf, "यह एक उदाहरण है\n");
    fclose(tmpf);

    return 0;
}
```
साउटपुट:
```
अगर शीर्षक कोड सही रूप से चलता है, तो यह कोई साउटपुट नहीं देगा। लेकिन एक नई अस्थायी फ़ाइल उत्पन्न होती है जिसमें "यह एक उदाहरण है" लिखा होता है।
```

## गहरा डाइव:

अस्थायी फ़ाइलें एक तकनीक हैं जिन्हें प्रोग्रामिंग की शुरुआत से ही उपयोग किया जा रहा है। जैसे-जैसे सिस्टम की क्षमताएं बढ़ीं, यह अधिक सुविधाजनक और उपयोगी होते गए। इसके अलावा, अस्थायी फ़ाइलों के साथ काम करने के लिए फ़ाइल पड़ने और लिखने के माध्यमिक कार्यों का आपको अच्छा ज्ञान होना चाहिए।
