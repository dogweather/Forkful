## क्या और क्यों?

नया प्रोजेक्ट शुरू करना मतलब है कि एक नया सॉफ़्टवेयर उत्पादन की प्रक्रिया को आरंभ करना। प्रोग्रामर इसे नई समस्याओं को हल करने और अपने कौशल को सुधारने के लिए करते हैं।

## कैसे करें: 

```Lua
--स्ट्रिंग को प्रिंट करना
print("नमस्ते, दुनिया!")

--परिवर्तनीयों के साथ काम करना
local var1 = "नमस्ते"
print(var1)

--फ़ंक्शन बनाना
local function सलाम()
  return "नमस्ते, दुनिया!"
end
print(सलाम())
```
**आउटपुट:**
```
नमस्ते, दुनिया!
नमस्ते
नमस्ते, दुनिया!
```

## गहरी डाइव:

आधिकारिक रूप से 1993 में नील ग्रेगोर के द्वारा शुरू किया गया, Lua का उद्देश्य विभिन्न प्रकार के सॉफ्टवेयर उत्पादन की सरलता और बेहतरीन सहयोग का प्रदान करना था। 

Lua के विकल्पों में Python, JavaScript और Ruby शामिल हैं, जो सभी सामान्य कार्यों में Lua पर विजयी हो सकते हैं। फिर भी, Lua की सरलता और रचनात्मकता इसे वीडियो गेमों के लिए स्क्रिप्टिंग भाषा के रूप में एक लोकप्रिय विकल्प बनाती है।

Lua का कोड बाइटकोड में कम्पाइल होता है और Lua वर्चुअल मशीन द्वारा चलाया जाता है।
