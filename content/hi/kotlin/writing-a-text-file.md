# क्यों

आपने अनेक बार सुना होगा कि किसी प्रोग्राम को लिखने के लिए टेक्स्ट फाइल का उपयोग किया जाता है। इसका कारण है कि टेक्स्ट फाइल के माध्यम से हमारा डेटा कंप्यूटर में बने रहता है और हम उसे बाद में दोबारा उपयोग कर सकते हैं।

# कैसे करें

कोटलिन में टेक्स्ट फाइल लिखने के लिए ```Kotlin"file.writeText("यहां आपके टेक्स्ट होगा")``` सिस्टेम कॉल का उपयोग किया जाता है। यह आसानी से एक नया टेक्स्ट फाइल बना सकता है और उसमे दिए गए टेक्स्ट को लिख भी सकता है। नीचे दिए गए उदाहरण में हम "hello.txt" नाम के नए फाइल में "नमस्ते!" टेक्स्ट लिख रहे हैं:

```Kotlin
val file = File("hello.txt")
file.writeText("नमस्ते!")
```

यहां आपको नई फाइल में लिखा हुआ टेक्स्ट दिखेगा:

नमस्ते!

# गहराई में

टेक्स्ट फाइल लिखने का और भी कई तरीके हैं। आप ```Println()``` सिस्टेम कॉल का उपयोग भी कर सकते हैं। इसमें आपको एक फाइल और उसमें लिखने के लिए के लिए 2 ओवरलोड फंक्शन मिलेंगे।

यहां आपको दोनों फंक्शन की उपयोग करने का तरीका दिखाया गया है:

```Kotlin
val file = File("hello.txt")
file.printWriter().use {
    it.println("नमस्ते!")
    it.println("मैं Kotlin में टेक्स्ट लिख रहा हूँ!")
}
```

यहां आपको नई फाइल में लिखा हुआ टेक्स्ट दिखेगा:

नमस्ते!
मैं Kotlin में टेक्स्ट लिख रहा हूँ!
