## क्यों

डिबग आउटपुट प्रिंट करना एलिक्सिर प्रोग्रामर लोगों के लिए बहुत महत्वपूर्ण है। इसके माध्यम से हम अपने कोड में गलतियां और इससे होने वाले बग्स को पता लगा सकते हैं। यह हमें कोड को बेहतर बनाने और समस्याओं को सुलझाने में मदद करता है।

## कैसे करें

```Elixir
IO.inspect("Hello, world")
```

यदि आप अपने कोड में कोई चीज़ देखना चाहते हैं, तो आप `IO.inspect` का उपयोग करके उस चीज़ को प्रिंट कर सकते हैं। इससे आपको चीज़ देखने और समझने में मदद मिलेगी।

```Elixir
list = [1, 2, 3]
IO.inspect(list)
```
**आउटपुट:**

```
[1, 2, 3]
```

आप चाहे तो अपने कोड में एक की जगह भी `IO.inspect` का उपयोग कर सकते हैं। इससे आप अपने कोड के विभिन्न हिस्सों का पता लगा सकते हैं और समस्याओं को तुरंत सुलझा सकते हैं।

## गहराई में चलें
