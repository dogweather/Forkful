## क्यों

टेक्स्ट फ़ाइल पढ़ना क्यों महत्वपूर्ण है? आपको अपने पाठकों को कुछ मोटिवेशन देने की ज़रूरत है। यहाँ कुछ टेक्स्ट फ़ाइल पढ़ने से होने वाले हैं।

## कैसे करें

इस अनुभाग में, हम आपको कैसे एक टेक्स्ट फ़ाइल को खोलना, पढ़ना और उसकी जानकारी शोधने के लिए आसान तरीकों के बारे में बताएंगे। नीचे दिए गए कोड ब्लॉक में आप एक कोड संग्रह और नतीजा देख सकते हैं:

```Python
# टेक्स्ट फ़ाइल खोलना
file = open("example.txt", "r")

# सारी लाइनों को पढ़ना
for line in file:
    print(line)

# फ़ाइल बंद करना
file.close()

# खोजना और जानकारी प्राप्त करना
if "apple" in file.read():
    print("फलों के नाम में से एक़ है!")
```

उम्मीद करते हैं कि आपको समझ में आया होगा कि कैसे आप टेक्स्ट फ़ाइल को खोल सकते हैं और उसमें जानकारी खोज सकते हैं। हमारे उदाहरण में, हमने संभवतः एक चिकनी फ़ाइल खोली होगी जिसमें फलों के नाम थे। क्या आपको यकीन है कि आपको संदेश में बताया है? :)

## गहराई में

टेक्स्ट फ़ाइल एक बहुत ही आम प्रकार की फ़ाइल है। आप इसे अपने कंप्यूटर पर खोलकर अपने डेटा को आसानी से संगठित कर सकते हैं। इस अनुभाग में, हम बात करेंगे कि आप अन्य प्रकार की फ़ाइलें, जैसे कि CSV या JSON भी कैसे प्रोसेस कर सकते हैं।
