हम सभी जानते हैं कि प्रोग्रामिंग डायरेक्ट्री हमारे काम को सुगम बनाती है। लेकिन क्या आपको पता है कि उपयोगकर्ताओं को डायरेक्ट्री की उपस्थिति की जांच करना अत्यंत आवश्यक होता है। इस ब्लॉग पोस्ट में हम आपको बताएंगे कि डायरेक्ट्री की उपस्थिति को कैसे जाँचा जाता है और इसके लिए पायथन का उपयोग कैसे किया जाता है।

## क्यों

डायरेक्ट्री की उपस्थिति जांचना आपके कोड को और सुरक्षित बनाता है ओर स्टोरेज स्पेस को बचाने में मदद करता है। 

## कैसे

दी गई उदाहरण को अच्छे से समझने के लिए आपको अग्ले परिवेश में Python 3 इंस्टालेट किया होना चाहिए ओर आपको CD डीरेक्ट्री जो कि आपके सिस्टम पॉइंटिंग करना होता है।

``` Python
import os #आप जीस भी मोड्यूल को आपने इम्पोर्ट किया है ओर ही उसके बाद os मोड्यूल को जीसे आप एग्ज़ेक्सपा हो सकता है

if os.path.isdir("my_directory"): # आप कोड जीसे आप खुद बनाया है उसे चेक करना होता है ओर ही उसके बाद आप उसको बताता हो  की यूजर को डाइरेक्ट्री उसे करना है।
    print("यह डायरेक्ट्री महान है!") #अगर डायरेक्ट्री जीसे आपने आजीीय है उसे प्रिन्ट करना है।
else:
    print("डायरेक्ट्री मौजूद नहीं है।") #अगर डायरेक्ट्री गलत हो गई है तो आप कोड करेंगे तो उसे प्रिंट करना होता है।
```
