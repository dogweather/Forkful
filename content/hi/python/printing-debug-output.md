# क्यों

डिबग आउटपुट को प्रिंट करने का कारण है कि यह आपको आपके कोड में अस्थिरताएं ढूंढने एवं उन्हें सुलझाने में मदद करता है।

# कैसे करें

इस ब्लॉग पोस्ट के माध्यम से मैं आपको कुछ उदाहरण दिखाऊंगा जो आपको बेहतर समझने में मदद करेंगे कि कैसे आप अपने कोड में डिबग आउटपुट प्रिंट कर सकते हैं। नीचे दिए गए उदाहरण कोड ब्लॉक्स में आपको एक पाएंगे।

```
# उदाहरण 1:
num = 10
print("आपका दिया गया संख्या है:", num)

# आउटपुट:
आपका दिया गया संख्या है: 10
```

```
# उदाहरण 2:
name = input("अपना नाम दर्ज करें:")
print("नमस्ते,", name)

# आउटपुट:
अपना नाम दर्ज करें: अन्ना
नमस्ते, अन्ना
```

```
# उदाहरण 3:
list = [1, 2, 3, 4, 5]
for num in list:
  print("आपको पुराना लिस्ट है:", num)

# आउटपुट:
आपको पास है: 1
आपको पास है: 2
आपको पास है: 3
आपको पास है: 4
आपको पास है: 5
```
