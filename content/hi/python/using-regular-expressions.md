## क्यों

रेगुलर एक्सप्रेशन का उपयोग करना एक बहुत ही शक्तिशाली तरीका है जो कुछ भी टेक्स्ट मैचिंग और प्रोसेसिंग के लिए किया जाता है। यह पाठकों को अपनी पाठ्यक्रम लिखने और अपने कोड में एक प्रोफेशनल टच जोड़ने की सुविधा प्रदान करता है।

## कैसे करें

इस आगे, हम आपको संकेत देंगे कि आप कैसे पाइथन में रेगुलर एक्सप्रेशन का उपयोग कर सकते हैं।

```Python
import re

# एक चरण मैचिंग
pattern = "dog"
text = "I have a dog and a cat"

if re.match(pattern, text):
  print("चरण मिल गया!")
else:
  print("चरण पाई नहीं गया।")

# धाराएं खोजें
match = re.search(pattern, text)

if match:
  print("विकल्प मिला!")
  print("मिला पाठ शुरू: ", match.start())
  print("मिल गया पाठ समाप्त: ", match.end())
  print("मिला पाठ: ", match.group())
else:
  print("कोई मिलान नहीं मिला।")
```

आउटपुट:
```
चरण मिल गया!
विकल्प मिला!
मिला पाठ शुरू:  9
मिल गया पाठ समाप्त:  12
मिला पाठ:  dog
```

## गहराई में जाएँ

रेगुलर एक्सप्रेशन की गहराई के साथ काम करने का एक बहुत ही प्रभावी तरीका वास्तव में कैसे यह काम करता है और आप कोड के लिए भालू को लहराने में आपको कैसे सहायता करता है। हम आपको सलाह देते हैं कि आप अन्य धाराएं, पाइप और कैरेट जैसे अन्य रेगुलर एक्सप्रेशन के विकल्पों को जानें जो पाइथन में विशिष्ट हैं।
