## क्यों
क्या आप एक पाठ फाइल लिखने में शामिल होने के लिए कोशिश कर रहे हैं? शायद आप अपने कोड की स्थायित्व या एक्सेस नीतियों को बदलने के लिए उपयोग करना चाहते हों। एक पाठ फाइल में प्रतिस्थापित किए गए डेटा को संग्रहीत करना आसान हो सकता है और अनुभागों में आसानी से पहुंचने को सुनिश्चित कर सकता है।

## कैसे
इस प्रक्रिया को करना आसान है। निम्न उदाहरण को उल्लेख करें:

```python
# नई फ़ाइल बनाएं और उसका नाम दें
new_file = open("नया_फ़ाइल.txt", "w", encoding="utf-8")

# इसमें डेटा लिखें
new_file.write("हम इस फाइल को आसानी से खोल सकते हैं।")

# फाइल को बंद करें
new_file.close()

# फ़ाइल खोलें और उसका सामग्री पढ़ें
existing_file = open("नया_फ़ाइल.txt", "r", encoding="utf-8")
print(existing_file.read())

# कभी भी फ़ाइल को बंद जरूर करें
existing_file.close()
```

आउटपुट:

```
हम इस फाइल को आसानी से खोल सकते हैं।
```

## गहराई में
पाठ फ़ाइल के रूप में डेटा को संग्रहीत करने के लिए कई विकल्प हैं। कई भाषाओं में आप विभिन्न फ़ंक्शन का उपयोग कर सकते हैं जो आपको विशिष्ट परिणाम उत्पन्न कर सकते हैं। आप यहां से भी सीख सकते हैं कि कैसे आप अपने एप्लिकेशन में पाठ फाइल का उपयोग कर सकते हैं और अपने उपयोगकर्ताओं को समझाने के लिए गैर-प्रोग्रामर दोस्ताना समाचार डिजाइन कर सकते हैं।
