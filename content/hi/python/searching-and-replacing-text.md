## क्यों

टेक्स्ट खोज और बदलाव एक आम कार्य है जो हर Python डेवलपर को अपने प्रोग्रामिंग जोखिमों को कम करने और कोड के साथ स्थिरता बनाए रखने के लिए करना चाहिए। यह काम बहुत से टेक्स्ट संबंधित रूपों में किया जा सकता है जैसे कि डेटाबेस में स्ट्रिंग को एक स्ट्रिंग से दूसरी स्ट्रिंग बदलना, लोगों के नामों और पतों को एक सेक्शन से दूसरे सेक्शन में बदलना, इत्यादि। इसलिए, Python में टेक्स्ट को खोजने और बदलने की प्रक्रिया को समझने बहुत जरूरी है।

## कैसे करें

Searching और Replacing टेक्स्ट का सबसे आसान और सबसे अधिक उपयोगितरीन तरीका `str.replace()` है। इसका उपयोग करने से पहले, हमें `str.find()` या `str.index()` जैसे string methods का उपयोग कर के स्ट्रिंग नुकसानों को खोजने की आवश्यकता हो सकती है। उदाहरण के लिए, हम एक स्ट्रिंग में `John` को `Jane` बदलना चाहते हैं। नीचे दिए गए कोड ब्लॉक में हम यह देख सकते हैं:

```Python
# इस उदाहरण के लिए, हमारी स्ट्रिंग "My name is John" है
new_string = string.replace("John", "Jane")
print(new_string)
```

आउटपुट: "My name is Jane"

इस तरह से आप देख सकते हैं कि `str.replace()` कुछ hi समय बाद मेंये मदद कर सकता है जबकि य अन्य काम करने के लिए `str.find()` और `str.index()` का उपयोग करना हो सकता है। इसके साथ ही, हम मल्टीपल टेक्स्ट नुकसानों को एक साथ बदलने के लिए भी `str.replace()` का उपयोग कर सकते हैं।
