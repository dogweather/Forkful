## क्यों
आज की तारीख प्रोग्रामिंग में बहुत महत्वपूर्ण है। आप अपने प्रोग्राम में वर्तमान तारीख को प्राप्त करके उसे अपनी आवश्यकताओं के अनुसार उपयोग कर सकते हैं। वर्तमान तारीख को प्राप्त करने से आपकी प्रोग्राम में समय संबंधी समस्याओं को अनुकूलित करने में मदद मिलती है। 

## कैसे करें
सबसे पहले, हम `datetime` लाइब्रेरी को इंपोर्ट करेंगे। इसके बाद, हम `date` क्लास का एक ऑब्जेक्ट बनाएंगे और उसे `today()` फंक्शन के साथ जोड़ेंगे। यह फंक्शन वर्तमान तारीख और समय को `date` ऑब्जेक्ट के रूप में लौटाता है।

```Python
from datetime import date

today = date.today()

print("वर्तमान तारीख:", today)
```

आउटपुट:
```
वर्तमान तारीख: 2021-06-10
```

आप अन्य तारीख स्कीमों के साथ भी काम कर सकते हैं, जैसे कि महीने और दिन की वर्तमान तारीख मिलने पर:

```Python
print("महीना:", today.month)
print("दिन:", today.day)
```

आउटपुट:
```
महीना: 6
दिन: 10
```

## गहराई में
`date` ऑब्जेक्ट के साथ सबसे अच्छी बात यह है कि यह आपको एक साधारण तारीख रूप में भी लौट सकता है। इसका मतलब है कि आपको कोई स्ट्रिंग फॉर्मेटिंग करने की आवश्यकता नहीं होगी, आप इसे सीधे अपनी प्रोग्राम में शामिल कर सकते हैं। आप और भी तारीख स्कीमों के लिए [डॉक्यूमेंटेशन](https://docs.python.org/3/library/datetime.html#datetime.date) देख सकते हैं।
