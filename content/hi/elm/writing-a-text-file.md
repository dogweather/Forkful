## क्यों
क्या आपने कभी अपनी वेब ऐप या सॉफ्टवेयर में एक टेक्स्ट फाइल को निर्माण करने की आवश्यकता महसूस की है? आपको हिंदी समाचार पत्रों, लेखों या वेबसाइटों को समाकलित करने के लिए शैलियों की कमी हो सकती है? इसका समाधान एक टेक्स्ट फाइल पर लिखना हो सकता है! यह आसान, अंग्रेजी या अन्य भाषाओं में दिए गए निर्देशों के अनुसार रूपांतरित किया जा सकता है। इस ब्लॉग पोस्ट में, हम आपको बताएंगे कि आपको इस प्रकार की अंतर्जालीय स्वचालितता में हाजिर होने की आवश्यकता है कि एक टेक्स्ट फाइल कैसे बनाएं।

## कैसे
एल्म में टेक्स्ट फाइल बनाने के लिए, सबसे पहले एक text डेटा टाइप होना चाहिए, जो सामान्य लिखित जानकारी को represent करता है। यदि आप इसको एक भेदानक के रूप में सोचते हैं, तो यह कुछ इस तरह से दिख सकता है:

```
type alias Text = String
```

अब हमें यह text फाइल बनाना है, जिसका नाम हो सकता है hello.txt। इस लिखित विधि से, आप निर्दिष्ट फाइल नाम और उसमें लिखने की आवश्यकता है। हालांकि, यह करने से पहले, आपको इस फाइल को बनाने के लिए एक नया सत्र प्राप्त करने की ज़रूरत होगी। आप निम्न प्रकार हो सकता है:

```
import File exposing (write)

write "hello.txt" "नमस्ते दुनिया!"  -- Hello world in Hindi
```
