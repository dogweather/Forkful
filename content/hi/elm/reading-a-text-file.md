## क्यों

यदि आप एक विकसित प्रोग्रामर हैं और हिंदी में लिखा कोड पढ़ने और समझने में आसानी करना चाहते हैं, तो आप एल्म भाषा के बारे में अधिक जानने के लिए यह ब्लॉग पोस्ट ध्यान से पढ़ सकते हैं। यह आपको एक टेक्स्ट फाइल कैसे पढ़ते हैं और इसका उपयोग कैसे करते हैं, इसकी जानकारी देगा।

## कैसे करें

```elm
import Text

-- फ़ाइल पाठ पढ़ना
main =
  Text.readFile "sample.txt"
  |> Task.andThen handleFile
  |> Task.perform (\_ -> Text.default)

-- पढ़ने के लिए हैंडलर फ़ंक्शन
handleFile : Text.Text -> Text
handleFile text =
  text
  |> Text.lines
  |> List.filter (\line -> String.length line > 10)
  |> Text.unlines

```

पहले हम टेक्स्ट मॉड्यूल को इम्पोर्ट करते हैं जो फाइल पाठ पढ़ने को संभव बनाता है। फ़ाइल को पढ़ने के बाद, हमने `map` फ़ंक्शन का उपयोग करके पाठ को संशोधित किया है और `filter` फ़ंक्शन का उपयोग करके छोटे पाठ को निकाला है। इसके बाद हमने `unlines` फ़ंक्शन का उपयोग करके पाठ को फिर से लाइनब्रेक्स के साथ एक लम्बा पाठ के रूप में फॉर्मेट किया है। आप `sample.txt` के साथ अपनी स्वयं की प्रारंभिक सामग्री के साथ एक नमूना फाइल बना सकते हैं।

जब आप इस कोड को चलाएंगे, तो आपको नमूना पाठ के फाइल से पाया गया पाठ लॉग में दिखेगा। आप इसे अपनी प्रोग्राम में भी उपयोग कर सकते हैं।
