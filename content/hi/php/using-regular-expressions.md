# क्यों

आज की प्रोग्रामिंग में, एक मुख्य स्किल है यह कि आप अपने कोड में भिन्न-भिन्न पैटर्न्स को ढूंढकर निकाल सके और उनको मैच कर सके। इससे आपको बहुत से दिन-दहाड़े का काम बचता है और साथ ही आपके कोड में भी उस रूपांतरण को मैच करने के लिए संदर्भ मिलता है। इसीलिए रेगुलर एक्सप्रेशन्स का इस्तेमाल करना एक बहुत ही उपयोगी और आवश्यक है।

# कैसे करें 

```PHP
<?php
$string = "मेरा नाम हिन्दी है ।";
// से पहले और से बाद में स्पेस को हटाने के लिए रेगुलर एक्सप्रेशन का इस्तेमाल करें
$string = preg_replace('/[^A-Za-z\-]/', '', $string);
echo $string;
// नतीजा:
// मेरानामहिन्दीहै
?>
```

इसमें हमने `preg_replace()` फ़ंक्शन का इस्तेमाल किया है जो पहले दिए गये रेगुलर एक्सप्रेशन के मैच हुए हिस्सों को वही चीज़ से बदलता है जो आखिरी पैरामीटर में दी गयी होती है। इसके बाद हमने नए स्ट्रिंग को `echo` कराया जो हमारे उपरोक्त उदाहरण में `मेरानामहिन्दीहै` था। इसमें स्पेस नहीं जोड़े गए हैं और सभी व्यंजनों को इकट्ठा कर दिया गया है।
