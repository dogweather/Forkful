## क्या और क्यों?
PHP में वर्तमान दिनांक प्राप्त करना एक मौलिक कार्य है जो आपको सिस्टम की दिनांक और समय को पुनः प्राप्त करने और संशोधित करने की अनुमति देता है। यह लॉगिंग, समय-मुहर लगाने, कार्यक्रमों को अनुसूचित करने या आपके एप्लिकेशन्स में समय-संवेदनशील संचालन करने जैसे कार्यों के लिए महत्वपूर्ण है।

## कैसे:
### देशी PHP
PHP का निर्मित `date()` फ़ंक्शन वर्तमान दिनांक प्राप्त करने का सबसे सीधा तरीका है। आप प्रारूप पैरामीटर को निर्दिष्ट करके विभिन्न तरीकों से दिनांक को प्रारूपित कर सकते हैं।

```php
echo date("Y-m-d"); // आउटपुट: 2023-04-01 (उदाहरण के लिए)
echo date("l, F j, Y"); // आउटपुट: शनिवार, अप्रैल 1, 2023
```

समय क्षेत्र समर्थन के साथ दिनांक और समय प्राप्त करने के लिए, आप `DateTime` क्लास के साथ `DateTimeZone` का उपयोग कर सकते हैं।

```php
$dateTime = new DateTime('now', new DateTimeZone('America/New_York'));
echo $dateTime->format('Y-m-d H:i:s'); // आउटपुट: 2023-04-01 12:00:00 (उदाहरण के लिए)
```

### कार्बन का उपयोग करना (एक लोकप्रिय थर्ड-पार्टी लाइब्रेरी)
[कार्बन](https://carbon.nesbot.com/) `DateTime` के लिए एक सरल API विस्तार है जो तारीखों और समय के साथ काम करने के लिए एक साफ़ और अधिक प्रवाहित तरीका प्रदान करता है।

सबसे पहले, सुनिश्चित करें कि आपने कम्पोसर के माध्यम से कार्बन को स्थापित किया है:
```bash
composer require nesbot/carbon
```

फिर, आप इसका उपयोग वर्तमान दिनांक प्राप्त करने के लिए कर सकते हैं:

```php
use Carbon\Carbon;

echo Carbon::now(); // आउटपुट: 2023-04-01 12:00:00 (उदाहरण के लिए, डिफ़ॉल्ट प्रारूप में)
echo Carbon::now()->toDateString(); // आउटपुट: 2023-04-01
echo Carbon::now()->format('l, F j, Y'); // आउटपुट: शनिवार, अप्रैल 1, 2023
```

कार्बन PHP में तारीख-समय संभालने को समृद्ध करता है जिसमें पठनीयता बढ़ाकर और समय को संशोधित करने, तुलना करने, और प्रारूपित करने के लिए अतिरिक्त कार्यक्षमता जोड़ी जाती है।
