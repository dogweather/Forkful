## क्या & क्यों?
डिरेक्टरी मौजूद है या नहीं चेक करना इसका मतलब है कि हम जांचने की कोशिश कर रहे हैं कि क्या एक विशेष डिरेक्टरी उपयोगिता पाठ या फाइल सिस्टम में मौजूद है या नहीं। कार्यक्रमकर्ताओं को इसे ताकि उन्हें जांचें और समस्याओं से बचें जो डिरेक्टरी अनुपस्थित होने पर उठ सकती हैं।

## कैसे करें:
PHP में डायरेक्टरी को जांचने के लिए `is_dir()` फ़ंक्शन का उपयोग करें। इसकी वापसी मान `true` होती है अगर डायरेक्टरी मौजूद है, अन्यथा `false`.

```PHP
<?php
$dir = '/path/to/my/dir';

if (is_dir($dir)) {
    echo " डिरेक्टरी मौजूद है";
} else {
    echo "डिरेक्टरी अस्तित्व में नहीं है";
}
?>
```
ऊपरी कोड या तो "डिरेक्टरी मौजूद है" प्रिंट करेगा अगर डिरेक्टरी उपस्थित है, अन्यथा "डिरेक्टरी अस्तित्व में नहीं है" प्रिंट करेगा।

## गहराी से विचार:
PHP में `is_dir()` फ़ंक्शन PHP 4 के वेर्जन से उपलब्ध है। यह एक विशेष डायरेक्टरी की ओपन बेसडीर जांच करता है, स्थानीय फ़ाइल सिस्टम या URL। इसका विकल्प फ़ंक्शन `file_exists()` हो सकता है, जो फ़ाइल या डिरेक्टरी दोनों की जांच कर सकता है, लेकिन अगर आप सुनिश्चित करना चाहते हैं कि यह वास्तव में एक डिरेक्टरी है, तो `is_dir()` का प्रयोग बेहतर होगा।
