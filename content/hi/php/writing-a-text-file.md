## क्यों

क्या आपने कभी सोचा है, आप आपके PHP कोड को कैसे स्टोर कर सकते हैं? टेक्स्ट फ़ाइल लिखना PHP में एक बहुत ही उपयोगी और सरल तरीका है अपने कोड को स्टोर करने का। इस लेख में, हम आपको टेक्स्ट फ़ाइल लिखने की प्रक्रिया के बारे में सीधे समझाएंगे और कुछ उदाहरण भी साझा करेंगे।

## कैसे करें

```PHP
<?php
// नया फ़ाइल बनाएं
$file = fopen("my_code.txt", "w");

// कुछ कोड टेक्स्ट में लिखें
$txt = "echo 'Hello World!';";
fwrite($file, $txt);

// फ़ाइल बंद करें
fclose($file);

// फ़ाइल से कोड पढ़ें
$code = file_get_contents("my_code.txt");
eval($code);
?>
```

उपरोक्त कोड स्निपेट में, हमने `fopen()` फ़ंक्शन का उपयोग करके नया फ़ाइल बनाई है और `fwrite()` का उपयोग करके उसमें टेक्स्ट लिखा है। फ़ाइल को बंद करने के लिए हमने `fclose()` का उपयोग किया है और `file_get_contents()` और `eval()` फ़ंक्शन का उपयोग करके फ़ाइल से कोड पढ़ा और चलाया है। इस तरह, आप अपने कोड को स्टोर कर सकते हैं और उसे बाद में फिर से इस्तेमाल कर सकते हैं।

## गहराई में
