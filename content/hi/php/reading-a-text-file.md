## क्यों

यदि आप एक PHP प्रोग्रामर हैं और आप एक भाषा में लिखे गए पाठ को पढ़ने या संपादित करने के लिए अपने कोड में इस्तेमाल करना चाहते हैं, तो आपको एक टेक्स्ट फ़ाइल को खोलने की आवश्यकता हो सकती है। इस लेख में हम आपको PHP में एक टेक्स्ट फ़ाइल को कैसे पढ़ें और संपादित करें के बारे में बताएंगे।

## कैसे करें

एक टेक्स्ट फ़ाइल को PHP में पढ़ने के लिए, हम `fopen()` और `fread()` फ़ंक्शन का उपयोग कर सकते हैं। नीचे दिए गए उदाहरण में, हम एक फ़ाइल को खोलते हैं, उसमें से डेटा पढ़ते हैं और फिर उसे बंद करते हैं।

```PHP
<?php
$file = fopen("example.txt", "r"); // फ़ाइल को खोलते हैं
$contents = fread($file, filesize("example.txt")); // फ़ाइल से डेटा पढ़ते हैं
fclose($file); // फ़ाइल को बंद करते हैं
echo $contents; // डेटा को प्रिंट करते हैं
?>
```

यदि हम फ़ाइल को संपादित करना चाहते हैं, तो हम `fopen()` फ़ंक्शन के दूसरे आर्ग्यूमेंट को `w` से बदल सकते हैं। इससे फ़ाइल को खोलने से पहले पूर्व मौजूदा डाटा हटा दिया जाएगा।

```PHP
<?php
$file = fopen("example.txt", "w"); // फ़ाइल को खोलते हैं
fwrite($file, "Hello world!"); // डेटा लिखते हैं
fclose($file); // फ़ाइल को बंद करते हैं
?>
```
