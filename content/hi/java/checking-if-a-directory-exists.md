## क्यों

जब हम किसी नए प्रोग्राम को बनाते हैं, तो ध्यान देना बहुत आवश्यक होता है कि क्या सभी आवश्यक फ़ॉल्डर और फाइल हमारे सिस्टम में मौजूद हैं या नहीं। किसी फ़ाइल या डिरेक्टरी के अनुपलब्ध होने से हमारा प्रोग्राम ठीक से काम नहीं कर सकता है, इसलिए इसे जांचना बहुत आवश्यक होता है।

## कैसे करें

जब हम अपने Java प्रोग्राम में कोड लिखते हैं, तो हमें यह जानने की जरूरत होती है कि कैसे हम फ़ाइल और फ़ोल्डर को जांच सकते हैं। यह काफी सरल है, इसके लिए हम कुछ कोड ब्लॉक का उपयोग करते हैं और उससे हम जांच सकते हैं कि दिए गए पथ पर कोई फ़ाइल या फ़ोल्डर मौजूद है या नहीं। नीचे दिए गए कोड ब्लॉक का उपयोग करके हम एक फ़ाइल के मौजूद होने की जांच कोड कर सकते हैं।

```Java
// फ़ाइल के अस्तित्व की जांच करें
File file = new File("C:/Documents/myfile.txt");
if (file.exists()) {
    System.out.println("फ़ाइल मौजूद है।");
} else{
    System.out.println("फ़ाइल मौजूद नहीं है।");
}
```

उदाहरण के लिए, यदि आपके सिस्टम में "C:/Documents" फ़ोल्डर में "myfile.txt" नाम की फ़ाइल है, तो पहला संरचना चलने के बाद, आपको फ़ाइल मौजूद है का संदेश मिलेगा। लेकिन अगर "myfile.txt" नाम की फ़ाइल मौजूद नहीं है, तो आपको "फ़ाइल मौजूद नहीं है" का संदेश मिलेगा। इ
