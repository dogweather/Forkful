## क्यों

क्या आप जानते हैं कि रैन्डम नंबर्स जावा में क्यों इतने महत्वपूर्ण हैं? वे खेलों और क्रिप्टोग्राफ़ि सहित कई प्रकार के कंप्यूटरीज़ एप्लिकेशन में उपयोगी हो सकते हैं। रैन्डम नंबर एक सरल तरीका है जिससे सुदूर भविष्य के जुड़ाव आ जाते हैं जो दिखायी करते हैं कि एक कंप्यूटर का कैसे होना चाहिए। रैंडम नंबर हेंडसैड्स, क्रैप्टोग्राफ़िक्स और आकस्मिक अनुसंधान में भी उपयोग में आते हैं।

## कैसे करें

अब जब की आप जानते हैं की रैन्डम नंबर्स उतने महत्वपूर्ण क्यों हैं, हम बताएंगे कि आप इसे कैसे जावा में प्रोग्रामिंग कर सकते हैं। जावा में रैन्डम नंबर्स बनाने के कई तरीके हैं। यहां हम दो उदाहरण देंगे जिसके माध्यम से आप रैंडम नंबर्स बना सकते हैं।

```
/*
 * यह एक साधारण उदाहरण है जिसमें आप एक सेट से रैंडम नंबर्स प्राप्त कर सकते हैं।
 */

import java.util.Random;

public class RandomExample {
    public static void main(String[] args) {
        Random rand = new Random(); // एक रैंडम ऑब्जेक्ट बनाएं
        int num = rand.nextInt(10); // 0 से 9 तक के बीच एक रैंडम संख्या प्राप्त करें
        System.out.println("रैंडम नंबर है: " + num);
    }
}
```
आउटपुट:
```
रैंडम नंबर है: 6
```
