## क्यों

किसी भी प्रोग्रामिंग भाषा में, गैर-इंटरेक्टिव कंटेंट में टेक्स्ट को फिंड और रिप्लेस करना एक अत्यंत महत्वपूर्ण कौशल है। यह आपकी कामकाज को आसान बनाता है और आपको समय की बचत करता है। यह भी कुछ स्थितियों में आवश्यक हो सकता है जब आपको बड़ी संख्या में डेटा में बदलाव करने की आवश्यकता होती है।

## कैसे करें

अगर आपको टेक्स्ट को फिंड और रिप्लेस करना है तो आपको `String` क्लास के `replace()` मेथड का उपयोग करना होगा। इसमें आपको दो पैरामीटर पास करने होंगे, पहला होगा उस शब्द को जिसे आप रिप्लेस करना चाहते हैं और दूसरा होगा उस शब्द को जिसे आप उस से रिप्लेस करना चाहते हैं। नीचे एक उदाहरण है:

``` java
String str = "मैंने बिस्तर पे यह शब्द लिखा है कि आप इसे देख सको";
String replaced = str.replace("बिस्तर", "मेज़");
System.out.println(replaced);
```

उपरोक्त कोड का आउटपुट `मैंने मेज़ पे यह शब्द लिखा है कि आप इसे देख सको` होगा। साथ ही, आप `replaceAll()` मेथड को भी प्रयोग कर सकते हैं जब आप एक regex के आधार पर रिप्लेस करना चाहते हो।
