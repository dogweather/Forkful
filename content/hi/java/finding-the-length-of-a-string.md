## क्यों

आपने शायद कभी सोचा है कि स्ट्रिंग की लंबाई को जानना कितना महत्वपूर्ण है? आमतौर पर, हम लोग शब्दों के साथ काम करते हैं और स्ट्रिंग लंबाई जानने की आवश्यकता क्यों होती है? उसके लिए आगे पढ़िए!

## कैसे करने के लिए

कई बार हमें यह पता नहीं चलता कि हमारी लिखी स्ट्रिंग की लंबाई क्या है। इस समस्या को हल करने के लिए, हम स्ट्रिंग का लंबाई निकालने के लिए बहुत स्ट्रेटेजी के साथ कोड कर सकते हैं।

यहां हम एक Java में "strlen()" फ़ंक्शन द्वारा स्ट्रिंग की लंबाई निकालने का उदाहरण दे रहे हैं।

*जवा कोड के लिए:*

```Java
public class FindStringLength {

	public static void main(String[] args) {
		String str = "हाय, मैं हूँ शुभम";
		//मदद की गई स्ट्रिंग की पहचान
		System.out.println("लंबाई निकालने के लिए फिर से : " + str.length());
	}
}
```
*आउटपुट:*

लंबाई निकालने के लिए फिर से : 17

## गहराई में खोज

स्ट्रिंग की लंबाई को निकालने का यह अभ्यास उस समय आता है जब हमें किसी काम के लिए स्ट्रिंग की लंबाई की आवश्यकता होती है। स्ट्रिंग लंबाई को जानना हमें उस स्ट्रिंग के कार्यों के साथ काफी मदद करता है जो हम कोडिंग में करते हैं। इससे हम अपने कोड को सुंदर और सरल बना सकते हैं।
