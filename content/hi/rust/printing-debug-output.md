व्यापक रूप से देखा जाए, रस्ट से शुरू करना मुश्किल हो सकता है और प्रोग्राम चलाने के बाद यह काफी समय ले सकता है। इसलिए, debugging को साइट पर debug output को छापने के लिए उपयोगी है।

## क्यों

यदि आप किसी भी प्रोग्रामिंग भाषा के साथ काम करते हैं, तो debugging output धारण या डीबग करने से आपको पता चलता है कि क्या गलत हो रहा है और आप कैसे उसे ठीक कर सकते हैं। Rust में, हम कुछ debug macros का उपयोग करके अपना debug output प्रिंट कर सकते हैं। इससे हमें अपनी कोड में त्रुटियां ढूंढने में मदद मिलती है और उन्हें ठीक करने में भी।

## कैसे करें

इस लेख में, हम दो तरीकों का उपयोग करके यह समझेंगे कि हम रस्ट में डीबग करने के लिए कैसे output प्रिंट कर सकते हैं।

### तरीका 1: `println!` मैक्रो

यह Rust में विशेष रूप से रची गई मैक्रो है जो हमारे code में अलग-अलग arguments को जोड़ता है और उनको स्क्रीन पर प्रिंट करता है। इसमें हम गंभीर त्रुटियों को पकड़ सकते हैं और अपने code को fix कर सकते हैं। यदि आपको अपने code में कोई गलती नज़र आती है, तो आप `println!` मैक्रो को `panic!` मैक्रो में बदल सकते हैं। इससे आपको उस गलती की स्थिति में अपने code को रोकने और ठीक करने का मौका मिलता है।

```rust
fn main() {
    let x = 5;
    let y = 2;
    println!("x / y = {}", x / y);
}
```

आउटपुट:
```
x / y = 2
```
