## क्यों

आज के समय में, सभी इलेक्ट्रॉनिक माध्यमों पर टेक्स्ट एक महत्वपूर्ण भाषा है। जब हम रस्ट में कोडिंग करते हैं, हम दो हँगामा खत्म करके हमारे स्ट्रिंग लंबाइ मापते हैं। स्ट्रिंग की लंबाइ को पता करना किसी भी प्रोग्राम में बहुत ही उपयोगी हो सकता है। चाहे आप अपने कोड में आवश्यकताओं को जांचने के लिए या दुबारा पसंदीदा स्ट्रिंग फंक्शन बनाने के लिए, स्ट्रिंग की लंबाई को पता करना आपके लिए उपयोगी होगा।

## कैसे करें

आइए, अब हम देखें कि आप अपने रस्ट कोड में स्ट्रिंग की लंबाई को कैसे निकाल सकते हैं। सबसे पहले, हम एक चर पद्धति बनाएंगे और उसमें स्ट्रिंग पास करेंगे। फिर हम वहां स्ट्रिंग बज़ुआशी के साथ इटरेट करेंगे और हर बार लूप के दौरान एक काउंटर बढ़ाएंगे। लूप के अंत में, हमारे काउंटर को वापस करेंगे जो हमारी स्ट्रिंग की लंबाई होगी। नीचे दिए गए कोड ब्लॉक में इस समस्या का समाधान दिखाया गया है:

```Rust
fn main() {
    let my_string = "नमस्ते"; // हमारी स्ट्रिंग
    let mut count = 0; // काउंटर की शुरुआत
    for _ in my_string.chars() { // हमारी स्ट्रिंग के साथ इटरेशन
        count += 1; // हर बार काउंटर बढ़ाएं
    }
    println!("हमारी स्ट्रिंग की लंबाई है: {}", count); // आउटपुट स्क्रीन पर दिखाएं
}
```
