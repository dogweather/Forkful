# क्यों

कभी-कभी हमारे प्रोग्राम में हमें किसी निर्दिष्ट निर्देशिका का उपयोग करना होता है, लेकिन हम पहले से ही नहीं जानते कि क्या वह मौजूद है या नहीं। इस समस्या का समाधान करने के लिए हमें निर्देशिका का अस्तित्व जांचने की आवश्यकता होती है। इस ब्लॉग पोस्ट में हम Rust में निर्देशिका का अस्तित्व जांचने के बारे में बात करेंगे।

## कैसे करें

जब हमें एक निर्देशिका का अस्तित्व जांचना होता है, तो हमने कुछ स्टैंडर्ड कोड लेखना होता है। आप निम्नलिखित कोड के द्वारा निर्देशिका के अस्तित्व की जांच कर सकते हैं:

```Rust
use std::fs;

let path = "/home/example/directory";
    
let dir_exists = fs::metadata(path).is_ok();
    
println!("Directory exists: {}", dir_exists);
```

यहां, हमने `fs` मॉड्यूल का उपयोग किया है जो Rust में फ़ाइल और निर्देशिका संबंधित ऑपरेशन को व्यवस्थित करता है। `is_ok()` फ़ंक्शन द्वारा हम निर्देशिका के अस्तित्व की जाँच करते हैं और इसके परिणाम को `bool` वापस लाते हैं। आप अपने निर्देशिका का पथ को अपनी आवश्यकतानुसार बदल सकते हैं और असल नाम वापस लाने के लिए `dir_exists` भी बदल सकते हैं।
