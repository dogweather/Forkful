## क्यों

यह प्रश्न आपके मन में उठ रहा हो सकता है कि हम अपने Go प्रोग्राम में जांचें कि क्या एक निर्दिष्ट निर्देशिका या फोल्डर मौजूद है। इसका उत्तर आपके काम के लिए बहुत उपयोगी हो सकता है। जैसे कि अगर आप अपने प्रोग्राम में एक निर्दिष्ट फोल्डर से फाइलें पढ़ते हैं और आपके पास उस फोल्डर की मौजूदगी की जांच नहीं होती है, तो आप चेक कर सकते हैं कि फोल्डर सही तरह से सेटअप है या नहीं।

## कैसे करें

इसमें Go में जांचें कि क्या दिए गए निर्देशिका में एक विशिष्ट फ़ोल्डर मौजूद है काफी आसान है। निम्नलिखित कोड ब्लॉक कोड स्निपेट आपको इसका एक उदाहरण देगा।

```Go
package main

import (
	"fmt"
	"os"
)

func main() {
	// निर्दिष्ट फोल्डर का पथ
	folderPath := "/home/user/Documents"

	// फोल्डर की मौजूदगी की जांच
	if _, err := os.Stat(folderPath); os.IsNotExist(err) {
		fmt.Println("फोल्डर मौजूद नहीं है।")
	} else {
		fmt.Println("फोल्डर मौजूद है।")
	}
}
```

उपरोक्त कोड ब्लॉक का हमारे द्वारा बनाई गई कुछ साउंड उत्पादन देगा:

```
फोल्डर मौजूद है।
```
