## क्यों

अगर आप अपने कोड में किसी टेक्स्ट को बदलकर और अपडेट करना चाहते हैं, तो आपको इसके लिए टाइम और कठिन प्रयास की आवश्यकता होती है। इसलिए, खोज और बदलने के उपकरण से आपको कंप्यूटर द्वारा स्वचालित रूप से कोड को अपडेट करने की सुविधा होती है।

## कैसे करें

गो में टेक्स्ट खोज और बदलने की कामना करने के लिए आपको `strings.Replace (टेक्स्ट, पुराना, नया, गिनती)` फंक्शन का उपयोग करना होता है। यह फंक्शन पुराने टेक्स्ट को नए टेक्स्ट से बदल देता है।

```Go
package main

import "fmt"
import "strings"

func main() {
    text := "हैलो दोस्तों, मैं गो में लिखा गया हूँ।"
    newText := strings.Replace(text, "गो", "Go", 1)
    fmt.Println(newText) // output: हैलो दोस्तों, मैं Go में लिखा गया हूँ।
}
```

आप दिए गए कोड को स्वयं एक्सप्लोर कर सकते हैं और कॉपी करके अपने कोड में उपयोग कर सकते हैं।
