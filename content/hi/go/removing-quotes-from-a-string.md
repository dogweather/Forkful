## क्या और क्यों?

एक स्ट्रिंग से कोट्स हटाने का मतलब है उन कष्टप्रद डबल या सिंगल कोट कैरेक्टर्स को दूर करना जो आपके असली टेक्स्ट को लपेट कर रखते हैं। हम यह डेटा को संवारने, पार्सिंग त्रुटियों को रोकने, या कोटेशन मार्क्स के अतिरिक्त फ्लफ के बिना आगे की प्रक्रिया के लिए टेक्स्ट को तैयार करने के लिए करते हैं।

## कैसे:

यहाँ गो में उन कोट्स को किनारे करने का सरल तरीका है:

```go
package main

import (
	"fmt"
	"strings"
)

func removeQuotes(s string) string {
	return strings.Trim(s, "'\"")
}

func main() {
	quotedString := "\"हेलो, वर्ल्ड!\""
	fmt.Println("मूल:", quotedString)

	unquotedString := removeQuotes(quotedString)
	fmt.Println("बिना कोट्स के:", unquotedString)
}
```

आउटपुट इस प्रकार दिखेगा, कोट्स सभी गायब हो जाएंगे:

```
मूल: "हेलो, वर्ल्ड!"
बिना कोट्स के: हेलो, वर्ल्ड!
```

## गहराई से जानकारी

पहले के दिनों में, जब डेटा फॉर्मेट्स और इंटरचेंज मानकीकृत नहीं थे, स्ट्रिंग्स में कोट्स हाहाकार मचा सकते थे। वे अब भी कर सकते हैं, खासकर JSON में या जब डेटाबेस में स्ट्रिंग्स को डाला जाता है। Go में `strings` पैकेज में एक `Trim` फंक्शन शामिल है, जो केवल व्हाइटस्पेस ही नहीं बल्कि जिन कैरेक्टर्स को आप पसंद नहीं करते, उसे भी हटा देता है।

Regex क्यों नहीं? खैर, `Trim` सरल कामों के लिए तेज है, लेकिन अगर आपकी स्ट्रिंग्स अजीब जगहों पर कोट्स के साथ छुपान छुपाई खेल रही हैं, तो regex आपकी भारी तोप हो सकती है:

```go
import "regexp"

func removeQuotesWithRegex(s string) string {
	re := regexp.MustCompile(`^["']|["']$`)
	return re.ReplaceAllString(s, "")
}
```

यह चुनने की तरह है कि कैंची और चेनसॉ में से कौन सा उपकरण काम के लिए फिट है।
