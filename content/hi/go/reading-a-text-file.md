## क्यों

टेक्स्ट फ़ाइल पढ़ने का कार्य आमतौर पर कोडिंग में जोड़ने के लिए किया जाता है। इसके अलावा, यह टेक्स्ट फ़ाइल से डेटा निकालने के लिए भी बहुत उपयोगी होता है। इस ब्लॉग पोस्ट में, हम आपको बताएंगे कि आप किस तरह से अपने गो प्रोग्राम के साथ टेक्स्ट फ़ाइल पढ़ सकते हैं।

## कैसे करें

```Go
package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    // फ़ाइल खोलें
    file, err := os.Open("sample.txt")
    if err != nil {
        fmt.Println("फ़ाइल खोलने में त्रुटि:", err)
    }
    // फ़ाइल को बन्द करें
    defer file.Close()

    // फ़ाइल से लाइन पढ़ें
    scanner := bufio.NewScanner(file)
    for scanner.Scan() {
        fmt.Println(scanner.Text())
    }
}
```

ऊपर दिए गए कोड का उपयोग करके, आप अपनी कंप्यूटर में संभवतः स्थित स्टेशनरी फाइल से डेटा पढ़ सकते हैं। आप भी एक फ़ाइल के बारे में अधिक जानकारी पढ़ सकते हैं, जैसे कि उसका आकार और मोड।

## गहराई से जानें

जब आप टेक्स्ट फ़ाइल से डेटा पढ़ते हैं, तो आपको कुछ बातों का ध्यान रखने की आवश्यकता होती है। पहले, आपको उस फ़ाइल को खोलने के लिए अनुमतियाँ होनी चाहिए। अगर फ़ाइल के पथ में त्रुटि होती है, तो `Open()` फ़ंक्शन  `err` मानवर्धन करेगा और आप उस त्रुटि को चेक कर सकते हैं और फ़ाइल बंद करने के लिए `Close()` आपके पास होना आवश्यक होगा।
