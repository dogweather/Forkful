## क्यों

क्या आप अपने एप्लिकेशन में भविष्य की तारीखों को हस्तांतरित करने की आवश्यकता है? यदि हां, तो Go विकास भाषा का उपयोग करके आप इसे आसानी से कर सकते हैं। आइए अधिक जानते हैं!

## कैसे करें

आप भविष्य की तारीख हिसाब करने के लिए `time` पैकेज का उपयोग करके आसानी से इस कार्य को कर सकते हैं। नीचे कोड ब्लॉक में देखें:

```Go
import (
    "fmt"
    "time"
)

func main() {
    // वर्तमान की तारीख हासिल करें
    currentDate := time.Now()

    // भविष्य की तारीख हिसाब करें
    futureDate := currentDate.AddDate(2, 0, 0)

    // नतीजा प्रिंट करें
    fmt.Println("वर्तमान तारीख:", currentDate)
    fmt.Println("भविष्य तारीख:", futureDate)
}
```

आपको अपनी आवश्यकतानुसार `AddDate()` के पैरामीटर बदलने की आवश्यकता हो सकती है, जैसे कि सालों, महीनों और दिनों की संख्या को किसी भी संख्या से बदलकर अपनी भविष्य तारीख को हस्तांतरित कर सकते हैं।

ऊपर दिए गए कोड को चलाने से निम्न आउटपुट मिलेगा:

```
वर्तमान तारीख: 2021-10-01 10:00:00 +0000 UTC m=+0.000000001
भविष्य तारीख: 2023-10-01 10:00:00 +0000 UTC
```

## गहराई तक

भविष्य की तारीखों की गणना करने के लिए Go में `AddDate()` का उपयोग करने के अलावा आप `time` पैकेज की अन्य सुविधाओं का भी उपयोग कर सकते हैं। आप भविष्य की तारीख से संबंधित अन्य जानकारी जैसे कि सप्ताह का दिन, वर्ष का पहला दिन, आदि `time` पैकेज
