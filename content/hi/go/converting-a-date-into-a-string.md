## क्या और क्यों?
डेट को स्ट्रिंग में बदलना यही है कि आप कैलेंडर की तारीख को पठनीय स्वरूप में परिवर्तित कर रहे हैं। प्रोग्रामर्स ऐसा इसलिए करते हैं ताकि वे समय और तारीख को लॉग्स, इंटरफेस और रिपोर्ट्स में आसानी से प्रदर्शित कर सकें।

## कैसे करें:
```Go
package main

import (
	"fmt"
	"time"
)

func main() {
	// वर्तमान समय
	now := time.Now()

	// स्ट्रिंग फॉर्मेट में डेट कन्वर्ट करना
	strDate := now.Format("2006-01-02 15:04:05")
	fmt.Println("स्ट्रिंग फॉर्मेट में तारीख़: ", strDate)
}

// सैंपल आउटपुट
// स्ट्रिंग फॉर्मेट में तारीख़:  2023-03-14 17:45:12
```

## गहराई में जानकारी:
Go में डेट को स्ट्रिंग में बदलने का इतिहास परंपरागत `printf` फंक्शन से शुरू होता है, लेकिन यहां हम `Format` मेथड का इस्तेमाल करते हैं। क्यों "2006-01-02 15:04:05" फॉर्मेट? क्योंकि Go के निर्माताओं ने यह स्टैंडर्ड रूप चुना है, जिसमें हर घटक (वर्ष, महीना, आदि) एक महत्वपूर्ण मानक का प्रतिनिधित्व करता है। इसके अलावा, आप `time` पैकेज के अनेक `Layout` स्ट्रिंग्स का उपयोग कर सकते हैं। इंप्लीमेंटेशन के विवरण में यह बात आती है कि जब आप `.Format()` का इस्तेमाल करते हैं, तो आप वास्तव में एक टेम्प्लेट प्रदान कर रहे हैं जिसके अनुसार तारीख़ को फॉर्मेट किया जाएगा।
