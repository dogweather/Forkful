गो प्रोग्रामिंग ब्लॉग पोस्ट को हिंदी पाठकों के लिए

## क्यों

कैसे: दिनांक को स्ट्रिंग में रूपांतरण के लिए निर्देश

```Go
package main

import (
	"fmt"
	"time"
)

func main() {
	// आज की तारीख को ले
	today := time.Now()
	// दिनांक को स्ट्रिंग में रूपांतरित करें
	dateString := today.Format("02-01-2006")
	// प्रिंट करें
	fmt.Println("आज की तारीख:", dateString)
}
```
आउटपुट:
आज की तारीख: 07-10-2021

## दीप डाइव

दिनांक को स्ट्रिंग में रूपांतरण करने का यह निर्देश तभी उपयोगी होता है जब हमें अपने प्रोग्राम में दिनांक को प्रासंगिक तारीख स्ट्रिंग में बदलने की आवश्यकता हो। यह काम करने के लिए हमें समय दिनांक को अपनी इच्छानुसार फॉर्मेट करने का विकल्प होता है, जिसमें उन सभी आइटमों के साथ स्ट्रिंग का विन्यास किया गया हो जो हमें अपनी आवश्यकताओं के अनुसार महत्वपूर्ण होते हैं।

## अधिक जानकारी

अगर आप अपने कोड में अन्य फॉर्मेट ऑप्शन या समय क्षेत्र के साथ काम करना चाहते हैं, तो आप गो की [डॉक्यूमेंटेशन](https://golang.org/pkg/time/#Time.Format) से अधिक जानकारी प्राप्त कर सकते हैं।
