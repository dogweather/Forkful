## क्यों

ऑनलाइन प्रोग्रामिंग में रेगुलर एक्सप्रेशन उपयोगकर्ताओं को टेक्स्ट में खोजने, परिवर्तन या बदलाव करने में सहायता करता है। यह पारंपरिक स्ट्रिंग मैनिपुलेशन के लिए एक शक्तिशाली और विश्वसनीय उपकरण है।

## कैसे

```Go
package main

import (
	"fmt"
	"regexp"
)

func main() {
	// टेक्स्ट पाठ स्ट्रिंग
	text := "आज एक अच्छा दिन है"

	// रेगुलर एक्सप्रेशन पैटर्न
	pattern := "अच्छा"

	// रेगुलर एक्सप्रेशन पैटर्न को अनुकूलित करना
	re := regexp.MustCompile(pattern)

	// टेक्स्ट से भेद सूचकक का उपयोग करके मिलान
	match := re.FindStringSubmatch(text)

	// मिले अंश को छपिए
	fmt.Println(match) // अच्छा

	// टेक्स्ट में संख्या की गणना करने के लिए समंजस्य तार्किक
	count := len(match)

	// संख्या छपाना
	fmt.Println(count) // 1
}
```

### पूर्ण आउटपुट:

```
[अच्छा]
1
```

## गहराई में जाएं

रेगुलर एक्सप्रेशन ऑपरेटर्स, टोकन्स, और क्वांटिफायर्स की गहराई में जाना बहुत महत्वपूर्ण है। इससे आप अपने प्रोग्राम में रेगुलर एक्सप्रेशन काम करने के लिए और अधिक समझ प्राप्त कर सकते हैं। आप अपने और कॉम्पलेक्स एक्सप्रेशन बनाने में सक्षम होंगे जो आपके प्रोजेक्ट को अधिक उपयोगी बनाएगा।
