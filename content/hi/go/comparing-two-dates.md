## क्यों

क्या आपको कभी दो तारीखों को तुलना करने की ज़रूरत पड़ी है? शायद आपको अपने ग्राहकों को तारीखों की तुलना करने के लिए उनकी डेटा को समझने की ज़रूरत हो या फिर आप अपने एप्लिकेशन में दो तारीखों के बीच कितने दिन हैं या कितने सेकंड हैं यह जानना चाहते हों। इस ब्लॉग पोस्ट में, हम आपको एक सरल तरीके से दो तारीखों को तुलना करने के बारे में बताएंगे।

## कैसे करें

आपको दो तारीखों को तुलना करने के लिए Go में बनाए गए कुछ फंक्शन्स का उपयोग करना होगा। नीचे एक सरल उदाहरण दिया गया है जिसमें हम दो तारीखों को तुलना करके उन दोनों के बीच के दिनों की संख्या प्राप्त करेंगे।

```Go
func main() {
  // दो तारीखों को बनाएं
  date1 := time.Date(2021, time.January, 1, 0, 0, 0, 0, time.UTC)
  date2 := time.Date(2021, time.February, 1, 0, 0, 0, 0, time.UTC)

  // दो तारीखों को तुलना करें
  days := int(date2.Sub(date1).Hours() / 24)

  // प्रिंट करें
  fmt.Println(days) // Output: 31
}
```
