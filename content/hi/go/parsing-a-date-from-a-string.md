## क्या और क्यों? (What & Why?)

किसी स्ट्रिंग से डेट (तारीख) पार्स करने का मतलब है उसे समझकर संरचनात्मक डेटा में बदलना। प्रोग्रामर्स इसलिए करते हैं ताकि वे तारीखों के साथ आसानी से गणितीय आपरेशन्स कर सकें और डेटा को सही फॉर्मेट में स्टोर कर सकें।

## कैसे करें? (How to:)

```Go
package main

import (
	"fmt"
	"time"
)

func main() {
	// स्ट्रिंग में दी गई तारीख को पार्स करने के लिए:
	dateString := "02-01-2023" // dd-mm-yyyy फॉर्मेट में
	layout := "02-01-2006" // गो के time पैकेज में यह फॉर्मेट स्वीकृत है
	parsedDate, err := time.Parse(layout, dateString)
	if err != nil {
		fmt.Println("तारीख पार्स करने में त्रुटि:", err)
	} else {
		fmt.Println("पार्स की गई तारीख है:", parsedDate)
	}
	// आपके पार्स की गई तारीख का उपयोग अब आगे कार्य में लिया जा सकता है।
}
```

सैंपल आउटपुट:

```
पार्स की गई तारीख है: 2023-01-02 00:00:00 +0000 UTC
```

## विस्तार से जानकारी (Deep Dive)

डेट पार्सिंग की सुविधा 1970 के दशक से मौजूद है, जब से कंप्यूटरों में डेट और टाइम के संचयन की जरूरत पड़ी। Go में इसका अपना `time` पैकेज है, जो RFC3339 स्टैंडर्ड्स का उपयोग करता है।

वैकल्पिक तरीके भी हैं, जैसे `strconv` पैकेज का उपयोग करके कस्टम पार्सिंग तंत्र बनाना, लेकिन `time` पैकेज की सुविधा और सुरक्षा इसे अधिकांश स्थितियों के लिए उत्तम बनाती है।

परदे के पीछे, Go डेट और टाइम को जाने-माने UNIX टाइमस्टैम्प में बदलता है, जिससे समय की गणना में सटीकता और संगतता बनी रहती है।
