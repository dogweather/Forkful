## क्यों

डेटा को स्ट्रिंग में परिवर्तित करने का कईं उद्देश्यों से निर्माण हो सकता है, जैसे कि यूजर को बेहतर उपयोगकर्ता अनुभव देने के लिए तारीख़ को आसानी से पढ़ा और समझा जा सके।

## कैसे

आप डेटा को स्ट्रिंग में परिवर्तित करने के कईं तरीकों से कर सकते हैं। ये कोड स्निपेट आपको इसका अच्छा समझने में मदद करेंगे:

```Javascript
// अपनी तारीख को बनाएं
let date = new Date();

// उसे स्ट्रिंग में परिवर्तित करें
let dateString = date.toString();

console.log(dateString); // Output: बुधवार सितंबर 8 2021 15:30:00 जीएमटी+0530
```

जैसा कि आप देख सकते हैं, हमने `new Date()` का उपयोग कर एक तारीख बनाई और उसे `.toString()` का उपयोग करके स्ट्रिंग में परिवर्तित किया। इसके अलावा, आप अपनी तारीख को स्ट्रिंग में परिवर्तित करने के लिए `toLocaleString()` भी उपयोग कर सकते हैं। इसमें आप अपने भाषा और समय क्षेत्र को निर्दिष्ट कर सकते हैं।

```Javascript
// अपनी तारीख को बनाएं
let date = new Date();

// उसे हिंदी भाषा में स्ट्रिंग में परिवर्तित करें
let dateString = date.toLocaleString("hi-IN");

console.log(dateString); // Output: बुधवार, 08 सितंबर 2021 15:30:00 जीएमटी+0530
```

इसके अलावा, आप `date-fns` जैसी लाइब्रेरी का भी उपयोग कर सकते हैं जो आपको अपनी तारीख को पसंद के अनुसार स्ट्रिंग में परिवर्तित करने की सुविधा देती है।
