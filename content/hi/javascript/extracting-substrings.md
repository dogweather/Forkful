## क्यों

कभी-कभी हमें अपनी बड़ी स्ट्रिंग को छोटी स्ट्रिंग में विभाजित कर नाप, खोज या कुछ और कामों के लिए सबस्ट्रिंग की आवश्यकता होती है। इससे हमें अपने कोड को और अधिक अनुप्रयुक्त और सुंदर बनाने में मदद मिलती है।

## कैसे करें

यदि आपको किसी विशेष शब्द को अपनी स्ट्रिंग से छोटा करना है, तो सबसे पहले आपको `substring()` फंक्शन को इस्तेमाल करना होगा। नीचे दिए गए उदाहरण में, हमने दो स्ट्रिंग बनाई हैं और एक का सबस्ट्रिंग निर्धारित किया है।

```Javascript
var string = "मेरा नाम जॉन है";
var substring = string.substring(7);
console.log(substring);
```

आप निचे की तरह कंडोल कंसोल में देख सकते हैं:

```
'जॉन है'
```

अगर आप शुरुआती और समाप्त अंकों के बारे में भी निर्धारित करना चाहते हैं, तो आप इस तरह से कोड को लिख सकते हैं:

```Javascript
var string = "मेरा नाम जॉन है";
var substring = string.substring(7, 10);
console.log(substring);
```

यह कंडोल कंसोल में दिखाएगा:

```
'जॉन'
```

इस तरह से आप `substring()` फंक्शन को इस्तेमाल करके अपनी स्ट्रिंग से छोटी स्ट्रिंग को आसानी से निकाल सकते हैं।

## डीप डाइव
