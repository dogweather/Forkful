## क्यों 

क्या आपको पता है कि जावास्क्रिप्ट में तार को संबद्ध करने का यह दृश्य स्थान आपको अपने कोड में फ़ायदे की सक्ति देता है? यदि नहीं, तो आइए हम आपको बताते हैं कि क्यों तारों को संबद्ध करना महत्वपूर्ण है। तार एक स्ट्रिंगों को एक साथ जोड़ने का काम करता है जो जावास्क्रिप्ट को पांच वर्षों तक अब अधिक स्मरणीय बनाता है।

## कैसे करें

तारों की संबद्धता को अपने कोड में संलग्न करने के लिए, आपको जावास्क्रिप्ट के + ऑपरेटर का उपयोग करना होगा। इसका उपयोग करके, आप विभिन्न स्ट्रिंगों को एक साथ जोड़ सकते हैं। यहां कुछ उदाहरण हैं:

```Javascript
var name = "अंजलि";
var lastName = "गुप्ता";
var fullName = name + " " + lastName;
console.log(fullName);

// आउटपुट:
// अंजलि गुप्ता
```

```Javascript
var greeting = "नमस्ते, ";
var userName = "John";
var welcomeMessage = greeting + userName + "!";
console.log(welcomeMessage);

// आउटपुट:
// नमस्ते, John!
```

## गहराई जाँच

जावास्क्रिप्ट में तारों को संबद्ध करने से पहले, आपको सुनिश्चित करना होगा कि आप उन्हें सही क्रम में रख रहे हैं। यदि आप गलत क्रम में तारों को संबद्ध करते हैं, तो आपको गलत परिणाम मिलेगा। इसके अलावा, जब आप तारों को संबद्ध करते हैं, तो आपको सुनिश्चित करना होगा कि आपके शब्दों के साथ सही अंतर दिया गया है। बस ध्यान रखें कि आप अपने कोड को समझ लें और सही समोचित दिनों को उ
