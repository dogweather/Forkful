## क्यों

जब हम लम्बे और अच्छी तरह से फॉर्मेटेड टेक्स्ट को बदलने की आवश्यकता होती है, तो हम अक्सर खोज और प्रतिस्थापन का सहारा लेते हैं। यह इसलिए भी महत्वपूर्ण है क्योंकि इससे हमारे कोड की पठनीयता और अनुकूलता में सुधार होता है। 

## कैसे

```Javascript
// टेक्स्ट खोज और प्रतिस्थापन का उपयोग करने का उदाहरण 
let text = "मेरा नाम है जॉन डो";
let changedText = text.replace("जॉन डो", "जेनिय्ार"); 

console.log(changedText); // मेरा नाम है जेनियर 
```

यदि हमारे पास बहुत से टेक्स्ट बदलने की आवश्यकता हो, तो हम ```.replace()``` के साथ ग्लोबल फ्लैग का उपयोग कर सकते हैं। यह हमें सभी मिलते-जुलते शब्दों को बदलने में मदद करेगा। 

```Javascript
let text = "आज मैं खेला देखा, मैं खेला देखा, मुझे भारतीय टीम सहारा दी।";
let changedText = text.replace(/खेला देखा/g, "मैच देखा");
console.log(changedText); // आज मैं मैच देखा, मुझे भारतीय टीम सहारा दी। 
```

## गहराई में जाएं

जावास्क्रिप्ट में टेक्स्ट को खोजना और प्रतिस्थापित करना बहुत आसान है। हम ```replace()``` को स्ट्रिंग में एक शब्द के साथ या ग्लोबल फ्लैग के साथ इस्तेमाल कर सकते हैं। इसके अलावा, हम अन्य फ़ंक्शन प्राप्त करने के लिए और अधिक समाकलीन करने के लिए विज्ञापन योग्य शब्द भी उपयोग कर सकते हैं।
