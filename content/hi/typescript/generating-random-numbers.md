## क्यों

जब हम प्रोग्रामिंग करते हैं, हमें कई बार रैन्डम नंबर्स की आवश्यकता होती है। ये नंबर्स हमारे कोड को दिनामिक बनाते हैं और उसे उचित तरीके से चलाने में मदद करते हैं। तो चलिए जानते हैं कि इन रैन्डम नंबर्स को कैसे जेनरेट किया जाता है।

## कैसे करें

तो TypeScript में रैन्डम नंबर्स जेनरेट करने के लिए हम "Math.random()" फ़ंक्शन का उपयोग कर सकते हैं। ये फ़ंक्शन 0 से 1 के बीच किसी एक रैन्डम नंबर देता है। इससे हमें एक रैन्डम नंबर के साथ उसको सीमा तक क्रमित करने की सुविधा मिलती है। नीचे एक उदाहरण है:

```typescript
// सीधे रैन्डम नंबर
let randNum = Math.random();

// सीमा तक क्रमित रैन्डम नंबर
let scaledNum = Math.random() * 10; // सीमा 10 तक
console.log(scaledNum); // उद्धरण मान: 7.2847596882

// सीमा तक क्रमित किए गए रैन्डम पूर्णांक
let integerNum = Math.floor(Math.random() * 20) + 1; // सीमा 20 तक
console.log(integerNum); // उद्धरण मान: 17
```
