## क्यों
क्या आपने कभी सोचा है कि आपके द्वारा स्ट्रिंग से सबस्ट्रिंग्स निकालने के प्रयास से आप किसी बहुत अधिक लाभ को हासिल कर सकते हैं? सबस्ट्रिंग्स को निकालना आपको अंतिम लक्ष्य तक पहुंचने के लिए आपके कोड को सरल बनाता है और भविष्य में उसे दोबारा उपयोग करने को आसान बनाता है। इस ब्लॉग पोस्ट में, हम आपको स्ट्रिंग से सबस्ट्रिंग्स निकालने के लिए निष्कर्ष तक पहुंचाने के लिए कदम-से-कदम तरीके बताएँगे।

## कैसे करें
```TypeScript
// string से substring निकालने का उदाहरण
let string = "मेरा नाम राहुल है";
let substring = string.substring(5, 10);
console.log(substring);

// आउटपुट: नाम

// indexOf() और slice() का उपयोग करके substring निकालना
let sentence = "मैंने आज दो साइकिलें खरीदी हैं";
let index = sentence.indexOf("दो");
let newString = sentence.slice(index);
console.log(newString);

// आउटपुट: दो साइकिलें खरीदी हैं
```
और भी ऐसे कई तरीके हैं जिनसे आप स्ट्रिंग से सबस्ट्रिंग्स आसानी से निकाल सकते हैं। TypeScript में substring निकालने के लिए विभिन्न फ़ंक्शन और मेथड हैं, जिन्हें आप समझ कर अपने काम को और भी सरल बना सकते हैं। 

## गहराई में जाएं
अगर हम गहराई से जाएं तो स्ट्रिंग से सबस्ट्रिंग निकालने की प्रक्रिया स्पष्ट होती है। स्ट्रिंग हमारे द्वारा उपयोग किए गए अक्षरों के कोड में होता है, और सबस्ट्रिंग एक स्ट्रिंग के अंतिम कोड तक पहुंचने के लिए उपयो
