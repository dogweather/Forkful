## कैसे करें:
PowerShell दिनांक और समय प्राप्त करने के लिए सीधे cmdlets प्रदान करता है। `Get-Date` cmdlet इस उद्देश्य के लिए मुख्य उपकरण है। यह पूरी दिनांक और समय को लौटा सकता है, जिसे आप अपनी जरूरतों के अनुसार प्रारूपित या संशोधित कर सकते हैं।

```powershell
# वर्तमान दिनांक और समय प्राप्त करें
Get-Date
```

**नमूना आउटपुट:**

```
मंगलवार, सितंबर 5, 2023 9:46:02 पूर्वाह्न
```

आप आउटपुट को केवल वह जानकारी दिखाने के लिए भी प्रारूपित कर सकते हैं जो आपको चाहिए, जैसे कि केवल दिनांक या केवल समय।

```powershell
# केवल वर्तमान दिनांक एक विशिष्ट प्रारूप में प्राप्त करें
Get-Date -Format "yyyy-MM-dd"
```

**नमूना आउटपुट:**

```
2023-09-05
```

```powershell
# केवल वर्तमान समय प्राप्त करें
Get-Date -Format "HH:mm:ss"
```

**नमूना आउटपुट:**

```
09:46:02
```

### .NET क्लास का उपयोग
PowerShell .NET क्लासों तक सीधी पहुंच की अनुमति देता है, जो दिनांकों और समयों के साथ काम करने के लिए एक वैकल्पिक तरीका प्रदान करता है।

```powershell
# .NET DateTime क्लास का उपयोग करके वर्तमान दिनांच और समय प्राप्त करें
[System.DateTime]::Now
```

**नमूना आउटपुट:**

```
मंगलवार, सितंबर 5, 2023 9:46:02 पूर्वाह्न
```

UTC समय के लिए:

```powershell
# .NET DateTime क्लास का उपयोग करके वर्तमान UTC दिनांच और समय प्राप्त करें
[System.DateTime]::UtcNow
```

**नमूना आउटपुट:**

```
मंगलवार, सितंबर 5, 2023 1:46:02 अपराह्न
```

ये आदेश और वर्ग PowerShell में दिनांकों और समयों के साथ काम करने के लिए शक्तिशाली और लचीले विकल्प प्रदान करते हैं, जो कई स्क्रिप्टिंग और स्वचालन कार्यों के लिए आवश्यक हैं।
