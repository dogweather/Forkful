## क्यों

टेक्स्ट को ढूंढ़ना और बदलना प्रोग्रामिंग में एक अहम हिस्सा है, क्योंकि यह हमें दिए गए डेटा में आवश्यक बदलाव करने की अनुमति देता है। यह कई समस्याओं को हल करने और प्रोग्राम को दूरस्थ करने में मदद करता है। 

## कैसे करें

जब हमें किसी दो ध्वनियों को बदलने की आवश्यकता होती है, हम टेक्स्ट को खोजने और बदलने के लिए छोटे रूबी कोड का उपयोग कर सकते हैं। इसके लिए हम `gsub` फ़ंक्शन का उपयोग कर सकते हैं। नीचे एक उदाहरण दिया गया है। 

```Ruby
# आदेश: सबसे पहले कौन होता है?
sentence = "सुलेखा, अमित, रजत, दीपा"
sentence.gsub("सुलेखा", "मेघा")

# उत्पादन: मेघा, अमित, रजत, दीपा
```

## गहराई से जाएँ

टेक्स्ट को बदलने के लिए आपके पास कई विकल्प होते हैं, जैसे कि `gsub!`, `sub` और `replace`. इनसे हम दिए गए स्ट्रिंग के सभी आम विभागों को बदल सकते हैं। यदि हमें किसी भी विशिष्ट आम विभाग को बदलने की आवश्यकता हो, तो हम `split` फ़ंक्शन का उपयोग कर सकते हैं। नीचे एक उदाहरण दिया गया है।

```Ruby
# आदेश: आपका नाम कैसा है?
name = gets.chomp
name_array = name.split("")
name_array[0] = "मृत्यु"
name = name_array.join("")

# उत्पादन: मृत्युत्य्रेश्वर, कृष्णा
```
