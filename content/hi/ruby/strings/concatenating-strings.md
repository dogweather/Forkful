## What & Why? (क्या और क्यों?)
स्ट्रिंग्स का सम्मिलन यानी दो या अधिक स्ट्रिंग्स को जोड़ना होता है। प्रोग्रामर्स इसे तब करते हैं जब उन्हें डायनामिक टेक्स्ट बनाना हो या अलग-अलग जानकारियों को आपस में मिलाना हो।

## How to: (कैसे करें:)
रूबी में स्ट्रिंग्स को कैसे जोड़ें:

```Ruby
# प्लस ऑपरेटर का इस्तेमाल करके
greeting = "नमस्ते " + "दुनिया!"
puts greeting # => नमस्ते दुनिया!

# इंटरपोलेशन का इस्तेमाल करके
planet = "दुनिया"
greeting = "नमस्ते #{planet}!"
puts greeting # => नमस्ते दुनिया!

# शाब्लिक (!) ऑपरेटर का इस्तेमाल करके
greeting = "नमस्ते "
greeting << "दुनिया!"
puts greeting # => नमस्ते दुनिया!
```

## Deep Dive (गहरी जानकारी)
स्ट्रिंग्स का सम्मिलन शुरुआती दिनों से ही प्रोग्रामिंग का हिस्सा है। इससे पहले कि रूबी आया, दूसरी भाषाएं जैसे C और Java में भी ये संभव था मगर रूबी ने इसे और आसान बना दिया।

प्लस ऑपरेटर (`+`) सबसे साधारण तरीका है पर यह हर बार नई स्ट्रिंग बनाता है, जो मेमोरी के लिए अच्छा नहीं है। शाब्लिक (`<<`) ऑपरेटर सोर्स स्ट्रिंग में ही जोड़ता है, इससे परफॉर्मेंस में फर्क पड़ता है। इंटरपोलेशन (`#{}`) बहुत लोकप्रिय है रुबी में क्योंकि यह क्लीन और फास्ट होता है।
