## क्यों

अगर आप एक रूबी प्रोग्रामर हैं तो आपने संभवतः सुना होगा कि डायरेक्ट्री की उपस्थिति की जाँच किसी कार्य को थोड़ी सी सुरक्षा प्रदान कर सकती है। यह आपको आपके कोड में गलतियों को पकड़ने और उन्हें ठीक करने का मौका देती है।

## कैसे करें

आइए रूबी में कैसे डायरेक्ट्री की उपस्थिति की जाँच कर सकते हैं इसका एक उदाहरण देखें:

```Ruby
if Dir.exist?("directory_name")
  puts "डायरेक्ट्री मौजूद है"
else
  puts "डायरेक्ट्री मौजूद नहीं है"
end
```

जैसा कि आप देख सकते हैं, हमने `Dir.exist?` मेथड का उपयोग किया है जो एक डायरेक्ट्री मौजूद होने की जाँच करता है और true या false के साथ हमें वापसी देता है। हमने उसे एक `if` शर्त के साथ कनेक्ट किया है जो हमें सही मूल्य लौटाता है जो हमें बताता है कि डायरेक्ट्री क्या मौजूद है या नहीं।

## गहराई में जाएं

रूबी में `Dir.exist?` का स्थानिक नाम `File.exist?` है। यह मेथड एक फ़ाइल की उपस्थिति की जाँच करता है। आप डायरेक्ट्री के अलावा किसी भी फ़ाइल को जाँचना चाहते हैं तो इस मेथड का उपयोग कर सकते हैं। आप फाइल और डायरेक्ट्री के अंतर को समझने के लिए इन दोनों को संबोधित कर सकते हैं:

```Ruby
if File.exist?("file_name")
  puts "फ़ाइल मौजूद है"
elsif Dir.exist?("directory_name")
  puts "डायरेक्ट्री मौजूद है"
end
```
