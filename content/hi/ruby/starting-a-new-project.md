## क्या और क्यों? ("What & Why?")

नया प्रोजेक्ट शुरू करना का मतलब है, एक नए सॉफ़्टवेयर ऐप्लिकेशन का विकास शुरू करना। प्रोग्रामर इसे करते हैं क्योंकि यह उन्हें नए विचारों को शापित करने और उन्हें हकीकत में बदलने का मौका देता है।

## कैसे: ("How to:")

Ruby में नया प्रोजेक्ट खोलने का एक उदाहरण देखते हैं:

```Ruby
mkdir my_new_project
cd my_new_project
```

ऊपर दिए गए कोड की मदद से, हमने एक नयी डायरेक्टरी `my_new_project` बनाई है और उसमें प्रवेश किया है। 

## गहराई में: ("Deep Dive")

Ruby (1995 में विकसित) ने प्रोग्रामरों को सॉफ़्टवेयर डेवलपमेंट में एक नया तरीका दर्शाया। इसका सरल सिंटैक्स और उच्च-स्तरीय फ़ीचर्स इसे एक महान विकल्प बनाते हैं। वैकल्पिक तौर पर, आप Python या Java जैसी अन्य भाषाओं का उपयोग कर सकते हैं, लेकिन Ruby की बात ही कुछ अन्य है। जब आप एक नया project शुरू करते हैं, तो आप संसाधनों, डेपेंडेंसीज़, और कन्फ़िगरेशन को प्रबंधित करने के लिए bundler और rake जैसे टूल्स का उपयोग कर सकते हैं।
