## क्या और क्यों?

स्ट्रिंग इंटरपोलेशन वर्णनात्मक मूल्यों को सीधे स्ट्रिंग्स में इंजेक्ट करने की प्रक्रिया है, अक्सर जटिल संयोजनों के बिना अर्थपूर्ण संदेश बनाने के लिए। प्रोग्रामर्स इसे स्वच्छ, पठनीय कोड के लिए और जटिल स्ट्रिंग संयोजनों में होने वाली त्रुटियों से बचने के लिए करते हैं।

## कैसे करें:

Dart में, स्ट्रिंग इंटरपोलेशन सरल होता है, स्ट्रिंग लिटरल्स के भीतर सीधे अभिव्यक्तियों को इंटरपोलेट करने के लिए `$` प्रतीक का उपयोग किया जाता है:

```dart
void main() {
  String name = 'Dart';
  int year = 2023;
  // साधारण वेरिएबल इंटरपोलेशन
  print('Learning $name in $year!');
  // आउटपुट: Learning Dart in 2023!
  
  // अभिव्यक्तियों का इंटरपोलेशन
  print('In two years, it will be ${year + 2}.');
  // आउटपुट: In two years, it will be 2025.
}
```

जहाँ आपके पास अधिक जटिल अभिव्यक्तियाँ होती हैं या स्ट्रिंग के भीतर ही ऑपरेशन्स करना चाहते हैं, अभिव्यक्ति को `${}` में बंद करें। Dart में कोई लोकप्रिय तृतीय-पक्ष पुस्तकालय विशेष रूप से स्ट्रिंग इंटरपोलेशन के लिए नहीं है क्योंकि यह स्वाभाविक रूप से विविध और जटिल परिदृश्यों को संभालने में अच्छी तरह से सुसज्जित है।
