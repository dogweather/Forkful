### क्यों

क्या आपने कभी बैश प्रोग्रामिंग में एक निर्देशिका की उपस्थिति को जांचने का प्रयास किया है? यदि हाँ, तो आप जानते होंगे कि यह आमतौर पर एक अनिवार्य टास्क है। यदि आप एक निर्देशिका के उपस्थिति का पता लगाते हैं, तो आप जान सकते हैं कि आपका स्क्रिप्ट शीघ्र ही त्रुटियों से बच सकता है और सही ढंग से काम करता है।

### कैसे करें

एक निर्देशिका के उपस्थिति को जांचने के लिए हम बैश के "इस्तेमाल" ऑपरेटर का उपयोग कर सकते हैं। यह ऑपरेटर एक विशिष्ट निर्देशिका की उपस्थिति को जाँचता है और यदि वह निर्देशिका मौजूद होती है, तो स्टेटस कोड 0 लौटाता है। उसके बाद हम `if` शर्त के भीतर हमें दिए गए पाठ को चलाने के लिए यूज़रनेम और पासवर्ड यूज़ कर सकते हैं।

एक निर्देशिका की उपस्थिति को जांचने के लिए निम्नलिखित बैश स्क्रिप्ट का उपयोग करें:

```Bash
if [ -d /path/to/directory ]; then
    echo "निर्देशिका मौजूद है"
    # यहां आप अपने कोड को लिख सकते हैं
else
    echo "निर्देशिका नहीं मिली"
fi
```

अगर आप एक निर्देशिका की उपस्थिति के साथ कुछ अन्य ऑपरेशन्स पर चलाना चाहते हैं, तो आप निम्नलिखित बैश स्क्रिप्ट को चला सकते हैं:
