## क्यों

कोई भी व्यक्ति तारीख की गणना करने में रुचि रखता है क्योंकि यह उन्हें आगामी या पूर्व में होने वाली स्पष्ट घटनाओं के बारे में जानकारी प्रदान करता है।

## कैसे करें

इस प्रकार तारीख की गणना करने के लिए, हमें साल, महीना और दिन के संज्ञानात्मक मानों को उपयोग करना होगा। हमें तारीख की गणना करने के लिए निम्नलिखित स्टेटमेंट्स का प्रयोग कर सकते हैं:

```C++
#include <iostream>

using namespace std;

int main() {
  // साल, महीना और दिन के संज्ञानात्मक मानों को आधार बनाकर तारीख की गणना करें
  int year, month, day;

  cout << "कृपया साल को दर्ज करें: ";
  cin >> year;
  cout << "कृपया महीना को दर्ज करें: ";
  cin >> month;
  cout << "कृपया दिन को दर्ज करें: ";
  cin >> day;

  // कोड लोजिक के अनुसार तारीख की गणना करें
  day = day + 7;
  month = month + 1;
  year = year + 1;

  // तारीख को सही फॉर्मेट में प्रिंट करें
  cout << "\nआगामी तारीख है: " << year << "/" << month << "/" << day << endl;

  return 0;
}

```

उपरोक्त कोड का आउटपुट निम्नलिखित हो सकता है:

```
कृपया साल को दर्ज करें: 2020
कृपया महीना को दर्ज करें: 5
कृपया दिन को दर्ज करें: 26

आगामी तारीख है: 2021/6/2
```
