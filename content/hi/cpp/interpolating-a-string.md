## क्या और क्यों?

वाक्यांश संवरधन से हमारा तात्पर्य होता है एक वाक्यांश के भीतर चर का सम्मिलन। प्रोग्रामर इसे उपयोग करते हैं जब वे एक तार के साथ विभिन्न मानों का मिलान करना चाहते हैं, आमतौर पर वे आउटपुट बनाते समय। 

## कैसे करें:

```C++
#include <iostream>
#include <string>

int main() {
    std::string name = "Ravi";
    std::cout << "नमस्ते, " << name << "!\n";
    return 0;
}
```
ऊपरी कोड का आउटपुट होगा:

```
नमस्ते, Ravi!
```

## गहराई से जानें

समवर्धन का उपयोग करना कम मात्रा में कोड और अधिक पढ़ने वाले कोड प्रदान कर सकता है। C++ में, नियमित वाक्यांश समवर्धन का समर्थन नहीं करता है, लेकिन आप `+` चिन्ह का उपयोग करके वाक्यांश को जोड़ सकते हैं।

निरंतरता, क्लाइंट साइड जावास्क्रिप्ट (जैसे कि नोड.जेएस) आदि भाषाओं में टेम्पलेट वाक्यांश संवर्धन का समर्थन करते हैं, जिह्वा में वाक्यांश समवर्धन का उपयोग करने के लिए बैकतिक (` `) चिन्ह का उपयोग करते हैं।
