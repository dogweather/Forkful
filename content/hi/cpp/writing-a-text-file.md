## क्यों

यदि आपको प्रोग्रामिंग से प्यार है और आप अपने कोड को सही से संरचित और सुव्यवस्थित रखना चाहते हैं, तो आपको एक टेक्स्ट फ़ाइल में कोड लिखना चाहिए। टेक्स्ट फ़ाइल आपके कोड को संग्रहीत करती है और आपको अपने कोड को स्पष्ट और सरल ढंग से प्रदर्शित करने में मदद करती है।

## कैसे करें

टेक्स्ट फ़ाइल में कोड लिखने के लिए, आपको कुछ सामान्य प्रोग्रामिंग कनसेप्ट्स की जानकारी होनी चाहिए। आपको `ofstream` द्वारा फ़ाइल को खोलना, `<<` ऑपरेटर का उपयोग करके फ़ाइल में कोड लिखना और फिर `close()` फ़ंक्शन का उपयोग करके फ़ाइल को बंद करना होगा।

इसके लिए, सबसे पहले हम `#include <fstream>` दिखाएँगे। फिर हम `ofstream` नाम के एक वेरिएबल को बनाएंगे जो हमारी फ़ाइल को खोलेगा। उसके बाद, हम `ofstream` द्वारा `open()` फ़ंक्शन को उपयोग करके अपनी फ़ाइल को खोलेंगे। अंतिम में, हम `<<` ऑपरेटर का उपयोग करके फ़ाइल में कोड लिखेंगे और फिर फ़ाइल को `close()` फ़ंक्शन का उपयोग करके बंद करेंगे। नीचे दिए गए उदाहरण को देखें:

```C++
#include <fstream>

using namespace std;

int main() {
    ofstream myfile;
    myfile.open("mycode.txt"); // यहाँ आप अपनी फ़ाइल का नाम दे सकते हैं
    myfile << "cout << \"Hello World!\";"; // यहाँ आप अपना कोड दे सकते हैं
    myfile.close();
    
    return 0;
}
```
