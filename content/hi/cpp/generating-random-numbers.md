## क्यों

यदि आप C++ प्रोग्रामिंग सीख रहे हैं, तो आपने शायद सुना होगा कि कुछ लोग यादें और संभावनाओं को रैंडमली देखने के लिए रैंडमली संख्याओं का उपयोग करते हैं। यह धोखा प्रमुख रूप से सिरफ़ आपके प्रोग्राम को आकर्षक और बेहतर बनाने के लिए होता है। इस ब्लॉग पोस्ट में, हम रैंडमली संख्याओं के उत्पादन के बारे में अधिक जानकारी प्राप्त करेंगे।

## कैसे करे

आप C++ में रैंडमली संख्याएं उत्पन्न करने के लिए `rand()` फंक्शन का इस्तेमाल कर सकते हैं। नीचे एक उदाहरण है जो आपको समझने में मदद करेगा।

```C++
#include <iostream>
#include <cstdlib> //यह बिल्ट-इन हेडर फाइल C++ में रैंडमली संख्याओं का उत्पादन करने के लिए आवश्यक है
using namespace std;

int main() {
	int num = rand(); //rand() सिस्टम समय के आधार पर एक रैंडम संख्या उत्पन्न करता है
	cout << num << endl; //इस समकक्षता के लिए, हम एक रैंडम संख्या को कॉन्सोल पर प्रिंट करते हैं
	return 0;
}
```

आप भी रैंडमली संख्याओं को निर्दिष्ट सीमा के भीतर उत्पन्न कर सकते हैं। निर्दिष्ट सीमा में से एक संख्या उत्पन्न करने के लिए `rand() % n` का उपयोग करें, जहां `n` सीमा है। इस हेडर फाइल (`cstdlib`) में, हमें `srand()` फंक्शन भी मिलता है जो आपकी स्वतंत्रता को बढ़ाता है कि आप रैंडमली संख्याएं कैसे उत्पन्न करने चाहते हैं। नीचे एक
